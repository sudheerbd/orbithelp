Multi-level Group Export to Excel

To work with multi-level group export to excel

 

1. Open the required data report.

2. On the Configurator tab, drag and drop the required columns.

3. On the Widget Settings tab, in the Tabular section, in the Grouping Mode box, type/select the required option from the following:

    • Multi
    • Single

Required Grouping Mode: Multi

Since, here grouping is being done at multiple levels, select Multi option

4. Here, multi-level grouping is explained using the following example:

a. Open the required data report with the required columns as shown below:

 

b. Set the properties for "Region" column as follows:

    i. To set the column properties, click on "Region" column and then click Properties.

    ii. Properties dialog appears.

 

      

 

    iii. In the Properties dialog, on, the right-hand side, in the Grouping & Summary section,perform the following:

      • In the Grand Total box, type/select the required option.  

Example: Grand Total : true

      • In  the Group Total box, type/select the required option.

Example: Group Total: true

      • In the Grouping box, type/select the required option.

Example: Group Total: true

In Format section, perform the following:

    • In the  DisplayText box {label} {value} {aggr} user can type any character after {aggr} to view the output

Example: {label} {value} {aggr}

{label} is the Column Name

{value} is the Grouped Value

{aggr} is the Aggregate

Note: {NOTEXT} doesnt display any Text

 c. For the Sales column, perform the following:

      • Set the Grand Total to true.

      • Set the Group Total to false.

 d. For the Product Category column, perform the following:

      • In the Grand Total box, type/select the required option.  

Example: Grand Total : true

      • In  the Group Total box, type/select the required option.

Example: Group Total: true

      • In the Grouping box, type/select the required option.

Example: Group Total: true

e. To save the changes, click Ok.

5. To run the report, click Run.

The report appears as follows:

To view the multi-level grouping in excel, perform the following:

1. On the right-hand side, click Options, point to Export, and then click Excel xlsx.

2. A process gets submitted with process id.

3. On the left-hand side, point to Monitoring, and then click Processes.

4. Under Scheduled Processes, a list of available processes appears.

5. Select the required process, and under Output(s) & Delivery section, click the output.

6. An excel sheet gets downloaded.

7. Open the excel sheet.

.

 

Note:

  • If the User doesn't mention any Display Test Separator in the String Meta Object then it considers the default value mentioned in Base Meta Object

  • If nothing is mentioned in Base and String Meta Objects then it provides a default output as "Space : Space" as shown in the above figure.
