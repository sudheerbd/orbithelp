Importing Users from Authority

To Import Users from Authority

 

  1. On the required authority, click icon (highlighted as “1”) under Actions, and then click Import users from authority (highlighted as “2”).

 

  1. Import Users dialog box appears.

  1. In the textbox before the Search button (highlighted as “1”), type the user name or the search term, and then click Search.

 

  1. A list of available users appears under Available Users list.

  2. Drag the required user and drop it under Selected Users.
