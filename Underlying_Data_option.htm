Underlying Data

This option allow user to view under line data of an report.

 

To work with underlying data

 

  1. To view underlying data of an visualization widget, right-click on the required visualization and then select Underlying Data.

 

Note: Underlying Data available in both Edit/View Mode

  1. Underlying Data window appears with data as show below:

 

 

  1. To disable this option, uncheck Underlying Data under Data Interaction Options.

  2. When user right-click on the required visualization widget, Underlying Data option will be disable.
