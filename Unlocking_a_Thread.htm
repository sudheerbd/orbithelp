Unlocking a Thread

To unlock a thread

 

1.In Orbit Application, at the top of the page, click Collaboration Threads  

2. A window appears with a list of collaboration threads.

3.    From the list of threads, select the thread you want to unlock.

Note: In the collaboration threads list, thread status indicator appears grey for locked threads.

4.    In the chat window of the selected thread, click  Thread Chat Settings Menu, and then click Unlock Thread.

5. Change Thread Status dialog box appears.

6. To unlock the thread, click Yes.

 

7.    To unlock the thread, click Yes.

Note: A Collaboration thread status change request processed notification appears briefly at the top of the ORBIT application window.

8.    To cancel the unlock thread request, click No.

9. When the thread is unlocked, color of thread status indicator changes to green.

10. Also, in Threads section Thread Status appears as “OPEN”.
