Dynamic Dimension

Creating Dynamic Dimension

To add dynamic dimension to a dashboard report:

1. Open a dashboard reports page in Edit Mode.

2. Right click on required fields, click Pick Visualization and then click Dynamic Dimension

(OR)

Drag the required fields from the Business Object into the Run Result area, click the expander arrow in the Report Configurator area and then click Dynamic Dimension.

Hold Ctrl and click on required fields to make multiple selections.

3. Dynamic Dimension Widget appears in the Run Result area.

4. Drag desired fields into the List of dimensions section (Highlighted as 1) in the report configuration area.

 

Visual Settings of Dynamic Dimension

Customize the look of Dynamic Dimension

Title

Customize the Dynamic Dimension widget title.

Default Value: None

Auto: When Title is set to Auto, Orbit application generates the title.

Custom: Select Custom to set a custom title to the widget. When you select Custom, a Text box appears where you can enter the title.

None: Select None to hide widget title.

Display Type

Select display type of Dynamic Dimension widget.

Default Value: Drop Down

Drop Down: Shows dynamic dimensions in a drop down list. Click the expander arrow on the Dynamic Dimension widget to see list of dimensions.

Button List: Shows dynamic dimensions as a button list. When you select Button List, Layout option appears.

Layout

select the Layout of the Button List

Default Value: Horizontal

Horizontal: Shows Dynamic Dimension as a horizontal button list.

Vertical: Shows Dynamic Dimension as a vertical button list.

Resize Dynamic Dimension Widget

Move the cursor to any edge or corner of the widget until a double-headed arrow appears. When this arrow appears, click-and-drag it to make the widget larger or smaller.

Working with Dynamic Dimension Widget

Dynamic Dimension widget allows you to filter the data displayed on the chart widgets in the dashboard reports page.

Click any one of the buttons in the Dynamic Dimension widget; data on the chart widgets updates.
