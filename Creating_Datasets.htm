Creating a Report using Datasets

 

To create a report

  1. For the selected dataset, click Create Report.

 

  1. Reports menu appears with the following types of reports:

    1. Data Report

    2. Analysis

    3. Dashboard

    4. Publisher Report

    5. Script Report

    6. Custom Report

  1. To select the required type of the report, double-click the report option.

(Or)

Select the required report type, and then click Ok.

 

 

  1. The selected report opens with the dataset object under Data Explorer in Reports.
