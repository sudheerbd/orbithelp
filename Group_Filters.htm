Group Filters

Group Filters enable you to group multiple filters using the logical operators (AND/OR).

 

Logical Operator Name

Description

AND

Returns true, if both the operands
evaluate to true.

OR

Returns true, if at least one of the
operands evaluates to true.

 

Group Filters can have the values either from the user filters or the data filters or both. Here, the data report has been considered.

 

To create group filter

 

1. Open the data report, and place required columns as user/data filters.

2. Assign the values to the user/data filters.

3. When you run the report, the report appears with OR condition applied between the parameters.

4. To work with group filters/change the Logical conditions, perform the following:

5. On the menu above the run reports area, click  Options, and then click Group Filters.

6. Group Filters dialog appears.

7. In the Group Filters dialog, the specified parameters appear with the default logical operator assigned.

 

¥ AND operator is assigned, by default.

 

 

8. To change the logical operator, click the already existing logical operator as shown above.

 

  

                        

 

9. A menu appears with the list of available logical operators.

10. From the menu, click the required logical operator, and then click Ok.

Creating a New Group

 

To create a new group under the Filters section

 

1. In the Group Filters dialog, right-click the Filters section, and then click New Group (highlighted as “1”).

 

  

 

This creates a new group – OR/AND group (highlighted as “2”). When AND is selected, logical AND operation is done between the two operands. When OR is selected, logical OR operation is done between the two operands.

 

2. To save the changes, click Ok.

3. To reset the created group filters, click Reset.
