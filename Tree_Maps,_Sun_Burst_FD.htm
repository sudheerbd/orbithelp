Tree Maps, Sun Burst

 

Tree Maps display hierarchical data using nested figures, usually rectangles. Tree Maps display quantities for each category via area size.

Sun Burst charts display hierarchical data through a series of rings, that are sliced for each category node.

In this topic, visualization options and visual settings are explained using Tree Maps examples. These visual settings are also same for Sun Burst charts.

 

Visualization (Chart type) Options

1. Tree Maps has the following visualization options

  • Size

  • Slices

2. Drag and drop the required fields from the selected Business Object into the Size and Slices.

Drag fields with numerical values into the Values section for several types of calculations (Sum, Average, Min, Max, etc.) However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the tree maps analysis report dynamically in the run results area.

 

Visual Settings for Tree Maps

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for tree maps has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Slice: See Customize Slice Visual Settings for more details.
  3. Value: See Customize Value Visual Settings for more details.
  4. Tooltip: See Customize Tooltip Visual Settings for more details.

These visual settings are also same for Sun Burst charts.
