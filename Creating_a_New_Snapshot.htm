Creating a New Snapshot

 

To create a new snapshot

 

1.    On the left-hand side, point to   Collaboration menu, and then click Snapshot.

2.   Snapshots page appears with a list of available snapshots.

       

3.   To create a new snapshot, click New.

       

4.   NewSnapshot dialog box appears.

 

       

 

5.   In the New Snapshot dialog box, do the following:

a.      In the Name box, enter new snapshot name.

b.      In the Description box, enter new snapshot description.

c.      To upload the snapshot, click Upload next to Snapshot Image box.

d.      In the Open dialog box that appears, browse your computer, select the desired file, and then click Open.

e.      In the Snapshot Image box, the path of the selected file appears.

 

 

f.       To save the changes, click Save.

Note: A Snapshot saved successfully notification appears briefly at the top of the ORBIT application window.

g.      To discard the changes, click Cancel.

6. In the Snapshots section, the newly created snapshot appears.

 

      

7. To refresh the snapshots, click  Refresh.
