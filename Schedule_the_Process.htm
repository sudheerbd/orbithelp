Scheduling the Process

To schedule the process

 

  1. In the ORBIT XLEdge application, in the Submit/Schedule tab, click Expand panel (highlighted as "1").

 

   

  1. Submit/Schedule panel expands.

  2. To schedule the process at a required interval, remove the checkmark from the   Run Immediately checkbox (highlighted as "2").

   

      

  1. The various fields to schedule the request appear under Submit/Schedule tab.

 

 

  1. In the Start Date box, type/select the required start date of the request.

  2. In the Start Time box, type/select the required start time of the request.

  3. In the End Date box, type/select the required end time of the request.

  4. In the End Time box, type/select the required end time of the request.

  5. The user can schedule the request using the following two options:

    • Simple

    • Advanced

  1. 10. If the Simple tab is selected, perform the following:

    1. Under Recurrence, select the required option from the following:

      1. Never Repeat  

      2. Simple Repeat

      3. Repeat   

      4. Calendar Dates    

The user can schedule the recurrence of the request using "Never Repeat" and "Repeat" options. "Repeat" option enables the user to schedule the request at a required interval. When the user selects Never Repeat, the request is never repeated

    1. To submit the request, click Submit.

  1. If the Advanced tab is selected, perform the following:

    1. Under the Months section, select the checkbox before the required month to select the required month.

    2. Under the Dates section, select the required dates.

    3. Under the Days section, select the required days.

    4. Under the Hour section, select the required hours.

    5. Under the Minute section, select the required minutes.

 

 

To notify the user about the end of the process, select the Notify me checkbox.  

    1.  To submit the request, click Submit.
