Logical Tables Properties under Business Objects

To configure the Logical Tables Properties

 

  1. Under the Business Objects, the business object appears.

        

            

  1. Right-click the business object, and then click Open. 

  2. Two sections – Logical Tables and Business Objects –appear.

  3. Under the Logical Tables, tables under it appear.

  4. Right-click the required table and then click Properties (highlighted as “1”).   

 

 

                                               

  1. Properties <table name> dialog box appears.

 

 

7. On the left hand side section, click the database object.

8. The properties of the database objects appear on the right hand side.

9. To edit the properties of the database object, perform the following:

    1. In the Name box, edit the required Name.

    2. In the Description box, type the required description.

    3. Under the Table section, perform the following:

      • From the Table Type list, select the required table type from the following:

        1. Fact

        2. Dimension

        3. Other

      •  In the Target Schema box, the required schema appears.

      •  In the Target Table box, the required target table appears.

      •  From the Target Table box, select the required option from the following:

        1. TABLE

        2. VIEW

        3. SYNONYM

        4. INLINE_SQL

    1. Under the Miscellaneous section, perform the following:

    • In the Relative Size box, enter/edit the required relative size.

Relative size can be assigned at the Physical Model level, logical model level and in the logical tables at Business object level.

    1. Under the Refresh Options section, perform the following:

      1. In the Refresh File Pattern box, select the required option.

        • Recently Uploaded

        • Sort By File Name

      1. In the Refresh Type box, select the required option.

        • Full Load

        • Incremental

 

10. To edit the properties of the columns, perform the following:    

  1. In the Name box, edit the required Name.

  2. In the Description box, type the required description.

  3. In the Field section, perform the following:

    • From the Default Aggregation box,  select the required option from the following:

  1. NONE

  2. SUM

  3. AVG

  4. COUNT

  5. COUNT_DISTINCT

  6. MIN

  7. MAX

    • From the Field Type box, select the required option from the following:

  1. OTHER

  2. DIMENSION

  3. FACT

  4. KEY

  5. ATTRIBUTE

    • In the List of Values box, double click the box to open the List of values dialog.

In the List of values dialog, provide the list of values for the required column.

    • From the Searchable box, select the required option from the following:

  1. True

  2. False

    1. In the Formula section, perform the following:

      • In the Column Type box, click the following option from the following:

        1. Column

        2. Formula      

      • In the Datatype box, click the following option from the following:

        1. Unknown

        2. String

        3. Date

        4. DateTime

        5. Boolean

        6. Integer

        7. Decimal

      • In Target Column/Expr box, the required target column or expression appears.

Target Column/Expr can be edited as required.

    1. In the Format section, perform the following:

      • In the Currency Code box, enter the required code.

      • In the Display Text, enter the required text,.

      • In the Format box, click the box, and then click to format the required column.

        1. In the Format dialog, perform the following:

          •  In the Decimal Places box, click /to increment/decrement the decimal places.

          • In the Currency Symbol  box, click to select the required currency.

          • To use the 1000 Separator, select Use 1000 Separator (,) checkbox.

          • To show the negative numbers in brackets, select Show Negative Numbers In Brackets.

          • To save the changes, click Ok.

        2. In Null Display box, enter the required text to display.

        3. From the Round Decimal Value box, select the required option from the following:

          • True

          • False

        4. From the Year Prior to 1900 box, select the required option from the following:

          • True

          • False

      • The changes made in the Format dialog appear in the Format box under Format section.

      • Fields in the Format changes based on the Datatype. The above figure displays the format for number datatype.

    2. In the Style section, perform the following:

      • In the Alignment box, type/select the required option from the following options:

        1. left

        2. center

        3. right

      • In the Background Color box, click in the textbox, and then click downward arrow to select the required background color from the color palette.

      • In the Cell Wrap box, select the required option from the following to wrap the contents of the cell.

        1. True

        2. False

      • In the Column Width box, type the required width or click  /to increment/decrement the column width.

      • In the Font Family box, click the box and then click downward arrow to make the required changes to the fonts as shown below:

To save the changes, click Ok in the above figure.

      • From the Hidden box, select the required option from the following to either hide or display the selected column

        1. true

        2. false

 

11. To save the changes, click Ok.
