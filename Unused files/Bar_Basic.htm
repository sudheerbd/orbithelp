Pivot/Matrix Charts

Pivot chart enables data analysis by visualizing a pivot table.

Widget Settings of Pivot

To set the widget settings of pivot

1.      On the report edit page, click Visual Settings.

2.       The following tabs appear:

  1. General

  2. Sub Totals

  3. Grand Totals

 

General

Under the General tab, perform the following:

1.      In the Detail Data box, type/select the required option from the following: 

             Example: true

 

     2.      In the Layout box, type/select the required option from the following:

 

a.       Outline

b.      Compact

c.    Tabular

Sub-Totals

1.      Under the Sub Totals tab, perform the following:

a.  .In Column Totals box, select/type the required option from the following:

                                      i.      Last

                                    ii.      First

                                  iii.      None

Example: Row Totals: Last

 

b.    In Row Totals box, select/type the required option from the following:

                                      i.      Last

                                    ii.      First

                                  iii.      None

Example: Row Totals: First

 

  •  In the Row Totals box, when Last option is selected, the total of the values in the rows appears in the last.

  •  In the Row Totals box, when First option is selected, the total of the values in the rows appears in the First. 

 

c.    When the above widget setting is applied, the report appears as shown below:

 

 

In the above figure the row is highlighted as “1” and row total is highlighted as “2”. Column is highlighted as “3” and Column Total is highlighted as “4”.
