Setting Deliveries

To set the Deliveries

1.      On the Outputs & Delivery tab, in the Deliveries area, click  (highlighted as “1”) to set a new delivery type.

 

                                          Figure: Setting New Delivery Type

 

 

  • Once you select the textbox that appears under Delivery Type, after clicking, delete option (highlighted as “2”) is enabled.

 

 

                                              Figure: Displaying Delete Option

 

2.      In the Delivery Type box, type/select the required option.

3.      In the Server box, type/select the required option.

4.      In the Users box, type/select the required option.

5.      To save the output and delivery type, click Save.

6.      To close the dialog box, click Cancel.
