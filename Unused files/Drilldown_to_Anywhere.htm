Drilldown to Anywhere

To Drilldown to Multiple Reports

 

  1. Create a Report with parameter, on which the drilldown must be performed.

  2. Save the report by clicking Save.

  3. The report (highlighted as “1”) appears under Reports on the left-hand side. 

 

  1. Open the newly created report.

  2. On the top right-hand corner of the Run Results area, click Options, and then click Drilldowns (highlighted as "2").

 

  1. Drilldowns dialog box appears.

  2. Under Drilldowns dialog box, perform the following:

  1. Drag and drop the required drilldown report that has the required parameters (highlighted as “3”) under the Drilldowns area.

 

  1. Drilldown Column and Navigate To options appear.

  2.  Under Parameters area, the required parameter (highlighted as “4”) appears with a list of available values for the parameter.

 

  1.  From the required report, drag and drop the required columns from report configurator area of the required report to the Drilldown Column box (highlighted as “5”).

  2.  In the Navigate To box (highlighted as “6”), click the required option from the following:

    • New Tab

    • Window

  1. From the Parameters Value box (highlighted as “7”), select the required option from the list.

  1. To save the changes, click Ok.

  1. The drilldown report appears as selected in Navigate To box.
