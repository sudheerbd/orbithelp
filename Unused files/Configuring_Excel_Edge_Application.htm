Configuring Excel Edge Application

To Configure Excel Edge Application

 

1.      On your local machine create a new folder, and rename it with an appropriate name (highlighted as “1”).

 

2.      Share the newly created folder with everyone and provide them read- write permissions. 

3.      To share the folder with everyone, perform the following:

4.      Right-click the newly created folder, point to Share with, and then click Specific people (highlighted as “2”).

 

 

5.      File Sharing dialog box appears.

6.      From the Choose people to share with list, choose Everyone, and then click Add.

 

 

7.      The selected option appears in the box below the list.

8.       In the box, for Everyone option, select the required permission under Permission Level, and then click Share.

 

 

 

? Example: Permission Level: Read/Write

 

¥    The folder now has the share permissions for everyone.

 

9.      Copy the downloaded XML file (highlighted as “3”), and then copy it in the Shared Folder (highlighted as “4”).

 

 

10.  Launch MS Excel 2016, and open a new worksheet.

11.  On the File Menu, click Options.

12.  Excel Options dialog box appears

 

    

 

13.  On the Trust Center tab, click Trust Center Settings.

14.   Trust Center dialog box appears.

15.  In the Trust Center dialog box, click Trusted Add-in Catalogs.

16.  On the right-hand side under Trusted Catalogs Table, in the Catalog Url box, type/paste the catalog URL, and then click Add catalog.  See Catalog URL to generate the Catalog URL.

 

    

 

17.  The URL appears under Trusted Catalog Address (highlighted as “6”).

 

¥   Ensure that a check mark is present in the check box under Show in Menu for the selected Catalog URL.   

 

 

18.  To save the changes, click OK.

19.  A message “We’ve saved your settings, they’ll be applied next time you Start Office” appears.

20.  To continue, click OK, and then close the excel workbook.
