User Filters

 

To apply user filter on a report

1.      To enable User Filters, click Hide/Show User Filters (highlighted as “1”).

 

                                         

2. An area for User filters appears.

  

3. From the available columns, drag and drop the required column the user filter area , which is above the Run Results area.

 

 

¥   The area above the Run Results area is called User Filter area

 

 

3.      Parameter (highlighted as “2”) appears in the user filter area (highlighted as “3”).  

 

                                                                

4.      To view the details of the selected filter column, click (highlighted as “4”)

5.      Filter - <Column Name> dialog box (highlighted as “5”) appears.

 

                                                                                

6.      To apply filter on the selected report, perform the following:

a.      Under Filter – <Column Name> dialog box, in the Filter Label box, type the filter label.

b.      To make the parameter required, select Required checkbox.

c.       In the Name box, type the name of the parameter.

d.      To apply the required filter, select the required radio button from the following:

·         Select from list

·         Match a specific string

·         Specify a condition

e.       Based on the selected radio button, fields underneath change.

f.       If no condition is required, select No Condition.

7.      Type the required fields, and then click OK.

8.      To close the dialog box, click Cancel.

 

¥  In View Mode, Hide/Show User Filters option is available. This enables the user to select the list of values in the parameter.
