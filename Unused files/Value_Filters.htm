Value Filters

Value filters filter by values. Value filters are available to Row labels and Column labels. Value filters contain the following filters:

·         Equals

·         Does not equal

·         Greater than

·         Greater than or equal to

·         Less than

·         Less than or equal to

·         Between

·         Not Between

·         Top 10

 

To set the Value Filters

 

1.      Under Row labels/Column labels, click the required field, and then point to Value filters, and then click the required filter value.

 

               

                                           

 

 

2.      Value filter <field name> dialog box appears.

 

       

                                

3.      In the Show items for which list, click the required option.

4.      In the filters list, select the required option.

5.      In the box next to the filters list, type the value.

6.      Select the Case sensitive checkbox, to make the applied filter case sensitive.

 

                                    

7.      To save the changes, click OK.

8.      To close the dialog box, click Cancel.
