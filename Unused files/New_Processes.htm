New Processes

To work with New Processes

1.      On the left-hand side menu, point to Monitoring, and then click Processes.

2.      Scheduled Processes page appears with available processes.

3.      On the right-hand side, click New Process (highlighted as “1”).

 

 

4. The following jobs come under New Processes.

 

  • Import Discoverer Object(s)

  • Import MetaObject Security

  • Purge Processes

  • Export MetaObject Security

  • Change Date Format

  • Import Domain/Report

  • Change Ownership

  • Import Users,Role and Groups from Excel

 

 

# More processes can appear in this menu, if they are added under Jobs in Security section
