Navigating to Drilldowns from Dashboard

 

To navigate to drilldowns from dashboards

 

1. Create a new dashboard report.

2. Drag and drop a report that has drilldown applied on it.

3. The report appears as shown below:

 

 

Note: In the above report, drilldown is applied on the region.

 

4. To save the report, click Save.

5. To run the dashboard report, click Run.

6. To view the drilldown in a report, select the required value under the required column that has drilldown applied on it.

7. Here, select the “West”, then click Drill to Report, and then click “Drilldown 1”.

 

  

8. This navigates to the report “Drilldown 1” as shown below:

 

 

9.  To export the report, click Export.
