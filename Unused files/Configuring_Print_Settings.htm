Configuring Print Settings

To Configure Print Settings

1.      On the Security menu, click Settings.

2.      Email tab appears by default.

3.      To create print settings, click Print tab.

4.      To create new print settings, click New.

5.      Textboxes (highlighted as “1”) appear under Name, UserName, Password, and Print Server Alias.

 

6.      On the Print tab, perform the following:

·         In the Name box, type the name of the printer.

·         In the UserName box, type the user name for the printer setting.

·         In the Password box, type the password for the printer setting.

·         In the Print Server Alias box, type the alias name of the print server.

·         To save the changes, click Save.

·         In Actions, click Remove (highlighted as “2”) to delete the print settings.

 

·         To modify the required print settings row, double-click on it.

·         Edit/make the required changes, and then click Update.
