Creating a Data Report using Dataset

To Create a Data Report using Dataset

1.      On the Reports section, click New Report, and then click Data Report. 

2.      In the Data Explorer, click Upload File.

3.      Upload File dialog box appears.

4.      To create a data set, perform the following:

a.       In the Dataset Name box, type the name of the dataset.

 

?   Example: Dataset Name: Demo Dataset

 

b.      In the Description box, type the description for the dataset.

 

          ?   Example: Description: This is demo dataset

 

c.       To navigate to the next window, click Next.

d.      Upload the required file in the required format, and then click Next.

 

?   Example: Required File Type: CSV or Other Delimited Text Format

                   Required File: SuperStore1.csv                           

 

 

e.       The columns present in the uploaded file appear.

f.        To import the file, click Import.

g.      Under Business Object section, the business object associated with the newly created dataset appears with columns under it. 

h.      Drag and Drop the required columns from Business Objects area to Report Configurator area.

i.        The report appears in the Run Results area.
