Creating discoverer Data Source

To create <discoverer> data source

 

1.  On the left-hand side menu, point to Business Models, and then click Data Sources.

2.  Data Sources page appears with a list of available data sources.

3.  To create a new data source, click New (highlighted as "1").

     

 

4.  Data Source dialog box appears. 

5. Select the required data sources, available in the Data Source dialog box and click Ok.

6. New Data Source dialog box appears. 

7. In the Name box, type the data source name.

           Example: Name: discoverer

8. In the Description box, type the description, if needed.

Note:   Description field is optional.

9. In the Data Source Type, select the required option from the following:

    • JDBC

    • JNDI

Example: Data Source Type: JDBC

10. In the Select Database list, select the required option.

            Example: Select Database: Oracle JDBC (Oracle Thin Driver)

11. In the Username box, type the required username.

            Example: Username: apps

12. In the Password box, type the password.

            Example: Password:apps
       Note: The username and password are required to connect to the apps database.

13. In the Oracle JDBC Settings, perform the following:

a.       In the Host Name box, type the required host name.

Example: Host Name : localhost

b.      In the Port box, type the required port.

         Example: Port : 1521

c.       In the Database Name box, type the required database name.

         Example: Database :orcle

Note: Driver Source and Data Source Class fields are automatically populated based on the Select Database option.

d.      To test the connection, click Test Connection.

e.       To save the changes, click Save.
