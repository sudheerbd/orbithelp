Charts in Pivot

 

To View the Charts in Pivot

1. Open the required data report.

2. In the reports Visual Settings,  perform the following:

    a. On the General tab, in the Default View box, select the required option from the following :

             i. table

             ii. Pivot

          Example: Default View: pivot

     b. On the Pivot tab, in the Show Chart box, select the required option from the following :

             i. True

             ii. False

          Example: Show Chart option: True

       To view the chart in pivot, set the Show Chart option to "True".

 3. The Pivot report appears as follows with Chart option (highlighted as "1") as shown below:

 

 

4. To expand the chart panel, click   Expand panel.

5. To view multiple chart options, click .

6. The following chart options appear:

    • Column
    • Line
    • Area

Exporting Pivot with Excel

To Export Pivot with Excel

 

1. Open the required data report.

2. Change the report to pivot.

3. To export the pivot with excel, perform the following:

      a. On the top-menu, click Options, point to Export, and then click Excel xslx.

      b. Process gets submitted.

  4.  To view the submitted process, perform the following:

      a. On the left-hand side menu, point to  Monitoring, and then click Processes.

      b. A list of processes appears.

      c. From that, select the required process.

      d. The output appears on the right-hand side under Output(s) & Delivery.           

      e. Select the required output.

       f. The output gets downloaded as an excel file.

       g. Open the excel file to view the output.         

             

 

The output appears in the sequence the columns that are present under All fields section.
