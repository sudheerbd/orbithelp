Creating Conditions

 

To create conditions

1.      Under Selected Columns area, the Conditions area appears.

2.      Drag and drop the required column from the Business Objects area to Conditions area.

3.      Under the Conditions area, perform the following:

a.       In the Conjunction box, select/type the required operator from the following:

·         AND

·         OR

·         AND_NOT

·         OR_NOT

b.      In the Aggregation box, select/type the required aggregate function from the following:

·          NONE

·         SUM

·         AVERAGE

·         COUNT

·         COUNT_DISTINCT

·         MINIMUM

·         MAXIMUM

 

c.       In the Column Name box, the column name appears automatically.

d.      In the Operator box, type/select the required operator from the following:

·         =

·         < 

·         > 

·         <=

·         >=

·         <> 

·         LIKE

e.       In the Value box, type a value to set the condition.

4.      To remove the condition from the Conditions area, click  Remove selected rows.
