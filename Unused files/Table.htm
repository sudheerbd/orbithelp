Tabular Charts

In Tabular Charts, data is presented in rows and columns. The following type of charts come under tabular charts:

·         Table

Widget Settings of Tabular Charts

To set the widget settings

1.         On the report edit page, click Widget Settings.

2.         The following tabs appear:

i.    General

ii.   Tabular

iii. Paging

 

General

 

1.      Under General tab, perform the following:

a.      In the Row Number box, select/type the required option from the following:

i. True

ii. False

Row Number: True

This field displays the row number on the left-hand side (highlighted as “1”).

b.       In the Column Lines box, select/type the required option from the following:

i. True

ii. False

Column Lines: False

When Column Lines option is set to False, the columns in the report disappear. Columns Lines appear only when the Column Lines option is set to True

c.      In the Row Lines box, select/type the required option from the following:

i. True

ii. False

Row Lines: True

 When Row Lines option is selected as False, the rows in the report disappear. Row Lines appear only when the Row Lines option is set to True

d.      In the Show Grid Filter box, select/type the required option from the following:

i. True

ii. False

Show Grid Filter: True

When Show Grid filter option is selected as False, the grid filter in the report disappear. Show grid filters appear only when the Row Lines option is set to True

Paging

1.      Under the Paging tab, perform the following:

a.       In the Paging box, type/select the required option from the following:

i. True

ii. False

Paging: True

When Paging is set to true, this enables the paging option at the bottom, by default (highlighted as “1”)

b.   In the Type box, the type appears as “Client” by default.

c.      In the Position box, type/select the required option from the following:

i.  Bottom

ii.  Top                                                 

Position: Top


When the Position is set to Top, the paging option is displayed on the top
By default, the paging option is displayed in the Bottom

d.       In the Size box, type/select the required option.

Size:5

The size option provides the number of records that must be displayed from the existing report. Here, the size is mentioned as 5, this displays 5 records from the current report (highlighted as “2”)

 

 

4.      To apply the widget settings, click Done.
