Configuring FTP Settings

To configure FTP Settings

1.      On the Security menu, click Settings.

2.      Email tab appears by default.

3.      To create FTP settings, click FTP tab.

4.      To create new FTP settings, click New.

5.       Textboxes (highlighted as “1”) appear under Name, ServerName, UserName, Password, RemoteFile, and RemoteDirectory.

 

6.      Under the FTP tab, perform the following:

·         In the Name box, type the name of the FTP setting.

·         In the ServerName box, type the name of the server name.

·         In the UserName box, type the user name.

·         In the Password box, type the password.

·         In the RemoteFile box, type the path of Remote File.

·         In RemoteDirectory box, type the path of the Remote Directory.

·         To enable the security for the required FTP Setting, select Secured checkbox.

·         To delete the required FTP Settings, click Remove.

 

·         To save the fields, click Save.

·         To modify the required FTP settings row, double-click on it.

·         Edit/make the required changes, and then click Update.
