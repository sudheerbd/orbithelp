Custom Widgets

To View Custom Widgets

 

1. Open the required analysis report.

2. In the report configurator area, click Change Widget, and then click More Widgets.
3. Custom Widgets dialog appears.


 
 

4. To create a new custom widget, on the New tab, perform the following:

 a. In the Name box, type the required name.

     Example: Name: Pyramid Test Chart

 b. In the Description box, type the required description.

      Example: Description: This is a test chart

      Note: Description field is optional.

 c. From the chart library select the required chart from the following:

  • D3 Js

  • Goggle charts

  • C3 Js(Extension of D3 Js)

  • Chartist charts

  • Chart charts

    • Other

d. If “Other” option is selected, then CDN Path field appears.

e. In the Icon box, upload the icon file.

 

 

Choose the required axes that need to appear for the customized chart.

  • X -Axis

  • Y-Axis

  • Size Axis

  • Color Axis

 

Example: Axis : X -Axis

                            Y-Axis

Properties

Properties of Custom Widgets should be configured by the developer; developer must be aware of the variables using the script and the values to be populated to be applicable on the respective charts.
