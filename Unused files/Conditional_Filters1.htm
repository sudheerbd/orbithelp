Data Filters

 

To apply a data filter on a report

1.      Drag and drop the required columns from Data Explorer into Report Configurator under Configurator tab.

2.      To apply the data filter, click Data Filters tab (highlighted as “1”) next to Configurator tab.

 

        

        

3.      From Data Explorer, drag and drop the required column (highlighted as “2”) that can be applied as filter on the report, into Data Filters  tab.

4.      To view the list of values (highlighted as “3”) under the selected filter, click  Search for items to add.

 

      

                                 

5.      From the available list of values, select the required value.

6.      The value appears under the selected filter in Data Filters tab.

 

                              

7.      To run the report, click Run.

Procedure 2:

 

To filter the data, perform the following

 

1.      In the Report Configurator area, under Data Filters tab, the selected column appears.

2.      On the selected column, click (highlighted as “1”).

 

       

             

3.      From the conditions list, select the required condition (highlighted as “2”).

                          Example: Required Condition: Contains

4.      In the textbox beneath, select the filter value by clicking  (highlighted as “3”).See Selecting Default Values for more information.   

                           Example: Filter Value: West

 

           

5. The selected value appears in the text box (highlighted as “4”).

6. To run the report, click Run.
