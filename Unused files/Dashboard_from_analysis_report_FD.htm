Dashboard from analysis report

To create a dashboard report from existing analysis report

1. Navigate to the Reports page.

2. On the Reports page, click New, click Dashboard, and then click Ok.

3. The Dashboard report page appears in Edit Mode. The Dashboard report page has a Data and Visual Settings area (highlighted as ”1”), a Run Result area (highlighted as ”2”), and a Report Configurator area (highlighted as “3”).

Note: See Menu Options in Analysis Reports for information about menu options in the analysis report page.

4. Click Add Widget (highlighted as "4"), and then click Visualization.

5. Add Visualizations window appears.

6. Add visualization window contains list of all Analysis Reports.

7. Click against the report you want to add to dashboard.

Note: A Visualization added to dashboard notification appears briefly at the top of the ORBIT application window.

8. Add desired visualizations and click Done.

9. Selected visualizations appear in the dashboards page.
