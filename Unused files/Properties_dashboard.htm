Properties

 

To set the Properties 

 

  1. Under Reports area, right-click the Dashboard report, and then click Edit.

Note: Properties appear only in Edit Mode.

  1. To set the properties, click   Options menu, and then click  Properties.

  2. Properties dialog window appears.

    

 

  1. On the left-hand side, the Name, Description, and Auto Refresh fields appear.

  2. On the right-hand side, the properties of report name and description of the report appear.

  3. For the selected report, the following properties appear:

a.   Under Name, type the name of the report. 

b.    Under Description, type the description of the report.

c. Under the Auto Refresh tab, enter the duration time to run the report automatically.
