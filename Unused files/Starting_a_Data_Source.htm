Starting a Data Source

To start a data source

 

1.       Under Data Sources, a list of data sources (highlighted as “1”) appears.

 

 

2.   To start a data source, click  for the required data source.

 

    • Example: Required Data Source: Test Data Source

 

3.       By default, the data source comes in Stop Mode.
