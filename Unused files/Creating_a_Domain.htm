Creating a Domain To create a Domain

  1. On the left-hand side menu, point to Business Models, and then click Domains.
  2. Domains & Data Models page appears with a list of available domains. 

 

 

 

  1. To create a new domain, click New.
  2. New Domain dialog box appears.

 

                                 

  1. In the New Domain dialog box, perform the following:

·         In the Name box, type the name of the domain.

 

?   Example: Name: Sample Domain

 

·         In the Description box, type the description.

 

¥   Description is optional.

 

  1. To save the domain, click Save.
  2. To close the dialog box, click Cancel.
  3. The newly created domain (highlighted as “1”) appears under Domains & Data Models area.

 

 

 

  1. To view the objects under the required domain, double-click the domain name.
  2. The required domain page appears with a Domain Hierarchy that has two options: Physical Models and Logical Models.
