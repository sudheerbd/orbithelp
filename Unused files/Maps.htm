Maps

Maps are used to display geographical data by category like sales by states.

 

To set the Visual settings

1.      On the report edit page, click Visual Settings.

2.      Under the Visual Settings the following tabs appear:

                                                  i.      General

 

General

1.    Under General Tab, perform the following:

a.       In the Title box, type the title.

       Example: Title: Map Chart

 

This title appears when viewing the report in the Dashboards.

b.             In the Map Region box, select the country.

Example: Map Region: United States of America

 

     c.       In the Field Code box, select the required option from the following:

  1. Name

  2. Postal Code

  3. HC-Key

  4. ISO-A3

  5. ISO-A2

  6. HC-A2

Example: Field Code: Name

 

When the Field Code is set to Name, the state names in the map are displayed completely (highlighted as “1”).

d.      In the Draw Type box, select the required option from the following:

                                                        i.      Color Map

                                                      ii.      Bubble

     Example: Draw Type: Bubble

 

e.  In the Color & Style box, select the required theme for the map.

 

 Example: Color & Style: CycleMap

 

  f. In the Hide box, type/select the required option from the following:

 

                                                        i.            True

                                                      ii.            False

  

   Example: Hide Option: False.

 

 When Hide is set to false, the legend appears on the map.
