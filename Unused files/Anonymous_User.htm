Anonymous User

An Anonymous user doesn't have the login credentials of the ORBIT Analytics application. An Anonymous user has limited privileges as opposed to other seeded users.

 

To view the reports as an anonymous user, perform the following:

 

1. In the browser, type the following URL.

 

     # Example: URL: http://orbit.polaxian-05:5050/reporting/public

 

2. Home page for the anonymous user appears.

 

    

 

3. In the Home page, two sections -- Categories and Home appear.

4. Under the Categories section, the report folders shared with anonymous user appear.  

5. On the right-hand side section (highlighted as "2") reports under the shared report folders appear.

6. To view the reports of the anonymous user, perform the following:

7. On the left-hand side, under Categories section, click the required folder.

    

      # Example: Required Folder: test

8. The reports under the selected folder appear on the right-hand side.

9. To view the report, click on the required report from the list of available reports.

 

      # Example: Required Report: Test Invoices Report

 

10. The report appears with the  column names.

11. To run the report and view the values under the available column names, click Run.

12. The report appears as shown below:  

 

 

# Paging facility for the report appears, if it is enabled in the Widget Settings of the original report.

 

13. For the data report, following menu appears above the report.

 

 

S. no. Menu Items Description
1. Run  This option is used to run the report.
2. Hide/Show User Filters This option is used to hide/show the user filters. This option is useful where parameters are available for the reports.
3. Hide/Show Grid Filters This option is used to hide/show the grid filters. Grid filters are available only for data filters.
4. Clear Grid Filters  This option is used to clear the grid filters applied on a report.
5. Table  This option enables the user to view the report in Table format.
6. Pivot This option enables the user to view the report in Pivot Format
7. Options  Options menu provides the following:

  • Auto Column Size
  • Export  
    • Excel xlsx

 Anonymous user can export the report in Excel xlsx format.  

 

14. For the Analysis report, following menu appears above the report.

S. no. Menu Items Description
1. Run  This option is used to run the report.
2. Hide/Show User Filters This option is used to hide/show the user filters. This option is useful where parameters are available for the reports.
3. Options  Options menu provides the following:

  • Underlying data - this option displays the underlying data of the report.

 

15. For the Dashboard report, following menu appears above the report.

S. no. Menu Items Description
1. Run  This option is used to run the report.
2. Hide/Show User Filters This option is used to hide/show the user filters. This option is useful where parameters are available for the reports.

 

16. For the dashboard report, click Options, and then click Underlying Data to view the underlying data.
