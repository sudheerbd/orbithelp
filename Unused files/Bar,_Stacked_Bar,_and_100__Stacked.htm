Widget Settings of Bar Charts

To set the Visual settings

1.      On the report edit page, click Visual Settings.

2.      Under the Visual Settings the following tabs appear:

  • General

  • Category Axis

  • Value Axis

  • Data Labels

  • Legend

  • Axis Label Styles

 General Tab

 

1. Under General tab, perform the following:

a)      In the 3D box, select/type the required option from the following:

                                i.            true

                              ii.            false

Example: 3D: True

 

b)      In the Theme box, select the required theme.

Example: Theme: Sky

 

c)     In the Horizontal Grid box, select the required option.

Example: Horizontal Grid: False

 

d)      In the Vertical Grid box, select the required option.

Example: Vertical Grid: False

 

e)       In the Zoom box, select the required option from the following:

a.       True

b.      False

¥    If Zoom is set to True, the user can zoom into the report. To zoom into an area in the report, select the area and drag the mouse. To zoom out of the area, double click the area.

 

f)      In the Background box, pick the required color from the color palette.

Example: Background: d3c8c8

 

g)  In Allow measures in dimensions, type/select the required option from the following:

a.     True

b.    False

 

  •  The report appears as follows when the above widget settings in the General tab are set:

 

Category Axis

1.  Under the Category Axis, perform the following:

a.       In the Title box, type the required title. 

Example: Title: X- Axis  

b.      In the Display box, type/select the required option from the following:

                                            i.            True

                                          ii.            False

 

   When Display is set to True, the Category Axis Title appears in the report.

 

c.       In the Rotate box, type/select the required value.

Rotate: -45

 

d.      In the Position box, select the required option from the following:

                                            i.            Left

                                          ii.            Right

  Example: Position: Right

 

By default, the Category Axis appears in the left.

Value Axis

1.      Under the Value Axis, perform the following:

a.       In the Position box, type/select the required option from the following:

                                i.            Top

                              ii.            Bottom

By default, the Value Axis appears in the bottom.

 

b.      In the Title box, type the required title.

Example: Title Box: Y-Axis

c.       In the Display list, select the required option from the following:

                                i.            True

                              ii.            False

   When Display is set to true, Value Axis title is displayed.

d.   In the Y2-Axis box, select the required option from the following:

                                i.            True

                              ii.            False

  When Y2-Axis is set to True, second Y-axis is displayed.

e.    In Y2-Axis Title box, enter the title for Y2-Axis.

f.  In the Rotate box, type/select the required value.

  Example: Rotate: 45

g.  In the Scale box, type/select the required option from the following:

                                i.            Thousands

                              ii.            Millions

                            iii.            Billions

                            iv.            Trillions

                              v.            None

Example: Scale: Millions

 

h. In the Ticks Count box, type/select the required value.

 

  • The report appears as follows when the above widget settings in the Value Axis tab are set:

 

 

Data Labels

1.      Under Data Labels tab, perform the following:

a.   In the Bar Labels box, type/select the required option from the following:

      1. None

      2. Inside Start

      3. Inside End

      4. Outside 

Example: Bar Labels: Outside

 

When Bar Labels is set to Outside, Bar labels appear outside the bar.

 

b.      In the Color box, pick the required color from the color palette.

Example: Color: d43535

 

 The bar label color changes to the above-mentioned color.

 

c.       In the Weight box, type/select the required option from the following:

      1. Normal

      2. Bold

      3. Bolder

      4. Initial

      5. Inherit

      6. Lighter

Example: Weight: Bolder

 

d.      In the Orientation box, type/select the required option from the following:

      1. Horizontal

      2. Vertical

Example: Orientation: Vertical

 

The bar labels are displayed vertically.

 

  • The report appears as follows when the above widget settings in the Data Labels tab are set:

 

Legend

1.      Under the Legend tab, perform the following:

a.      In the Hide tab, select the required option from the following:

      1. True

      2. False

Example: Hide: False

 

  • When Hide is set to false, then the legend is displayed (highlighted as”1”).

  • Legend is displayed only when two or more columns are present in the Values (Y-Axis) section.

 

b.      In the Position box, type/select the required option from the following:

      1.  Bottom

      2. Right

      3. Left

      4. Top

c. In the Title box, type the title for the legend.

 


Styles

1.   Under the Axis Label Styles tab, perform the following:

a. In the Size box, type/select the required size.

Example: Size: 20

 

b. In the Weight box, type/select the required option from the following:

      1. Normal

      2. Bold

      3. Bolder

      4. Initial

      5. Inherit

      6. Lighter

 

Example: Weight: Bolder

 

c.   In the color box, pick the required color from the color palette.

 

Example: Color: 514bd8

 

  • The report appears as follows when the above widget settings in the Styles tab are set:

 

Here, the styles are applied to the axes labels (highlighted as “1”).

2.      To apply the widget settings, click Done.
