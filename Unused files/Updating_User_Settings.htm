Updating User Settings

To update the user settings

 

  1. In the Processes page, on the right-hand side, click New Process, and then click Update user settings.

  2. Update user settings dialog appears.

 

  1. Under the Parameters tab, perform the following:

    1. To apply the settings to all the users, click All Users checkbox.

    2. From the Language box, type/select the required language.

      • Example:  Language: Japanese

    3. In the Date Format box, type/select the required date format.

      • Example: Language: yyyy/MM/dd

    4. In the Number Format box, type/select the required number format.

      • Example: Number: 10 000.00

    5. In the Default Page box, type/select the required page.

      • Example: Default Page: Reports

 

  1. To schedule the process immediately, select Run Immediately check box.

  2. To schedule the process at a later date, deselect the Run Immediately check box.

  3. Schedule tab becomes available. View Scheduling for more information.

  4. To set the notifications, click Notifications tab. View Notifications for more information.

  5. To submit the process, click Submit.

 

These settings override the settings at the user level and can be applicable to all the users.
