Radar

Radar charts are similar to line charts but they use a circular axis instead of a linear grid.

 

To set the Visual Settings

1.      On the report edit page, click Visual Settings.

2.      Under the Visual Settings the following tabs appear:

a.       General

b.      Value Axis

c.       Data Labels

d.      Legend

 

General

 

1.      Under the General Tab, perform the following:

a.       In the Title box, type the required title.

?     Example: Title: Radar Chart

b.      In the Theme box, type/select the required theme.

?     Example: Theme: Category4

 

?   When the widget settings in the General tab are applied, the report appears as follows:

 

Value Axis

To view the common widget settings, see Value Axis tab under Polar Charts.

 

Data Labels

To view the common widget settings, see Data Labels tab under Polar Charts.

 

Legend

To view the common widget settings, see Legend tab under Polar Charts.
