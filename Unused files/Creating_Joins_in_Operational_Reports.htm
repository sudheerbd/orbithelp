Creating Joins in Data Reports

To create joins between two tables

1.      In the Reports area, click, and then click Data Report.

2.      Business Object area appears along with Report Configurator and Run Results area.

3.      In the Business Objects area, click Upload File, to upload the first table (highlighted as “1”).

 

                                           

4.      To upload the second table (highlighted as “2”), click Upload File again.  

 

       

                                                                                 

 

 

5.    In the Business Objects area, the newly imported tables (highlighted as “3”) appear.

          

        

                  .      From the Business Objects area, drag and drop the required fields (highlighted as “4”) from the two tables, on to the Configurator tab, in the Report Configurator.

 

   

                        

7.      To create joins between the tables, click Relationships (highlighted as “5”).

8.      Relationships dialog box appears with two imported tables.

9.       To create a join between the two tables, select the source table.

10.  The source table appears highlighted and a hand icon appears when the mouse is hovered on it.

 

¥    When a hand icon appears on table, it implies that a join can be created from the table.

 

11.  Drag the arrow from source table to the destination table to create a join.

12.  Relationship (source table => destination table) dialog box appears.

 

 

                            

 

13.  Under Relationship (source table => destination table) dialog box type the required fields, and then click OK. See Establishing joins between tables

14.  The join appears between the two tables.

 

       

                                  

 

15.  To save the changes, click Ok.

16.  To close the dialog box, click Cancel. 

17.  A message “Relations saved successfully” appears.
