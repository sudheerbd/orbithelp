Creating a Drill-through in Dashboard

To Create a Drill-through in Dashboard

 

  1. Create a New Dashboard Report.

  2. From the Widgets section, drag and drop the report onto the dashboard area.

  3. On the report, click  Options, and then click Drilldowns.

 

 

  1. On the Drilldowns box, perform the following:

a. Drag and drop the required drilldown report.

b. Under the drilldown report, in the Drilldown Column box, drag and drop the required column.

     Example: Required Column: Region

c. Under the Parameters, enter the required parameter value

      Example: Required Parameter Value: Region

d. To save the changes, click Ok.

 

  1. To save the report, click Save.

  2. To run the report, click Run.

  3. To view the drilldown in a report, select the required value under the required column that has drilldown applied on it.

  4. Here, select value, then click Drill to Report, and then click “drilldown 1”.
