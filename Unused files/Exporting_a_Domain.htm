Importing a Domain

To import a domain

  1. On the left-hand side menu, point to Business Models, and then click Domains .

  2. Domains & Data Models page appears with a list of available domains.

  1. To import a domain, click Import.

  2. Import Domain File dialog box appears

 

          

                                 

  1. To select a domain file, click Select Domain File.

  2. Open dialog box appears.

  3. Select the required file, and then click Open.

  4. The path of the domain file (highlighted as a”1”) appears in the textbox before Select Domain File button.

 

           

                                   

  1. To replace the existing domain, select Replace existing domain checkbox (highlighted as “2”).

  2. To import the domain file, click Import.
