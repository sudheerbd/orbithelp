Uploading Files

To upload files

1.      In the Business Object area, click Upload File (highlighted as “1”).  

 

           

 

¥   This procedure for uploading files is same as Import from File option.

 

2.      Upload File dialog box appears.

 

 

a.       In the File Type box, type/select the required option.

File Type field has the following options:

o   CSV or Other Delimited Text Format

o   Excel

?  Example: File Type: CSV or Other Delimited Text Format

 

¥     Here, CSV or Other Delimited Text Format option is selected.


        b. To upload a file, click Select File.

                   c. The selected file appears in the textbox next to File.

                   d. To navigate to the next section, click Next.

                   e. The following screen appears:

 

     

 f. The columns in the CSV file appear with their associated data types.

 

¥     Edit the data types as required.

¥     Update the fields as required.

 

a.       To import the fields, click Import.

b.      To navigate to the previous screen, click Previous.

c.       To cancel the import and close the dialog box, click Cancel.

3.      The columns in the selected CSV file appear in the Business Objects area.

 

    

 

4.      Drag and drop the required columns from Business Object area to the Configurator area under Report Configurator, and then click Apply.

    5.      Report appears in the Run Results area.

 

¥     In the Query Settings, if the Auto Refresh check box is selected, the report is automatically generated in the Run Results area. 

¥     In the Query Settings, if the Auto Refresh check box is not selected, you need to run the report explicitly by clicking Run.
