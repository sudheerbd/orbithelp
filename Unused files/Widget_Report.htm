Widget Report

To View the Widgets

 

1. On the Reports section, click New Report, and then click the required report.

2. Data Explorer (highlighted as “1”) and Widgets (highlighted as “2”) tabs appear.

 

       

                Note:  Data Explorer appears open by default

3. To work with widgets, click Widgets tab.

4. On the Widgets tab, the following appear:

  • My Folders

  • Shared Folder

                Note: Under My Folders, the folders and the associated reports created using the current user/role appear.

 

 

                 

5. Under Shared Folders, the reports that were shared by a specific user/role appear along with the parent folder.
