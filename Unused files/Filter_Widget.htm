Filter Widget

 

Filters Widget are parameter-based filters that allow the user to input a value, to filter the data in the report. However, in Dashboard report the filter widget will sort the data of all the widgets of Dashboard report.

 

Creating a Filter widget

 

  1. On the top-menu, from the Add Widget list, select Filter Widget.

  2. Filter Widget appears with a message “Drag and Drop Fields here to filter data” in dashboard, as shown below.

  1. Drag and drop the required column in Filter Widget from the selected business object.

  2. After dragging the column, the following options appear as shown below:

  • Combobox

  • Checkbox Group

  • Radio Group

Filter Options

 

Filter widget appears with two options:

    • Layout

    • Refresh

 

Layout

  1. In the Filter widget, click Options in Filter widget header, and then point to Layout.  

  2. Layout menu appears with the following two options:

  • Horizontal

  • Vertical

 

Note:  Layout option is applicable on multiple filters.

 

 

 

Horizontal Layout

 

a. From the Layout menu, select Horizontal.

Refresh

 

  1. In the Filter widget, click Options in Filter widget header, and then point to Refresh.

  2. Refresh menu appears with the following two options:

  • Auto

  • Manual

 

Note:  Refresh option selected as Auto by default .
