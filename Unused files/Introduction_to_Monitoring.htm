Introduction to Monitoring

 

In the section, you can schedule new processes, re-start the processes, or cancel processes.  

Scheduled Processes contain the following fields:

 

S.no

Fields

Description

1.       

Process Id

This describes the process id

 

2.       

 Name This describes the Report name

3.       

Phase

This describes the phase of the process.

4.       

Status

This describes the status of the process.

5.       

Start Time

This describes the start time of the process.

6.       

End Time

This describes the end time of the process.

7.       

Output

This provides the output of the process.

8.    

Log

This provides the log of the process
