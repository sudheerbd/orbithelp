Gauge Charts

Gauge charts, also known as dial charts or speedometer charts, use needles to show information as a reading on a dial. The goal, or target value, is represented by the line (needle).

 

 

To set the Visual settings

1.      On the report edit page, click Visual Settings.

2.      Under the Visual Settings the following tabs appear:

·         General

·         Range

·         Value Axis

·         Data Labels  

General

1.   Under the General Tab, perform the following:

a.       In the Title box, type the title.

Example: Title: Gauge Chart

 

This title appears when viewing the report in the dashboard.

 

     b.         To set the donut size, slide the Donut slider left or right.

 

?    Example: Donut Size: 74

 

c.       In the Theme box, type/select the required theme.

?    Example: Theme: Category 4

 

d.      In the Show Needle box, type/select the required option from the following:

                                                                          i.   True

                                                                        ii.   False

?    Example: Show Needle Option: True

¥   When Show Needle is set to True, the needle appears showing the target value. (highlighted as “1”)

When the above-mentioned widget settings are applied, the report appears as follows:

 

 

 

Range

Here, the settings in the range tab have been explained using an example.

Let us say there’s a cloth retailer, where the target average sales per month is set to 2500.  Then 2500 is the target value or the maximum value, and the minimum possible value is 0.

So, if the cloth retailer has an average sale of 2035, then the retailer has achieved 80.72% of sales, and needs 19.28% of average sales to achieve the target average sale of 2500 per month.

 

1. Under the Range tab, perform the following:

 

¥   In the Min Value box, type/select the required value.

?     Example: Minimum Value: 0

¥   In the Max Value box, type/select the required value.

?    Example: Maximum Value: 2500

¥   In the No of segments, type/select the required value

    ?     Example: No of segments:4

 

¥    Based on the values in the No of segments box, the percentage is calculated as per the number of segments mentioned. (highlighted as “1”).

?      When the above-mentioned widget settings are applied, the report appears as follows:
