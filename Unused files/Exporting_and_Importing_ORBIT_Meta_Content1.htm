Exporting and Importing ORBIT Meta Content

 

Prior to exporting report, you must export domain from source instance to target instance.  

 

 

 

 

 

The order of the export/import process is as follows:   

• Exporting Domain

• Importing Domain

• Exporting Report  

• Importing Report

 

If the domain and reports exported/imported in the source and target instances are pointing to the same data source, then there is no need to edit the data source while importing the domain/reports; however, if they are pointing to different data sources, then you need to edit the xml file with the appropriate data source details while importing into target instance.
