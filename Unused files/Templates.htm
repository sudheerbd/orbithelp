Working with XSLFO Template

In the Templates, the user can attach the XSLFO template. XSLFO template comes with an extension .fo.

 

In this section, the following sections appear:

  • Adding a Template

  • Configuring Template

  • Viewing the Output

  • Editing the Template

  • Deleting a Template

 

Adding a Template

To add a Template

1.      For the required report, click  Options above the Run Results area, and then click Template.

 

?     Example: Required Report: Store

   

     

 

2.      Templates window appears.

 

3.      On the right-hand side, upper corner of the Template window, click Add Template (highlighted as “1”).

4.      Various fields to create a new template appear.

 

 

5.      In the Templates window, perform the following to create a new template:

a.       In the Name box, type the name of the template.

 

        ?   Example: Template Name: Store

 

b.      In the Description box, type the description of the template.     

 

   ?   Example: Description: Creating Template for Store

 

 

¥    Description field is optional.

 

c.       In the Type box, type/select the required template from the following options:

                                                              i.      XSLFO

 

?   Example: Description: XSLFO

 

d.      To upload file, click Upload file next to Upload textbox. 

e.       The required template file appears in the Upload text box. 

 

 ?   Example: Required Template: Store.fo

 

f.        To save the template, click Save.

6.     The newly created template appears as shown below:

 

 

 

Configuring Template

To view the template

1.      On the Reports tab, under the Reports section, right-click the required report, and then click Schedule.

 

            ?   Example: Required Report: Store

 

2.      Schedule Report dialog box appears.

3.      In the Schedule Reports dialog box, click Output & Delivery tab.

 

                                              

 

4.     In the Output & Delivery tab, under the Outputs and Deliveries section (highlighted as “1”), perform the following:

 

5.      On the right-hand side, click Add output (highlighted as “2”).

6.      A row to add the output appears.

7.      In the Output Type list, click the required option from the following:

·         CSV

·         XML

·         PDF

·         RTF

·         HTML

?   Example: Output Type: PDF

 

8.      In the Template list, click the required option (highlighted as “1”).

 

?   Example: Template: Store

 

 

 

         Here, the Default template comes for every output type. Default template is a basic template.

 

         9. The required template appears as shown below:

 

                                              

 

¥   This template is created using Templates options under in Reports section.

10. To submit the template, click Submit.

 

Viewing the Output

To view the output with the attached template

 1. On the left-hand side menu, point to Monitoring, and then click Processes.

 2. Scheduled Processes page appears with a list of available processes (highlighted as “1”).

 

 

 

3. Under Scheduled Processes, select the required process (highlighted as “2”).

4. The output for the required process appears on the right-hand side in the Output(s) & Delivery. 

5. In the Output(s)& Delivery section, under the Output column, select the required output (highlighted as “3”).

6. The output appears with the applied template as shown below:

   

     

 

Note:  Templates can be used to apply various conditional formats, including changing the background color, font color, font weight and so on.
