Tabular Charts

In Tabular Charts, data is presented in rows and columns. The following type of charts come under tabular charts:

  • Table

Visualization (Chart type) Options

1. Table chart has one visualization option - Columns

2. Drag and drop the required fields from the selected Business Objects into the Columns.

3. Orbit application generates the tabular analysis report dynamically in the run results area.

 

Visual Settings for Tabular Charts

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for tabular chart has following tabs:

  1. General
  2. Tabular
  3. Paging

General

General tab has the following visual settings:

A. Row Number: Show/hide row numbers on the table.

Default Value: False

In the Row Number box, select True to show the row number on the left-hand side of the table.

B. Column Lines: Show/hide column lines.

Default Value: False

In the Column Lines box, select True to show the column lines in the table.

C. Row Lines: Show/hide row lines.

Default Value: False

In the Row Lines box, select True to show the row lines in the table.

D. Show Grid Filter: Show/hide grid filter.

Default Value: False

In the Show Grid Filter box, select True to show filter in the table.

 

Tabular

Tabular tab has the following visual settings:

A. Row Banding: Show/hide shading to every other row in the table.

Default Value: False

In the Row Banding box, select True to show banding on alternate rows.

 

B. Grouping Mode:

Default Value: Single

 

C. Locked Column:

Default Value: 0

 

Paging

Paging tab has the following visual settings:

A. Paging: Enables/Disables display of table report in multiple pages

Default Value: False

In the Paging box, select True to enable display of report in multiple pages.

 

B. In the Type box, the type appears as “Client” by default.

 

C. Position: Changes position of page numbers.

Default Value: Bottom

In the Position box, select Top to display paging at top of the page.

 

D. Size: Controls number of rows to show on each page

Default Value: 100

In the Size box, select 10 to display only 10 rows of data per page.
