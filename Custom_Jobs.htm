Custom Jobs

A Custom job in Orbit is a process that is used to customize the business requirements and implement it as a job in the application.

Apart from deployment and development of Custom Jobs, Orbit also supports the scheduling of the jobs.

 

Scheduling of jobs is done using the following two ways:

  1. Orbit allows to configure scheduling jobs in Orbit application for every report in Reports view pane. Currently, it is a part Orbit application.

  2. It supports to write a separate Orbit job for custom functionality as per customer requirements and place it in Orbit configured java class path directory for the deployment.
