Uploading Snapshot in Thread Chat

 

To upload a snapshot in Thread Chat

 

1. In the chat window of the selected thread, click  in the textbox at the bottom of the window.

2. A menu appears with the following options:

a.    Snapshot

b.    Document

c.    Voting Request

3.    To upload a snapshot, click Snapshot.

 

4.    Snapshot upload dialog box appears.

 

 

5.    In the Snapshot upload dialog box, do the following:

a.    In the Description box, type the description for the snapshot.

b.    To upload a snapshot, click Upload next to the Snapshot box.

c.     In the Open dialog box that appears, browse your computer, select the desired file, and then click Open.

d.     In the Snapshot box, the path of the snapshot appears.

 

       

 

e.    To save the changes, click Save.

Note: A Snapshot saved successfully notification appears briefly at the top of the ORBIT application window.

 

f.   To discard the changes, click Cancel.

7. To Refresh the thread chat, click  Refresh.
