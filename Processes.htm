Processes

In the section, you can see the running processes, re-start processes, or canceled processes.

Processes contain the following fields:

S.No

Fields

Description

1.

Process Id

This describes the process id;

2.

Name

This describes the Report name

3.

Phase

This describes the phase of the process.

4.

Status

This describes the status of the process.

5.

Start Time

This describes the start time of the process.

6.

End Time

This describes the end time of the process.

7.

Job Type

This describes the job type of the process.

8.

Log

This provides the log of the process
