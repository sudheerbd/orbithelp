Creating a New Report

To create a new report, perform the following:

 

 1. Under Reports section, perform the following to create various reports.

  • To create a data report, right-click on existing folder in My Folders, point to New, and then click New Data Report.

  • To create an analysis, right-click on existing folder in My Folders, point to New, and then click New Analysis.

  • To create a dashboard,right-click on existing folder in My Folders, point to New,and then click New Dashboard.

  • To create a publisher report, right-click on existing folder in My Folders, point to New, and then click New Publisher Report.

  • To create a script report, right-click on existing folder in My Folders, point to New, and then click New Script Report.

  • To create a finance report, right-click on existing folder in My Folders, point to New, and then click New Finance Report.

  • To create a custom report,right-click on existing folder in My Folders, point to New, and then click New Custom Report.

 

 

Alternate Procedure

1. On the Orbit Header, click New.

2. The following reports appear:

  • Data Report

  • Analysis

  • Dashboard

  • Publisher Report

  • Script Report

  • Finance Report

  • Custom Report

 

3. From the available reports, select the required report, and then click Ok.
