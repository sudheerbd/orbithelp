Configuring Instance

To Configuring the instance

In order to logon to the application, initially configure the instance.

  1. On Orbit XLEdge menu, in the Set-up group, click URL.

  1. Instance Configuration dialog box appears.

  1. The Instance Configuration dialog box, has the following options:

    • Add: this option is to add a server

    • Delete: this option is to delete a selected server

    • Close: this option is to close Instance Configuration dialog.

To Add Instance, perform the following:

  1. Under the URL Name, type the name of the instance.
    Name: orb

  1. Under the URL Address, type the instance URL.
    URL: http://orbdev07.orbit8.com:5050/reporting/

  1. To add the record, click Add.

  1. Once URL is configured, a message "Record Added Successfully" appears.

 

To delete the instance details, perform the following:

  1. Select the required instance details, and then click Delete.

  1. Upon clicking Delete, a success message "Record Deleted Successfully" appears.

  1. To close the Instance Configuration dialog, click Close.
