Schedule Refresh

Schedule Refresh option is to fetch the data in reports when a dataset is newly created or modified.   

 

Procedure 1:

To perform schedule refresh

 

  1. In the Datasets section, a list of available datasets appears.

  2. For the required dataset, under Actions, click  icon, and then click Schedule Refresh.

  3. Refresh Dataset dialog appears.

 

 

  1. To refresh the dataset, click Submit.

  2. Once Schedule Refresh is performed on a dataset, a process id is generated.

  3. This appears as a process in the Monitoring section.


a.  In order to display schedule refresh for a dataset, there must be joins between the data objects, when there is more than one data object. There must be no orphan data objects. 
b.  When Schedule Refresh option is performed, the data is fetched from the dataset in the Reports.
c. Schedule Refresh option is also available when creating a dataset.

 

Procedure 2:

Schedule Refresh

Once a new dataset is created or when a dataset is modified, click Schedule Refresh.

To retrieve the data in the reports section using the dataset, it is essential to perform a Schedule Refresh.

To perform the Schedule Refresh

  1. In the dataset workspace, add or delete an object.

  2. On the dataset menu, click Save to save any changes.

Once the changes are saved, Schedule Refresh button becomes available.

  1. To perform schedule refresh, click Schedule Refresh.

Once the Schedule Refresh option is selected, the status of the selected dataset changes from Not refreshed to the date when it is last refreshed. And, the options Create Report, Schedule Refresh, and Share also become available, when the user clicks  .
