Creating an Analysis Report

 

Creating an Analysis Report

To create a new analysis report

1. Navigate to the Reports page.

2. On the Reports page, click New, click Analysis, and then click Ok.

3. The Analysis report page appears in Edit Mode. The Analysis report page has a Data and Visual Settings area (highlighted as ”1”) and a Run Result area (highlighted as ”2”).

Note: See Menu Options in Analysis Reports for information about menu options in the analysis report page.

4. On the Data tab, click the expander arrow (highlighted as ”3”) next to Select Business Object.

5. Search Business Object window appears with available business objects list (highlighted as “4”).

6. Select the required business object, and then click Ok.

(OR)

Double-click the required business object.

7. Under the Data tab, the selected business object appears.

8. To create analysis report do one of the following:

  • Drag the required fields from the selected Business Object into the Run Result area. Orbit creates best visualization based on the selected fields.

  • Right click on required fields and select Create Best Visualization. Orbit creates best visualization based on selected fields.

  • Right click on required fields and select Pick Visualization. Select desired chart type from the chart options window. Orbit creates analysis report based on selected chart type.

Hold Ctrl and click on required fields to make multiple selections.

9. Analysis Report appears in the Run Result area and Report Configurator area (highlighted as “5”) appears next to the Data and Visual Settings area.

10. Report Configurator area has the following sections:

  • Visualization (Chart type) Options (highlighted as “6”)
  • Local Filters (highlighted as “7”)

  • My Calculations (highlighted as “8”)

 

Note: Sections in the Visualization Options change based on the selected visualization type.

See Visualization Options for more details.

11. Drag and drop desired fields in report configuration area to update the report. Orbit application generates the analysis report dynamically in the run results area.

Note: Data in analysis reports may not come in the same order always. If the user needs the same order every time, then enable sorting. See Applying Sort for more information.

 

Changing Visualization Type

1. To change the visualization type, click the expander arrow or the visualization name (highlighted as ”9”) in the Report Configurator area.

2. Select the required visualization type from the available options and then click Ok.

(OR)

Double click the required option.

3. Report with selected visualization appears in the run result area.

 

Saving Analysis Report

1. To save the report, click Save.

2. Save Report dialog box appears.

3. If the required report folder already exists, then perform the following:

  • In the Name box, type the report name.

  • In the Description box, type the description.

  • To save the report, click Save.

4. If the required report folder does not exist, then create a new reports folder, and then save the report in it. See Creating a Reports Folder.

5. To create a report under an existing folder see Creating a Report Under Existing Folder.
