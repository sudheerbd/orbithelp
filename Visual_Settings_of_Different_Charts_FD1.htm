Visualization Options in Analysis Reports

 

List of visualization options available in Analysis Reports

Auto Visualization

Auto visualization creates visualizations based on the fields dragged into report configuration area.

For more information about Auto Visualization, see Auto Visualization.

Bar and Column Charts

A bar chart displays data using rectangular bars. One axis of a bar chart measures a value, while the other axis shows the specific categories being compared.

Types of Bar and Column charts available in Analysis Reports:

  • Bar Basic

  • Stacked Bar

  • 100% Stacked

  • Column Basic

  • Stacked Column

  • 100% Stacked

  • Multi Axis

  • Negative Values

For more information about bar charts, see Bar Charts.

    

Combination Charts

A Combination chart combines a column chart and a line chart.

Visual Settings of Combination Charts are similar to Bar Charts. For more information about visual settings, see Bar Charts.

Line Charts

A line chart displays information as a series of data points connected by line segments.

Types of line charts available in Analysis Reports:

  • Line Basic

  • Spline

  • Step Line

Visual Settings of Line Charts are similar to Bar Charts. For more information about visual settings, see Bar Charts.

Area Charts

An Area Chart is a line chart with the areas between axis and lines filled with colors.

Types of area charts available in Analysis Reports:

  • Area

  • Area Spline

Visual Settings of Area Charts are similar to Bar Charts. For more information about visual settings, see Bar Charts.

Pie Charts

Pie charts use "pie slices" to show relative sizes of the data.

For more information about visual settings of pie charts, see Pie Charts.

Doughnut Charts

Doughnut charts are similar to Pie charts. Doughnut charts display information about the data as a parts to a whole.

Visual Settings of Doughnut Charts are similar to Pie Charts. For more information, see Pie Charts.

Spiderweb Charts

A spiderweb chart or radar chart displays different series types on a radial axis. Spiderweb charts are commonly used to compare multivariate data sets.

Types of spiderweb charts available in Analysis Reports:

  • Spiderweb Line

  • Spiderweb Area

Visual Settings of Spiderweb Charts are similar to Pie Charts. For more information, see Pie Charts.

Tree Maps

Tree Maps display hierarchical data using nested figures, usually rectangles. Tree Maps display quantities for each category via area size.

For more information about visual settings of tree maps, see Tree Maps.

Sun Burst

Sun Burst charts display hierarchical data through a series of rings, that are sliced for each category node. The inner elements are parent nodes, with child nodes distributed on the outer rings.

Visual Settings of Sun Burst Charts are similar to Tree Maps. For more information, see Tree Maps.

Scatter and Bubble charts

The Scatter and Bubble chart displays pairs of numerical data on the graph as a dot, with one variable on each axis. This type of chart can be used in to visually describe relationships between two numerical parameters or to represent distributions.

For more information about visual settings of scatter and bubble charts, see Scatter and Bubble Charts.

  

Packed Bubble Charts

Packed bubble charts display data in a cluster of circles. Dimensions define the individual bubbles, measures define the size, Color defines color of the individual circles.

For more information about visual settings of packed bubble charts, see Packed Bubble Charts.

Tabular Charts

In Tabular Charts, data is presented in rows and columns.

For more information about visual settings of tabular charts, see Tabular Charts.

Pivot/Matrix

Pivot tables are interactive tables that allow the user to group and summarize large amounts of data in a concise, tabular format for easier reporting and analysis.

For more information about visual settings of pivot/matrix charts, see Pivot/Matrix Charts.

Heat Map

Heat Maps are commonly used to visualize hot spots within data sets, and to show patterns or correlations.

For more information about visual settings of Heat Maps, see Heat Maps.

Maps

Maps associate both categorical and quantitative information with locations.

Types of maps available in Analysis Reports:

  • Map

  • Google Map

For more information about visual settings of maps, see Maps.

Word Cloud

Word cloud displays text data, typically used to depict keywords, or to visualize free form text. Words are usually single words, and the importance of each word is shown with font size or color.

For more information about visual settings of word cloud, see Word Cloud.

Sankey Chart

Sankey charts are used to visualize data flow and volume between nodes. The wider lines indicate larger volumes.

For more information about visual settings of sankey chart, see Sankey Chart.

Value Pair

Value Pair visualization displays numbers (value) such as total sales, or total profits compared against a series category.

For more information about visual settings of value pair, see Value Pair.

KPI Card

A Key Performance Indicator (KPI) Card displays the amount of progress made toward a measurable goal.

For more information about visual settings of KPI Card, see KPI Card.

Solid Gauge

A gauge chart displays a single value that measures progress toward a goal or a Key Performance Indicator (KPI).

For more information about visual settings of Solid Gauge charts, see Solid Gauge.
