Forgot Password

To recover the Password

  1. Navigate to the ORBIT URL.

  2. ORBIT Analytics Login Page appears.

  3. To recover the password, click Forgot Password.

  4. Username and Email fields appear, with warning message "Please enter Username and registered email".

  5. In the Username box, type the required user name.

  6. In the Email box, type the registered email, and then click Submit.

  7. "Reset password link sent to register email." message appears.

  8. To reset the password, email will be received to the registered email.

  9. To Reset the password, click Reset Password.

  10. Below page appears.

  11. Email will be received from Orbit with reset password.

  12. Login to the Orbit application using the new password.

  13. Change system generated password with a password of your choice. See Changing Password for more information.
