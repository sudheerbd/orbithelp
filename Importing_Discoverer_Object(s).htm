Importing Discoverer Object(s)

Exercise: Importing Discoverer Object(s)

Using Import Discoverer Object(s) option, import schedule xml which loads the business areas into ORBIT Analytics application. Also, importing the eex files uploads the worksheet for a specific business area.

 

Importing Schedule XML

Schedule XML is an XML file generated using ORBIT Discoverer Migration. The contents of the file include the selected business areas in the discoverer migration. Once the schedule.xml file is generated using ORBIT discoverer migration, this file can be imported into the ORBIT Analytics.

 

To Import Schedule XML

 

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded and Custom jobs.

  3. Under Jobs sections, on Import Discoverer Object(s), click , and then click Schedule.

  4. Import Discoverer Object(s) dialog appears.

 

 

  1. Under Parameters, click Upload (highlighted as “1”) next to the Upload File box.

  2. Open dialog box appears with required files.

  3. Select the required xml file, and then click Open.

Required XML File: disco_job_ba (2).xml

  1. In the Upload File box, the selected file path appears.

 

  1. To submit the file, click Submit.

  2. Under Scheduled Processes, the newly submitted file appears.

  3. On the left-hand side menu, point to  Business Models, and then click Domains.

  4. Domains page appears with a list of available domains.

  5. Under the Domains & Data Models, the newly created domain appears.

    • Example: Newly Created Domain: Cash Forecasting

 

 

Importing EEX Files

 

Importing EEX Files migrates the worksheets of selected business area into ORBIT Analytics.

 

To import EEX Files

 

Generate EEX file from discoverer desktop administrator. Once the EEX file is generated, you can import the file into the monitoring section of ORBIT Analytics application.

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded and Custom jobs.

  3. Under Jobs sections, on Import Discoverer Object(s), click , and then click Schedule.

  4. Import Discoverer Object(s) dialog appears.

 

 

  1. Under Parameters, click Upload (highlighted as “1”) next to the Upload File box.

  2. Open dialog box appears with required files.

  3. Select the required eex file, and then click Open.

Required EEX file: CE_Cash_Forecasting.eex

  1. In the Upload File box, the selected file path appears.

 

  1. To submit the file, click Submit.

  2. Under Scheduled Processes, the newly submitted file appears.
