Scheduling a Finance Report

To schedule a finance report

  1. On the selected report, click Options, and then click Schedule.

  2. Schedule - Report dialog appears with the parameters.

  3. In the Schedule-Report dialog, perform the following:

    1. On the Parameters tab, in the Finance Cubes box, type/select the required finance cube.

    2. In the Ledgers box, type/select the required ledger associated with selected finance cubes.

    3. In Periods box, select the required period.

    4. In the Currencies box, select the required currency.

    5. To enable offline drilldowns, select Generate Drilldowns check box in the Drilldowns section.

    6. When the Generate Drilldowns check box is selected following options appear.

      1. Drilldowns with authentication: When Drilldowns with authentication is selected, the user needs to provide valid credentials to drilldown on the data.

      2. Drilldowns without authentication: When Drilldowns without authentication is selected, the user can drilldown on data without authentication.

    7. On the Output & Delivery tab, in the Outputs and Deliveries section, under Output Type, select the required output type.

      Example: Output Type: Excel(xlsx)

  4. To submit the process, click Submit.

  5. The process gets submitted successfully.

  6. The submitted report output available in Processes section.
