Working with Roles

ORBIT Analytics Application provides the following seeded roles:

 

  • ORBIT ADMIN ROLE

  • ORBIT SUPER USER ROLE

  • ORBIT USER ROLE

 

      Roles section consists of the following:

 

    • Creating a New Role

    • Assigning Users to the Role

    • Assigning Groups to the Role

    • Filtering Roles

    • Configuring Permissions for the Roles

    • Editing Roles

 

Delete role option will be enable only for imported users, from different Authority.
