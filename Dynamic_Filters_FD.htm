Dynamic Filters

Dynamic filters are user interface widgets (such as check boxes, category pickers, and range sliders) that allow you to quickly narrow down and filter the data displayed on the dashboard.

Dynamic filters create a dynamic report depending on the user's selection and enhance the report presentation and usability.

Types of dynamic filters available in dashboard reports:

  • Dynamic Dimension

  • Dynamic Measure

  • List Filter

  • Range Filter

 

The dynamic filter allows you to apply one of the predefined list filters to display date values that fall within a specified time period (next, this or last week, month, year, etc.), or to display numbers that are above or below the average.
