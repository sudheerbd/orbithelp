Creating a New Domain

To create a new Domain

 

  1. On the left-hand side menu, point to Business Models, and then click Domains .

  2. Domains & Data Models page appears with a list of available domains. 

      

                                                         

  1. To create a new domain, click New.

  2. New Domain dialog box appears.

 

                                            

  1. In the New Domain dialog box,  perform the following:

    • In the Name box, type the name of the domain.

    • In the Description box, type the description.

  1. To save the domain, click Save.

  2. To close the dialog box, click Cancel.

  3. The newly created domain opens in a new tab.

    • The domain page has Domain Hierarchy on the left side of the ORBIT window.

    • Domain Hierarchy consists of two folders:

      1. Physical Models

      2. Logical Models

 

  1. The newly created domain appears under Domains & Data Models area.

           

                                                   

  1. By default, the following columns appear in Domains & Data Models page.

    • Name

    • Description

    • Created By

    • Modified Date

    • Last Refresh Date

    • Actions

  1. You can show/hide following columns.

    • Created By

    • Modified Date

    • Last Refresh Date

    • Last Modified By

    • Actions

 

  1. To show or hide columns, click the  column header arrow next to any column name, point to Columns and then select or clear check boxes against column names.

 

  

  1. By selecting the check box next to the columns, the columns appear under Domains & Data Models section.
