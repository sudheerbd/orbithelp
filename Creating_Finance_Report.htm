Creating Finance Report

Prerequisites: You must have a report created in Excel using Orbit GL Sense add-in

To create a finance report

  1. In the Orbit application, on the left-hand side, click Reports.

  2. Reports page appears with a list of reports available to the logged-in user.

  3. In the Reports page, on the Orbit Main Menu, click New, click Finance, and then click Ok.

  4. Finance Reports page appears.

  5. In the Finance Reports page, perform the following:

    1. In the Name box, enter the required finance report name.

    2. In the Description box, enter the required description.

    3. In the Cube Name box, select/enter the required cube name.

      Note: Associated cubes list appears in the dropdown.

    4. In the Password box, enter the required password.

      Note: Password protection is applicable on uploaded template file, if user enters any password in the Password box.

    5. In the Upload box, upload the template file that has been created using Orbit GLSense add-in, by clicking Upload file….

    6. Select the file from your local machine and then click Open.

    7. File appears in the Upload box.

    8. Under Parameters section, perform the following:

      1. In Finance Cubes box, select the required finance cubes. 

      2. In the Ledgers box, select the required ledgers associated with the finance cubes.

      3. In the Periods box, select the required period.

      4. In the Currencies box, select the required currency.

      5. To override existing segments values in the template file, click in the Segment Override box.

      6. Select Segment values dialog box appears.

      7. select the required values in the Select Segment values dialog box and click Ok.

        Note: The parameters values are associated with the selected cube.

    9. In the Drilldowns section perform the following:

      1. To enable offline drilldowns, select Generate Drilldowns check box in the Drilldowns section.

      2. When the Generate Drilldowns check box is selected following options appear.

      3. Drilldowns with authentication: When Drilldowns with authentication is selected, the user needs to provide valid credentials to drilldown on the data.

      4. Drilldowns without authentication: When Drilldowns without authentication is selected, the user can drilldown on data without authentication.

  6. To save the changes, click Save.

  7. Save Report As dialog appears.

  8. In the Name box, report name entered in the Name box appears.

  9. Select the required folder from the navigation pane.

  10. Click Save to save the report.

  11. Once the report is saved, a success message “Report saved successfully” appears.

  12. Navigate to the Reports page and select the folder where the newly created report is saved.

    Note: Here, the newly created report “DemoQA1” is saved under the folder “Demo Finance Reports”.
