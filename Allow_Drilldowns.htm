Allow Drilldowns

This option allow user to configure drilldowns to the required report.

 

  1. To work with allow drilldowns, refer Configure Drill-through.
  2. To disable this option, uncheck Allow Drilldowns(Drill-through, Drillby) under Data Interaction Options.
  3. When user right-click on the required visualization widget, Configure Drill-through option will be disable.
