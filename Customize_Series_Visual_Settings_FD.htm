Customize Series Visual Settings

This topic explains different ways to customize the Series visual settings of your charts.

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Series to open Series visual settings.

 

Series

Series tab has the following visual settings:

  1. Series Label: Change Series Label name.
  2. Value Prefix: Adds text before data labels.
  3. Value Suffix: Adds text after data labels.
  4. Chart Type: Change chart type of selected series.

    1. Line Type: Change line type of the line chart.

    2. Line Weight: Change line weight of the line chart.

    3. Line Dashes: Change line dashes of the line chart.

    4. Show Point Markers: Show/Hide point markers on the line chart type.

  5. Y2 Axis: Show/hide Y2 Axis.
  6. Data Labels: Show/Hide data labels on the bars of the chart.
    1. Show Percentage: Show/Hide percentage value of the slice in the data label.
    2. Show Value: Show/Hide value of the slice in the data label.
    3. Show Label: Show/Hide name of the slice.

    4. Label Position: Change data label position.

    5. Label Angle: Changes angle of the data labels.

    6. Label Align: Change label alignment.

    7. Label Connector Shape: Change label connector shape

  7. Format: Change data label format.

Note: The visual settings available depend on the visualization type selected.

For Bar and Column charts, Series tab has child tabs based on number of fields in Value Axis section. If Value Axis section has two fields, Series tab is further divided into two tabs: each having field name as title.

Series Label

In the Series Label box enter your desired name to change the default series name.

Value Prefix

Note: Text entered in the Prefix is visible only when Data Labels toggle is ON.

Enter text in the Prefix box to append before data labels.

Value Suffix

Note: Text entered in the Suffix is visible only when Data Labels toggle is ON.

Enter text in the Suffix box to append after data labels.

Data Labels (Bar, Column, Line and Area Charts)

Default Value: OFF

  1. To show data labels, turn on Data Labels toggle.

  2. To hide data labels, turn off Data Labels toggle.

When you turn on Data Labels toggle, following options appear

  • Label Position

  • Label Angle

Label Position

Default Value: Outside

  • To show data labels on inside of the slice, select Inside.

 

Label Angle

Default Value: ---

Select desired label angle from the below options:

  1. - - - (Horizontal)

  2. | | | (Vertical)

Data Labels (Pie, Doughnut, and Spiderweb charts)

Default Value: ON

  1. To show data labels, turn on Data Labels toggle.

  2. To hide data labels, turn off Data Labels toggle.

 

When you turn on Data Labels toggle, following options appear

  • Show Percentage

  • Show Value
  • Show Label

  • Label Position

  • Label Align

  • Label Connector Shape

Show Percentage

  1. Select Show Percentage check box to show percentage value of the slice in the data label.

  2. Clear Show Percentage check box to hide percentage value of the slice in the data label.

Show Value

  1. Select Show Value check box to show value of the slice in the data label.

  2. Clear Show Value check box to hide value of the slice in the data label.

Show Label

  1. Select Show Label check box to show name of the slice.

  2. Clear Show Label check box to hide name of the slice.

Label Position

Default Value: Outside

  • To show data labels on inside of the slice, select Inside.

Label Align

Default Value: Default

Label Align has following options

  1. Default: Select Default to keep labels in default alignment.

  2. Plot Edges: Select Plot Edges to align labels to plot edges.

  3. Connectors: Select Connectors to align label connectors.

Label Connector Shape

Default Value: Fixed Offset

Select any one of the following options:

  1. Fixed Offset

  2. Straight

  3. Crooked Line

Chart Type

Change chart type of selected series. When there is more than one field in Value Axis (Y Axis), you can change chart style of each series individually.

Default Value: Bar

 

To change chart type to line, select Line, in the chart type.

 

To change chart type to area, select Area, in the chart type.

 

When chart type is set as Line or Area, following options appear:

  1. Line Type

  2. Line Weight

  3. Line Dashes

  4. Show Point Markers

Line Type

Change line type of the line chart

Default Value: Line

To change line type, select Spline or Step from Line Type options.

 

 

Line Weight

To change line weight, select any one of line weights from Line Weight options.

Line Dashes

To change line dashes, select any one of line dashes from Line Dashes options.

 

Show Point Markers

  1. Select Show point markers check box to show point markers on the line chart.

  2. Clear Show point markers check box to hide point markers on the line chart.

Y2 Axis

  1. Select Y2 Axis check box, to show the Y2 Axis.

  2. Clear Y2 Axis check box, to hide the Y2 Axis.

Format

Changes data format of the data labels.

Note: Formatted data is visible only when Data Labels toggle is ON.

Refer to Format Options from Value Axis for detailed information.
