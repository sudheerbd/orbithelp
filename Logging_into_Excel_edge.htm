Logging into XLEdge

To Log onto Orbit XLEdge

  1. On the Orbit XLEdge tab, various groups with related controls and commands appear.

    Before you log on to the application, only two options-Login and URL option are available on the Excel ribbon.
    The remaining features are unavailable and appear grey. Once you log on to the application, all the remaining features become available.

  2. To log onto the Orbit XLEdge application, click Login (highlighted as "1").

  3. Orbit XLEdge - Instance dialog box appears.

  4. In Instance box, list of available instances appears.

  5. From the instances list, select the required instance.

  6. The associated URL for the selected instance appears.

    Associated URL: http://orbdev7.orbit8.com:5050/reporting

  7. Orbit Login page appears with the selected instance.

  8. In the UserName box, type the required username.

  9. In the Password box, type the password.

  10. To log on to the application, click Login.
