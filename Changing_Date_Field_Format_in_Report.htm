Changing Date Field Format in Report

To change Date field format in report

  1. On the left-hand side menu, point to Business Models, and then click Meta Objects.

  2. Meta Objects page appears with a list of Meta Objects (highlighted as “1”) on the left-hand side.

  3. To change the formatting of Date Field in the reports, perform the following:

    1. Under the Base Objects (highlighted as “2”), select the Date Field (highlighted as “3”).

    2. Name and Value columns (highlighted as “4”) for the Date Field appear.

    3. Under the Format tab, in the Display Text box, enter the required text to display.

    4. Under the Format tab, in the Format box, type/select the required format.

      Required Date Format: dd/MM/yyyy

    5. Under the Style tab, perform the following:

      1. In the Alignment box, type/select the required option

        Required option: left

      2. In the Column Width box, type/select the required value.

        Required Value: 130

      3. In the Font Family box, type/select the required values by clicking

      4. A dialog window appears with multiple fields.

      5. Edit the required fields, and then click Ok.


        Font Family: Arial
        Font Size(px: 14)
        Effects: Underline
        Color: 638888

      6. In the Hidden box, type/select the required value.

        Hide: false

    6. Under Miscellaneous tab, in the format box, type the required format.

      1. In the nnbr box, type/select the required option.

  4. To save the changes, click Save.
