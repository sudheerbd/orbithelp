Dashboard Options

To view the dashboard report option

 

  1. Open the required dashboard report in run results area.

Dashboard report Options available in both View and Edit Mode

  1. On the top right-hand corner of the run results area, click Options, and then click Options.

  2. Dashboard Options dialog appears.

 

 

  1. Under Dashboard Options dialog, the following options appears:

    • Themes

    • Auto Refresh

    • Data Interaction Options
