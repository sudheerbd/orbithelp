Multiple Drill-through in a Report

To perform multiple drill-through

 

  1. In the Drill-through dialog box, under the first drill-through report, drag and drop the second report (highlighted as “1”).

 

 

Drill profit is already existing drill-through report.

  1. Drill-through Column and Navigate To appear for the second report, along with the available parameters of the report.  

  2. From the report Configurator area, drag and drop the required column into Drill-through Column box.

  3. In the Navigate To box, type/select the required option.

Navigate To: Window

  1. In the box under the Parameters (highlighted as “2”), type/select the required parameter value.

 

                                               

  1. To save the changes, click Ok.

  2. The drill-through report appears as shown below:

 

                  

Example:

    • To view the drill-through report State-wise, click NewYork, under State Column.

    • The drill-through column name appears beside the report name (highlighted as “3”).

 

                                                  

    • To view the drill-through report Region-wise, click East, under Region Column.
