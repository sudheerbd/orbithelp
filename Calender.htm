Scheduler Calendar

Working with scheduler calendars

 

  1. On the left-hand side menu, point to   System, and then click Maintenance.

  2. Scheduler page appears with Operations tab.

  3. To view calendar list, click Calendars.

  4. Under calendar list below options appears:

    • Name

    • Description

  5. To create a new calendar, click Add New Scheduler Calendar.

  6. New Scheduler Calendar dialog appears.

 

  1. Enter the required Name and Description fields.

  2. To add the calendar dates, click Calendar Date (highlighted as "1").

  3. Under calendar date below options appears:

    • Simple Date

    • Advanced Date

  4. To add simple date, click Simple Date.

    • Simple Calendar Date dialog appears as shown below:

    • Select the required Day, Month and Year from the dropdown list.

    • To select the entered date as holiday, select Is Holiday? checkbox.

From the selected year, the date display as holiday.

    • To add the entered date, click Add.

    • To cancel/close the dialog, click Cancel/.

    • To save the selected dates, click Save
