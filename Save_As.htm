Save As

To copy a report using save as option

 

  1. Open the required report in run results area.

  1. On the top right-hand corner of the run results area, click Options, and then click Save as (highlighted as “1”).

 

 

  1. Save Report As dialog appears.

By default Copy of <Report Name> appears in Name field.

  1. To save the copy report, select the required folder under Folders section as shown below:

 

 

  1. To save the copy report, click Save.

  2. To cancel the changes,click Cancel.
