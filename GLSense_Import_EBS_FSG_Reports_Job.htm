GLSense Import EBS FSG Reports Job

To import FSG reports from EBS

Prerequisite: You must have the EBS Finance Cube.

  • By using this job, the imported FSG reports are created as financial reports in Orbit application. To work with the imported reports, refer Finance Reports imported from EBS FSG.

  1. On the left-hand side menu, point to , and then click Jobs.

  2. The Jobs page appears with a list of Seeded, Custom and Plugin jobs.

  3. Under the Jobs sections, on GLSense Import EBS FSG Reports Job, click , and then click Schedule.

  4. Schedule Job dialog box appears.

  5. In the Schedule Job dialog box, on the Parameters tab, perform the following:

a.     In the Finance Cubes box, select the required finance cube from the list.

Only EBS Finance Cubes are shown in the list.

b.     In the Ledgers box, select the required ledgers.

You can select multiple ledgers.

c.      In the FSG Report box, select the required report to import from EBS FSG.

Actual/Budget Income Statement

d.     In the Periods box, select the required period.

e.    In the Currencies box, select the required currency.

 

  1. To schedule the job, click Submit.

  2. To cancel the job, click Cancel.
