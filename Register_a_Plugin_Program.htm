Register a Plugin Program

To run a plugin job, user need to register the plugin with Plugin Programs. Before registering a plugin program, it is mandatory to save the uploaded zip file.

To register a new plugin

  1. To register a plugin programs, click  New Program.

  2. A tab appears under Plugin Programs.

  3. Select the tab to enter the program name as shown below:

 

 

  1. To save the changes, click Update.

  2. Once the plugin program is register, it will appear under Jobs section.

For a single plugin, user can register multiple programs as shown below:

  1. For Fusion plugin, orbit supporting the below three jobs:

    • GLSense Fusion Balance Job

    • GLSense Fusion Setup Job

    • GLSense Fusion Import User Job

  1. For Netsuite plugin, orbit supporting the below three jobs:

    • GLSense Netsuite Balance Job

    • GLSense Netsuite Setup Job

    • GLSense Netsuite Import User Job

  1. For EBS Plugin Orbit supporting the below job:

    • GLSense EBS Generate View Script Job

    • GLSense Import EBS FSG Report Job
