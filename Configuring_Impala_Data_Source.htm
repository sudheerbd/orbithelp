Configuring Impala Data Source

To Configure Impala Data Source

 

  1. In the New Data Source dialog, perform the following:

 

 

  1. In the Name box, type the data source name.

Name: Test Data Source  

  1. In the Description box, type the description for the data source.

  2. In the Data Source Type, select the required data source type radio button from the following:

    • JDBC

    • JNDI

Data Source Type: JDBC

 

  1. In the Select Database box, select or type the required option.

Required Database Option: Impala (Impala)

  1. Under the Impala Settings section, perform the following:

 

 

  1. In the JDBC URL box, type the URL.

URL: jdbc:impala://192.168.150.129:21050;AuthMech=0;


a. To configure a connection without authentication, set the AuthMech property to 0.
b. To configure Kerberos authentication, set the AuthMech property to 1.
c. To configure User Name authentication, set the AuthMech property to 2.
d. To configure User Name and Password authentication, set the AuthMech property to 3.

When the user selects the Impala data source Driver Class and Data Source Class appears, by default.    

 

  1.  To test the data source connection, click Test Connection 
  2.  To save the data source, click Save.
  3. To close the dialog box, click Cancel.
  4. The newly created data source appears under data source list.
