Finance cube

To Create a Finance Cube

  1. On the left-hand side menu, click Business Models and then click Finance Cube.

  2. Finance Cubes page appears with two sections â€” Finance Cubes and Data Loads.

  3. In the Finance Cubes section, click Add new cube to create a new finance cube.

  4. Finance Cube dialog box appears.

  5. In the Finance Cube dialog box, perform the following:

    1. In the Cube Name box, enter the required cube name.

    2. In the Description box, enter the description.

      Description field is optional.

    3. In the Source Connection box, select the required source connection.

      1. For EBS ERP, select an EBS data source connection.

      2. For Fusion, select a fusion data source connection.

      3. For NetSuite, select the staging schema data source created for the NetSuite ERP.

    4. In the Target Connection box, select the required target connection.

    5. Select the data source, where to create the finance cube.

      For EBS ERP, you can use the Source Connection EBS datasource as Target Connection datasource. This will provide real time materialized views in the datasource itself and eliminates export and import of data from source and target datasources.

    6. To save the finance cube, click Save.

    7. To cancel, click Cancel.
