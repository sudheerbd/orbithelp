Displaying Join Order View

To display Join Order View

 

1. Create a Simple/Comple Join in Logical Model. Refer Simple Join /Complex Join

2. Navigate to Relationship (Source table =>Destination table) dialog box.

3. In the Join Order Key box, type the order key for the join.

 

4. To Display the entered Join Order key, click   Show Join Order View.
