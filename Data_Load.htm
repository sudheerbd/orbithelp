Data Loads

Two ways to extract data

  1. Creating Data Load manually

  2. Automating Job Creation

    • Extract GL Setup Data (Fusion),

    • Extract GL Balance Data (Fusion)

    • Process Finance Cube Batch

    • Import User Roles (Fusion).

The procedure below describes the manual way of extracting data load.

Prerequisite:

  • You must have a finance cube to which you need to associate the data load.

  • You must have files on Ledgers, Periods, Segments, Segment Values, Budgets, Budget Versions, Currencies, Journal Sources, journal categories, encumbrances, segment rules, segment hierarchy, role ledger association, code combination, and balances.

  • Supported file formats are .CSV, and .ZIP. For large size file, we recommend zipping it and upload. Zipped file should contain only one file in it.

 

To Create a Data Load

  1. On the left-hand side menu, point to Business Models, and then click Finance Cube.

  2. Finance Cubes page appears with two sectionsâ€” Finance Cubes and Data Loads.

  3. Under Finance Cubes, select the required cube for which the data load must be associated, and then click Add new data load in the Data Loads section to create a new data load.

  4. Cube Batch dialog box appears.

  5. In the Cube Batch dialog box, perform the following:

    1. In the Cube Name box, enter the required cube name.

    2. In the Description box, enter the description.

      Description field is optional.

    3. To upload ledgers in the Ledgers box, click Upload.

    4. To upload periods in the Periods box, click Upload.

    5. To upload segments in the Segments box, click Upload.

    6. To upload segment values in the Segment Values box, click Upload.

    7. To upload budget file in the Budgets box, click Upload.

    8. To upload budget versions file in the Budget Versions box, click Upload.

    9. To upload currencies file in the Currencies box, click Upload.

    10. To upload journal sources file in the Journal Sources box, click Upload.

    11. To upload journal categories file in the Journal Categories box, click Upload.

    12. To upload the encumbrances file in the Encumbrances box, click Upload.

    13. To upload the segment rules file in the Segment Rules box, click Upload.

    14. To upload the segment hierarchy file in the Segment Hierarchy box, click Upload.

    15. To upload role ledger association file in the Role Ledger Association box, click Upload.

    16. To upload the code combinations in the Code Combinations box, click Upload.

    17. To upload the balances in the Balances box, click Upload.

    18. To save the finance cube, click Save.

    19. To cancel, click Cancel.
