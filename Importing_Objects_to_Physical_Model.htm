Importing Objects to Physical Model

Importing objects in physical model

 

  1. To import the tables into the physical model, do one of the following:

    • Right-click the <Physical Model Name>, and then click Import Objects.

Or

    • Click arrow next to the <Physical Model Name>, and then click Import Objects.

         

           

  1. Import Object(s) - <Physical Model Name> dialog box appears.

 

          

   

Search objects with schema name:                        

  1. To search objects with schema name, enter the schema name or click arrow beside Select Schema box.

  2. Select the required schema and then click Search.

  3. All the available objects in the selected schema appears under Available Objects.

Search objects with specific condition:

  1. To search objects with specific value do the following:

    • Select the required schema from Select Schema List.

    • Select the required condition from condition list

 

List of conditions

Description

Contains

Filters the data for the objects which contains the search string.

Not Contains

Filters the data for the objects which does not contains the search string.

Equals

Filters the data for the objects which are exact match of the search string from beginning to end.

Not Equals

Filters the data for the objects which does not match the search string.

Begins With

Filters the data for the objects which starts with the search string.

Not Begins With

Filters the data for the objects which does not starts with the search string.

Ends With

Filters the data for the objects which ends with the search string.

Not Ends With

Filters the data for the objects which does not end with the search string.
