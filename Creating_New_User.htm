Creating New User

To create a new user

 

  1. On the left-hand side menu, point to  User Management, and then click  Users.

  2. User List appears with a list of available users.

      

         

                                                                  

  1. To create a new user, click New (highlighted as "1").

  2. New User dialog box appears.

           

                                        

  1. To create a new user, perform the following :

    • In the User Name box, type the user name.

    • In the First Name box, type the first name.

    • In the Last Name box, type the last name.

    • In the Email box, type the email.

    • To save the changes, click Save.

    • To discard the changes, click Cancel.

 

To enable Last Modified By, in the header, point the cursor over a column, and then click  beside the column name. A menu appears column Last Modified Date, Modified By and Actions.
