Extract Orbit MetaData Job

This job is used to synchronize Orbit meta statistics. This job is essential for viewing the newly created reports or objects under Advanced search option. Once the reports or objects are created, it essential to run this job to view those objects under Advanced Search.

 

To Extract Orbit MetaData Job

 

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded and Custom jobs.

  3. Under Jobs sections, on Extract Orbit MetaData Job, click , and then click Schedule.

  4. Extract Orbit MetaData Job dialog appears.

  5. To extract whole metadata, check Reload checkbox.

  6. To extract modified metadata, uncheck Reload checkbox.

 

 



If the process must be run immediately, select the check box before Run Immediately.
If the process must be run a later time, then remove the checkmark from the Run Immediately checkbox. 
When the Run Immediately checkbox is deselected, the Schedule tab becomes available.
Using the Schedule tab, schedule the process at a required time.

 

  1. To set any notifications, click Notifications tab. View Notifications for more information.

  2. To submit the process, click Submit.
