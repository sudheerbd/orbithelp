Editing a Data Source

To Edit a Data Source

 

  1. In the Data Sources page, a list of available data sources appears.

  2. In the Data Sources page, under Actions, click  against the thread you want to edit, and then click Edit.

  3. Edit <Data Source Name> dialog box appears.

 

  1. Edit the required fields, and then click Save.

A Datasource saved successfully notification appears briefly at the top of the ORBIT application window.

  1. To discard the changes, click Cancel.
