Editing Roles

To edit Roles

 

  1. On the left-hand side menu, point to  User Management, and then click  Roles.

  2. Roles appears with a list of available roles.

  3. To edit a role, click icon (highlighted as “1”) under Actions, and then click Edit (highlighted as “2”).

 

 

For the seeded roles –ORBIT ADMIN ROLE, ORBIT SUPER USER ROLE, ORBIT USER ROLE, ORBIT REPORT SECURITY TOKEN and Imported Roles -- and the imported roles, the option to edit is unavailable.

  1. Edit Role- < Role Name> dialog box appears.

 

  1. In the Edit Role dialog box, perform the following: .

    • In the Name box, type/edit the role name.

    • In the Description box, type/edit the role description.

  1. To save the changes, click Save.

  1. To close the dialog box, click Cancel.
