Search Base User

  1. Under User Search Information section, perform the following:

  • Here, the values in the Search Base box yields all the users under People in LDAP.

  1. In the Search Base box, type the required search attribute and value.

     Example: ou=People

  1. The above condition retrieves all the users. Here, “ou” is the attribute Organizational Unit and “People” is its value.

  • Search Base defines the starting point of the search. When “ou=People” is provided in the Search Base, then the starting point of the search is People (not the entire domain as with Base Domain).
