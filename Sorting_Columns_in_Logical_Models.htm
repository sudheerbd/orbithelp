Sorting Columns in Logical Models

To Sort the Columns in Logical Models

 

1. In the logical model area, right-click the required logical model, and then click Open.

2. The selected logical model appears in the logical model workspace (highlighted as “1”).

 

3. Above the logical model workspace, on the extreme right-hand side, click Details (highlighted as “2”).

4. Two sections Columns - <Logical Model Name > and Filters appear.

5. To view the columns under Columns- <Logical Model Name>, click the logical model in the workspace.

6. The columns appear under Columns- <Logical Model Name> (highlighted as “5”).

7. To sort the columns under Columns- < Logical Model Name>, click  Sorting Columns (highlighted as “6”).
