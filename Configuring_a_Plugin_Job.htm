Configuring a Plugin Job

To configure a plugin job

 

  1. On the left-hand side menu, point to , and then click Jobs.

  2. A list of jobs appears.

  3. To configure a plugin job, click the  icon (highlighted as “1”) under Actions, and then click Edit (highlighted as “2”).

 

 

  1. Plugin job window appears with Name, Description and Base Plugin as shown below:

 

  1. User need to enter the job configuration details manually under Job Configuration section.

 


Example:

  • For a GLSense Fusion Setup Job user need to enter the below fields.

module_type=DATA
loadType=SETUP

  • For a GLSense Fusion Balance Job user need to enter the below fields.

module_type=DATA
loadType= BALANCE

  • For a GLSense Fusion Import User Job user need to enter the below fields.

module_type=USER
loadType=USER


i.   The above configuration values are same for Netsuite epr.
ii.  The configuration values are provided by orbit in Plugin.txt.
iii. This plugin text file resides in plugin zip folder.

 

Job configuration details for EBS erp plugin

 

  • For a GLSense EBS Generate View Script Job user need to enter the below fields.

 

module_type=DATA
load_type=DATA

  • For a GLSense Import EBS FSG Report Job user need to enter the below fields.

 

module_type=FSG_REPORT
load_type=FSG_REPORT

The configuration values are provided by orbit in Plugin.txt.
This plugin text file resides in plugin zip folder.

  1. All the scheduled screens pulled dynamically based on job configuration.
