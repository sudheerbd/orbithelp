Create a Publisher Report

 

To create a Publisher Report

 

  1. In the Bands section, right-click MainBand, and then click New Band.

New Band: ReportData.

  1. Rick-click “ReportData” band, and then click New Query.

  2. A new query “Query 2” appears.

  3. Select “Query 2”.

  4. Query Details section appears.

  5. Under Query Details section, on the Configurator tab, drag and drop the columns from the business object area.

  6. Query section appears when the columns are dropped onto the Configurator tab.

  7. In the Query section, the query is built. 
     
       
      

  8. On the right-hand side, click Options, and then click Templates.

  1. Templates box appears.

  2. Upload the required template, and then click Close.

  3. To save the changes, click Save.
