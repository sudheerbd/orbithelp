Setting Advance Option

To set the Advanced Option

 

  1. On the Advanced tab (highlighted as “1”), under Months section (highlighted as “2”), select the month in which the request has to be run.

  1. Under Dates area (highlighted as “3”), select the required dates of every month.

 

 

  1. Under Days area (highlighted as “4”), select the required days of every month.

  2. Under Hour area (highlighted as "5"), select the required hour of the day.

  3. To select all the hours, select All Hours check box.  

 

 

  1. Under Minute area (highlighted as "6"), select the required minutes of the hour.

  2. To select all the minutes, select All Minutes checkbox.

 

 

  1. To submit the request, click Submit.
