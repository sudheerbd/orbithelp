Editing the thread

To Edit a Thread 

   1. In the Threads section, under Actions, click  against the thread you want to edit, and then click Edit.

       

 

   2. The Edit Thread- <Thread Name> dialog box appears.

       

       

 

   3. In the Edit Thread -<thread name> dialog box, edit the required fields and then click Save.

Note: A Thread update successfully notification appears briefly at the top of the ORBIT application window.
