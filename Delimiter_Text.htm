Delimiter Text

Delimiter text is used to delimit the fields .It Supports any single character like comma (,), dollar($), pipe (|) Alphabet (A) or anything.

Note: It has a special case only for /t for the tab (i.e. it allows 2 characters)

To work with Delimiter Text

1. Under the Reports area, select the required folder to view the reports under it.

2. Right-click the required report, and then click Schedule.

      (or)                    

   For the required report, click icon, and then click Schedule.

3. Schedule-Report dialog box appears.

4. In the Schedule-Report dialog box, click the Output & Delivery tab, in it click Add Output.

5. Under Output and Deliveries section, in the Output Type box, select the required output.

6.  In the textbox beside the Output Type box, type the required delimiter as shown below:

7. In the Delimiter Text drop down click below figure appears.

 

 

a. Delimiter text configuration dialogue box appears with Delimiter Text box,Quote Character and Escape Character.

Note: Default Characters in the Delimiter, Quote Character and Escape Character are mentioned below.

Delimiter:

b. In the Delimiter textbox user can type any character based on the requirement.

E.g: Alphabet (A), " or (@) etc.

Quote Character:

In the Quote Character user can type any value as it separates each column as individual/independent once. It provides a quote at the start and end of the value.  

c. In the Quote Character textbox user can type any required character

E.g:  ^, &, + etc.

Escape Character:

In the Escape Character if the same alphabet or characters are existing in the report it appears before it.(i.e. it repeats again)

d. In the Escape Character textbox user can type any character which is existing in the report.

E.g ( /) or (&)

 

8. Click OK

9. Click Submit and message briefly appears on the top of the process page.

 

Verifying Delimiter Text Option

  1. On the left-hand side, point to Monitoring, and then click Processes.  

  2. Scheduled Processes section appears with a list of available processes.

  3. Select the required process.

  4. To download the output file click Output file, under Output(s) & Delivery section, .

  5. The text file appears with the required delimiter between the values as shown below:

 

E.g: ^East^@^10-Jun-2012 00:00:00^@^DS//HD IBM Formatted Diskettes

@ is the Delimiter

^ is the Quote Character

/ is the Escape Character

Note: If the user is not providing any value in the Quote and Escape character the above Suppress Quote Character and Suppress Escape character check box should be checked or else it considers " as default and provides the output.
