Configuring Widget Settings

To Configure Visual Settings

 

1.      To configure report widget settings, click  Widget Settings (highlighted as "1") .

2.      The following tabs appear:

    • General (highlighted as “2”)

    • Tabular  (highlighted as “3”)

    • Pivot  (highlighted as “4”)

    • Paging (highlighted as “5”)

 

 

3.  Under General tab, perform the following:

a.      In the Row Number box, type/select the required option.

·        true

·        false

b.    In the Column Lines box, type/select the required option.

·        true

·        false

c.      In the Row Lines box, type/select the required option.

·        true

·        false

d.    In the Show Grid Filter box, type/select the required option from the following:

·        true

·        false

     When Show Grid Filter is selected as true, then Grid Filter appears on the report.

e.   In the Default View box, type/select the required option from the following:

·        table - Table for operational grid

·        pivot - This is for pivot grid

 

Prior to this set the grouping to true at column level properties.

 

    • When the Grouping is set to true for multiple columns (dimensions) and when group total and grand total is selected as true for dimension (example: sales) and when "Multi" is selected here, the output appears as follows:

 

 

 

Grouping will work only when we export the report using Options -> Export -> Excel xlsx in report section.

 

When the Grouping is set to true for multiple columns (dimensions) and when group total and grand total is selected as true for dimension (eg. sales) and when "Single" is  selected here, the output appears as follows:

 

          

f.      To save the changes, click Done.

 

4.      Under Tabular tab, perform the following:

            a.     In the Row Banding box, type/select the required option.

      • true

      • false

            b.     In the Grouping Mode box, type/select the required option from the following:

i.   Single - Select single for grid grouping

ii.   Multi - Select multi for multi-level subtotal

            c.     In the Locked Columns box, type/select the required number.

            d.     To save the changes, click Apply.

 

5.   Under Pivot tab, perform the following:

            a.     In the Show Chart box, type/select the required option.

      • true

      • false

            b.     In the Show Pivot Mode box, type/select the required option.

      • true

      • false

In order to view Pivot option, Show Pivot Mode must be set to True when the report is being shared to user

            c.     To save the changes, click Apply.

 

6.   Under Paging tab, perform the following:

a.        In the Paging box, type/select the required option.

      • true

      • false

b.      In the Type box, by default Client option is available.

c.   In the Position box, type/select the required option from the following:

      • top

      • bottom

d.       In the Size box, type/select the required number.

e.      To save the changes, click Apply.

Paging is unavailable, when report is in Pivot Mode.

 

For more information see Table Widget Settings
