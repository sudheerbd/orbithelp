Dynamic Dimensions, Measures

Introduction

Dynamic dimensions and measures allow the user to update the analysis report by changing the axis/columns of the chart, even in view mode, with just a click.

Dynamic dimensions and measures eliminates creating multiple charts/reports for different dimensions/measures.

Add dynamic measures to a visualization

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

3. Click the expander arrow next to the Values section in the Report Configurator area and then click Alternative Measures (Highlighted as "1").

4. Alternative Values section (Highlighted as "2")  appears below the Values section.

5. Drag the desired value fields into the Alternate Values section.

Note: Values won't accept non-numeric fields. So if you drag a Category field into the Values section, it changes to Count of Category.

 

Add dynamic dimension to a visualization

1. In an analysis report, click the expander arrow next to the Categories/Colors section (any type of data section) in the Report Configurator area and then click Alternative Dimension (Highlighted as "3").

2. Alternative Category section (Highlighted as "4")  appears below the Category section.

3. Drag the desired data fields into the Alternate Category section.

 

Working with Dynamic Dimensions/Measures

Tip: Dynamic Dimensions/Measures works in view mode also.

1. Add desired fields to alternative dimensions/measures section in the report configuration area.

2. Click  Dynamic Dimensions/Measures.

3. Change Dimensions and Measures window appears.

4. Change Dimensions and Measures window shows all the fields added to the visualization.

5. Select desired alternate dimensions/measure from the Change Dimensions and Measures window.

6. Visualization updates with selected dimensions/measures dynamically.
