Viewing the Output

To view the output

 

1. Switch to View Mode from Edit Mode.

2. In the View Mode, from the available templates (highlighted as “1”), select the required template.

3. From the Select Output box, select the required output, and then click Run.

 

    

 

4. The output gets downloaded as HTML and TXT.

5. To view different output scenarios, refer to Creating a Word Template with Grouping and Page Break.

6. The process of viewing Excel or HTML outputs is same as this process.
