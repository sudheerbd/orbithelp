Creating a Sub Folder in a Folder

To create a sub folder in a folder

 1.      On the Reports menu, under the Reports area, the required folder appears.

         2.     Right-click the required folder, and then click New Folder (highlighted as “1”). 

 

         3.      New Folder dialog box appears.

 

                   

4.      In the New Folder dialog box, perform the following:

a.  In the Name box, type the folder name.

b.  In the Description box, type the description.

 

     Note: Description filed is optional.

  c. To save the new folder, click Save.

  d. To close the dialog box, click Cancel.
