Accessing Account Details

To access the account details

  1. On the top menu, click the icon or icon, on the top right-hand side of the window.

  2. A list of options appears as shown below:

  3. To update the account details, click Profile (highlighted as “1”).

  4. User - <UserName> dialog box appears.

    You cannot edit User Name and Email fields.

  5. To update the account details, perform the following:

    • In the First Name box, edit the required first name.

    • In the Last Name box, edit the required last name.

    • To change the profile picture, point the mouse over the picture, and then click Change.
    • Select Profile Photo dialog box appears.
    • In the Select Profile Photo dialog box, click Select a photo from your computer.
    • Open dialog box appears.
    • In the Open dialog box, select the required picture, and then click Open.
    • The selected picture appears in the Select Profile Photo dialog box.
    • To save the picture, click Save.
    • To cancel the changes, click Cancel.
    • The selected picture appears in the User-<username>dialog box on the left-hand side.

  6. To save the changes, click Save (highlighted as “2”).

  7. To close the dialog box, click Cancel (highlighted as “3”).
