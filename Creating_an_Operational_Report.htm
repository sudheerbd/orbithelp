Creating a Data Report

To create a data report

 

1.      In Reports, click  New Report, click Data Report (highlighted as “1”), and then click Ok.

 

               

 

2.      Run Result area (highlighted as “3”) appears along with Select Business Object (highlighted as “1”) and Configurator area (highlighted as “2”).

 

 

3.      On the Select Business Object, click  (highlighted as “4”) next to Select Business Object to select the required business object.

4.      Search area (highlighted as “5”) appears with available business objects (highlighted as “6”).

 

                

 

5.      In the Search box, type the business object name to search. 

(OR)

Select the required business object, and then click OK.

(OR)

Double-click the required business object.    

6.      The Selected Business Object (highlighted as “7”) appears with the associated columns under it, in the Business Object area.

7.      Drag and drop the required columns (highlighted as “8”), from the required business object to the Columns in the configurator area.

 

 

8.  To run the report, click Run above the Run Results area.

9.  A data report appears in the Run Results area.

 

 

10.  To save the report, click Save.

11.  Save Report dialog appears.

12.  select  the required report folder (highlighted as “9") already exists, then perform the following:

a.       In the Name box (highlighted as “10"), type the report name.

b.      In the Description box (highlighted as “11"), type the description.

c.       The selected folder appears in the Selected Folder box (highlighted as “12").

d.      To save the report, click Save.

 

                   

 

13.  If the required report folder does not exist, then create a new reports folder, and then save the report in it. See Creating a Reports Folder

 

    • The reports can be categorized using Folders when saving the reports.
