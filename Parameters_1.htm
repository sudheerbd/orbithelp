Parameters

1. This is the first tab in the dialog box. Here, the parameters for the specific report appear. For example, the filters applied to the report appear as parameters as shown below :

2. On Parameters tab, the required parameter appears.

3. On the required parameter, click Search for items to add (highlighted as "1") to view the list of values of parameters, which appears on the right-hand side.


If, Skip validation while searching option is selected at User Filter, the parameter column retrieve the search data without validation.
This option applicable for String and Number fields. 

4. In the list of values box, click  (highlighted as "2") to view the available parameters value.

5. From the available parameters, select the required parameter value (highlighted as "3").

6. The selected parameter value appears under the parameter (highlighted as "4").

 

 

This process is for the parameter type "string".


Parameter type as Date
To schedule the report with date increment value, select the checkbox increment date with schedule date (highlighted as "5") next to date parameter.
