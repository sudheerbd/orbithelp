AND Filter

The AND Filter is used when there are more than one condition and where all the conditions in the series must be true. It is represented by an ampersand ‘&’.

Consider the following example, where the there is a need to retrieve all the users with different last names and whose first name is “Test”.

  1. Under the User Search Information section, in the Search Filter box, type the required search filter.

  Example: Search Filter: (&(objectClass=person) (givenName=U*) (sn=A*))

  1. The following figure illustrates the AND Filter.

  1. The above search filter retrieves the users with givenName(first name) starting with U and sn(last name) starting with A; for example, “User Azzuolo”.
