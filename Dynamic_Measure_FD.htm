Dynamic Measure

Creating Dynamic Measure

To add dynamic measure to a dashboard report:

1. Open a dashboard reports page in Edit Mode.

2. Right click on required fields, click Pick Visualization and then click Dynamic Measure

(OR)

Drag the required fields from the Business Object into the Run Result area, click the expander arrow in the Report Configurator area and then click Dynamic Measure.

Hold Ctrl and click on required fields to make multiple selections.

3. Dynamic Measure Widget appears in the Run Result area.

4. Drag desired fields into the List of measures section (Highlighted as 1) in the report configuration area.

Visual Settings of Dynamic Measure

To customize the look of Dynamic Measure see Visual Settings.

Working with Dynamic Measure Widget

Dynamic Measure widget allows you to filter the data displayed on the chart widgets in the dashboard reports page.

Click any one of the buttons in the Dynamic Measure widget; data on the chart widgets updates.
