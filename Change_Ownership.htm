Change Ownership

Change ownership job is used to change the ownership of all the meta objects (Folders, Jobs, Datasets, DataSources).

To change the ownership

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded and Custom jobs.

  3. Under Jobs sections, on Change Ownership, click , and then click Schedule.

  4. Change Ownership dialog appears.

 

 

  1. In the Parameters tab, perform the following:

    1. In the From User box, type/select the required user.

From User: Orbit

    1. From the Select Object list, select the required report folders, Jobs, Datasets, or DataSources.

Select Object: test, test2

    1. For the above-mentioned report folders, the ownership is changed from orbit to another user.

    2. In the To User box, type/select the required user. 

To User: test

 

 

 If the process must be run immediately, select the check box before Run Immediately. If the process must be run a later time, then remove the checkmark from the Run Immediately checkbox.   
When the Run Immediately checkbox is deselected, the Schedule tab becomes available. Using the Schedule tab, schedule the process at a required time.

  1. To set any notifications, click Notifications tab.  View Notifications for more information.

  1. To submit the process, click Submit..
