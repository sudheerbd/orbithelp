This topic explains different ways to customize the axes (X-axis and Y-axis) of your charts. Not all charts have axes. Pie charts, for example, don't have axes.

 

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Axes to open Axes visual settings.

 

Axes

Axes tab is further divided into 3 tabs:

  1. Label Axis
  2. Value Axis
  3. Y2 Axis

 

 The visual settings available depend on the visualization type selected.

Label Axis

Label Axis has the following visual settings:

  1. Title: Customize the chart title.

  2. Show axis line: Show/Hide axis line.

  3. Show labels: Show/Hide labels on the axis.

  4. Show Max Labels: Set maximum number of labels to display on the chart.

  5. Reversed direction: Reverse the direction of chart on Label axis.

  6. Label Rows: Display labels on 1, 2, or 3 rows.

  7. Label Drop: Shows only selected Labels. Drops (Hides) other Labels.

  8. Label Angle: Changes label angle.

  9. Show tick marks on axis: Show/Hide tick marks on the axis.

  10. Show grid lines: Show/Hide grid lines on the chart.

  11. Date Format: Customize the date format.

  12. Label Format: Change the label name by adding prefix and/or suffix.

  13. Axis Position: Change axis position (left to right, and vice versa).

  14. Category Items: Group/Ungroup chart visual elements and enable drilldown.

 

Title

Default Value: Auto

Title setting has the following options:

  1. Auto: When Title is set to Auto, Orbit application generates chart title based on the field names dropped in the visualization (chart type) options section.

  2. Custom: Select Custom to set a custom title.

  3. None: Select None to hide axis title.

Text

When you  select Custom in Title options, Text box appears. Enter a custom title in the text box.

Show Axis Line

  1. Select Show Axis Line check box to show the axis line.

  2. Clear Show Axis Line check box to hide the axis line.

 

Show Labels

  1. Select Show Labels check box to show the axis labels.

  2. Clear Show Labels check box to hide the axis labels.

Show Max Labels

Set maximum number labels to show on the chart window. If the number of labels are more than maximum number set, you can still view them by scrolling up/down or left/right.

Reversed Direction

  1. Select Reversed Direction check box to reverse the direction of chart on Label axis.

  2. Clear Reversed Direction check box to bring chart back to default status.

 

Show Tick Marks on Axis

  1. Select Show Tick Marks on Axis check box to show tick marks on the axis.

  2. Clear Show Tick Marks on Axis check box to hide tick marks on the axis.

Show Grid Lines

  1. Select Show Grid Lines check box to show grid lines on the chart.

  2. Clear Show Grid Lines check box to hide grid lines on the chart.

Select to display labels in 2 rows.

Select to display labels in 3 rows.

 

 

Label Drop

Default value: Don't drop labels

Label Drop has following options:

  1. Don't drop labels - Shows all labels

  2. Show every 2nd label - Shows every 2nd label and hides remaining labels.

  3. Show every 3rd label - Shows every 3rd label and hides remaining labels.

  4. Show every 4th label - Shows every 4th label and hides remaining labels.

  5. Show every 5th label - Shows every 5th label and hides remaining labels.

  6. Show every 6th label - Shows every 6th label and hides remaining labels.

  7. Show every 7th label - Hides every 7th label and hides remaining labels.

 

Label Angle

Default value: Auto

Label angle has following options:

  1. Auto

  2. --- (Horizontal)

  3. /// (-45°)

  4. ||| (Vertical)

  5. \\\ (45°)

Date Format

Customize the date format. Select any one of the date format from the drop down list.

Label Format

In the Label Format box enter prefix and/or suffix to labels.

 

Axis Position

Default value: Left

Logarithmic Scale

A Logarithmic Scale is a nonlinear scale. It is useful when analyzing a large range of quantities. Instead of increasing in equal increments, each interval is increased by a factor of the base of the logarithm. Typically, a base ten scale is used.

  1. Select Logarithmic Scale check box to plot bar chart on a logarithmic scale.

  2. Clear Logarithmic Scale check box to plot bar chart on a linear scale (Default).

 

Format Type

Default Value: Auto

Format Type has following options:

  1. Auto: Orbit auto formats values based on the field properties

  2. Number: Formats value axis values as numbers

  3. Currency: Formats value axis values as currency

  4. Percent: Formats value axis values as percentage

 

When Format Type is set as Number, following fields appear.

  1. Display Units

  2. Decimals

  3. Separators

Display Units

Default Value: Auto

Display Units has following options:

  1. Auto: Orbit auto formats values

  2. None: Complete number is shown on value axis. No format is applied.

  3. Thousands: Numbers are shown in thousands

  4. Millions: Numbers are shown in millions

  5. Billions: Numbers are shown in billions

  6. Trillions: Numbers are shown in trillions

Decimals

Default Value: .#(1)

Decimals has following options:

  1. .#(1): Shows 1 fractional number after decimal point

  2. .##(2): Shows 2 fractional numbers after decimal point

  3. .###(3): Shows 3 fractional numbers after decimal point

  4. .####(4): Shows 4 fractional numbers after decimal point

  5. .#####(5): Shows 5 fractional numbers after decimal point

  6. .######(6): Shows 6 fractional numbers after decimal point

Separators

Default Value: 1,234.56

Separators has following options:

  1. 1,234.56: Comma (,) is used as thousands separator and period (.) is used as decimal point

  2. 1.234,56: Period (.) is used as thousands separator and comma (,) is used as decimal point

  3. 1`234.56: Apostrophe (') is used as thousands separator and period (.) is used as decimal point

  4. 1 234,56: Space ( ) is used as thousands separator and comma (,) is used as decimal point

  5. 1 234-56: Space ( ) is used as thousands separator and hyphen (-) is used as decimal point

  6. 1 234.56: Space ( ) is used as thousands separator and period (.) is used as decimal point

  7. 1234.56: No thousands separator and period (.) is used as decimal point

Y2 Axis

When there are two or more fields in Value Axis (Y Axis) section, they can be separated as Value Axis and Y2 Axis in Bar chart.

By Default, Y2 Axis is hidden.

To show Y2 Axis in chart:

  1. Go to Visual Settings -> Series.

  2. Series tab has child tabs based on number of fields in Value Axis. If Value Axis section has two fields, Series tab is further divided into two tabs: each having field name as title.

  3. Select the tab you want to change as Y2 Axis and select Y2 Axis check box.

  4. Y2 Axis appears in the chart area.

 

Visual Settings for Y2 Axis are same as Value Axis. Refer to Value Axis.
