Getting Started with Script Report

To get started with a Script Report

 

1.  In the Data Explorer section, click  Choose business object next to Select Business Object.

2.  A list of business objects appear.

3.  From the list, double click the required business object.

4.  The columns under the selected business object appear.

6.  On the Data section, click  Add New Query.

7.  A new query (highlighted as “1”) appears.

    

 

 

8. On the Query tab, a section (highlighted as “2”) drop the required columns.

9. From the Data Explorer area, drag and drop the required columns onto the Query tab under COLUMNS.

 

 Example: Required Columns: State

Sales

 

This automatically displays a Query section (highlighted as “3”) with the query for the dragged columns.

              

 

10.  To display the output in the desired format, perform the following:

  • In the Editor section, enter the relevant script.

  • In the Select script list, click the required script.

In the above script, the line csv_in=‘${Query-0}’, is to specify the input for the script. Internally, the input changes to the CSV file and the file path is specified in place of ‘${Query-0}’, behind the scenes.  from where the columns are picked up into the report.

The line py.image.save_as(fig, filename='${PNG}')is to specify the output as png. You can replace JPG as well in place of PNG.

·   To save the changes, click Save.

·  To view the output in the PNG format, click View Mode.

·   The report now switches to View Mode from the Edit Mode.

·   In the view mode, click Run.
