Schedule

To Schedule a report

  1. By default, Schedule tab is not available.

  2. To enable the Schedule tab, clear Run Immediately check box.

  3. In the Schedule Name box, enter the required schedule name.

    Note: If Schedule Name is empty, by default report name is taken as schedule name.

  4. In the Start Date box, enter or select the required date.

  5. In the End Date box, enter or select the required date.

  6. In the Start Time box, enter or select the required time.

  7. In the End Time box, enter or select the required time.

  8. User Timezone displays the time zone of the user's machine.

  9. Schedule section provides two options to configure recurring schedules:

    • Simple

    • Advanced
