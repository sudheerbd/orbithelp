Filtering Data Source

To Filter a data source

 

  1. In the Data Sources page, click .

  1. A filter toolbar appears. Filter toolbar has textboxes under Name, Description, and Created By.

          

 

User can filter the data sources using any one or combination of the following fields: Name, Description, and Created By.

  1. Under the required column, enter the search term to filter.

  2. Based on the filter value, the search results appear as shown:

 

           

When the filter is active, filtered column name color changes to blue and font style changes to italics.

  1. To clear the filter, delete search term in each field or click  on the filter toolbar.

  2. To hide the filter tool bar, click .

Clicking   Filter icon only shows/hides the filter tool bar. It does not clear the filter.

  1.  To refresh the list of data sources, click Refresh .
