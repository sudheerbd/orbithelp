Data Interaction Options

To work with data interaction options

 

  1. Open the required dashboard report in run results area.

Note: Dashboard report Options available in both View and Edit Mode

  1. On the top right-hand corner of the run results area, click Options, and then click Options.

  2. Dashboard Options dialog appears.

 

  1. Data Interaction Options  have the following option

    • All

    • Allow Highlights Others

    • Allow Cross Filters

    • Allow Drilldown(Drill-through, Drill by)

    • Allow Filter Others

    • Underlying Data

  1. When user selects All from Data Interaction Options, all the option are available for the visualization widgets.

  2. By uncheck the above option, it disable the functionality on Visualization option. For more info refer Visualization Options.

  3. To save the changes, click Ok.

  4. To cancel the changes, click Cancel.
