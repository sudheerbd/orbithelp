User Management

ORBIT Analytics application provides the following seeded users:

 

  • ORBIT ADMIN ROLE ( Developer)

  • ORBIT SUPER USER ROLE (Super User)

  • ORBIT USER ROLE ( End User)

 

This chapter introduces you to the User Management feature of the application. Specifically, this chapter teaches you about:

 

  • Working with Users

  • Working with Groups

  • Working with Roles

  • Working with Authorities

  • Working with Permissions
