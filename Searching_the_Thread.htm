Filtering the Thread

To filter threads

1. In the Threads section, click .

 

 

2. A filter toolbar appears. Filter toolbar has textboxes under Name, Created By, and Thread Status columns.

 

 

Note: User can filter the threads using any one or combination of the following fields: Name, Created By, and Thread Status.

3. In the textbox under required field, enter the search term.

Example: Required Field: Name

                Search Term: Test

 

4. The filter results appear as above.

Note: When the filter is active, filtered column name color changes to blue and font style changes to italics.

5. To clear the filter, delete search term in each field or click  on the filter toolbar.

6. To hide the filter toolbar, click .

Note: Clicking  only shows/hides the filter toolbar. Clicking does not clear the filter.
