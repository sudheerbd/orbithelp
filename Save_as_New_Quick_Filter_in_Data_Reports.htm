Save As New Quick Filter

To Save the New Quick Filter

 

  1. Open the required report in run results area.

Required Report: Report Doc

    • The report here has the parameter "Region" and the parameter values "East" and "Mountain"

  1. On the top right-hand corner of the run results area, click Options, and then click Save as New Quick Filter (highlighted as “1”).

  

   

  1. Report Quick Filter dialog box appears.

 

 

  1.  Under the Report Quick Filter, perform the following:

 

    1. In the Lable box, type the name of the quick filter.

Quick Filter Name: Mountain and East Zones

    1. To save the changes, click Save.

 

      •  The quick filter is saved as "Mountain and East Zones"

      • To view the quick filter , click Options, and then click Quick Filters (highlighted as “2”).

 

        

 

  1. Quick Filter dialog box appears with a list of Search Label (highlighted as “3”) as shown below:

 

 

To Apply the Quick Filter

 

  1. On the top menu, click Quick Filters, on the top right-hand side of the run report area.

  2. A list of Quick Filters appears as shown below:

 

 

  1. To apply the Quick Filter, select the required filter.

 

When user select the required filter, the filter values appears in User Filter as shown above and the Quick Filter appears in blue color.

  1. To close the Quick Filter window, click Close.

  2. The report appears with the selected quick filter as shown below:

 

    

  1. To clear the filter values in user filter/a Quick Filter, click Clear.
