Creating a Dashboard Report

To Create a Dashboard Report

 

  1. In the Reports page, click   New Report, and then click Dashboard.

  2. Dashboard report opens with two options- Data (highlighted as “1”) and Visual Settings (highlighted as “2”).

 

  

 

  1. Under the Data area, the Select Business Object area appears along with Run results area.

  2. Under the Visual Settings area, the required options appears.

  3. To change the related report visuals,refer Visual Settings.

  4. Dashboard reports can be created in either of the following ways:

      1. Adding existing reports from Visualization.  

        • On the top-menu, from the Add Widget list, select Visualization.

        • A list of reports appears, under Add Visualization dialog.

        • To add the visualization to the report, click (highlighted as 3) as show below:

 

 

        • To close the dialog, click Done or   Close dialog.

 

Example: Analysis Report 1: Sales by Region

      Analysis Report 2: Sales,Profit by State

 

 

Note: Below options available, when the user mouse hover on the visualization widget.

To view the Visualization option

        • To view the applied filters on the visualization, click Applied Filter (highlighted as "4").

        • To change the visualization, click Change Visualization (highlighted as "5"). See Visualization.

        • To maximize/minimize the visualization , click Maximize/Minimize (highlighted as "6").

        • To view options of visualization , click Options (highlighted as "7").

 

 

(OR)

      1. Select a business object from Select Business Object under Data section.

        • Drag the required columns from the selected Business Object to the Run Result area. Orbit creates best visualization based on the selected fields.

(OR)

        • To create a visualization in dashboard report, right-click on the required fields and then click Create Best Visualization

(OR)

        • To create a visualization in dashboard report, right-click on the required fields and then click Pick Visualization.

        • A list of chart appears under Pick Visulaization, select the required chart from the list.

Note: Hold Ctrl and click on required fields to make multiple selections.

 

 

  • Multiple reports can be displayed on the Run Results area of the Dashboards

 

  1. The reports appear on the Run Results area.

  1. To save the dashboard, click Save Report.

  2. Save Dashboard dialog box appears.

 

 

  1. To save the report, perform any one of the following:

                                            i. Save the report in an existing folder.

(OR)

                                          ii. Create a new folder, and then save the report under it.

  1. To save the report, click Save.

  2. To close the dialog box, click Cancel.

Note: Only Analysis reports can be used in Dashboard (No Data reports)

       Business Objects which have Oracle ERP as the Query Executor, must not be used for Dashboard reports.
