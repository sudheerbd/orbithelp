Applying Filters in Logical Model

Filters set here in the logical model are pre-defined conditions and the user must enter these conditions

 

Applying Filters in Logical Model

 

1. For the selected object, Columns (highlighted as "1") and Filters (highlighted as "2") appear in the Logical Model workspace.  

 

                                            

2. The columns for the selected table/view appear under the Columns - <Table Name>.  

  • To view the columns under the selected table/view/synonym, you need to first save the logical model.

3. The columns appear as show below:

 

    

                                         

4. Drag and Drop the required column from Columns area to Filters area to set the filter conditions.

5. To view the filter conditions for the selected column, click for the selected column.

6. Filter-<Column Name> dialog box appears.

 

                                                                              

7. By default, Select a range option is selected.


If you want to change a condition, click Specify a condition.
Apply the desired filter.

8. To save the changes. click Ok.

9. To save the domain, click Save on the menu area.

        

 The report appears with the specified filter column when it is run. The user must enter the condition to run the report.
