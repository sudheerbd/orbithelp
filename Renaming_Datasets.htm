Renaming Datasets

To rename a dataset

 

  1. In the Datasets area, a list of available datasets appears.

  2. For the required dataset, under Actions, click  icon, and then click Rename.

 

                                                  

  1. Rename <Dataset name> dialog box appears.

 

                              

 

  1. In the Rename<Dataset name> dialog box, perform the following:

a.       In the Name box, type the new name for the dataset.

b.      In the Description box, type/edit the description for the dataset.

  1. To save the changes, click Save.

  2. To close the dialog box, click Cancel.
