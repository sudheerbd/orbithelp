Configuring Query Settings

 

To Configure Query Settings

 

1. To apply query setting, select Query Settings (highlighted as “1”) next to Configurator  .

   

2. The following two sections appear:  

 

·        Run Now

·        Schedule

 

           

 

Schedule option in Report -Query Settings is applicable for Data Reports only.

 

3. In the Run Now section, perform the following:

 

    • To make the fields distinct, select the Select Distinct checkbox.

    • To set the row limit, select the Row Limit checkbox. 

    • Select/type the number of rows in the box beneath the Row Limit checkbox.

    • To set the timeout session of the query, select Query Timeout checkbox.

    • Select/type the number of seconds in the box beneath the Query Timeout checkbox.

Example: 12000

    • To auto-refresh the report, click Auto Refresh.

    

              

 

 

4. In the Schedule section, perform the following:

 

    • To set the row limit, select the Row Limit checkbox. 

     

            

 

      Note: When Row Limit checkbox is selected, two radio buttons appear

    • First, to select/type the number of rows beneath the Row Limit checkbox.

    • Second, the All Rows radio button to retrieve all the rows in a data report.

 

                 

    • To set the timeout session of the query, select Query Timeout.

    • Select/type the number of seconds in the box beneath the Query Timeout checkbox.

8.      To save the changes, click Save.
