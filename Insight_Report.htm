Insight Report

 

An insight report is a child report that originates from a data report. Insight report can have fewer columns compared to its parent report.

Prerequisite: In order to view the insight reports, create a data report and save the report.

Insight report option is available, when the permission is given in role level.

To Create an Insight Report

 

  1. Create a data report with the required columns.

  2. Run the data report.

  3. Save the data report with the desired name.

  4. On the tool bar above the run results area, click Show/hide insights.

  5. Insights window appears on the right-hand side.

  6. In the Insights window, point to  icon.

  8. Create new insight option appears with the following two report types:

  • Data Grid

Scheduling the Insight Report:

1. To schedule the Insight report, select the required report from Reports, right-click and then click Schedule.

2. Schedule report dialogue box appears.

3. In the Parameters tab, Under Select Insight (Optional), select the Insight report from the drop down and click Submit.
