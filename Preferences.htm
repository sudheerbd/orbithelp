Preferences

The preferences section enables the user to set different preferences. Under preferences section the following options are available:

 

  • Audit Level

  • Force Detail Data For Pivot

  • Zip output file by default

  • Report Statistics

  • Row Limit

  • Query Time Out

  • Report Output File Format

 

Audit Level

To enable audit data option

  1. To enable Audit Level option, select Audit Level value as "All".

By default, the Audit Level is in Off state.

  1. To disable Audit Level, select Audit Level value as "OFF".

 

Force Detail Data For Pivot

To enable details data option

  1. To enable Detail Data option in reports, select Force Detail Data For Pivot as "Y".

By default, the force detail data for pivot is in enable state.

  1. To disable Detail Data option in report , select Force Detail Data For Pivot as "N".

 

Zip output file by default

Using this option the user can zip the output files

To enable the zip output file by default

  •  On the Settings menu, under the Preferences tab,  in the Enable Zip output file by default box, select "Y".

By default, the zip output file by default is in the enabled state.

To disable the zip output file by default

  •  On the Settings menu, under the Preferences tab,  in the Enable Zip output file by default box, select "N".

In case, the Zip output file by default box under Settings -> Preferences is set to "N", then Send attachment as zip file checkbox in Schedule-Report -> Output & Deliverye ->Deliveries ->Delivery To ->Email appears unchecked.

 

 

 

Report Statistics

Using this option the user can either enable/disable the report statistics

To enable the report statistics

  • On the Settings menu, under the Preferences tab,  in the Enable Report Statistics box, select "Y".

:  By default, the report statistics is in the enabled state.

 

To disable the report statistics

  • On the Settings menu, under the Preferences tab, in the Enable Report Statistics box, select "N".

: The reports that are scheduled/modified after the report statistics is set in disabled state,do not appear in the Report Statistics section under Monitoring.   

 

 

Row Limit

Using this option, the user can set the row limit for the reports.

To set the row limit

  • On the Settings menu, under the Preferences tab,in the Row Limit box, type the required value.

 

Here, the row limit for one report is set to 10,000. This implies, one report can fetch 10,000 rows at once.

 

 

Query Time Out

This option enables the session/query  time out in seconds.

To set the time out

  • On the Settings menu, under the Preferences tab, in the Time Out box, type the required value.

 

  • Row Limit and Time Out Preferences are limited to Data Reports.

  • If Max row count is not set in report definition-Query Settings.

 

 

Report Output File Format

This option helps the user to store the files locally with Unique Names.

To set the Report Output File Format

On the Settings menu, under the Preferences tab, in the Report Output File Format type the required value.

E.g: {REPORTNAME} {PROCESSID} {REPORTID} {RTDF=yyyy-MM-dd HH:mm:ss.SSS}

  • {REPORTNAME} : for report name

  • {PROCESSID} : for process id

  • {REPORTID} : for report Id

  • {DTDF=date_format} : date will be replace as per the date format .

  • {RTDF=date_format} : date will be replace as per the date format
    Dev note : ( schedule date will be applied if no schedule found created date will be applied)

  • {RAND} / {RAND=4} : random string with length default(3) or as given (=4) will be replaced
