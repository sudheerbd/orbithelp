Visualization options

To work with Visualization option perform the following

 

  1. To view visualization option, click Options (highlighted as "1") on the dashboard widget.

(OR)

Right-click on the dashboard widget.

  1. A list of options appears, on visualization widget as shown below (highlighted as "2").

    • Use as Filter

    • Ignore in Interactivity

    • Change Dimensions and Measures

    • Duplicate Visualization

    • Delete Visualization

    • Configure Drill-through

    • Underlying Data

    • View SQL
