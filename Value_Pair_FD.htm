Value Pair

 

Visualization (Chart type) Options

1. Value Pair has the following visualization options

  • Values

  • Group By

2. Drag and drop the required fields from the selected Business Object into the Values and Group By.

3. Orbit application generates the packed bubbles analysis report dynamically in the run results area.

 

Visual Settings for Value Pair

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for Heat Maps has the following options:

  1. Title: Show/hide value pair chart title. See Customize Chart Visual Settings for details.
  2. Layout: Change layout of values in the chart.
  3. Value Tab: Each field dropped in values section has following settings.

    1. Label: Change Label Name. See Customize Series Visual Settings for details.

    2. Prefix: Adds text before labels. See Customize Series Visual Settings for details.

    3. Suffix: Adds text after labels. See Customize Series Visual Settings for details.

    4. Format: Change label format. See Customize Axes Visual Settings for details.

Layout

Default: Horizontal

Layout setting has the following options:

  1. Horizontal: When Layout is set to Horizontal, values are shown next to each other.

  2. Horizontal List: Select Horizontal List to show values in a list form.

  3. Vertical: Select Vertical, to show values in a vertical list form.
