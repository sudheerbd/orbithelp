Creating Object from SQL

To create an object from SQL

 

 

  1. Under Domain Hierarchy, Physical Models folder appears.  

  2. Under the folder, right-click the required folder, and then click  Create Object From SQL

 

 

                       

  1. Create Object From SQL <physical model name> dialog box appears.

 

                      

  1. Under Create Object From SQL <physical model name> dialog box, perform the following:

    1. In the Schema Name list, click the required schema.

Schema Name: XXFO

      • Schema Name is mandatory.  

    1. In the Object Name box, type the object name.

Object Name: INVOICES_ALL

      •  Object Name is mandatory.

    1. In the Description box, type the description.

Description: This is test description

    1. In the Sql query box, type the SQL Query.

Sql query : Select * from INVOICES_ALL

      •   A semicolon (;) should not be placed at the end of the SQL Query

    1. To create an object, click Create.

    2. To close the dialog box, click Cancel.
