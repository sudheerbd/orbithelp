Customize Legend Visual Settings

This topic explains different ways to customize the legends of your charts.

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

Note: You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Legend to open Legend visual settings.

Legend

Legend tab has the following visual settings:

  1. Legend: Show/Hide data legend on the chart.

  2. Show Values: Show/Hide value of the slice in the legend.

  3. Show Percentage: Show/Hide percentage value of the slice in the legend.

  4. Legend Alignment: Changes legend alignment on the chart.

  5. Legend Format: Change the legend name by adding prefix and/or suffix

Note: The visual settings available depend on the visualization type selected.

Legend

  1. Select Legend check box to show legend on the chart.

  2. Clear Legend check box to hide legend on the chart.

Show Value

  1. Select Show Value check box to show value of the slice in the legend.

  2. Clear Show Value check box to hide value of the slice in the legend.

Show Percentage

  1. Select Show Percentage check box to show percentage value of the slice in the legend.

  2. Clear Show Percentage check box to hide percentage value of the slice in the legend.

 

Tip: Click on a Legend Name in the chart area to filter out its data from the pie chart.

Legend Alignment

Default Value: Bottom

Select Legend Alignment from the following options:

  1. Left

  2. Right

  3. Bottom

  4. Top

 

Legend Format

In the Legend Format box enter prefix and/or suffix to the legends.
