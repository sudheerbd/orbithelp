Use as Filters

To use visualization as filter

  1. To view visualization option, click Options (highlighted as "1") on visualization tab.

(OR)

Right-click on the visualization widget.

  1. A list of options appears, on visualization widget as shown below (highlighted as "2").

 

 

  1. To use the visualization value as filter, right-click on the required visualization and then click Use as Filter.

Note: Use as Filter option available in both Edit/View Mode

  1. When Use as Filter is enabled, a checkmark appears before Use as Filter option and the visualization appears with a filter icon as shown below:

  1. By selecting the required value field (highlighted as "3") from first widget are applied as filter values in second widget (highlighted as "4").

 

Note: This option available in both View/Edit Mode.
