Changing Base Field Format in Report

To change the Base field format in the report

  1. On the left-hand side menu, point to Business Models, and then click Meta Objects.
  2. Meta Objects page appears with a list of Meta Objects on the left-hand side.

3. To change the formatting of Base Field in the reports, perform the following:

    1. Click the Base Objects Name and Value columns of  the Base field appear.

    2. Under the Format tab,perform the following

      •  In the Display Text box, enter the required text to display.

      • In the Display Test Separator box, enter the required character to be displayed.

      • In the Null Sorting Priority select the priority from the drop down ( i.e DEFAULT, FIRST or LAST)

     c.   Under the Style tab, perform the following:

      • In the Alignment box, type/select the required option.

      • In the Column Width box, type/select the required value.

      • In the Font Family box, type/select the required values by clicking.

      • A dialog window appears with multiple fields.

      • Edit the required fields, and then click Ok.

  •  In the Hidden box, type/select the required value.

4. To save the changes, click Save.
