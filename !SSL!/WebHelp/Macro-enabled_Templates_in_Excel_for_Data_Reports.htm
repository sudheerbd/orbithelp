Macro-enabled Templates in Excel for Data Reports

 

Prerequisites

It is essential to have knowledge of creating Macro-enabled templates in Excel.

 

Introduction

Here, the “Export with Template” feature has been explained with the macro-enabled templates in Excel similar to Publisher Report Templates.

This feature is explained using multiple scenarios. Each scenario describes a template that can be uploaded into Data Report.

 Consider the following scenarios:

Scenario 1­ :When columns exist in excel sheet and names exist in Name Manager

Scenario 2: When no columns exist in Excel Sheet

Scenario 3: When no names exist in Name Manager

Scenario 4: When there are no Names in the Name Manager and no Columns in the Excel template.
