Assigning a Thread to a Snapshot

To assign a thread to a snapshot

 

               1.    Under Snapshots, select the required snapshot.

2.    In the Threads section, click   Search for items to add to view the available threads.

 

3.    A list of available threads appear.

    

 

4.    Select the threads from the list, to assign to the selected snapshot.

 

 

5.    Under Threads section, the selected threads appear.

6.    To save the changes, click Save.

Note: A Changes saved successfully notification appears briefly at the top of the ORBIT application window.
