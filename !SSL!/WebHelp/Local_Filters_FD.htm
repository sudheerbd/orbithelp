Local Filters

To apply Local Filter on a report

1. On an existing analysis report, drag and drop the required fields from the selected Business Object into Local Filters under the Report Configurator area.

2. To configure the local filter, under Local Filters, click Edit on the field name.

3. <Field Name> dialog box appears.

4. In the <Field Name> dialog box, click  Search for items to add.

5. From the available list of values, select the required values and then click Ok.

6. The Analysis Report updates according to the Local Filter selection.

7. To remove the Local Filters, click Remove this item on the field name.

 

Procedure 2:

To filter the data, using Specify a condition perform the following:

1. In the Report Configurator area, under Local Filters, click Edit on the field name.

2. <Field Name> dialog box appears.

3. In the <Field Name> dialog box, click Specify a condition.

4. Select the required condition from the conditions list.

5. In the textbox below the condition list, click Select value. See Selecting Default Values for more information.

6. The selected value appears in the text box.

7. Click Ok to apply the filter.
