Custom Job Registration

To register the custom job

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of available seeded and custom jobs.

    Note: If there are no custom jobs currently registered then only seeded jobs appear.

  3. To create a new job, click (highlighted as “1”).

  4. New Job pane appears.

  5. Under the New Job pane, perform the following:

    1. In the Name box, type the required name.

    2. In the Description box, type the required description.

    3. In the Job Class box, type the fully qualified Job class name, including the package name.

    4. In the Job Configuration box, type any configuration-related information.

      For example, if there is web service integration, we need Web Service Definition Language (WSDL) location and so on.

    5. On the Parameters section, click New (highlighted as “2”).

    6. New Parameter dialog appears.

    7. In the New Parameter dialog, perform the following:

      1. In the Name box, type the required name.

      2. In the Label box, type the required label.

      3. In the Type list, select the required option from the following list:

        • Integer

        • String

        • Date

      4. To set the filter and value fields, select a business object from the Select Business Object section, by clicking   to view the available business objects.

      5. Search Business Object section (highlighted as “3”) appears with a list of available business objects.

      6. To view the filter and value fields, click List of values checkbox.

      7. The following fields appear as shown below:

      8. Drag and drop the required columns from the selected business object to the Value Field, Display Field(s), and Filter(s) section.

      9. To save the parameter, click Ok.

    8. Under the Parameters section, the newly created parameter appears.

    9. To save the job, click Save.

  6. The newly created job appears under Jobs section (highlighted as "4").
