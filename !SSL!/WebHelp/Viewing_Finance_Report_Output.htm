Viewing Finance Report Output

To view the scheduled output

  1. On the left-hand side menu, click Monitoring.

  2. On the Monitoring menu, click Processes.

  3. A list of processes appears with Processes Id.

  4. To view the output file, select the report process (highlighted as “1”).

  5. Under the Parameters area (highlighted as "2"), parameters of the selected report appear.

  6. Output of the report displayed under Output(s) & Delivery area (highlighted as "3").

  7. To download the output file, click Download Excel (highlighted as “4”).

  8. To view the download file, click Open.

Working with Offline Drilldowns

Note: Offline Drilldowns work when a user selects Generate Drilldowns check box while scheduling the report.

If a user selects Generate Drilldowns and chooses Drilldowns with authentication, then the user needs to provide valid credentials to enable drilldowns.

To work with Drilldowns with authentication:

  1. Open the downloaded report file.

  2. Go to the Add-ins tab. Custom Toolbars group appears.

    Note: Before log in to the application, only the Login option is available in the Custom Toolbars group. The remaining options are unavailable and appear grey.

  3. On the Add-ins tab, in the Custom Toolbars group, click Login.

  4. Login Form dialog box appears.

  5. In the Login Form dialog box, select the required Orbit application instance name from the Instance Name drop-down box.

  6. Enter valid credentials to log into the selected Orbit application instance.

  7. After successful log-in, the Login option becomes unavailable and the remaining options in the Custom Toolbars group become available.

  8. To perform Offline Drilldown, Select a cell with balance data and then click Drilldowns > Balance Drilldown.

  9. Balance drilldown of the selected cell opens in a new sheet in the excel file.

  10. To view the journal details, select a cell in the Balance Drilldown sheet and click Drilldowns > Journal Drilldown.

    Note: Journal Drilldown works only when the user selects YTD or PTD values.

  11. Journal drilldown of the selected cell opens in a new sheet in the excel file.

  12. To view the Subledger details, in the Journal Drilldown sheet, select a cell from the SUB_DRILL_DOWN column and click Drilldowns > Subledger Drilldown.

  13. Subledger drilldown of the selected cell opens in a new sheet in the excel file.

 

To work with Drilldowns without authentication:

If a user selects Generate Drilldowns and chooses Drilldowns without authentication, then the user can perform drilldowns without providing credentials.

  1. Open the downloaded report file.

  2. Go to the Add-ins tab. Custom Toolbars group appears.

    Note: Before log in to the application, only the Login option is available in the Custom Toolbars group. The remaining options are unavailable and appear grey.

  3. On the Add-ins tab, in the Custom Toolbars group, click Login.

  4. Login Successful message appears. Click Ok.

  5. The Login option becomes unavailable, and the remaining options in the Custom Toolbars group become available.

  6. To perform Offline Drilldown, Select a cell with balance data and then click Drilldowns > Balance Drilldown.

    Note: Refer to To work with Drilldowns with authentication for detailed information on all drilldown options.
