Thread Chat Settings Menu

To work with thread setting

1. To view the menu options for the selected thread, click  Thread Chat Settings Menu.

 

 

2.  The menu appears as follows:

  1. Lock Thread

  2. Archive Thread

  3. Snapshots

  4. Change Thread View

  • Thread Chat

  • Documents

  • Voting Requests

   3.   Initially, Archive Thread option appears unavailable; this option becomes available, when a thread is locked.
