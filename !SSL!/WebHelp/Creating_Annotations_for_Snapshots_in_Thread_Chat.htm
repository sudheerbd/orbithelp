Creating Annotations for Snapshots in Thread Chat

 

To create Annotation for Snapshots in Thread Chat  

 

1.  In the chat window of the selected thread, click  Thread Chat Settings Menu, and then click Snapshots.

2.  Snapshot window appears with available snapshots for the selected thread.

 

 

3.  In the Snapshot window, select the snapshot to annotate.

4.  Under Snapshot Image section, the selected snapshot appears.  

5.  To annotate on the image, click and drag the cursor.

6.  A message box appears.

7.  In the message box, type the required comment/message, and then click Save.

 

 

Note: To close the message box, click Cancel.

8.  The annotation appears o the image and annotation details appear under Annotations section.

 

 

 

 

Editing the annotation

To edit the annotation

1.  To edit the annotation, point to the required annotation.

2.  The annotation options appears.

 

 

 

3.  On the annotation options, click Edit. 

4. This enables the Edit mode.

5.  Make the necessary changes in the annotations, and then click Save.

6.  To close the message box, click Cancel.

 

 

Deleting the annotation

To delete the annotation

1. To delete the annotation, point to the required annotation.

2.The annotation options appears.

 

 

3. On the annotation options, click   Delete.

 

 

Changing Thread View

To change the thread view

 1. In the chat window of the selected thread, click  Thread Chat Settings Menu, and then point to Change Thread View.

Changing Thread View has the following options:

·        Thread Chat

·        Documents

·        Voting Requests

1)    Thread Chat

Thread Chat view is the default view. When a thread is opened, thread chat view appears, by default.

 

 

Note: In case, the user is in different view and wants to navigate to Thread Chat View, click Thread Chat Settings Menu, point to Change Thread View, and then click Thread Chat.

 

2)    Documents
