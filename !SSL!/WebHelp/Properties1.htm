Properties

Data Reports, Analysis, Publisher and Script Report have the same properties. Publisher Reports and Script Reports do not have Style section in report Properties.

 

To set the Properties 

 

1.  Under the Reports area, select the required folder to view the reports under it.

2. Double click on the required report, and then click Edit Mode.

Report Properties appear only in Edit Mode.

3. To set the properties, click Options menu, and then click Properties above the Run Results area.

4. Properties dialog box appears.

5. On the left-hand side, the selected report name (highlighted as “1”) and the columns (highlighted as “2”) under it appear.

 

 

6.      On the right-hand side, the properties of report name and the columns appear.  

7.      For the selected report, the following properties (highlighted as “3”) appear:

a. Under Name, type the name of the report. 

b.  Under Description, type the description of the report.  

c.  Under Output section , the following options appears:

      1. Default Output

This option have the following output types

        1. CSV

        2. DELIMITER TEXT

        3. TABULAR/JSON

        4. XML

        5. HTML

        6. PDF

        7. Excel(xlsx)

The selected default output appears in report schedule window.

      1. Delimiter 

For more information refer Delimiter Text

At the report level, in the properties dialog, the "date/time" field can have the various formats which can be used using the following notation:

 

        • y : Year ( eg. 12); also use either yy or yyyy

        • M: Month (eg: Nov); also use either MM, MMM, or MMMM

        • d : Day in a Month ; can also use dd

        • h: hour of the day- 1 to 12 (AM/PM); can also use hh

        • H : Hour of the day - 0 to 23 ; can also use HH

        • m: minute in hour ( 0-59 ); can also use mm

        • s: seconds in a minute (0-59); can also use ss

        • SSS: Milliseconds in a second; 0-999

        • E: Day of the week; also use either EE, EEE, EEEE, or EEEEE

        • a: Uppercase Ante Meridiem (AM) /Post  Meridiem (PM)

        • Z: Difference to Greenwich time (GMT) in hours and minutes

 

The user can either type the above notations or simply select the required date/time options from the format box.
