Importing Data from SOAP

 

To import data from SOAP Web Service

  1. Under the newly-created dataset, click Web Service.

  2. Web Service dialog appears.

  3. In the Web Service dialog, perform the following:

  4. In the Name box, type the web service name.

  5. In the Description box, type the description for the web service.

  6. From the Web Service Type, select the required radio button from the following:   

    1. REST

    2. SOAP

Web Service Type: SOAP

  1. In the URL box, type required URL, and then click Go.

 URL: http://webservices.oorsprong.org/websamples.countryinfo/CountryInfoService.wso?WSDL

Note: When the user enters the URL, and then clicks Go, the methods from the URL are automatically populated in the Select Method list.

  1. From the Select Method list, select the required option from the available method list, and then click Get Soap Body.

URL: FullCountryInfoAllCountries

The Soap body appears in the Soap Request tab.

  1. In the Result Node box, type the required result node.

Result Node: m:FullCountryInfoAllCountriesResult/m:tCountryInfo

  1. To navigate to the next screen, click Next.

 

  1. Under Columns section, the required columns appear.   

   

 

  1. To import the columns, select the checkbox before the required columns, and then click Import.

  2. The dataset appears in the workspace as shown below:

 

      

  1. To view the data for the required dataset, click the required dataset.

  2. Columns -<object name> section appears with a list of columns.

  3. To view the data for the required columns, click View Data.
