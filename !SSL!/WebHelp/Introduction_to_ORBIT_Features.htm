Introduction to ORBIT Analytics Features

ORBIT Analytics application comprises of the following features:

 

  • Home

Home section contains the following:

    • Full Screen

    • Collaboration Threads

    • Settings

    • Profile

    • Change Password

    • Lock Screen

    • Help

    • Download Addons

    • Logout

    • Favorites

    • Recent

    • Unlock home widget

 

  • Reports

Reports section displays following types of reports

    • Data Report

    • Analysis

    • Dashboards

    • Publish Reports

    • Script Reports

    • Finance

    • Custom Reports

 

  • Jobs

Jobs consists of Seeded jobs, Plugin jobs, and Custom jobs.

 

  • Business Models

Business Models consists of the following:

    • Domains

    • Datasets

    • Data Sources

    • Security

    • Meta Objects

    • Finance Cube

 

  • User Management

User Management section consists of the following:

    • Users

    • Groups

    • Roles

    • Permissions

    • Authorities

 

  • Monitoring

Monitoring section consists of the following:

    • Processes

    • Report Statistics

    • Report Tokens

    • Nodes

 

  • Collaboration

Collaboration section consists of the following:

    • Threads

    • Snapshots

 

  • System

System section consists of the following:

    • Settings

    • Plugins

    • Maintenance

 

  • Business Units

Business Units section enables to create new business units and perform various actions on Business Units.
