Editing Users

To edit users

 

  1. On the left-hand side menu, point to User Management, and then click Users.

  2. The User List appears.

 

 

  1. To edit a user, click icon (highlighted as “1”) under Actions, and then click Edit (highlighted as “2”).

 

 

  1. Edit User <user name> dialog box appears.

  1. Under Edit User <user name> dialog box, perform the following:

    • In the First Name box, type the first name of the user.

    • In the Last Name box, type the last name of the user.

    • In the Email box, type the email of the user.

You cannot edit the User Name.

  1. To save the changes, click Save.

  2. To close the dialog box, click Cancel.
