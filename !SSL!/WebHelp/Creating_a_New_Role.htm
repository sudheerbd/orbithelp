Creating a New Role

To create a new role

  1. On the left-hand side menu, point to  User Management, and then click  Roles.

  2. A list of available roles (highlighted as "1") appears.

 

  1. To create a new role, click New.

  1. New Role dialog box appears.

  1. To create a new role, perform the following:

    • In the Name box, type the name.

    • In the Description box, type the description.

Name and Description are mandatory.

  1. To save the changes, click Save.

  1. To close the dialog box, click Cancel.
