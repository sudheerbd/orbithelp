This topic explains different ways to customize the values of tree map.

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

 You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Value to open Value visual settings.

 

Format Type

Default Value: Auto

Format Type has following options:

  1. Auto: Orbit auto formats values based on the field properties

  2. Number: Formats value axis values as numbers

  3. Currency: Formats value axis values as currency

  4. Percent: Formats value axis values as percentage

 

When Format Type is set as Number, following fields appear.

  1. Display Units

  2. Decimals

  3. Separators

 
Display Units

Default Value: Auto

Display Units has following options:

  1. Auto: Orbit auto formats values

  2. None: Complete number is shown on value axis. No format is applied.

  3. Thousands: Numbers are shown in thousands

  4. Millions: Numbers are shown in millions

  5. Billions: Numbers are shown in billions

  6. Trillions: Numbers are shown in trillions

 
Decimals

Default Value: .#(1)

Decimals has following options:

  1. .#(1): Shows 1 fractional number after decimal point

  2. .##(2): Shows 2 fractional numbers after decimal point

  3. .###(3): Shows 3 fractional numbers after decimal point

  4. .####(4): Shows 4 fractional numbers after decimal point

  5. .#####(5): Shows 5 fractional numbers after decimal point

  6. .######(6): Shows 6 fractional numbers after decimal point

 

Separators

Default Value: 1,234.56

Separators has following options:

  1. 1,234.56: Comma (,) is used as thousands separator and period (.) is used as decimal point

  2. 1.234,56: Period (.) is used as thousands separator and comma (,) is used as decimal point

  3. 1`234.56: Apostrophe (') is used as thousands separator and period (.) is used as decimal point

  4. 1 234,56: Space ( ) is used as thousands separator and comma (,) is used as decimal point

  5. 1 234-56: Space ( ) is used as thousands separator and hyphen (-) is used as decimal point

  6. 1 234.56: Space ( ) is used as thousands separator and period (.) is used as decimal point

  7. 1234.56: No thousands separator and period (.) is used as decimal point

 

Prefix

Enter text in the Prefix box to append before data labels.

 

Suffix

Enter text in the Suffix box to append after data labels.
