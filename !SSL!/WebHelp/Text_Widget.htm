Styled Text Widget

Styled Text widget is used to display the text in Dashboard.

 

To add a styled text widget in the dashboard, perform the following:

 

  1. On the top-menu, from the Add Widget list, select Styled Text widget.

  2. Styled Text widget appears in dashboard as shown below:

     

 

  1. In the Styled Text, a text area appears to enter the required text.

 

Example: Text: Sales by Region

 

 

  1. To format the text, click the tools on tool bar (highlighted as “1”) as shown below:

 

 

  1. For Styles Text widget, the following Visual Settings appears.

    • Image

      1. Title

        • Show: Show the title on styled text widget.

        • None: Hide title on styled text widget

 

The tools in the toolbar are as follows:

 

Tool

Tool Name

Description

Bold

 Displays the text in bold.

Italic

This option is to display the text in italics.

Insert Unordered List

This option creates the bulleted list.

Insert Ordered List

This option creates the ordered list.

Font size

Displays a list of font size to select.

Fill color or set the text color

This is used to set background and text color.

Insert format block

Insets Format tags such as Normal, Heading1, Heading 2 and so on.

Align

This option provides the following:

  - Align Center – This aligns the selected text in center.

  - Align Left – This aligns the selected text to the left.

  - Align Right – This aligns the selected text to the right.

 - Align Justify – This distributes the text evenly between the left and the right margins.

--

This option displays the following:

- Strike through – This option strikes through the text in the text area.

- Underline- This option is to underline the text in the text area.

- Decrease Indent- This option moves the text closed to the left margin of the text area.

- Increase Indent - This option moves the text away from the left margin of the text area.

- Font Family – Displays a list of fonts to pick.

- Insert Horizontal Line - This option inserts a horizontal line.

- Insert Special Character- This option enables inserting special character in the text area.
