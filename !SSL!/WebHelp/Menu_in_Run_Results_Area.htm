Menu in Run Results Area

Field

Description

 Run

To run the report

Save

To save the report.(This option is available in Edit Mode.)

 View Mode/  Edit Mode

In View Mode, you can only view the report, but you cannot edit/save the report.

In Edit Mode, you can edit the report. Edit Mode is available only to report owner. If the report is shared, then you only have View Mode.

Export to Excel xlsx

To export report data to excel. This option is available only in view/Edit mode.

Quick Filters

To access Quick filters. This option is available only in view/Edit mode.  

Hide/Show User Filters

To hide/show User Filters

Show/Hide Insights

To hide/show Insights

Close

To close the reports

Options

Options provides the following options:

  • Save As (Available in Edit/View Mode)

  • Save As New Quick Filter  (Available in Edit/View Mode)

  • Save Current Quick Filter  (Available in Edit/View Mode)

  • Quick Filters (Available in Edit/View Mode)

  • Properties (Available in Edit Mode )

  • Auto Column Size (Available in Edit/View Mode)

  • Export (Available in Edit/View Mode)

  • Excel xlsx

  • With Template

  • Templates (Available in Edit Mode )

  • Conditional Format (Available in Edit Mode )

  • Configure Drill-through (Available in Edit Mode )

  • Group Filters (Available in Edit Mode )

  • Generate Link (Available in Edit/View Mode)

  • Add to Dataset (Available in Edit/View Mode)

  • View SQL (Available in Edit/View Mode)

  • Pre Executive Block (Available in Edit Mode )

  • Delivery Plan (Available in Edit Mode )
