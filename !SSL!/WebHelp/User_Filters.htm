User Filters

  • User filters are parameter-based filters that allow the user to input a value, to filter the data in the report.

  • To view the user filters area, click  Hide/Show User Filters, if it is hidden.

  • From the available columns in the Configurator area, drag and drop the required column in the area above the Configurator and Run results area.

 

  1. The column appear as a parameter (highlighted as “1”) above the Configurator and Run Results area

 

                                              

  1. To view the values for a column, click the user filter.

  2. A list of values for the selected user filter appears.

  2. The selected value appears under the Selected item(s).

  3. To apply the user filter, click Ok.

Two Reports with Same Parameters

 

  • If there are two reports with same parameters, then the parameters in the first report appear in the second report when the first report is open and it appears in the opened reports section, provided the it fulfill the following conditions.

  • The parameters should have the same label or name.

(and)

  • The type of select -- Multi-select or Single-select should match for both the reports

(and)

  • The return type: Both the parameters in two reports should have same return type.
