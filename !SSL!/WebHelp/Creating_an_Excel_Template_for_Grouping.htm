Creating an Excel Template for Grouping

Steps to Create an Excel Template for Grouping

1. Open the Excel Sheet. 

2. In the Excel sheet, enter the required Columns in the Header section as shown in the below figure.

.3. To extract the column values under the required columns, enter the column numbers as ${COL0}, ${COL1}, and so on (highlighted as “2”).

 

4. Now select the first two rows and group them under one band as “Header” in the Name Manager as shown in the figure below.
