Configure Drill-through

To configure drill-through for a dashboard report

 

  1. To view visualization option, click Options (highlighted as "1") on visualization tab.

(OR)

Right-click on the visualization widget.

  1. A list of options appears, on visualization widget as shown below (highlighted as "2").

 

 

  1. To configure drilldown,select Configure Drill-through.

  2. Drill-through Reports dialog box appears.

  1. Under Drill-through Reports dialog box, perform the following:

    • To drag and drop the required drill-through report, click Add drill-through reports.

    • Reports popup window appears as shown below (highlighted as “3”) :

    • Drag and drop the required drill-through report that has the required parameters (highlighted as “4”) under the Drill-through Reports area.

    • Drill-through Column and Navigate To options appear.

    • From the required report, drag and drop the required columns from report Configurator area of the report to the Drill-through Column box (highlighted as “5”).

    • Under Parameters area, the required parameter appears with a list of available values for the parameter.

    • From the Parameters Value box, select the required column from the list (highlighted as “6”).

    • In the Navigate To box (highlighted as “7”), click the required option from the following:

    1. New Tab

    2. Window

    • To save the changes, click Ok.

 

To view the configure drill-through

 

  1. To view the configure drill-through, select the value field (highlighted as “8”) in the visualization.

  2. A list of options appears as shown below:

 

 

  1. To view the drilldown report, select Drill-through option and select the report (highlighted as “9”).

  2. The drill-through report appears as selected in Navigate To box.
