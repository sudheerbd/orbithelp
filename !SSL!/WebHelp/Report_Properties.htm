Report Properties

To view the report properties

 

 1. On the Reports section, right-click the required report, and then click Open.

 2. The report appears in the View Mode.

 2.   3. To switch to edit mode, click Edit Mode.

       4. The report appears in Edit Mode.

       5. In the Edit mode, on the right-hand side, click Options, and then click Properties.

     

            

         

        6.     Properties dialog appears.

        7.   In the Properties dialog, on the right-hand side, under the General section, in the Default Output box, select the required option from the following:

·         CSV

·         Delimiter Text

·         Tabular/JSON

·         XML

·         HTML

·         PDF

·         Excel(xlsx)

 

Default Output: PDF

 

              

     

Delimiter is only applicable when the default output is “Delimiter Text”.         

       8. To save the changes, click Ok.

 

     Verifying the Output Type

 

      1.    In the Reports section, right-click the required report, and then click Schedule.  

      2.  Schedule -Report dialog appears.

      3.  In the Schedule-Report dialog, click Output & Delivery tab.

      4.  Output and Deliveries section appears.

      

     5.    In the Output and Deliveries section, click Add Output.

     6.  Under Output Type, the default output type appears as “PDF”.

This default output “PDF” has been set using report properties
