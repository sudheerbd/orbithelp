Generating Report Link

Orbit allows to use public tokens to show embedded analytic. Public token reports show output with default report owner login.

 

To generate the required report links, perform the following:

 

  1. In the Reports section, select the required folder under My Folders.

  2. A list of reports appears on the right-side of the report section.

  3. Right-click the required report, and then click Generate Link.

 

  1. Generate Report Link dialog appears.

  2. In the Generate Report Link dialog, perform the following:

    • To generate report links, select Navigation from the Link Type list (highlighted as “1”).

Navigation appears by default.

    • The following two options available under Navigation (highlighted as “2”):

      • Private

      • Public

    • Private radio button is selected by default.

 

    • To generate the public link, select the Public radio button, and then click Generate.

    • The public link appears in the Link box.

    • To generate the private link, select the Private radio button, and then click Generate.

    • The private link appears in the Link box.

    • The link appears as follows:

 


• To copy the link, click Copy beside the Link box.
• Paste the private/public link in the desired browser.

  • To enable this option, select the checkbox next to Auto Run.

Downloading Reports

To download the report

 

  1. In the Reports section, select the required folder under My Folders.

  2. A list of reports appears on the right-side of the report section.

  3. Right-click the required report, and then click Generate Link.

  4. In the Generate Report Link dialog, perform the following:

    • To download the report, select Download from the Link Type list (highlighted as “1”).

 

 

    • In the Select Template box, select the required template.

    • In the Select Output box, select the required output type.

The output type depends on the selected template.

    • In the Expiry Date box, type/select the required date.

    • In the Expiry Time box, type/select the required time.

    • To include the parameter value, select Include current parameter values in link checkbox.

    • To generate the script or link for downloading the report, click Generate.

    • In the Link box, the generated link appears.

 

 

• To download the report using the GET method, copy the link from the Link box.


• To copy the script/link, click Copy beside the Script/Link boxes.
• To save the report links, click Done.
• To discard the changes, click Cancel.

Once Done is selected, the generated links (tokens) appear in Monitoring->Report Tokens.

 

Embedding Required Report

 

To embed the required report, perform the following:

 

  1. In the Reports section, select the required folder under My Folders.

  2. A list of reports appears on the right-side of the report section.

  3. Right-click the required report, and then click Generate Link.

  1. Generate Report Link dialog appears.

  2. In the Generate Report Link dialog, perform the following:

    • To embed the report, select the Embed (highlighted as “1”) radio button.

    • The following two options appear (highlighted as “2”):

      • Private

      • Public

    • Private radio button is selected by default.

 

  1. Under Embed, perform the following:

    •  In the Expiry Date box, enter the required date.

(or)

In the Expiry Date box, click to select the required date.

    • In the Expiry Time box, enter the required time.

(or)

In the Expiry Time box, click to select the required time.

If Expiry Date and Expiry Time are not mentioned, the link works, based on application level session time out configuration.

    • To include the parameter values in the link, select Include current parameter values in link checkbox.


a. For Include current parameter values checkbox to be available, the report should be in Edit/View Mode and then navigate to Options->Generate Link.
b. If a user, generating a link with special character parameter values, refer Appendix A.

    • To generate the script and link, click Generate.

    • In the Script box, the generated script appears.

Scripts for POST method appear in the Script box.

    • In the Link box, the generated link appears.

This generated link is a public link, which is the GET method.

 

GET Method

GET: The data/query string (name/value pairs) is sent in the URL itself.

 

http://orbit.qawinlocaltest1.orbit8.com:1030/reporting/externalReportView?portal=Y&token=48a436a5-55fd-406b-9bf9-19ad88fd9441&OH=Y&FR=Y&AR=N

 

 

POST Method

POST: The data sent to the server with POST is stored in the request body of the HTTP request.

Post Method uses script to generate output. Once the script is generated in Orbit, place the following code beneath the script to access the id “divTarget” in the method viewReport(). The following code is to view the report only. This is not for downloading the report.

 

<div class="cnt-div">

 <iframe src="" id="divTarget" name="iframe_a" frameborder="0" style=" width:100%" height="100%" width="1024px"></iframe>

 </div>
