Introduction

When several fields in a business object form a natural hierarchy, it is possible to group them and create a hierarchy drilldown.

For example, a report that shows sales revenue by region can allow the user to select a state within the region, click on it and see sales revenue by city within that state.

Using hierarchy drilldown, you can explore in-depth details about your data in a single chart.

 

List of charts that support grouping and hierarchy drilldown

  • Bar and Column charts

  • Line charts

  • Area charts

  • Pie, Doughnut charts

  • Spiderweb charts

  • Scatter and bubble charts

Charts support the grouping and hierarchy drilldown for dimension (data) fields dropped in the category section - that is, for each chart type, the visual elements in a chart (columns in column charts, pie slices in pie charts, etc.) can act as drilldown hot-spots.

 

Set up Grouping and Hierarchy Drilldown in a report

1. To set up grouping and drilldown, create an analysis report.

2. On the analysis report page, from the selected business object, drag the fields for which you want to enable grouping and hierarchy drilldown into the Category section of the Report Configurator area.

Note: Grouping and hierarchy drilldown works when the Category section of the Report Configurator area has more than one field.

3. Click the expander arrow next to Category in the Report Configurator area and then point to Grouping.

4. Grouping options appear.

5. Select Concatenate Items from Grouping options, to display visual elements in the chart separate from each other.

Note: Concatenate Items is the default selection.

Note: When you select Concatenate Items, it joins string names of the visual elements.

6. Select Group from the Grouping options, to group visual elements.

7. Select Hierarchy Drilldown, to enable hierarchy drilldown. Drill controls appear at the top of the chart.

 

Ways to access the drill features

You have two ways for accessing the drill down, and drill up features for charts that have hierarchies.

  • Use Drill controls and breadcrumb navigation at the top of the chart.

  • Right-click a visual element (like a bar, pie slice, or bubble) to reveal and use the Drill menu.

Drill Controls

/ Enable/Disable drilldown

indicates drilldown is enabled.

indicates drilldown is disabled.

1. Select Enable/Disable drilldown to disable drilldown.

2. Select Enable/Disable drilldown to enable drilldown.

Note: When you disable drilldown, drilldown controls won't work. But Move to hierarchy top level and Breadcrumb navigation menu controls work.

Drill up

Click Drill Up to drill up one level at a time.

 

Drill down all items

Click Drill down all items to drill down all items of the chart one level at a time.

 

Move to hierarchy top level

Click Move to hierarchy top level to drill up to the chart top level at once.

 

 

Drill Menu on the Visual Elements

Right-click on a visual element to reveal Drill menu.

Drill Down

Select Drill Down to drill down selected visual element one level at a time.

Drill down all items

Select Drill Up to drill up one level at a time.
