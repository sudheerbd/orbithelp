Creating a Physical Model

To create a new Physical Model

 

  1. On the newly created domain page, Domain Hierarchy appears.

  2. Under Domain Hierarchy area, Physical Models and Logical Models (highlighted as “1”) appear.

 

                              

  1. To create a new physical model, right-click Physical Models, and then click New Physical Model.

         

                               

  1. New Physical Model dialog box appears.

  1. In the New Physical Model dialog box, perform the following:

    • In the Name box, type the name of the new physical model.

    • In the Description box, type the description for the new physical model.

    • In the Data Source box, select or type the required data source.

    • To create a new data source, click Create new datasource .  See Create new datasource.  

    • To save the changes, click Save.

    • To close the dialog box, click Cancel.

  1. The newly created physical model appears under Physical Models.
