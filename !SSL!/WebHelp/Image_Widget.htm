Image Widget

Image widget is used to display an image in Dashboard.

 

To add a Image widget in the dashboard, perform the following:

 

  1. On the top-menu, from the Add Widget list, select Image widget.

  2. Image widget window appears in dashboard (highlighted as "1") as shown below:

 

 

  1. To upload an image, navigate to Visual Settings next to Data tab.

 

Upload Image from URL:

To Upload Image from URL

  1. Under Visual Settings the following options appears

    • Image

      1. Title

        • Show: Show title on Image widget

        • None: Hide title on Image widget

      1. Upload Type

        • URL: Enter the copied URL, to the URL text box.

 

 

Upload Image from computer:

To Upload Image from computer

  1. Under Visual Settings the following options appears

    • Image

      1. Title

        • Show: Show title on Image widget

        • None: Hide title on Image widget

      1. Upload Type

        • Local: Select the file path, using Upload option.

  1. To upload a image from local machine follow the below steps:

    • Select Local option in Upload Type.

    • Under upload type, File Path appears with Upload button.

    • To upload a image from local machine, click Upload.
