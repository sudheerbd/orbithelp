Editing Groups

To edit group

 

  1. On the User Management menu, click Groups.

  2. Group list (highlighted as "1") appears with a list of available groups.

 

  1. To edit the required group, click icon (highlighted as “2”) under Actions, and then click Edit (highlighted as “3”).

 

  1. Edit Group <group name> dialog box appears.  

 

  1. In Edit Group <group name> dialog box, perform the following:-

    • In the Name box, type a new name for the group.

    • In the Description box, type the description for the group.                                                            

  1. To save the changes, click Save.

  2. To close the dialog box, click Cancel.
