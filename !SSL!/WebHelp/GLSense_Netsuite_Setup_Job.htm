GLSense NetSuite Setup Job

This job primarily extracts all the GL Setup data from NetSuite instance.

To extract GL setup data from NetSuite

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded, Custom and Plugin jobs.

  3. Under Jobs sections, on GLSense Netsuite Setup Job click , and then click Schedule.

  4. Schedule Job dialog box appears.

  5. In the Schedule Job dialog box, on the Parameters tab, perform the following:

    • In the Finance Cubes box, select the required finance cube from the list.

    • In the Ledgers box, select the required ledger.

    • In the Start Date box, Select the required year from the list.

      A finance cube could have multiple years of data. For, better performance it is recommended to extract the data for one or two years.

    • Under the Custom Segments section, click to add custom segments.


      If the user does not add custom segment table names, this job only extracts seeded segments.
      To extract custom segments from NetSuite ERP, the user should enter all the required custom segment table names manually under the Custom Segments section to extract the data.
      Users can add all the custom segments existing in their ERP.

    • If the user enters an incorrect custom segments table name, the job ignores this table as such a table does not exist in the system.

    • Incremental Load: this loads only the latest changed data, instead of loading the entire load from the beginning.

    • Truncate Cube Dimensions removes the existing tables in the cube.

  6. To schedule the job, click Submit.
