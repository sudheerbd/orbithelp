Creating List Filter

To add list filter to a dashboard report:

1. Open a dashboard reports page in Edit Mode.

2. Right click on required fields, click Pick Visualization and then click List Filter

(OR)

Drag the required field from the Business Object into the Run Result area, click the expander arrow in the Report Configurator area and then click List Filter.

3. List Filter Widget appears in the Run Result area.

 

Visual Settings of List Filter

Customize the look of List Filter

Title

Customize the List Filter widget title.

Default Value: None

Auto: When Title is set to Auto, Orbit application generates the title.

Custom: Select Custom to set a custom title to the widget. When you select Custom, a Text box appears where you can enter the title.

None: Select None to hide widget title.

Max Item Count

Limit the number of items to display in List Filter widget.

Default Value: 100

Enter desired number in the Max item count box.

Display Type

Select display type of List Filter widget.

Default Value: List

List: Displays list filter widget as a list.

Drop Down: Displays list filter as a drop down list. Click the expander arrow on the List Filter widget to see list of data filters.

Button List: Displays list filters as a button list.

Multi Selection

Enable/Disable selection of multiple items.

Select Multi Selection check box to enable multi selection.

Clear Multi Selection check box to disable multi selection. Radio buttons appear in the list filter widget when you clear Multi Selection check box.

Enable Search

Enable/Disable search in the List Filter widget.

Select Enable Search check box to enable search in the List Filter widget. Search box appears in the List Filter widget.

Clear Enable Search check box to disable search in the List Filter widget. Search box disappears in the List Filter widget.

Number of Columns

Set number of columns to display on the List Filter widget.

Enter the desired number in the Number of Columns box.

Number of Columns option works only in Vertical layout.

Layout

select the Layout of the Button List

Horizontal: Shows list filters as a horizontal button list.

Vertical: Shows list filters as a vertical button list.

Resize List Filter Widget

Move the cursor to any edge or corner of the widget until a double-headed arrow appears. When this arrow appears, click-and-drag it to make the widget larger or smaller.

Working with List Filter Widget

List Filter widget allows you to filter the data displayed on the chart widgets in the dashboard reports page.

Select any one of the check boxes in the List Filter widget; data on the chart widgets updates.
