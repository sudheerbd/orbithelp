Visual Settings for Auto Visualization

 

Auto visualization creates visualizations based on the fields dragged into report configuration area.

Visualization (Chart type) Options

1. Auto Visualization has the following visualization options

  • Measures

  • Rows

  • Columns

2. Drag and drop the required fields from the selected Business Object into the Measures, Rows, and Columns sections.

Drag fields with numerical values into the Measures section for several types of calculations (Sum, Average, Min, Max, etc.) However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the analysis report dynamically in the run results area.

4. In Auto Visualization, type of visualization is also dynamic and changes based on the fields placed in the Report Configurator area.

Visual Settings for Auto Visualization

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

As visualization is dynamic in Auto Visualization, Visual Settings change based on the type of visualization. See Visual Settings of Analysis Reports for more information.
