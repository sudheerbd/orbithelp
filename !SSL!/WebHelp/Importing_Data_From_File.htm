Importing Data from File

To import data from File

 

  1. Under the newly-created dataset, click File.

  2. Upload File dialog box.

  3. Under Upload File dialog, perform the following:

            a.     In the File Type box, type/select the required option from the following:

                                      i.        CSV or Other Delimited Text Format

                                     ii.        Excel

                                   iii.        Zip

Required File Type: CSV or Other Delimited Text Format

b.     To upload the required file, click Select File.

c.     The path of the selected file appears in the textbox before Select File.

d.     To navigate to the next screen, click Next.  

 

  1. The columns present in the uploaded file appear.

            

 

  1. To import the file, click Import.

  2. The imported file appears as an object (highlighted as “1”) shown below with its associated columns (highlighted as “2”) on the right-hand side.

 

  1. To view the data for the columns, click View Data.
