Assigning Roles to the Specific User

To assign roles to the specific user

  1. Under the Users section, select the required user, and then click  next to Assigned Roles (highlighted as “1”).   

 

By default, for a user, ORBIT USER ROLE appears.

  1. A list of roles (highlighted as “2”) appears.

 

        

                                                

  1. Select the required role from the list.

Required Role: ORBIT User Role

     4. Under Assigned Roles, the selected role appears.

 

 

     5. To save the changes, click Save.
