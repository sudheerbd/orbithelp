GLSense EBS Generate View Script Job

This job generates materialized view or standard view script for the EBS Finance Cubes which has same datasource in both source and target connection.

To Generate EBS Materialized view script

Prerequisite: You must have the EBS Finance Cube.

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded, Custom and Plugin jobs.

  3. Under Jobs sections, on GLSense EBS Generate View Script Job, click , and then click Schedule.

  4. Schedule Job dialog appears.

  5. In the Schedule Job dialog, on Parameters tab, perform the following:

    1. In the Finance Cubes box, select the required finance cube from the list.

Only EBS Finance Cubes are shown in the list.

    1. In the Ledgers box, select the required ledgers.

You can select multiple ledgers.

    1. In the Start Date box, select the required year from the list for each ledger.

A finance cube could have multiple years of data. For, better performance it is recommended to extract the data for one or two years.

    1. Select Materialized View or Standard View from the Generate option.

    2. To execute the script after generating it, select “Execute the generated script in the target connection”.

 

  1. To schedule the job, click Submit.

  2. To cancel the job, click Cancel.
