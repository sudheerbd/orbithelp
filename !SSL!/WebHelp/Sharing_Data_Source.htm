Sharing Data Source

To Share a Data Source

 

  1. In the Data Sources page, a list of available data sources appears.

  2. In the Data Sources page, under Actions, click  against the thread you want to share, and then click Share.

  3. Share < Data Source Name > dialog box appears.

 

 

  1. In the Share <Data Source Name> dialog box, you can do following:

    • Assign roles and/or users to the datasource

    • Delete roles and/or users from the datasource

    • Modify permissions to roles and/or users.

 

To assign roles to the data source

 

  1. To assign a role to the data source, click  Assigned Roles section.

  2. The available roles list appears.

  3. Select a role from the list to assign to the data source.

To search a role, enter the search term in the Search box.From the search results, select the required role.

  1. The selected roles appear under Assigned Roles section.

The selected data source is shared to all the users who have the selected roles

 

 

  1. To save the changes, click Ok.

A Changes done successfully notification appears briefly at the top of the ORBIT application window.

  1. To discard the changes, click Close.

 

 

To assign users to the data source

 

  1. To assign a user to the data source, click   Assigned Users section.

  2. The available users list appears.

  3. Select a user from the list to assign to the data source.

To search a user, enter the search term in the Search box. From the search results, select the required user.

  1. The selected users appear under Assigned Users section.

  2. To save the changes, click Ok.

A Changes done successfully notification appears briefly at the top of the ORBIT application window.

6.   To discard the changes, click Close.

 

 

 

 

To delete roles/users from data source

 

  1.  In the Share <Data Source Name> dialog box, click  Delete against the role/user you want to delete.

  2. To save the changes, click Ok.

A Changes done successfully notification appears briefly at the top of the ORBIT application window.

  1. To discard the changes, click Close.

 

Permissions

  • Roles/users could be given permissions to edit and/or delete data sources assigned to them.

  • Following permissions can be assigned to roles/users

 

Option

Description

Edit

Users can edit data sources assigned to them.

Delete

Users can delete data sources assigned to them.

 

 

 

To modify permissions

 

  1. In the Share <Data Source Name> dialog box, select check boxes under Edit and/or Delete, against the role/user you want to grant permissions.

  2. In the Share <Data Source Name> dialog box, clear check boxes under Edit and/or Delete, against the role/user you want to revoke permissions.

  3. To save the changes, click Ok.

A Changes done successfully notification appears briefly at the top of the ORBIT application window.

  1. To discard the changes, click Close.
