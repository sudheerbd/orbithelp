Finance Reports imported from EBS FSG

To work with imported FSG reports

Once GLSense Import EBS FSG Report Job scheduled with the required FSG Report, the imported reports appear under Imported FSG Report folder in My Folders section.   

Note: By default, Imported FSG Report folder created once we schedule the GLSense Import EBS FSG Report Job with FSG Report.

  1. To view the imported FSG reports, click on Imported FSG Reports folder.

  2. Imported report appears as shown below:

  1. To edit the report, right-click on the report and click Edit.

  2. Report appears with Name, Description, Cube Name and Template file.

  3. To update any changes in the existing template, click on the template.

  1. Template downloads successfully to the local machine as shown below:

  1. To view the report data, login to Orbit GLsense add-on.

  1. Once user login to the application, select the required Cubes and the related Ledgers (highlighted as ‘1’) to execute the report.

Note: The parameter values (highlighted as ‘2’) appear in the sheet, are the values where user given in the submitted job.

  1. To view the formula, select the required cell and the formula appears under function field fx (highlighted as ‘3’).

  1. To view the report output, click Sheet under calculate section.

  1. To upload modified /updated template, user can remove existing template and upload the updated template.

  1. To Schedule the saved report, refer Schedule a Finance Report.
