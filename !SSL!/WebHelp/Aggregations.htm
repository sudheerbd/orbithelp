Aggregations

The following aggregations can be performed on report columns:

  • Sum - this is an aggregate function where the sum of expression is assessed over a set of rows.

  • Avg - this is an aggregate function where the average of an expression is assessed over a set of rows.

  • Count: this is an aggregate function that counts the number of rows in an expression.

  • Distinct Count :This eliminates the repeated appearance of the same data.

  • Min: this function returns the minimum value of an expression.

  • Max: this function returns the maximum value of an expression.  

  • Standard Deviation:  this is a measure of the spread of scores within a set of data.  StdDev is the Standard  Deviation -Sample and StdDevp is the standard deviation- population. It is the square root of variance.

  • Variance:  this is the average of squared differences from the Mean.  Variance is of two types: Var and Varp, where Var is the Variance sample and Varp is the Variance population.  

 

To View the Aggregate Functions

 

1. In the Configurator tab, drag and drop the required columns.

2. To apply aggregate function, select the required column and the click Aggregation.

3.  In the Aggregation menu, the following options appear:

  • Sum

  • Avg

  • Count

  • Distinct Count

  • Min

  • Max

  • StdDev

  • StdDevp

  • Variance

  • Variancep
