Create a Business Object for a Column

To create a Business Object for a Column

 

  1. For the selected domain, under Domain Hierarchy, Physical Models folder appears.

  2. Under Physical Models folder, import the required columns.

  3. Right-click the required column, and then click Create Business Object.

 

 

 

  1. Create Business Object dialog appears.

 

  1. To  create a new business object, click Yes.

  2. To close the dialog, click  or No.
