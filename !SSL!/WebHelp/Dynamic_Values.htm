Dynamic Values

1. To pick up the default values from the query, click Select Default Value(s), and then click Dynamic values.

2. Choose Value dialog appears with the following options:

      • Static

      • Expression

      • Query

              

 

By default, Static option is selected.

 

Static

 

To choose the Static Value, perform the following:

1. On the Values section, click  Add.

2. Textboxes to enter Values and Display Value appear.

3. In the Value box, type the required value for the selected parameter.

Value: Central

 4. In the Display Value box, type the required value that must be displayed as the default values in the report.

Display Value : Central

 5. To save the changes, click Ok.

 

 

6. The values appear as the default values.
