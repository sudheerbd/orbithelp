Importing Data from REST

To import data from REST Web Service

  1. Under the newly-created dataset, click Web Service.

  2. Web Service dialog appears.

  3. In the Web Service dialog, perform the following:

  4. In the Name box, type the Web Service name.

  5. In the Description box, type the description for the web service.

  6. From the Web Service Type, select the required radio button from the following:

    1. REST

    2. SOAP

Web Service Type: REST

  1. In the Request Method list, select the required method from the following: 

    • GET

    • POST

  1. In the URL box, type the request URL of the webservice.

URL: http://hostname:portname/get/all

  1. In the Result Node box, type the required result node.

Result Node: /

  1. In the Response Type, select the required response type:

·         XML

·         JSON

Response Type: JSON

  1.  On the Parameters tab, select the required radio button:

    1. FORM

    2. RAW

By default, Parameters and Header tabs have FORM radio button in selected mode. In case, there is no need to select the Parameters/Headers as FORM/RAW, then click Next.

  

 

  1. To navigate to the next screen, click Next.

 

  1. To import the fields, click Import.

  2. The dataset appears in the workspace.

  3. To view the columns on the right-hand side, click the required dataset.

  4. The columns appear under Columns- {Webservice name}.
