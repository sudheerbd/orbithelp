View SQL

To view the SQL Query

 

1.  From the Select Business Object area, drag and drop the required columns into the report configurator area.

2.  The report automatically appears in the Run Results area, if Auto Refresh is selected in Query Settings.

       

            

3.  If Auto Refresh is not selected in Query Settings, click Apply to run the report.      

       

4. To view the SQL Query for a given report, click Options, and then click View SQL (highlighted as “1”).

5.  Query box opens with SQL query for the given report.

                                      

6.  To close the Query box, click Close.
