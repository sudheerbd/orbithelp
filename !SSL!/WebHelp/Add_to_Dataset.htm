Add to Dataset

Adding report data as a new Dataset

 

  1. Select a required report.

  1. To create dataset from the report, click Options and then click Add to Dataset.

 

 

 

Note: Add to Dataset option available in report view mode and edit Mode

  1. Add to dataset dialog appears:

  1. In Add to dataset you will have two option:

    • New Dataset

    • Existing Dataset

 

To create a New Dataset

 

  1. In the New Dataset , perform the following:

    1. In the Name box, type the required name.       

     Note: Name is a mandatory field.

    1. In the Description box, type the required description.

    2. To save the details, click OK.

 

  1. Once dataset details saved, report page navigates to Dataset window as shown below:

 

Note: Before using the above dataset, make sure the dataset is Schedule Refresh.

  1. Using the above dataset  you can create reports.

 

 

 

To create a Existing Dataset

 

  1. If you want to add dataset to existing, select Existing Dataset radio button.

  2. A list of datasets appears with Name and Description.

 

  1. To add the dataset, select the required dataset and click OK.

 

  1. Report page navigates to Dataset window as shown below:

 

  1. Using the above dataset, you can create reports.
