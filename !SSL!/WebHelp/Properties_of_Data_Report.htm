Properties of Data Report

To set the Properties 

 

1.      Under Reports area, right-click the required report, and then click Edit/Run.

Properties appear only in Edit Mode

2.      To set the properties, click   Options menu, and then click Properties (highlighted as “1”) above the Run Results area.

 

         

 

    • Alternatively, in the report configurator area, under Configurator, click   Settings on the required column, and then click Properties.

 

3.      Properties dialog window appears.  

4.      On the left-hand side, the selected report name (highlighted as “1”) and the columns (highlighted as “2”) under it appear.

 

  

 

5.      On the right-hand side, the properties of report name and the columns appear.  

6.      For the selected report (highlighted as “3”), the following properties (highlighted as “4”) appear:

a.   Under Name, type the name of the business object. 

b.    Under Description, type the description of the business object.

c. Under the Report Run Type tab, in the Run Type box, type/select the required option from the following:

      • Schedule Only

      • Run & Schedule (BOTH)

 

  

 


In the Run Type box, when Schedule Only is selected, the user can only submit the report as a process and view the output. The user cannot run the report.
In the Run Type box, when Run & Schedule (BOTH) option is selected, the user can submit the submit the report as a process, and can as well run the report.

  

7.      For the required Attributes/Dimensions/Measures (highlighted as “5”), the following properties (highlighted as “6”) appear:

 

      1. Under Name, type the name of the attributes/dimensions/measures.

      2. Under Description, type the description of attributes/dimensions/measures.

      3. Under Column Header tab, perform the following:

        1. In the Font Family box, type/select the font family.

          1. In the Font Size box, type/select the font size.

          2. In the Font Style, select the required checkbox from the following:

            • Italic

            • Bold

            • Underline

          1. To set the color for the underline, click the required color from the color palette.

          2. Under Preview area, the preview of the selected changes appears.

          3. To save the changes, click Ok.

        2. In Wrap list, click one of the following options:

          • True

          • False

      4. Under Custom tab, perform the following:

        1. In the Client Formula box, type the required text in client formula dialog.

      5. Under Grouping & Summary tab, perform the following:

        1. In Grand Total list, click one of the following options:

          • True

          • False

Set this option on numeric column. Calculates grand total and shows values.

        1. In Group Total list, click one of the following options:

          • True

          • False

Set this option on numeric column. Calculates grouping values.

        1. In Grouping list, click one of the following options:

          • True

          • False

Set this option on grouping column


If the column property is True for any of the grouping property
        i. Grand Total
       ii. Group Total
      iii. Grouping
The report is consider as grouping report.

        1. In the Summary list, click any options from the following:

          • SUM

          • AVG

          • MIN

          • MAX

If it is a dimension, then only COUNT aggregation is available.

      1. Under Page Filter tab, perform the following:

        1. In the Default Value box, enter the default value.

        2. In the Multi Select list, click anyone of the following:

          • True

          • False

        3. In the Page Filter list, click anyone of the following:

          • True

          • False

      2. Under Sub Total (Multilevel grouping) tab, perform the following:

        1. In the Report Group list, click anyone of the following:

          • True

          • False

      3. Under Format tab, perform the following:

        1. In Display Text, enter the required values to display. See Using Display Text in Grouping

        2. In the Format box, type the format of attribute/measure/column.

        3. In the HTML Encoding box, type/select the required option from the following:

          • True

          • False

        4. In the Null Display box, type the required text.

        1. In the Round Decimal value box, type/select the required option from the following:

          • True

          • False

        2. In the Sort Column Order box, type/click  to enter the column order.

        3. In the Special Character Handle box, type/select the required option from the following:

          • True

          • False

      1. Under Style tab, perform the following:

        1. In the Alignment box, type/select the required option from the following:

          • Left

          • Center

          • Right

        1. In the BackGround Color box, type/select  the background color.

        2. In the Cell Wrap box, type/select the required option from the following:

          • True

          • False

        1. In the Column Width box, type/click  to enter the column width.

        2. In the Font Family box, type the font family. (OR) Select  to perform the following:

          1. In the Font Family box, type/select the font family.

          2. In the Font Size box, type/select the font size.

          3. the Font Style, select the required checkbox from the following:

            1. Italic

            • Bold

            • Underline

          4. In set the color for the underline, click the required color from the color palette.

          5. Under Preview area, the preview of the selected changes appears.

          6. To save the changes, click Ok.

        3. In the Hidden box, select the required option from the following, to hide the dimension/attribute/measure

          • True

          • False

    1. To save the changes, click OK.

    2. To close the dialog box, click Cancel.
