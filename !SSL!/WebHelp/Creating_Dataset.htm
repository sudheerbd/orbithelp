Creating Dataset

To create a new dataset

  1. On the left-hand side menu, point to  , and then click Datasets.

  2. Datasets page appears with a list of available datasets.

  3. To create a new dataset, click (on the right-hand side of the dataset window).

  1. New Dataset box appears.

 

  1. In the New Dataset box, perform the following:

    1. In the Name box, type the required name.       

Name is a mandatory field.

    1. In the Description box, type the required description.

               c. To save the changes, click Save.

 

This opens the newly created dataset.

  1. To import data from file, click File .

  2. To import data from web service, click  Web Service.

  3. To import data from SQL Query, click   SQL Query.
