Schedules

In the section, you can see the created schedule processes.

 

Scheduled Processes contain the following fields:

 

S.no

Fields

Description

1.       

Schedule Id

This describes the Schedule id 

2.       

Name

This describes the Report name

3.       

Status

This describes the status of the process.

4.       

Start Time

This describes the start time of the process.

5.       

End Time

This describes the end time of the process.

6.       

 Next Fire Time

This describes the next fire time of the process.

      7.

Submitted By

 This provides the details of the owner
