Managing the Collaborators

 

To manage the collaborators

1.      In the Threads section, under Actions, click  against the thread you want to manage the collaborators, and then click Collaborators.

 

 

         2. The Manage Collaborators- <Thread Name> dialog box appears.

      Note: By default, the username of user who created the thread appears under Selected item(s) list.

 

                                  

         3. To add collaborators to the thread, do one of the following:

    • Double-click the username from the left-hand side.

(or)

    • Drag the username from the left-hand side to the Selected item(s) list.

Note: Filter usernames by typing the first few alphabets in Search box and then double-click or drag the required username.

    4. To add multiple collaborators to the thread, hold Ctrl, select the usernames, and then drag to the Selected item(s) list.

Note: You can use shift key instead of Ctrl key if you are selecting usernames adjacent to each other.

5. To add all the available collaborators to the thread, select Cntrl+A and drag the usernames from the left-hand side to the Selected item(s) list.

The selected user appears under Selected item(s) list.

 

        

4. To remove collaborators from the thread do the following:

  • Under Selected item(s) list, click Remove this collaborator  next to the collaborator you want to remove.

        

 5. To save the changes, click Ok.

Note: A Thread collaborators update processed notification appears briefly at the top of the ORBIT application window.

         6. To discard the changes, click Cancel.
