Save Current Quick Filter

To save the report as new search

  1. Open the required report in run results area.

Example: Required Report: Report Doc

Note:

  • The report here has the parameter "Region"  and the parameter values "East" and "Mountain".

  • Save Current Quick Filter appears greyed by default; it becomes available only when an already created quick filter is selected.

  • If you want to make changes to the existing parameter, and save the quick filter with same name, click Save Current Quick Filter on the Options menu.

  1. On the top right-hand corner of the run results area, click Options, and then click Save Current Quick Filter (highlighted as “1”).

    • By selecting Save Current Quick Filter, the changes are made to the report which is currently selected from the Quick Filter list.

    • In the below report, the parameter value is changed to " Mountain"  from "Mountain, East" and this is saved with same quick filter name (Mountain and East Zones) using. Save Current Quick Filter option. The report changes and appears as follows:

          

  1. A success  message appears "Report view is saved successfully".
