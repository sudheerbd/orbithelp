Assigning a Snapshot to the Thread

To Assign a Snapshot to the Thread

 

1.    Under Threads, select the required thread.

2.    In the Snapshots section, click    Search for items to add to view the available snapshots.

 

 

3.    A list of available snapshots appears.

 

 

      4.    Select the snapshots from the list, to add to the selected thread.

 

 

      5.    Under Snapshots section, the selected snapshot appears.

      6.    To save the changes, click Save.

Note: A Changes saved successfully notification appears briefly at the top of the ORBIT application window.
