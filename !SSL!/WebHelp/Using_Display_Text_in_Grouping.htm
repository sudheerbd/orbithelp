Using Display Text in Grouping

Grouping is available in the following two modes:

  • Single
  • Multi

 

Configuring the Grouping Mode

To configure the type of grouping mode  

1. In the Report Configurator area, click Widget Settings tab.

2. In the Widget Settings tab, under the Tabular section, in the Grouping Mode box, type/select the following options:

  • Multi
  • Single

3. To apply the changes, click Apply, and then click Save.

 

Display Text Option in Multi Grouping Mode

Once grouping mode is set to "Multi", now set the display text for the required column. For example, consider the following example where the sum of sales are displayed region wise.

So, the first level of grouping is done on "Region" column, then on "state" column.

 

1. In the Configurator tab, under COLUMNS, click Settings on the required column, and then click Properties.  

Required Column: Sales.

2. Properties dialog appears.

3. In the Properties dialog, on the right-hand side, under Grouping & Summary section, in the Display Text box, enter the required display text.

Required Display Text: {aggr} of {value} {label}

Note:

  • Here, value, label, and aggregation are dynamic values which must be present between the flower braces {}. Eg. {value}, {label}, {aggr}

  • label represents the column name, value represents the column value, and aggregation represents the aggregation.

For example, as shown in the figure below "SUM" is the aggregation, "Central"/"Texas" is the value and "Sales" is the label.

 

       

 

4. To save the changes, click Ok.

 

Display Text Option in Single Grouping Mode

 

Once grouping mode is set to "Single", now set the display text for the required column.

 

Since, the grouping is set to "Single" mode, you can perform the grouping on only one column, i.e "Region".

For example, consider the following example where the sum of sales are displayed region wise.

1. In the Configurator tab, under COLUMNS, click Settings on the required column, and then click Properties.  

Required Column: Region

2. Properties dialog (highlighted as "1") appears.

3. In the Properties dialog, on the right-hand side, under Grouping & Summary section, in the Display Text box, enter the required display text.

Required Display Text: {aggr} of {value} {label}

 

            

 

  • Here, value,label, aggr are dynamic values which must be present between the flower braces {}.

{value}, {label}, and {aggr}

  • label represents the column name, value represents the column value, and aggr represents the aggregation. For example, as shown in the above figure, "sum" is the aggr, "Central" is the value on which the grouping is done , "Sales" is the label.
