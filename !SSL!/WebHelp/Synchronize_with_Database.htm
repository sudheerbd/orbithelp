Synchronize with Database

This feature is useful when there is a change in the database and that has to reflect in the database object in ORBIT application.

 

To Synchronize with Database

 

  1. For the selected domain, under Domain Hierarchy, Physical Models folder appears.

  2. Under Physical Models folder, import the required columns.

  3. Right-click the required column, and then click Synchronize with Database.

  4. Synchronize with Database dialog appears.

 

  

 

  1. To synchronize with database, click Yes.

  2. To close the dialog, click  or No.

 

This synchronizes with any new changes in the database for the selected database object.
