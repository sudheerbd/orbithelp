Splitting Date Dimensions

An available date dimension is split into various date dimensions.

For example, if order date is the only available date dimension in the report and you need to create a report year and quarter-wise, then you can split the order date into multiple date dimensions, including calendar year, month, quarter, and so on.

To split the date dimensions

  1. Under Data, right-click a date dimension, and then click Split Date Dimensions.

  2. Date Dimensions dialog box appears with the following options:

  3. Select split type from Table Based or Expression Based.

  4. Select the required date dimensions, and then click Ok.

  5. The selected dimensions in Date Dimensions dialog box appear under the selected date dimension with <Date Dimension Name> Hierarchy.
