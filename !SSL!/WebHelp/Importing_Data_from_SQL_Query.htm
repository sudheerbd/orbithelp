Importing Data from SQL Query

 

To import data from SQL Query

  1. Under the newly-created dataset, click SQL Query.

  2. From Query dialog box appears.

  3. In the From Query dialog, perform the following:

  4. In the Data Source box, type/select the required data source.

  5. In the Name box, enter the data source name.

  6. In Description box, type the required box.  

  7. In the Query box, type the required query.

Creating object from SQL requires schema name in query to build appropriate object

Select * from AP.AP_INVOICES_ALL

 

  1. To import the data, click OK.

  2. To cancel the changes, click Cancel.

  3. The object appears on the work area as shown below:

 

  

 

To view the columns for the required object (highlighted as "1"), click once; the associated columns (highlighted as "2") appear on the right-hand side.

   

In case, any columns are not required, uncheck the checkbox before columns that must be removed.  

  1. To view the data for the columns, click View Data (highlighted as "3").

Data of only selected columns appears in Query Results.

  1. To refresh the columns, click Refresh.

  2. To save the changes, click Save
