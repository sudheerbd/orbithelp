Properties for Tables/Views/Synonym

For the required table/view/column, perform the following:

  1. Under the Name field, type the name of the table/view/synonym.

  2. Under the Description field, type the description for the table/view/synonym.

  3. Under the Table tab, perform the following:

    1. In the Table Type box, type/select the required option.

      • Dimension

      • Fact

      • Other

    2. Target Schema is automatically picked up from the data source.

    3. In the Target Table box, type the target table or select the target table textbox, and then click .

    4. In the Target Table Type box, type/select the required option.

      • Table

      • View

      • Synonym

      • Inline_SQL

    5. Under the Miscellaneous section, perform the following:

      • In the Relative Size box, enter/edit the required relative size.

Note: Relative size can be assigned at the Physical Model Level, logical model level and in the logical table at Business object level.

  1. Under Refresh Options section, perform the following:

    1. In the Refresh File Pattern box, select the required option:

      • Recently Uploaded

      • Sort By File Name

    2. In the Refresh Type box, select the required option:

      • Full Load

      • Incremental
