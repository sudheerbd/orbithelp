Bursting for File

To perform bursting for File

 

1. In the Delivery Plans dialog, perform the following:

a.  In the Plan Name box, type the required delivery plan name.

Plan Name: Bursting

b. In the Description box, type the required description.

Description is an optional field.

c. In the Delivery Type box, select the required delivery type from the following:

    • EMAIL

    • FTP

    • FILE

Required Delivery Type: FILE

d. To maintain the history of the delivery plan, select the Maintain History checkbox.

By default, Maintain History checkbox is selected.

e. In the Split/Delivery By box, drag the required field from the business object section.

Since, bursting is done region-wise, drag Region from the business object section.

Split/Delivery By: Region

f.  In the Report Mapping Field list, select the required field that needs to be mapped to the field with which the bursting is done.

Since, bursting is done region-wise, and “Sample-Superstore”. “Region” appears in the Split/Delivery By box, the Report Mapping Field must be mapped appropriately.

Split/Delivery by and Report Mapping fields are mandatory fields.

Report Mapping Field: Region

 

 

       2. Under File tab, perform the following:

  • In the Directory box, drag and drop the required field from the business object or type the required directory by clicking .

Directory: “C:\File”

  • In the File Name box, drag and drop the required field from the business object or type the required file name by clicking .

File Name: “File Bursting”

  • From the Zip list, select the required option from the following:

    • False

    • True

Zip: True

  • In the Password box, drag and drop the required field from the business object or type the required password by clicking .

Password protection is applicable only for PDF, Excel and Word.

3. To save the changes, click Save.

4. To cancel any updates, click Cancel.
