Validate Condition

Validate Condition: This allows conditional execution of an occurrence of a job. If no data is returned, then the occurrence of the job is skipped.

 

To Work with Validate Condition

 

  1. In the Schedule -Report dialog, on the Schedule tab, select the Validate condition check box.

Once the Validate condition checkbox is selected, a text box appears (highlighted as “1”).

 

 

  1. To build a query, click Build Query.

  2. Query Builder dialog appears.        

 

  1. In the Query Builder dialog, perform the following:

  • In the Select Business Object box, click   Choose business object.

  • Search Business Object dialog appears.

  • In the Search Business Object dialog, in the textbox box, type the required search term.

 

               

  • A list of search items appears.

  • From the list of items, double-click the required business object or select the required business object, and then click Ok.

  • The selected business object appears with columns under it as shown below:

 

  • From the available columns, drag and drop the required columns.

  • In the data filters section, enter the required filter criteria.

   

  • In the Query section, click Show results.

  • The query returns the following results, with the filter criterion applied.

 

 

  • To close the Query Results dialog, click or  Close dialog.

In the above example, the filter criterion returns the region containing “w” in Query Results dialog.

  • To save the changes in the Query Builder dialog, click Ok.

  • The query appears as shown below:

 

 


When Validate condition is selected, and when the report returns any data, only then the submitted process becomes successful.
In case the report does not return any data, then the submitted process becomes unsuccessful.
