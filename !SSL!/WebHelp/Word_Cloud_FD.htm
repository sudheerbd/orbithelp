Visualization (Chart type) Options

Visual Settings for Word Cloud

 

Word cloud displays text data, typically used to depict keywords, or to visualize free form text. Words are usually single words, and the importance of each word is shown with font size or color.

Visualization (Chart type) Options

1. Word Cloud has the following visualization options

  • Values (Size)

  • Category

  • Color

2. Drag and drop the required fields from the selected Business Object into the Values (Size), Category, and Color sections.

3. Orbit application generates the packed bubbles analysis report dynamically in the run results area.

 

Visual Settings for Word Cloud

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for Word Cloud has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Series: See Customize Slice Visual Settings for more details.
  3. Value: See Customize Series Visual Settings for more details.
  4. Legend: See Customize Legend Visual Settings for more details.
  5. Tooltip: See Customize Tooltip Visual Settings for more details.
