Exporting Analysis Report

To export Analysis Report

  1. On the top of the Run Results area, click Options (highlighted as “1”), point to Export (highlighted as “2”), and then click any one of the following options (highlighted as “3”):

    Report Export option is available in both Edit and View mode.

    • PDF

    • Image

    • HTML

    • With Template

  2. When you select export option as PDF, Image, or HTML a notification "Process submitted successfully. Process Id=>XXXX" appears briefly at the top of the ORBIT application window.

    XXXX is process id number.

  3. Once the process is complete a notification "<Report Name> <Export option> export is completed successfully. Please check in downloads" appears briefly at the top of the ORBIT application window.

  4. When you select Template option, Export With Template dialog box appears with the following two fields:

    • Select Template

    • Select Output

  5. Select Template field displays templates that have been uploaded.

  6. Select Output field displays multiple outputs such as the following:

    • CSV

    • Text

    • Tabular/JSON

    • XML

    • IMAGE

    • HTML

    • PDF

    • Excel (xlsx)

    • Excel (xls)

  7. Click Export to start export process.
