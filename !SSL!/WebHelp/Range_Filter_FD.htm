Creating Range Filter

To add range filter to a dashboard report:

1. Open a dashboard reports page in Edit Mode.

2. Right click on required fields, click Pick Visualization and then click Range Filter

(OR)

Drag the required field from the Business Object into the Run Result area, click the expander arrow in the Report Configurator area and then click Range Filter.

3. Range Filter Widget appears in the Run Result area.

 

Visual Settings of Range Filter

Customize the Range Filter

Title

Customize the Range Filter widget title.

Default Value: None

Auto: When Title is set to Auto, Orbit application generates the title.

Custom: Select Custom to set a custom title to the widget. When you select Custom, a Text box appears where you can enter the title.

None: Select None to hide widget title.
