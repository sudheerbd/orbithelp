Deliveries option currently provides FTP (File Transfer Protocol) connection configuration to deliver the files.

Creating a new FTP Connection

To create an FTP delivery option:

  1. On the left-hand side menu, point to System, and then click Settings.

  2. Settings page appears.

  3. In the Settings page, click the Deliveries tab.

  4. Deliveries tab opens with FTP configuration options.

  5. On the FTP tab, click New to create a new FTP connection.

  6. New FTP Configuration dialog box appears.

  7. In the New FTP Configuration dialog box, perform the following to configure the FTP connection.

    1. In the Name box, enter the FTP connection name.

      Example: Demo FTP

    2. In the Server Name box, enter the server name.

      Example: DemoFTP1

    3. In the User Name box, enter the username.

      Example: FTPuser

    4. In the Password box, enter the password.

    5. In the Port box, enter the port number.

      Example: 21

      Note:
      The default port for FTP is 21.
      The default for Secure FTP (SFTP) is 22.

    6. In the Remote Directory box, enter the remote directory.

      Example: \test

    7. To enable the connection as secured, select the Secured check box.

    8. Select the desired check boxes against Delivery Config to customize the file names in FTP delivery.

    9. To save the FTP connection, click Ok.

      Note: A 'Changes saved successfully' notification appears at the top of the ORBIT application window.

  8. The saved FTP connection appears as follows:

  9. To test the FTP connection, perform the following:

    1. Select the required FTP connection and then click Test FTP Connection.

    2. If the FTP configuration is valid, a notification ' FTP Connection established successfully ' appears at the top of the ORBIT application window.

    3. If the FTP configuration has any issues, an error message 'Error occurred connecting to FTP server' appears.

    4. Edit the FTP connection if necessary. Refer to Editing an FTP Connection for more details.

  10. To make an FTP connection as default, select the required FTP connection and then click Make As Default.

  11. A check mark appears under the Default column for the required FTP connection.

 

Deleting an FTP Connection

To delete an FTP connection:

  1. Click icon under the Actions column against the required FTP connection and then click Delete.

  2. Delete dialog box appears.

  3. Click Yes in the Delete dialog box to delete the FTP connection.

 

Editing an FTP Connection

To edit an existing FTP connection:

  1. Click icon under the Actions column against the required FTP connection and then click Edit.

  2. Edit - <FTP Name> dialog box appears.

  3. Modify required fields and click Ok.

 

Configuring FTP Delivery

To configure the FTP delivery:

  1. Navigate to the Reports page and select a report you want to schedule.

  2. Right-click the required report, and then click Schedule

    OR

    For the selected report, click icon, and then click Schedule.

  3. Schedule - Report dialog box appears.

  4. In the Schedule -Report dialog box, perform the following:

    1. On the Output & Delivery tab, in the Outputs and Deliveries section, click Add output (highlighted as "1") to create a new output type.

    2. A new Output and Deliveries section appears.

      Note: Deliveries section is permission-based. If Deliveries is unavailable under Outputs and Deliveries, navigate to, User Management > Permissions, and then assign the permission Allow Delivery under Reports to the desired role.

    3. Under the Deliveries section, Delivery Type and Delivery To options appear.

    4. To create a new delivery type, click  Add delivery.

    5. A new delivery option appears.

    6. Click under 'Delivery Type' and select FTP from the drop-down list.

    7. To select the FTP connection, click under 'Delivery To.'

    8. FTP dialog box appears.

    9. In the FTP dialog box, perform the following:

      1. In the FTP Server box, click icon to see the list of available FTP connections.

      2. Select the required FTP option from the drop-down list.

      3. In the File Name box, enter the required file name.

      4. To customize the file name, select Customize file name check box.

      5. By default, Date, Time, and Random String options are not available and appear dimmed.

        Check marks appear beside Date, Time, and Random String options if the user selected them while configuring the FTP connection.

      6. When a user selects Customize file name check box, Date, Time, and Random String options become available.

        Note:
        a. Formats of date and time are java convention based.
        b. Avoid spaces in file name and formats.
        c. Random String is alphanumeric in lowercase with six char length.

      8. To save the changes, click Ok.

    10. The path of the selected FTP connection appears in the FTP box.

    11. To submit the process, click Submit.
