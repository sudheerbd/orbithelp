Setting Data Security

To set the Data Security in the Logical Model

1. Under Domain Hierarchy, in the Logical Models folder a list of logical models appears.

2. To set data security in a logical model:

Right-click the <Logical Model Name>, and then click Data Security.

Or

Click the arrow next to the <Logical Model Name>, and then click Data Security.

 

 

3. Data Security- <Logical Model Name> dialog box appears.

4. Data Security has the following three options:

  • No Security

  • Global

  • Role Based

  • Initialization Block

 

To Enable security, select Global, Role Bases or Initialization Block

 

NO Security

  • By default, the data security is selected as No security

  • When No security is applied the complete data in the reports is visible to every user/role.
