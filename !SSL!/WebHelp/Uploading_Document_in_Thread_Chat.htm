Uploading Document in Thread Chat

 

To upload a Document in Thread Chat

 

1. In the chat window of the selected thread, click  in the textbox at the bottom of the window.

2. A menu appears with the following options:

a.    Snapshot

b.    Document

c.    Voting Request

3.    To upload a document, click Document.

 

4.    Document upload dialog box appears.

 

 

5.    In the Document upload dialog box, do the following:

a.    In the Description box, type the description for the document.

b.    To upload a document, click Upload next to the document box.

c.     In the Open dialog box that appears, browse your computer, select the desired file, and then click Open.

d.     In the Document box, the path of the document appears.

 

       

 

e.    To save the changes, click Save.

Note: A document saved successfully notification appears briefly at the top of the ORBIT application window.

f.   To discard the changes, click Cancel.

 

6.    The uploaded document appears in the thread chat window as shown below:

 

      

7. To Refresh the thread chat, click  Refresh.
