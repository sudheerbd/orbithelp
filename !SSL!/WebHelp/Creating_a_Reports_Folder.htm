Creating a Reports Folder

 

To create a Reports Folder

1.      To save a report, click Save.

2.      Save Report dialog box appears.                                           

3.      Under the Folders area, right-click the Folders, and then click New Folder.

4.      New Folder dialog box appears.

 

                                    

5.      To create a new folder, perform the following:

a.       In the Name box, type the name of the folder.

b.      In the Description box, type the description for the folder.

c.       To save the changes, click Save.

Created folder appears under Folder section.

d.      To close the dialog box, click Cancel.

6.      By selecting the newly created folder (highlighted as “1”), the Name box appears on the right-hand side.

7.      In the Name box, type the report name.

 

                                     

8.      In the Description box, type the description, and then click Save.
