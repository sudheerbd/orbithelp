Using Calculation Formula

To use Calculation Formula, perform the following

  1. In any required box, click Select value.

  2. Calculation Formula dialog box appears.

  3. In the Name box, type the required name.

    Required Name: test4

  4. In the Datatype box, select the required option from the following:

    • Number

    • Text

    • Date

    • Integer

  5. In the expression box, type the required expression.

    Required expression: “Test Email”

    The expression such as “subject”/"body"/ “email”/ “cc” / “bcc” should be in double quotes.

    The above figure demonstrates the subject field. Here, the subject is entered in between double quotes.

  6. To validate the expression, click Evaluate Expression.

  7. To save the changes, click Ok.

  8. The expression appears in the required field box.
