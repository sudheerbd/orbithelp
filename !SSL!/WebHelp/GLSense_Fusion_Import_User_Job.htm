GLSense Fusion Import User Job

This job primarily imports the users, roles and groups from Fusion.

Prerequisite: You must have the required authority, user credentials and an excel file to upload.

Sample excel file will be available in GLSense Fusion Import User Job schedule dialog.

 

To import user roles from Fusion

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded, Custom and Plugin jobs.

  3. Under Jobs sections, on GLSense Fusion Import User Job click , and then click Schedule.

  4. Schedule Job dialog appears.

  5. In the Schedule Job dialog, on the Parameters tab, perform the following:

    • In the Authorities box, type/select the required option from the list.

To download a sample excel file, click Download Sample Excel. In the sample excel file (shown below), enter the required Login, First Name, Last Name, Email, AuthKey, and AuthUserId of the authority from which the roles/users must be imported into Orbit application.   

 

    • To upload the excel file, in the Upload Excel File box, click Upload.

    • To schedule the job, click Submit.
