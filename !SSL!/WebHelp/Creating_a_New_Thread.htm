Creating a New Thread

 

To Create a New Thread

1. On the left-hand side, point to  Collaboration, and then click Threads.

2.Threads page appears with a list of available threads.

 

3. To create a new thread, click  New in the upper-right corner of the threads page.

4. The New Collaboration Thread dialog box appears.

5. In the New Collaboration Thread dialog box, do the following:

a. In the Name box, enter the new thread name.

b. In the Description box, enter the new thread description.

 

 

Note: Description is an optional field

c. To save the changes, click Save.

Note: A Thread saved successfully notification appears briefly at the top of the ORBIT application window.

d. To discard the changes, click Cancel.

6. Under Threads section, newly created collaboration thread appears.

 

7. To refresh the Threads, click  Refresh in the upper-right corner of the threads page.
