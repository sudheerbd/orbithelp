Importing Oracle Reports(Concurrent Programs)

To Import Oracle Reports

 

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded and Custom jobs.

  3. Under Jobs sections, on Import Oracle Reports (Concurrent Programs), click , and then click Schedule.

  4. Import Oracle Reports (Concurrent Programs) dialog appears.  

 

 

  1. Under Parameters tab, perform the following:

    1. In the Select Authorization box, type/select the required authorization.

    2. In the Search box, type the required search term, and then click .

(or)

    1.  In the Search box, click to display all the available reports under the selected authority.

    2. Select the required report and then click  to move the selected report to the right-hand side.

    3. To move all the available reports to the right-hand side, click.

 

If the process must be run immediately, select the check box before Run Immediately. If the process must be run a later time, then remove the checkmark from the Run Immediately checkbox.  When the Run Immediately checkbox is deselected, the Schedule tab becomes available. Using the Schedule tab, schedule the process at a required time.

 

  1. To set any notifications, click Notifications tab. View Notifications for more information.

  2. To submit the process, click Submit.
