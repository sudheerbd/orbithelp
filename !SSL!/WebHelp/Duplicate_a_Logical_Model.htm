Duplicate a Logical Model

To duplicate a Logical Model

 

1. In the Domain Hierarchy section, under the Logical Models, right-click the required logical model, and then click Duplicate.

 

  

2. A message "Logical Model copied successfully" appears. 

3. Copy of  logical model is automatically created as shown below:

 

  

Role-based security and global security will not be copied when a copy of logical model is created; they must be reapplied after copying.
