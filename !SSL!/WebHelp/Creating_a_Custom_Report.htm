Creating a Custom Report

To Create a Custom Report

 

1. On the left-hand side, click  Reports.

2. Reports pages appears. 

3. On the Reports section, click New Report (highlighted as "1"), and then click double-click Custom Report.

     (OR)

    On the Reports section, click New Report, click Custom Report, and then click Ok.

4. A new Custom Report page (highlighted as "2") appears as shown below:

 

 

5. In the Custom Report page, perform the following:

 

a. In the Name box, type the name of the custom report.

Required Name: Payables Posted Invoice Register

b. In the Description box, type the description.

Description: Payables Posted Invoice Register

c. In the Job class box, type the required job class.

Job class: com.orbit.reporting.job.OracleConcurrentProgramJob

  • This job class comprises of the class that is used in the implementation code for custom report.

d. In the Output Type list, select the required option.

Output Type: XML

e. In the Job Configuration box, type the required job configuration.

6. To create a new parameter, on the Parameters section, click .

7. New Parameter dialog appears.

8. In the New Parameter dialog, perform the following:

 a.      In the Name box, type the name of the parameter.        

Required Parameter Name: p_security_id_int_1

 

 b.      In the Label box, type the label of the parameter

Required Label: Operating Unit

 

 c.      In the Type list, select the required option from the following list:

    • INTEGER

    • STRING

    • DATE

Type : STRING

 9. To enable the list of values, select the List of values checkbox.

 

10. When the List of values checkbox is selected, perform the following:

 

    a. Drag the columns from the Choose business object section and then drop them into Value Field and Display Field(s).

    b. In the Filter(s) section,  drag and drop the required columns from the Choose business object section.
   

   c.  Apply the required filter condition, and then click Ok to save the parameter.

 

11. The newly created parameter appears under the Parameters section.

 

  

 

12. To save the report, click Save and then save the custom report with an appropriate name.

13. A success message appears.
