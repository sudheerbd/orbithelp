Visualization (Chart type) Options

Visual Settings for Sankey Charts

 

Sankey charts are used to visualize data flow and volume between nodes. The wider lines indicate larger volumes.

Visualization (Chart type) Options

1. Sankey Charts has the following visualization options

  • Weight

  • From

  • To

2. Drag and drop the required fields from the selected Business Object into the Weight, From, and To sections.

3. Orbit application generates the packed bubbles analysis report dynamically in the run results area.

Visual Settings for Sankey Charts

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for Sankey Charts has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Series: See Customize Series Visual Settings for more details.
  3. Legend: See Customize Legend Visual Settings for more details.
  4. Tooltip: See Customize Tooltip Visual Settings for more details.
