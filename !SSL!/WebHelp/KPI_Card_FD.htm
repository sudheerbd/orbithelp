KPI Card

Visualization (Chart type) Options

1. KPI Card has the following visualization options

  • Actual

  • Target/Goal

  • Trend/Compare By

  • Group By

2. Drag and drop the required fields from the selected Business Object into the Actual, Target, Trend, and Group By.

3. Orbit application generates the packed bubbles analysis report dynamically in the run results area.

 

Visual Settings for KPI Card

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for KPI Card has the following tabs:

  1. Card

  2. Format. See Customize Axes Visual Settings for details.

 

Card

Card tab has the following visual settings:

  1. Title: Show/hide word cloud chart title. See Customize Chart Visual Settings for details.

  2. Layout: Change Layout of the KPI Card.

    1. Template: Change KPI Card Template.

    2. Actual Value Label: Show/hide value label.

    3. Target Value: Show/hide target value.

    4. Percent Variation: Show/hide percentage variation.

    5. Absolute Variation: Show/hide absolute variation.

    6. Group Title: Show/hide group title.

  3. Delta Options: Show/hide delta options on the KPI Card.

    1. Result Indication: Change result indication style.

    2. Threshold Type

    3. Threshold Value

  4. Sparkline

    1. Sparkline view type

    2. Highlight Min/Max Points

    3. Highlight Start/End Points

 

Template

Default Value: Compact

Template setting has the following options:

  1. Compact

  2. Simple

  3. Target Range

  4. Gradient Card

 

Actual Value Label

Default Value: Selected

Clear Actual Value Label check box to hide value label.

Target Value

Default Value: Selected

Clear Target Value check box to hide target value.

Percent Variation

Default Value: Selected

Clear Percent Variation check box to hide percentage variation in KPI Card.

Absolute Variation

Default Value: Selected

Clear Absolute Variation check box to hide absolute variation in KPI Card.

Group Title

Default Value: Selected

Clear Group Title check box to hide group title in KPI Card.

 

Delta Options

Default Value: ON

  1. To hide delta options, turn off Delta Options toggle.

When you turn on Delta Options toggle, following options appear

  • Result Indication

  • Threshold Type

  • Threshold Value

 

Result Indication

Select one of the options from the Result Indication

  • Greater is Good

  • Less is Good

  • Warning if Greater

  • Warning if Less

Threshold Type

Default Value: Percent

Select one of the options from the Threshold Type

  • Percent

  • Absolute

Threshold Value

Enter any numerical value in the Threshold Value box.

Sparkline

Default Value: ON

  1. To hide Sparkline, turn off Sparkline toggle.

When you turn on Sparkline toggle, following options appear

  • Sparkline View Type

  • Highlight Min/Max Points

  • Highlight Start/End Points

 

Sparkline View Type

Default Value: Area

Select one of the options from the Sparkline View Type

  • Line

  • Area

  • Column

Highlight Min/Max Points

Default Value: Selected

Clear Highlight Min/Max Points check box to hide minimum/maximum points.

Highlight Start/End Points

Default Value: Selected

Clear Highlight Start/End Points check box to hide Start/End points.
