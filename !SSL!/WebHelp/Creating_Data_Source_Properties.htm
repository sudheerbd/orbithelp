Creating Data Source Properties

To attach the data source properties

 

  1. In the New Data Source dialog, click Datasource properties.

 

  1. Datasource Properties dialog appears.

 

     

  1. To add a new data source property, click   Add property.

  2. Under Name and Value sections, textboxes to enter the key and value pair of the property appear.

 

 

  1. To save the data source properties, click Ok.

  2. To save or update the data source, click Save.
