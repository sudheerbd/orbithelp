Visual Settings for Pie Charts

 

Pie charts use "pie slices" to show relative sizes of the data.

These visual settings are also same for Doughnut charts, Spiderweb Line, and Spiderweb Area.

Visualization (Chart type) Options

1. Pie chart has the following visualization options

  • Values (Slice)

  • Category (Color)

  • Variable Size

  • Split By

2. Drag and drop the required fields from the selected Business Object into the Values (Slice), and Category (Color).

Note: Drag fields with numerical values into the Values section for several types of calculations (Sum, Average, Min, Max, etc.) However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the pie chart analysis report dynamically in the run results area.

4. Drag and drop any size fields from the selected Business Object into the Variable Size section to generate variable size pie chart.

5. Drag and drop any dimension fields from the selected Business Object into the Split By section.

 

Visual Settings for Pie Charts

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for pie chart has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Slice: See Customize Slice Visual Settings for more details.
  3. Series: See Customize Series Visual Settings for more details.
  4. Legend: See Customize Legend Visual Settings for more details.
  5. Tooltip: See Customize Tooltip Visual Settings for more details.

 

These visual settings are also same for Doughnut charts, Spiderweb Line, and Spiderweb Area.
