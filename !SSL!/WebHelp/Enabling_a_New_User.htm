Enabling a New User

To enable a new user

 

  1.  In the Users list, under Status, for the required user, click  Deactivated (highlighted as "1") to activate the user.  

 

 

  1. Status Change dialog box appears with a message “Are you sure you want to activate selected user?” 

 

 

  1. To change the status, click Yes.

  2. A message “User activated successfully” appears.

  3. The required user now gets Activated status.

  4. An e-mail is triggered to the registered email id.

  5. In the generated email, click Activate Account.

  6. An account activation success screen is generated.

  7. An email with login details is triggered to the registered email.

  8. In the account activation success screen, click Login.

  9. Login page appears.

  10. In the Login page, type the login credentials, and then click Login.

When a new user is created, a system-generated password is sent to your registered email. Initially, you need to login to the application, by entering the system-generated password. Once you are logged in, you can change the password. See Changing Password for more information.
