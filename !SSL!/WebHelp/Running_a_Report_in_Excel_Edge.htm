Running a Report in XLEdge

To Run a Report in XLEdge

 

  1. In ORBIT XLEdge application, Reports section appears with the available report folders. 

Reports available under the Reports section are the data, analysis, publisher, and script reports already present in the ORBIT Analytics application.

  1. Under the Reports section, select the required report (highlighted as "1”) and then click next to the report name.

 

 

 

  1. To open the report, click Open (highlighted as "2").

  1. If the report does not have any parameters, the following section appears indicating there are no available parameters for the selected report.

 

 

 

  1. To run the report, click Run (highlighted as “3”).

  2. The output of the report appears on the left-hand side of the excel edge application (highlighted as “4”).

 

 

 

  1. When a report with parameters is run, then Parameters - <report name> tab appears with already existing parameters as shown below:

 

 

 

  1. In the parameters box, click .

  2. A list of available parameter values appears.

  3. For the required report, type the required parameter value, and then click Run.

Required Parameter: Central.               

  1. The output of the report appears in the excel worksheet on the left-hand side of ORBIT Excel Edge application.
