Sorting Null Values

Before applying sort on a data report with null values, configure the Nulls Sorting Priority property to get desired sort order for null values.

Configuring Nulls Sorting Priority Property:

  1. Open a data report which has null values in the data in the Edit mode.

    Or

    Create a data report with null values in the data.

  2. Click Run, to run the report.

    Example: A data report with null values in the data.

  3. Do either of the following to open the Properties dialog:

  4. On the top right-hand corner of the run results area, click Options, and then click Properties.

    or

    In the Report Configurator section, click on a column and then click Properties.

  5. In the Properties dialog, on the left-hand pane, select the column name on which you want to apply sorting.

  6. Properties of the selected column appear on the right-hand pane.

  7. In the properties list, Under the Format section, click Nulls Sorting Priority.

  8. Against the Nulls Sorting Priority, click under Values and select any one of the following:

    Note: By default, Null sorting Priority is Blank which is equivalent to DEFAULT.

    • DEFAULT: Select DEFAULT to retain the default sort order.

    • FIRST: Select FIRST to show null values at the top of the report, when sorting is applied to the column.

    • LAST: Select LAST to show null values at the bottom of the report, when sorting is applied to the column.

  9. Click Ok to save the changes.

 

To sort null values in a data report:

Note: Perform this after configuring Nulls Sorting Priority property.

  1. In the Report Configurator section, click on a column containing number fields and then click Sort 0 to 9.

  2. The selected column is sorted from 0 to 9 and null values are shown first in the report.

    Example: In the example below, Nulls Sorting Priority is set as FIRST and Sort 0 to 9 is applied on the DEPTNO column.

  3. In the Report Configurator section, click on a column containing number fields and then click Sort 9 to 0.

  4. The selected column is sorted from 9 to 0 but null values are still shown first in the report.

    Example: In the example below, Nulls Sorting Priority is set as FIRST and Sort 9 to 0 is applied on the DEPTNO column.

  5. Sorting A to Z, Z to A, New to Old, or Old to New on a column works similarly.

    Example: In the example below, Nulls Sorting Priority is set as LAST, and Sort A to Z is applied on the JOB column.

Note:
A metaobject config property “Nulls Sorting Priority” is also added at Base level in the Meta Objects screen.
If the user does not configure Nulls Sorting Priority at report level, then Base level property is considered for sorting.
