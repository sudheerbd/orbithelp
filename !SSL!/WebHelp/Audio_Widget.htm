Audio Widget

To add a audio widget in the dashboard, perform the following:

 

  1. On the top-menu, from the Add Widget list, select Audio widget.

  2. Audio widget appears in dashboard as shown below:

     

 

Note: By default a sample audio file appears, along with audio widget.

  1. To download the audio file, click next to audio play file.

  2. To play the audio, click option.

  3. To pause the audio, click option.

  4. To mute the audio, click option.

  5. For Audio widget, the following Visual Settings appears.

    • Image

      1. Title

        • Show: Show the title on styled text widget.

        • None: Hide title on styled text widget

      2. File Type

        • File Type: Allows audio/mpeg files

 

 

  1. To upload a audio file, enter the required file path in Audio File URL.

  2. The uploaded audio file appears in Audio widget.
