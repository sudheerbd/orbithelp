Audit Archive

To archive the audit data

  1. On the left-hand side menu, point to , and then click Jobs.

  2. Jobs page appears with a list of Seeded and Custom jobs.

  3. Under Jobs sections, on Audit Archive, click  , and then click  Schedule.

  4. Audit Archive dialog appears.

 

  1. In the Parameters tab, perform the following:

a. In the Number of days to retain box, type/select the required value.

Number of days to retain: 1


Day will be calculated as before the current data of database. Here, calendar days ignore current day of this job
No of day to retaindetermine the job to archive the records in the database, which are greater than the given value.

  1. To run the job immediately, select the check box before Run Immediately.


This job can be scheduled periodically using Schedule option.
When the Run Immediately checkbox is deselected, the Schedule tab becomes available.    

  1. Using the Schedule tab, schedule the process at a required time.

  1. To set any notifications, click Notifications tab.  View Notifications for more information.

  2. To submit the process, click Submit.

  1. Process submitted successfully pop message will appear with process id.

  2. To view the process status, navigate to Monitoring->Processes.

  1. To view process log, click on Log file.

  2. Log file download successfully to the local machine.
