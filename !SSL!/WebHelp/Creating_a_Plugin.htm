Creating a Plugin

To create a new plugin

 

  1. On the left-hand side menu, point to , and then click  Plugins.

  2. Plugins page appears as shown below:

  1. To create a new plugin, click  in the right corner of the plugins page.

  2. An Untitled plugin window appears, with the following option:

    1. In the Name box, type the plugin name.

    2. In the Description box, type the plugin description.

    3. In the Package(.Zip) box, upload the required zip file.

Package zip file will be provided by Orbit and (initially we are providing for 4 modules) it consists of Salesforce, EBS, NetSuite and Fusion.
