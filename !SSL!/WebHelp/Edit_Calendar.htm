Edit Calendar

To edit a scheduler calendar

 

  1. On the left-hand side menu, point to   System, and then click Maintenance.

  2. Scheduler page appears with Operations tab.

  3. To view calendars list, click Calendars.

  4. To edit a calendar, click  icon (highlighted as “1”) , and then click Edit (highlighted as "2").

 

  1. Edit Scheduler Calendar dialog appears.

  2. Edit the required changes and then click Save.
