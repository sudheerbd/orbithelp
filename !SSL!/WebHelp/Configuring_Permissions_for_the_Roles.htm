Configuring Permissions for the Roles

To configure permissions for the roles

 

  1. On the left-hand side menu, point to  User Management, and then click  Roles.

  2. Role List appears with a list of available roles.

  3. For the required role, click icon (highlighted as "1") under Actions, and then click Permissions (highlighted as "2").

 

  1. Permissions - <Role Name>  dialog box appears.

 

      

  1. In the Permissions - <Role Name> dialog box, under the Menu Root folder (highlighted as "3"), various menu options (highlighted as "4") appear.

  2. For the required role, select the menu options from the available menu permissions.

  3. To save the changes, click OK.

  4. To close the dialog box, click Cancel.
