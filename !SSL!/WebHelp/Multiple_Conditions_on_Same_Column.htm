Multiple Conditions on Same Column

To set multiple conditions on same column

1.      For a selected report, on the reports menu, click Options, and then click Conditional Formats. 

 

 

2. Conditional Formats dialog box appears.

 

 

3. To add the required field to the Conditional Formats dialog, drag and drop the required field from the Columns section under Configurator.

4.  The selected field appears as shown below:           

Selected Field: Sales

 

 

6. For the selected field, set the required conditions and values (highlighted as “1”).

 

You can set multiple conditions on the same field. For this, repeat the above-mentioned process; change the values of the following:

      • Operator

      • Value

      • Font Family

      • Font Size

      • Font Color

      • Background

      • Alignment

      • Style

The condition appears as follows (highlighted as “2”):

 

 

 

  • In the above scenario, the conditional formatting for sales greater than or equal to 10000 is highlighted as “1”. The conditional formatting for sales less than 10000 is highlighted as “2”.
