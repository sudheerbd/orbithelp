Editing a Snapshot

 

To edit a snapshot

 

1. In the Snapshots section, under Actions, click   against the snapshot you want to edit, and then click Edit.

 

    

2.Edit Snapshot -<snapshot name> dialog box appears.

   

    

 

 

3. In the Edit Snapshot - <snapshot name> dialog box, edit the required fields, and then click Save.

Note: A Snapshot update successfully notification appears briefly at the top of the ORBIT application window.

 

4. To cancel edit snapshot request, click Cancel.
