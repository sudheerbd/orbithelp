Creating a New Group

To create a new group

 

  1. On the left-hand side menu, point to  User Management, and then click Groups.

  2. Group section appears with a list of available groups.

  3. To create a new group, click New.

  4. New Group dialog box appears.

 

                       

 

  1. To create a new group, perform the following:

    • In the Name box, type the name.

    • In the Description box, type the description.



i.   Name and Description are mandatory.
ii.  The Name must be of at least 4 characters.
iii. Description is mandatory to enable Save button.

  1. To save the changes, click Save.

  1. To close the dialog box, click Cancel.
