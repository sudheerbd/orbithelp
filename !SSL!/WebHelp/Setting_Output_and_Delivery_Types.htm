Setting Output and Delivery Types

The Output & Delivery tab can be seen under Schedule option in Reports

1. On the Output & Delivery tab, on the Outputs and Deliveries section, click   Add output (highlighted as “1”) to set a new output type.

 

Delivery plan option is unavailable for Analysis and Dashboard Reports

2. Templates section with its associated output types appear under Outputs and Deliveries.

3. The following are the available output types for a data report:

·        CSV

·        DELIMITER TEXT

·        TABULAR/JSON

·        XML

·        HTML

·        PDF

·        Excel(xlsx)

      For Analysis reports, the following output formats are available:

·        PDF

·        HTML

·        IMAGE

      For Dashboards, the following output formats are available:

·        PDF

·        HTML

      For Publisher Reports, the following output formats are available:

·        HTML

·        PDF

·        Excel(XLS )

·        Excel(XLSX)

·        DOCX

·        DOC

·        IMAGE

Output type changes depend upon the template.

     For Script Reports, the following output formats are available:

      • CSV

      • XSLX

      • XML

      • PDF

      • HTML

      • DOCX

      • Tabular

Searching for Users

To Search for an Already Existing User

1. In the Search box, type the required search term, and then click .

2. The required user appears in the box below:

 

    

 

3. Once the required email is entered in To, Cc, or Bcc, perform the following:

a. In the Subject box, type the required subject for the e-mail.

b. In the Body text area, type the required body for the e-mail.

c. In the File Name box, type the file name for the report/process that is being emailed.

4. The File Name can be of the format “myFile{currentDate:dd-MM-yyyy}”; for example, “myFile01-31-2017”.

5. The filename should not contain the following:

< (less than)

> (greater than)

: (colon)

" (double quote)

/ (forward slash)

\ (backslash)

| (vertical bar or pipe)

? (question mark)

* (asterisk)

 

6. To send the attachment as zip file, select the Send attachment as zip file checkbox.

If the zip file name is empty, then the process id would be the report name in the zip file.

7. To save the changes, click Ok.

8. In the Schedule - Report dialog, under Delivery To the selected email ids appear.

 

  

 

9. To submit the process, click Submit.

10. The process gets submitted.

11. To view the process, navigate through the following:

  • Monitoring-> Processes

12. In the Processes page, under Processes tab, the newly submitted process appears.    

13. Also, the scheduled report is sent to the respective email ids that are mentioned in To, Cc, and Bcc.
