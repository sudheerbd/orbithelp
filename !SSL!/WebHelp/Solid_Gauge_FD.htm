Solid Gauge

A gauge chart displays a single value that measures progress toward a goal or a Key Performance Indicator (KPI).

Visualization (Chart type) Options

1. Solid Gauge has the following visualization options

  • Value

  • Minimum

  • Maximum

  • Target

2. Drag and drop the required fields from the selected Business Object.

3. Orbit application generates the analysis report dynamically in the run results area.

Visual Settings for Solid Gauge

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for Solid Gauge has following tabs:

  1. Gauge

  2. Format

Gauge

List of visual settings available in the Gauge tab:

  1. Title: Customize the chart title.

  2. Minimum: Enter minimum value.

  3. Maximum: Enter a maximum value.

  4. Target: Enter a target value.
