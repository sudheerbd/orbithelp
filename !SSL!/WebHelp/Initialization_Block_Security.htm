Initialization Block Security

1.  Right-click the required Logical model, and then click Data Security.

2.  Data Security-<Logical Model> box appears. 

3. In the Data Security box, select Initialization block radio button (highlighted as “1”).

4. In the formula box, click to create a formula.  

5. Expression Builder (Data Filters)dialog box appears.

6. In the formula box enter the required formula.

7. To save the expression, click Ok.

8. The formula appears in the formula box.

 

 9. To apply the initialization block formula, click Ok.

10. To save the changes, click Save.

Since, the data security is applied at the logical model level, it applies to its business objects under it. This data security reflects on the reports built on this business object.

 

Viewing the applied Data Security

 

1. Open the required report with business object that has the “initialisation block” data security applied on it.

2. To run the report, click Run.

3. The condition/information provided in the initialisation block is executed.

4. A table with “AP_INVOICES_ALL1” is created in the database with all the records of “AP_INVOICES_ALL” table.
