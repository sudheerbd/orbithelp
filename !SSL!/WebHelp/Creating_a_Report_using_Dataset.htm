Creating a Report

To create a report

  1. In the Datasets area, a list of available datasets appears.

  2. For the required dataset, under Actions, click  icon, and then click Create Report.

 

 

  1. Reports menu appears with the following types of reports:

    1. Data Report

    2. Analysis

    3. Dashboard

    4. Publisher Report

    5. Script Report

    6. Finance

    7. Custom Report

 

 

  1. To select the required type of the report, double-click the report option.

(Or)

Select the required report type, and then click Ok.

  1. The report open with the required dataset object under Data Explorer in Reports.

 

Options for Datasets

 

 

Dataset has the following options:

  1. Create Report

  2. Edit

  3. Remove

  4. Rename

  5. Schedule Refresh

  6. Share

  7. Upload Files
