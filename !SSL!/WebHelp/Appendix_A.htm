Appendix A

Generate link with special character parameter values

 

To generate the required report links, perform the following:

 

  1. In the Reports section, select the required folder under My Folders.

  2. A list of reports appears on the right-side of the report section.

  3. Right-click the required report, and then click Open (highlighted as "1").

  1. Selected report appears with run result area.

  2. Enter the required filter values as show below:

 

In the above report user enter a special character ' % '.

  1. To save the filter value, click Ok.

  2. To run the report, click Run and the report appears as shown below:

 

To generate a link

  1. On the top right-hand corner of the run results area, click Options, and then click Generate Link (highlighted as “1”).

  1. Generate Report Link dialog appears.

  2. In the Generate Report Link dialog, perform the following:

To generate a public link with include parameter values

    • To select link type, click the dropdown list and the select Navigation.

Navigation appears by default.

    • To generate public link, select Public link radio button.

    • To include parameters in the generate link, select Include current parameter values in link checkbox.

    • To generate the link, click Generate.

    • The public link with parameter values, appears in the Link box.

    • The link appears as shown below::

 

  1. To copy the link, click Copy beside the Link box.

  2. Paste the public link in the desired browser.

  1. When a user using special character values, the respect URL need to encode with values.

 

Before Encoding the URL:

Public Link URL: http://orbdev01.orbit8.com:7070/reporting/web/public/report?token=30e3ba70-5ae7-4713-ac60-05d6443bfad9&OH=Y&FR=Y&AR=N&region=%&profit=

 

 

  • Here, '%' (highlighted as "3") is the parameter value of region.

After Encoding the URL:

For the parameter value '%', the Code Points (Hexadecimal) is '25'. So we have replace '%' with '%25' (highlighted as "4")

Public Link URL: http://orbdev01.orbit8.com:7070/reporting/web/public/report?token=30e3ba70-5ae7-4713-ac60-05d6443bfad9&OH=Y&FR=Y&AR=N&region=%25&profit=
