Custom Job Implementation

To implement a custom job

 

 

  1. A custom job should be implemented in Java only.
  2. The Job class extends OrbitQuartzJob class
  3. It is required to override execute() method as part of the implementation.

 

For example

 

 public void execute(JobExecutionContext context) throws JobExecutionException

{

//write the Implementation hereâ€¦.

}

 

  1. The very first line of code in this method should be:

 

   OrbitJobContext ctx = getOrbitJobContext(context, this);

 

 

 

  1. OrbitContext is the job context, which is required to get all the details about ORBIT job.  This context is used to get processId, output stream, spring application context, output file location if any, log API  etc.,
