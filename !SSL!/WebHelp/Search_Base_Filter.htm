Search Base Filter

  1. Under Group Search Information section, perform the following:

    • In the Search Base text box, type the required search base.

Example: Search Base: OU=Orbit2             

  • In the search filter text box type the required search base.

Example: Search Filter: (&(objectCategory=Group)(objectClass=group)(cn=group6))

     2. This retrieves the required reports from the ObjectCategory equals to "Group" and ObjectClass equals to" User group6"

  1. Import the required users once they are retrieved.
