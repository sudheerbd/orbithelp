Introduction

Analysis reports have two view modes: Edit Mode and View Mode.

Users having a role with permission to create/edit reports can create/edit analysis reports in the Report Edit Mode. End users can interact with the reports in Report View Mode.

Report Edit Mode

The Report Edit Mode is divided into several main areas:

  • Menu Options Area (Highlighted as "1")

  • Data Explorer and Visual Settings Area (Highlighted as "2")

  • Report Configurator Area (Highlighted as "3")

  • Run Result Area (Highlighted as "4")

Data Explorer and Visual Settings Area has two tabs.

Data: The Data pane displays the business objects, and folders, fields in the selected business object that are available for you to use to create analysis reports.

Visual Settings: Select Visual Settings tab to display the Visual Settings (Highlighted as "5"). The options available depend on the type of visualization selected.

 

Report View Mode

The Report View Mode is divided into two main sections:

  • Menu Options Area (Highlighted as "1")

  • Run Result Area (Highlighted as "2")
