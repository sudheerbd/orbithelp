Visualization (Chart type) Options

1. Maps has the following visualization options

  • Values (Slice)

  • Category (Color)

2. Drag and drop the required fields from the selected Business Object into the Values (Slice), and Category (Color).

3. Orbit application generates the analysis report dynamically in the run results area.

Visual Settings for Maps

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for Maps has following tabs:

  1. Map

  2. Legend

Map

List of visual settings available in the Maps tab:

  1. Title: Customize the chart title.

  2. Map Region: Select an area from the list.

  3. Field Code: Select the area/region code.

  4. Draw Type: Customize the map plot.

  5. Legend Hide: Show/Hide data legend on the map.

  6. Plot Background Color: Customize the plot background color.

  7. Color Range Settings: Customize color settings on the map.

 

Title

Title setting has the following options:

  1. Show: Select Show to set a custom title

  2. None: Select None to hide chart title.

Text

When you  select Show in Title options, Text box appears. Enter a custom title in text box.

Map Region

Select the area/region to show from the list. You can select whole world, specific continents, or a specific country.

When you select a country, map shows major regions of that country.

Use search box to filter and find regions from the list.

Field Code

Select the format in which areas (countries/regions) are listed in database. Select the correct field code to identify all areas in the map.

Name: Name of the area

Postal Code: Postal code for the area, usually two letters wide

HC-Key: Unique hierarchical identifier for the area. Consistent across maps, and reflected in the naming convention of the maps.

ISO-A3: Country codes as described in the ISO 3166 alpha-3 international standard.

ISO-A2: Country codes as described in the ISO 3166 alpha-2 international standard.

HC-A2: Two letter abbreviation of area name. Usually same as postal code/ISO code, but not always. Not guaranteed to be unique within map.

Draw Type

Customize how the map is plotted. Select map draw type from

Color Map: Areas are filled with color

Bubble: Bubbles are shown on the area.

Legend Hide

Show/Hide data legend on the map.

Plot Background Color

Default color: White (RGB 255, 255, 255).

Select Background Color box to change the color of plot background. See Change default color in Negative Color for detailed information.

Color Range Settings

Color Mode

Customize colors on the map. Color intensity relates to the value.

Select color mode from

Min-Max Color: Creates a two color gradient

Gradiant 3 Colors: Creates a 3 color gradient

Gradiant 4 Colors: Creates a 4  color gradient

Gradiant 5 Colors: Creates a 5 color gradient

Colors

Customize individual colors in a color mode. Click on a color box under Colors and pick a color from the colort picker.

Legend

See Customize Legend Visual Settings for detailed information.
