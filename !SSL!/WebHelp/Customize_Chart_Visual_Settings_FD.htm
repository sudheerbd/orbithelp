1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

Note: You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Chart to open chart visual settings.

Chart

List of visual settings available in the Charts tab:

  1. Title: Customize the chart title.

  2. Padding: Adjust space around the chart.

  3. 3D: Enable/Disable 3D chart style.

  4. Negative Color: Enable/Disable specific color for negative values in the chart.

  5. Chart Background Color: Customize the chart background color.

  6. Plot Background Color: Customize the plot background color.

  7. Plot Background Image URL: Customize the plot background image.

  8. Pie Style: Changes Pie chart style. (For Pie and doughnut Charts only)

  9. Chart Type: Changes chart type. (For Scatter and bubble charts only)

  10. Point Size: Changes scatter point size. (For Scatter charts only)

 

Note: The visual settings available depend on the visualization type selected.

Title

Default Value: Auto

Title setting has the following options:

  1. Auto: When Title is set to Auto, Orbit application generates chart title based on the field names dropped in the visualization (chart type) options section.

  2. Custom: Select Custom to set a custom title.

  3. None: Select None to hide chart title.

Text

When you  select Custom in Title options, Text box appears. Enter a custom title in the text box.

 

Padding

Default Value: 15 pixels, Same for all sides

  1. Enter any numerical value in the padding box, to apply padding to the chart.

    Note: You can enter negative values also in padding box.

  2. Clear Same for all sides check box, to change padding individually on Top, Bottom, Left, and Right of the chart.

3D

To enable 3D chart style:

  1. Select 3D check box to change chart style to 3D.

  2. When you select 3D, following options appears: Alpha Angle, Beta Angle, and Depth

  3. Move the sliders on Alpha Angle, Beta Angle, and Depth to change the 3D settings for the chart.

Negative Color

  1. Select Negative Color check box to enable Specific color for negative values in the chart.

  2. When you select Negative Color check box, a Color box appears. Default color for Negative values is Red (RGB 255, 0, 0).

  3. To change default color:

    1. Click the Color box to open the Color Picker dialog box.

    2. Option 1: In the Color Picker, drag the vertical slider above H to choose a hue (color). Then click a spot in the large color box to choose the brightness and saturation of that color.

    3. Option 2: In the Color Picker, drag the vertical sliders above H (hue - color), S (Saturation), and V (Value - Brightness) to choose a color.

    4. Option 3: Enter exact color values into the HSV, RGB, or Hexadecimal color fields.

    5. To change opacity, drag the vertical slider above A to choose alpha (opacity).

    6. Click OK to close the Color. The color you chose appears in the Negative Color box.

Chart Background Color

Default color: White (RGB 255, 255, 255).

Select Background Color box to change the color of chart background. See Change default color in Negative Color for detailed information.

 

Plot Background Color

Default color: White (RGB 255, 255, 255).

Select Background Color box to change the color of plot background. See Change default color in Negative Color for detailed information.

Background Image URL

Default: Blank

Enter an Image URL in the Background Image URL box to set the image as plot background.

Note: Background Color does not have effect on the chart when you add Background Image.

Pie Style

Default Value: (Pie icon)

Select the donut icon in the Pie Style to change pie style chart to doughnut style chart.

Donut Size

Donut Size slider appears when you select the donut icon .

Move the sliders on to change the donut size settings for the chart.

Chart Type

Default Value:

Select the bubble icon to change chart type to Bubble.

Point Size

To change point size, select any one of point sizes from Point Size options.

Note: Point Size option is only applicable for Scatter Charts.
