Without Repeat Header Rows Option

The below scenario displays the output where “Repeat Header Rows” option is not applied.

Scenario1:

When Repeat Header Rows option is not selected in Microsoft Word; however, Table Grouping and Page break is applied in the template.

Repeat Header Rows appears in the Layout menu in Microsoft Word (highlighted as “1”).

 

Output:

The figure below illustrates the output of the above template, where the header is not repeated in the second page. Table Grouping is done State-wise, hence, the data from different states (Arizona and California) is displayed below. As Page Break is applied, the data from each state appears in a new page. In the figure below, the data for the state Arizona ends in the second page and as the page break is applied, the data for the state of California appears in the third page, instead of appearing in second page, where the data of Arizona state ends.

 

When no Table Grouping or no Page Break is applied, then it’s a normal table and the output is similar to Scenario 2 in the above section.
