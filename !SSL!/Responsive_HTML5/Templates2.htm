Templates

To Upload a New Template

1. On the right-hand side, click Options, and then click Templates (highlighted as "1").

  

2. Templates box appears.  

3. In the Template box, click New Template (highlighted as "2").  

4. The fields appear as shown in the box below:

 

5. In the Templates box, type the following :

    a. In the Name box, type the name of the template.

    b. In the Description box, type the appropriate description.

  •  In the Type box, the type of the template "ORBIT" appears by default.

    c. To upload the template, click Upload file.

  • Once the file is uploaded, it appears in the text box next to Upload.

       

  •  Save the template and close the Templates box.  

6. The newly created template appears as follows:

       

7. To make the template default, select the template, and then click Make Default.

     

8. A check mark appears in the check box under Default.

  

9. To remove the template as default template, select the required template, and then click Remove Default.
