Creating Notifications

To create notifications

1.        On the Notifications tab, in the Notifications area, click Add to set new notification. 

 

 

2.  In the Status list, click the required status option from the following:

a.      Success

b.      Failure                                                         

c.       Warning

     

                   Example: Status: Success

 

2.      In the Message box, type the required message.

      

                   Example: Message: This is a test notification.

 

3.      In the Users box, type/select the required option.

 

                   Example: Users: XXFORX

 

¥    Delete option becomes available only when a notification is created.

 

1.       To delete the notification, click (highlighted as “2”).

 

       

 

2.      To submit the notification, click Submit (highlighted as “3”).

3.      To notify about the end of the process, select the Notify me when this process ends checkbox (highlighted as “4”).

4.      In the Notes box (highlighted as “5”), type the notes for the notification.

 

Example: Notes:  Sample Notes

 

5.      To submit the changes, click Submit.

 The Notes entered in the Notes box, is sent as a notification when the process ends.
