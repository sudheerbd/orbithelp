Configuring Fax Settings

To Configure Fax Settings

1.      On the Security menu, click Settings.

2.      Email tab appears by default.

3.      To create Fax settings, click Fax tab.

4.      To create new Fax settings, click New.

5.      Textboxes (highlighted as “1”) appear under Name, Server Name, Server Alias, UserName, Password, and Number.

 

6.      On the Fax tab, perform the following:

·         In the Name box, type the name of the Fax Settings.

·         In the Server Name box, type the name of the Server Name.

·         In the Server Alias box, type the alias of the Server.

·         In the User Name box, type the user name.

·         In the Password box, type the password.

·         In the Number box, type the fax number.

·         To save the changes, click Save.

·         In Actions, click Remove (highlighted as “2”) to delete the Fax settings.

 

·         To modify the required fax settings row, double-click on it.

·         Edit/make the required changes, and then click Update.
