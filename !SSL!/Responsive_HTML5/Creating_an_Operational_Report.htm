Creating a Data Report

To create a data report

 

1.      In Reports, click  New Report, click Data Report (highlighted as “1”), and then click Ok.

 

               

 

2.      Run Result area (highlighted as “3”) appears along with Data Explorer (highlighted as “1”) and Report Configurator area (highlighted as “2”).

 

 

      3.    Widgets tab appears next to Data Explorer tab.

4.      On the Data Explorer tab, click  (highlighted as “4”) next to Select Business Object to select the required business object.

5.      Search area (highlighted as “5”) appears with available business objects (highlighted as “6”).

 

                

 

6.      In the Search box, type the business object name to search. 

(OR)

Select the required business object, and then click OK.

(OR)

Double-click the required business object.

             ?  Example: Required Business Object: Superstore

7.      The selected business object appears with the associated columns under it, in the Business Object area.

 

                     

 

 

8.      To view the columns under the selected business object, click  (highlighted as “7”) before the required business object.  

9.      Drag and drop the required columns (highlighted as “8”) from the required business object to the Configurator in the Report Configurator area.

 

 

10.  To generate a data report, click Apply.

(or)

To run the report, click Run above the Run Results area.

11.  A data report appears in the Run Results area.

 

       

 

12.  To save the report, click Save.

13.  Save Report dialog appears.

    

       

 

 

14.  If the required report folder (highlighted as “1”) already exists, then perform the following:

a.       In the Name box (highlighted as “2”), type the report name.

?  Example: Name: Sample Report - Superstore

b.      In the Description box (highlighted as “3”), type the description.

?  Example: Description: Sample Description for Superstore Report

c.       The selected folder appears in the Selected Folder box.

d.      To save the report, click Save.

 

                   

 

15.  If the required report folder does not exist, then create a new reports folder, and then save the report in it. See Creating a Reports Folder

 

  • The reports can be categorized using Folders when saving the reports.
