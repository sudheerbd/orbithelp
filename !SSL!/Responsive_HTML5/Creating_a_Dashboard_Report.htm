Creating a Dashboard Report

To Create a Dashboard Report

1.      In Reports, click New Report, and then click Dashboard.

 

        

                   

 

2.      Dashboard report opens with two options- Data Explorer (highlighted as “1”) and Widgets (highlighted as “2”).

 

 

3.      Under the Data Explorer area, the Business Object area appears along with Run Results area.  

4.      Under the Widgets area, the required reports appear.

 

5.      Dashboard reports can be created in either of the following ways:

a.       Drag and drop the already existing analysis reports from Reports area under Widgets to Run Results area.

Examples: Analysis Report: Test Analysis

   

                              

 

(OR)

b.      Select a business object and then drag and drop the required columns from Business Object area to Report Configurator area.

        Example: Required Columns: Y-axis: SUM(Sales)

                                                                   X-axis: State

                                                                           Region

 

The report appears as shown below:

 

 

¥        Multiple reports can be displayed on the Run Results area of the Dashboards

 

5.      The reports appear on the Run Results area.

·         To view the underlying data, click Options, and then click Underlying Data.  

·         To maximize the chart, click  Maximize widget .

·         To minimize the maximized chart, click  Minimize Widget.

 

                        

6.      To save the dashboard, click Save Report.

7.      Save Dashboard dialog box appears.

 

 

8.      To save the report, perform any one of the following:

                                            i.            Save the report in an existing folder.

(OR)

                                          ii.            Create a new folder, and then save the report under it.

9.      To save the report, click Save.

10.  To close the dialog box, click Cancel.
