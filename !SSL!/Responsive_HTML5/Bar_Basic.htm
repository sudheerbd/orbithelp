Pivot/Matrix Charts

Pivot chart enables data analysis by visualizing a pivot table.

Widget Settings of Pivot

To set the widget settings of pivot

1.      On the report configurator, click Widget Settings.

2.         The following tabs appear:

                                            i.            General

                                          ii.            Paging

General

Under the General tab, perform the following:

1.      In the Title box, type the title for the pivot chart.

 

             Example: Title: Pivot Chart

 

     2.      In the Layout box, type/select the required option from the following:

 

a.       Outline

b.      Compact

 

        a. When Outline is selected, the report appears as follows:

 

 

b. When Compact is selected, the report appears as follows:

 

 

3.      In the Measure on Row box, type/select the required field.

¥    When Measure on Row is selected as true, the measures column appears (highlighted as “1”) as shown below.

¥    Measures column appears in the report only when the following are fulfilled:

·         Values section must have two columns.

·         Columns section in Configurator tab must have at least one columns under it. Here, Product Category is mentioned under Columns.

 

 

Sub-Totals

1.      Under the Sub Totals tab, perform the following:

a.       In Row Totals box, select/type the required option from the following:

                                      i.      Last

                                    ii.      First

                                  iii.      None

?     Example: Row Totals: First

 

¥ In the Row Totals box, when Last option is selected, the total of the values in the rows appears in the last.

¥ In the Row Totals box, when First option is selected, the total of the values in the rows appears in the First. 

b.      In the Column Totals box, select/type the required option from the following:

                                                        i.      Last

                                                      ii.      First

                                                    iii.      None

?     Example: Column Totals: Last

 

?   When the above widget setting is applied, the report appears as shown below:

 

 

 

¥    In the above figure the row is highlighted as “1” and row total is highlighted as “2”. Column is highlighted as “3” and Column Total is highlighted as “4”.
