Setting Permissions

To set permissions for each role

 

1. On the left-hand side menu, point to User Management  ,  and then click Permissions.  

2. Two sections -- Roles and Menu Permissions appear.

3. In the Roles section, under Name , select the required role (highlighted as “1”).

 

       

                                                    

2.      Under Menu Permission, the permissions for the selected role (highlighted as “2”) appear.

3.      To select permissions for a specific role, select the checkbox before the required permissions, and then click Save.

4.      To deselect permissions for a specific role, remove the checkmark before the required permission, and then click Save.

 

  • Each Role can have limitations on the Menu Permissions; For example, an ADMIN can have all the menu permissions, however, the menu permissions for a user can be limited.
