Generating Excel Output in Schedule Report

To generate Excel Output in Schedule Report

1. On the required report, right-click and then click Schedule (highlighted as "1").

 

2. Schedule Report dialog appears.

 

3. In the Parameters tab, click to enter the required value for the existing parameter.

  • Example: Required Value for the Parameter: Texas

4. In the Output & Delivery tab, on the Outputs and Deliveries section, click Add output (highlighted as "2") to add an output.

5. Under Outputs and Deliveries section, an output is added along with the Deliveries section.

6. Under the Output Type, select the required output type, and then click Submit.

To view the submitted processes

1. On the Monitoring menu, click Processes.

2. The submitted processes appear under the Scheduled Processes.  

3. Select the required processes, and the output for the required process (highlighted as "3") appears under Output(s) & Delivery section.

4. The output is downloaded as an excel sheet.

Viewing the Output

1. Open the downloaded excel sheet.

2. The excel sheet opens with the report data as well as parameter and timestamp details.
