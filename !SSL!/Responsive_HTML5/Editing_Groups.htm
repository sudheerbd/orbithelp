Editing Groups

To edit group

1.      On the User Management menu, click Groups.

2.      Group List (highlighted as "1") appears with a list of available groups.

 

       

                                                                             

 

3.      For the required group, under Actions, click Edit  

4.      Edit Group <group name> dialog box appears.

 

      

                                  

5.      In Edit Group <group name> dialog box, perform the following:

a.       In the Name box, type a new name for the group.

b.      In the Description box, type the description for the group.

6.      To save the changes, click Save.

7.      To close the dialog box, click Cancel.
