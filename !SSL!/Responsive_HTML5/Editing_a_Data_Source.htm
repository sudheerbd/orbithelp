Editing a Data Source

To Edit a Data Source

 

  1. Under Data Source area, a list of available data sources appears.
  2. From the available data sources, select the required data sources, and then click Edit.
  3. Edit <Data Source Name> dialog box appears.

             

                 

             4. Edit the required fields, and then click Save.
