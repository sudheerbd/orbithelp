Resubmitting the Process

To resubmit the process

1.      On the menu bar, click Monitoring.

 

      

                

 

2.      Scheduled Processes page appears with a list of processes.   

3.       For the required process, click Resubmit (highlighted as “1”).

 

 

                                

 

  4.  Schedule Report dialog box appears.

  5.  In the Schedule Report dialog box, select the Run Immediately checkbox , and click Submit to resubmit the report immediately.

  6.  To schedule the report at a later time,deselect the Run Immediately checkbox, select the required scheduling options and then click Submit.

 

  •  By default, Run Immediately checkbox is selected.
