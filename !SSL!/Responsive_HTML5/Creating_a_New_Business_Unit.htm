Creating a New Business Unit

To create a new business unit

 

  1. On the menu bar, click Business Units.

  2. Business Units page appears.

  3. In the Business Units page, click New.

 

           

                                               Figure: Displaying New Option

 

 

      4. New Business Unit dialog box appears.

 

           

 

                                                    Figure : New Business Unit Dialog Box

      

 

  1. To create a new business unit, perform the following:

 

o   In the Name box, type the name of the business unit.  

o   In the Name Key box, type the DNS label.

 

§  Name Key must be a DNS label, and unique within your Business Units

 

o   In the Description box, type the description of the customer.

o   In the Admin Email box, type the email of the admin.

o   In the Admin Username box, type the username of the admin.

o   In the Staging datasource list, click the required data source. 

o   In the Banner Text box, type the banner text.

o   In the Copyright Text box, type the copyright text.

  1. To save the changes, click Save.

  2. To close the dialog box, click Cancel.

  3. A message “Business Unit saved successfully” appears when clicked on Save.
