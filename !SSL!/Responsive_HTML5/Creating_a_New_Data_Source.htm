Creating a New Data Source

To create a Data Source

 

  1. On the left-side menu, click Business Models.  

  2. On the Business Models menu, click Data Sources .  

  3. Data Sources page appears with a list of available data sources (highlighted as “1”).

 

         

          

                                                   

  1. To create a new data source, click New.

  2. New Data Source dialog box appears.

 

          

                          

  1. To create a new data source, perform the following:

 

·         In the Name box, type the data source name.

    • Example: Name : Test Data Source  

·         In the Description box, type the description for the data source.

·         In the Data Source Type, select the required data source type radio button from the following:

  • JDBC

  • JNDI

    • Example: Data Source Type: JDBC

 

·         In the Select Database, select or type the required option.

 

    • Example: Required Database Option: Oracle JDBC (Oracle Thin Driver)

 

·         In the Username box, type the user name.

    • Example: Username:apps

 

·         In the Password box, type the password.

    • Example: Password:apps

 

·         Under the Oracle JDBC Settings area, select the Enter custom JDBC URL checkbox to enter the custom JDBC Url as shown below:

 

      

                                          

·         Remove the checkmark from Enter custom JDBC URL checkbox to view the Oracle JDBC Settings.

 

       

                                   

·         In the Host Name box, type the host name.

    • Example: Host Name: iorb1.orbit8.com

·         In the Port box, type the port number.

    • Example:Port:1521

·         In the Database Name box, type the name of the database.

    • Example: Database Name:ebsdb

·         To test the data source connection, click Test Connection (highlighted as “1”).

·         To save the data source, click Save (highlighted as “2”), once the data source connection is successful.

·         To close the dialog box, click Cancel.
