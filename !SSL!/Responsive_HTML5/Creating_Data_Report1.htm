Creating Data Report

To create a Data Report

 

1.      In Reports, click New Report, and then click Data Report .

2.      Run Result area  appears along with Data Explorer and Report Configurator area.

3.      To select the business object, click  next to Select Business Object.

4.      Search area  appears with available business objects.     

5.      In the Search box, type the business object name to search. 

(OR)

Select the required business object, and then click OK.

(OR)

Double-click the required business object.

 

            ?   Example: Required Business Object: Sample Business Object

 

6.      The selected business object appears with the available tables/views/synonyms and its associated columns under it, in the Business Object area.

      

7.      To view the columns under the selected business object, click  before the required table/view/synonym.  

 

      ?   Example: Required Table: Invoices

 

8.      Drag and drop the required columns from the required table/view/synonym to Columns under Configurator  in Report Configurator area.

 

9.      A data report appears in the Run Results area.

 

¥   Select columns from one or more data objects under one business object to generate reports.

 

(OR)

To generate a data report, click Run.  

 

10.  The report appears in Run Results area.

11.  To save the report, click Save.

12.  Save Report dialog box appears.

13.  If the required report folder  already exists, then perform the following:

a.       In the Name box , type the report name.

 

                      ?   Example: Name: Test Invoices Report

 

b.      In the Description box, type the description.

 

?   Example: Description: Sample Description for Invoices Report

 

c.       The selected folder appears in the Selected Folder box.

d.      To save the report, click Save.

 

        4.  If the required report folder does not exist, then create a new reports folder, and then save the report in it. See Creating a Reports Folder

 

¥     The reports can be categorized using Folders when saving the reports.
