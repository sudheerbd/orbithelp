Widget Settings of Card Charts

 

To set the widget settings of card chart

1.      On the report configurator, click Widget Settings.

2.   Under the Widget Settings section, the following tabs appear.

·         General

·         Value Columns

 

General  

1.Under the General tab, perform the following:

a.          In the Template box, type/select the required template.

  

                   ?     Example: Template: CARD2

 

Value Columns

1.      Under the Value Columns, perform the following:

 

a.       In the Value1 box, type the required value.

 

             ?   Example: Value 1: [Sales]

 

b.      In the Value 2 box, type the required value.

           

            ?   Example: Value 2: [Region]

 

    c. In the Value 3 box, type the required value.

 

            ?   Example: Value 3: [State]

 

 d.  In the Value 4 box, type the required value.

 

?   Example: Value 4: [Product Category]

 

¥  [Value 5], [Value 6], [Value 7], [Value 8], [Value 9], and [Value 10] can be filled based on the requirement in similar way.
