Deleting Datasets

 

To delete a dataset

 

1.      In the Datasets area, a list of available datasets (highlighted as “1”) appears.

 

                             

 

2.      To delete a dataset, click Remove.  

 

                                                      

 

3.      A confirmation message “Are you sure you want to delete selected domain?” appears.

 

       

                          

 

4.       Click Yes to delete the dataset.

5.      Click No to close the dialog box
