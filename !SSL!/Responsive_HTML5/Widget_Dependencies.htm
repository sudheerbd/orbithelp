Widget Dependencies

 

Widget Dependencies options allows to create a dependency between two or more reports with the need of enabling auto interaction. This is like the drilldowns option.

 

To work with Widget Dependencies

 

Prior to working with widget dependencies, create two analysis reports which have data filters.

For example, here two reports “Bar Chart” and “Column Chart” have been created with data filter region.

1. Drag and drop the two reports onto the workspace in the dashboard.

2. On the top-menu, click Options, and then click Widget Dependencies.

3. Widget Dependencies dialog appears with the following two sections in it.

·         Widgets

·         Refresh Parts

             4. Widgets displays the list of available charts.

5. To set the widget dependencies, perform the following:

6. On the Refresh Parts section, click Add part to refresh, and select the required chart (highlighted as “1”) from the menu.  

 

 

The required chart appears with the parameters under Refresh Parts.

Select the required parameter of the chart (highlighted as “2”), and then click Ok.
