Domains

Domains feature in ORBIT Analytics provides the following:

·         Physical Model

·         Logical Model

·         Business Objects

                

The above figure depicts the structure of a domain. A Physical Model can have multiple data sources. One logical model can have multiple physical models. One logical model can have multiple business objects. Using a business object, the following reports can be generated:

 

  • Data Reports 

  • Analysis

  • Dashboard Reports   

  • Publisher Reports

  • Script Reports
