Applying Sort

For the reports, the types of sort depend on the data types.

 

For Date fields, the following sort can be applied:

Sort Old to New - This sort displays the old dates first followed by the new dates

For example, if there is a data in the report from the year 2004 to 2016, then the data of the year 2004 is displayed first followed by 2005, 2006, and so on. the data of the year 2016 is displayed last.

Sort New to Old- This sort displays the new dates first followed by the old dates.  

 

For Number fields, the following sort can be applied:

Sort 0 to 9 :  This sort first displays the data that contains 0, then 1, 2 ,3 , and so on.  The data containing 9 is displayed at the end.

Sort 9 to 0 : This sort first displays the data that contains 9, then 8, 7 ,6 , and so on.  The data containing 0 is displayed at the end.

 

For String fields, the following sort can be applied:

Sort A to Z : This sort displays the data in the ascending order i,e. the value starting with A is displayed first, and then B, C and so on. The value starting with Z is displayed last.

Sort Z to A:  This sort displays the data in the descending order.
