Performing Grouping on Publisher Report

 

Prior to performing grouping on the publisher report in ORBIT and view the output in Excel, you must have the template ready with required filter (for example: State).  For more information, see Creating an Excel Template for Grouping.

 

Applying Grouping to the Publisher Report

 

To Apply Grouping to the Publisher Report

 

  1. Under the Bands section, MainBand appears.

 2. Under the MainBand, perform the following:

    a.  On the MainBand, right-click and then select New Band to create a new band.

           (OR)

        Hover the cursor on the MainBand, click , and then click New Band.

 

        Example: New Band: Header

   

   Here, “Header” is for the header band that we created in the excel sheet.

 b.  On the Header band, right-click, and then click New Query.


 c. From the Business object area, drag and drop the columns, based on which the grouping must be performed, on to the Configurator area.

        Example: Columns: State

                                        Sales

  • Here, perform the aggregation on the “Sales” as the sum of the sales is required state-wise.

  • Now, create a data band to extract the data under Header. Since Data band is needed for every group, place the Data band under Header

 d. On the Header band, right-click, and then select New Band to create a new band.

    (OR)

     Hover the cursor on the Header band, click , and then click New Band.

  

  • Example: New Band: Data

 

e. On the Data band (highlighted as “5”), right-click, and then select New Query.

 

        (OR)

      Hover the cursor on the Data band, click  , and then click New Query.


 f. A new query option (highlighted as “6”) appears.


  

g. Select the query option.

h. Query Details (highlighted as “7”) and Query (highlighted as “8”) sections appear.


i. For the query under Data band, drag and drop the columns from Business Object area to the Configurator under Query Details.

     

 j. Use the calculation column to create the own formula (highlighted as “9”) by clicking on New Calculation Column.

  

    
k. Under the Query section, a query is built based on the dragged columns.

   

  Since the grouping is done state-wise, add the data filters for the Query under Header.

 

l. Select the query under Data band, and then select the Data Filters tab under Query Details.


m. From the Business Object, area drag and drop the required column (highlighted as “11”), which works as a filter, onto the Data Filters tab (highlighted as “10”).

 

  • Example: Filter Column: State

 

n. For the required filter, click Show multi select filter.


    

    o. From the “Show items when the value” list, select the required option.

 

  • Example: Required Option: equals

 

p. Under the list, in the textbox, enter the value for the filter.

 

  • Example: Filter Value: ${Header.COL0}

 

q. State column in the configurator area is at COL0.

r. Save the report under appropriate folder, by clicking on Save.

s. Once you have saved the report, you must apply the template of the required format. See Applying Excel Template for Publisher Report  to apply template.

t. The report appears as follows:

 

 

  • Grouping is possible only in Excel and HTML; grouping cannot be done in Word templates.
