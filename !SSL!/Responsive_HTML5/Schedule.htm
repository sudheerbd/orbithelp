Schedule

Schedule tab is in disabled mode, by default.

 

To enable the Schedule tab, select Run Immediately checkbox (highlighted as “1”).

 

 

                                       Figure: Displaying Schedule Tab

 

To schedule a process/request

1.      In the Start Date box, type or select the required date.

2.      In the End Date box, type or select the required date.

3.      In the Start Time box, type or select the required time.

4.      In the End Time box, type or select the required time.

5.      Schedule section provides two options:

·         Simple (highlighted as “2”)

·         Advanced (highlighted as “3”)
