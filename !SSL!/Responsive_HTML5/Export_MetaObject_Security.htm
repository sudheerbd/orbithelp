Export MetaObject Security

Prior to exporting metaobject security it is essential to export the domains and reports having the corresponding securities.

For more information refer to Exporting Domain and Exporting Report.

 

To Export MetaObject Security  

 1. Open the source instance, and then login with the required credentials

 2. On the left-hand side menu, point to Monitoring, and then click Processes.

 3.  Scheduled Processes page appears with available processes.

       4.   On the right-hand side, click New Process , and then click Export MetaObject Security (highlighted as “1”).

 

            

 

 

      5. Export MetaObject Security dialog appears.

 

         

 

       5. To export the security details for metaobjects, click Submit.

       6. This generates a process (highlighted as "2") under the Scheduled Processes in the Processes section.
