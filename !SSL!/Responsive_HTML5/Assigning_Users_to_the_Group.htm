Assigning Users to the Group

To assign a user to the group

  1. Select the required group, and then click  next to Assigned Users.

  2. A list of users appears.

 

        

                                                

3.      Select the required user from the list.  

 

    • Example: Required User:admin

 

  1. Under Assigned Users, the selected user appears.

 

          

                                        

  1. To save the changes, click Save.
