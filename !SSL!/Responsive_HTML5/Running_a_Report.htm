Running a Report

 

To run a new Report

 

1.      Under the Reports area, the required reports appear.

2.      Right-click the required report, and then click Open (highlighted as “1”).

          

         

            

 

(OR)

 

3.      For the selected report, click  , and then click Open.

 

4.      The report opens in the Run Results area.

     

 

 # To run a report that is already run once, click Run (highlighted as “2”) on the top of the Run Results area.
