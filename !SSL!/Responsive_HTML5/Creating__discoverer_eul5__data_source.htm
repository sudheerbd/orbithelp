Creating discoverer_eul5 Data Source

To create <discoverer_eul5> data source

1.      On the left-hand side menu, point to Business Models, and then click Data Sources.

2.      Data Sources page appears with a list of available data sources (highlighted as “1”).

3.      To create a new data source, click New (highlighted as “2”).

 

       

4.      New Data Source dialog box appears. 

5.      In the Name box, type the data source name.

         ? Example: Name: discoverer_eul5

6.      In the Description box, type the description, if needed.

 

¥    Description field is optional.

 

7.      In the Data Source Type, select the required option from the following:

·         JDBC

·         JNDI

? Example: Data Source Type: JDBC

8.      In the Select Database list, select the required option.

           ? Example: Select Database: Oracle JDBC (Oracle Thin Driver)

 

 9.      In the Username box, type the required username.

            ? Example: Username: eul_us

10.  In the Password box, type the password.

              ? Example: Password:eul_us

 

¥    The username and password are required to connect to the discoverer database.

 

11.  In the Oracle JDBC Settings, perform the following:

a.       In the Host Name box, type the required host name.

         ? Example: Host Name : localhost

         b.      In the Port box, type the required port.

                   ? Example: Port : 1521

         c.       In the Database Name box, type the required database name.

                  ? Example: Database :orcl

 

¥    Driver Source and Data Source Class fields are automatically populated based on the Select Database option.

 

d.      To test the connection, click Test Connection.

e.       To save the changes, click Save.
