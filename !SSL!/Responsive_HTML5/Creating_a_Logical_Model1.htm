Creating a Logical Model

To create a New Logical Object

                

1.      Under Domain Hierarchy area, Logical Models option appears.

2.      To create a new logical model, right-click the Logical Models, and then click New Logical Model.

3.      New Logical Model dialog box appears.

4.      To create a new logical model, perform the following:

 

·         In the Name box, type the name of the new logical model.   

 

        ?   Example: Name: Sample Logical Model

 

·         In the Description box, type the description for the logical model.

 

?     Example: Description: Sample Logical Model Description

 

·         To save the logical model, click Save.

·         To close the dialog box, click Cancel.

 

5.      The newly created logical model appears as shown below:

 

6.      New work area (highlighted as “1”) is created for the newly created logical model as shown below:

 

7.      To pull the tables from physical model into logical model work area, perform the following:

 

 

8. Select the tables under the physical model and drag and drop the tables on to the logical model work area.

 

                         Example: Required Tables: AP_INVOICES

                                                       AP_INVOICES_ALL

 

# To create a simple or complex join between the tables, select the source table and drag the arrow from source table to destination table. You should see a hand symbol when you select the source table, before you drag the arrow. For more information on Joins, see Joins.

 

9. Once the join is created between the source and the destination table, save the logical model.

10. To save the logical model, click Save.
