Creating a Publisher Report

¥    In case, the data of a specific column value is required, then apply the user filter.

 

¥    Grouping is currently unavailable in Word Template.

 

 

 

1. Under the Bands section, MainBand appears.

 

2. Under the MainBand, perform the following:

 

·      On the MainBand, right-click and then select New Band to create a new band.

(OR)

·      Hover the cursor on the MainBand, click , and then click New Band.

 

?   Example: New Band: Header

 

 

 

¥   Here, “Header” is for the header band that we created in the excel sheet.

 

·      On the Header band, right-click, and then click New Query.

 

·      From the Business object area, drag and drop the columns, based on which the grouping must be performed, on to the Configurator area. 

 

? Example: Columns: State

                                                   Sales

 

 

¥    Here, perform the aggregation on the “Sales” as the sum of the sales is required state-wise.

 

·         To view the single-state data, add a user filter to the report.

 

·         If the User filters section is already available drag and drop the required column onto the user filters section.

·         If User filters section is not already available, on the report menu, click  Hide/Show User Filters.

 

·    The User Filters section appears.

 

·         Drag and drop the required column onto the user filters section.

 

·         Hover the cursor over the required column, and click Filter Definition.

·         Filter-Column Name dialog appears.

·         In the Filter-Column Name dialog, in the Name box, enter the name for the filter, and then click Ok.

?   Example: Name: state

 

     

 

 

·         Under the Bands section, under MainBand, under Header band, select the required query.

·         For the required query, select the Data Filters tab, and then drag and drop the required filter column.

·         Enter the required fields as shown below (highlighted as “1”):

 

      

?   Example: Filter Column: State

 

·         For the required filter column, click Show multi select filter.

 

·         From the “Show items when the value” list, select the required option.

 

?  Example: Required Option: contains

 

 

·         Under the list, in the textbox, enter the value for the filter.

 

 

?Example: Filter Value: :state

 

 

¥   Now, create a data band to extract the data under Header.

¥   Since Data band is needed for every group, place the Data band under Header band.

 

 

·   On the Header band, right-click, and then select New Band to create a new band.

 

(OR)

 

       Hover the cursor on the Header band, click , and then click New Band.

 

?  Example: New Band: Data

 

 

·   On the Data band (highlighted as “5”), right-click, and then select New Query.

 

    (OR)

                       Hover the cursor on the Data band, click , and then click New Query.

·   A new query option (highlighted as “6”) appears.

 

 

·   Select the query option.

·   Query Details (highlighted as “7”) and Query (highlighted as “8”) sections appear.

 

 

 

·   For the query under Data band, drag and drop the columns from Business Object area to the  Configurator under Query Details.

·         Since the grouping is done state-wise, add the data filters for the Query under Header.

 

·         Select the query under Data band, and then select the Data Filters tab under Query Details.

 

·         From the Business Object area, drag and drop the required column (highlighted as “2”), which works as a filter, onto the Data Filters tab (highlighted as 1).

 

?   Example: Filter Column: State

 

·         For the required filter, click Show multi select filter.

 

               

 

 

·         From the “Show items when the value” list, select the required option.

 

?   Example: Required Option: equals

 

 

·         Under the list, in the textbox, enter the value for the filter.

 

?   Example: Filter Value: ${Header.COL0}

 

 

·      State column in the configurator area is at COL0.

·      Save the report under appropriate folder, by clicking on Save.
