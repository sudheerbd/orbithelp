To set the Widget Settings

1.      In the report configurator area, click Widget Settings .

2.                  Under Widget Settings, the following tabs appear:

a.       General

b.      Category Axis

c.       Value Axis

d.      Data Labels

e.       Legend

f.        Styles 

 

General

The widget settings under General tab are similar to that of Column Basic Charts. See General Tab  under Column Basic Charts for more information.

 

Category Axis

The widget settings under Category Axis tab are similar to that of Column Basic Charts. See Category Axis under Column Basic Charts for more information.

 

Value Axis

The widget settings under Value Axis tab are similar to that of Column Basic Charts. See Value Axis under Column Basic Charts for more information.

 

Data Labels

To view the other common widget settings, see Data Labels tab under Column Basic Charts.

Apart from the widget settings in the above-mentioned link, the following are the additional settings available under the Data Labels tab.  

1.      In the Data Labels box, type or select the required option from the following:

a.      None

b.      Under

c.       Over

d.      Rotate

Legend

The widget settings under Legend tab are similar to that of Column Basic Charts. See Legend Tab of Column Basic Charts for more information.

 

Styles

The widget settings under Styles tab are similar to that of Column Basic Charts. See Styles Tab of Column Basic Charts for more information.
