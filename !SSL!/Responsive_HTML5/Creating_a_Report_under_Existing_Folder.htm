Creating a Report under Existing Folder

To create a new analysis report under an existing folder

 

1.      On the Reports menu, under the Reports section, right- click the required folder.

  

               Example: Required Folder: Superstore

 

  1. A menu appears as shown below (highlighted as “1”):

 

 

  1. From the menu, click New Analysis.
  2.  A new analysis report opens.
