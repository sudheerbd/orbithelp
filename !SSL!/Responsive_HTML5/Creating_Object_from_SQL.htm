Creating Object from SQL

To create an object from SQL

 

1.      Under Domain Hierarchy, Physical Models folder appears.  

2.      Under the folder, right-click the required folder, and then click  Create Object From SQL

 

                       

3.      Create Object From SQL <physical model name> dialog box appears.

 

                      

4.       Under Create Object From SQL <physical model name> dialog box, perform the following:

a.       In the Schema Name list, click the required schema.

 

  ?   Example: Schema Name: XXFO

 

  • Schema Name is mandatory.  

 

b.      In the Object Name box, type the object name.

 

   ?  Example: Object Name: INVOICES_ALL

 

    •  Object Name is mandatory.

 

c.       In the Description box, type the description.

 

             ? Example: Description: This is test description

 

d.      In the Sql query box, type the SQL Query.

 

              ?   Example: Sql query : Select * from INVOICES_ALL

    •   A semicolon (;) should not be placed at the end of the SQL Query

 

e.       To create an object, click Create.

f.       To close the dialog box, click Cancel.
