Calculation Column

To work with Calculation Column

 

1.      In the Reports section, right-click the required report, and then click Open.

               ?     Example: Required Report: Test Invoice Report

 

                        

 

2.      The required report opens in the Run Results area.

3.      To view the report configurator area, click Edit Mode.

4.       The report appears with the Business Object and Report Configurator areas.

 

 

5.      In the Report Configurator area (highlighted as “1”), click New Calculation Column (highlighted as “2”).

6.      Calculation Formula dialog box appears.

   

        

                                       

 

7.      In the Name box, type the name of the formula.

 

                ?     Example: Formula Name: Sum of Sales and Profit  

 

8.      In the Datatype list, click the required data type from the following:

 

  • Number

  • Text

  • Date

  • Integer

 

        ?     Example: Required Data type: Number

 

9.      In the Formula area (highlighted as “3”), enter the formula.   

10.  To enter a formula in the calculation area, perform the following:

a.       In the Function Types section on the right-hand side of the Calculation Formula dialog window, click the required function.

 

                   ?     Example: Required Function Types: Aggregation Functions

 

b.      The functions based on the function types appears in the Functions section below the Function Types section.

 

c. In the Function box, type the search term for the required function.

d. The relevant functions matching the search term appear (as shown below) under Functions.

 

 

e.       In the Functions section, double-click the required function.

 

          ?  Example: Aggregation Function: SUM

 

f.      In the Formula area, the selected function appears.

g.       In the formula, drag and drop the column name or type the column name.

 

                     ?  Example: Formula: SUM(["SuperStore"."Sales"]) + SUM(["SuperStore"."Profit"])

  

h.        The formula appears in the formula area as shown below:

 

 

i.      To validate the formula, click Evaluate.

j.     To save the changes to the formula, click Ok.

¥    The functions that are supposed to be used in “WHERE” clause of a query cannot be used in the Calculation Formula.     

 

Editing the Formula

 

 To edit the formula of the newly created column, perform the column:

1.      Click Settings , and then click Formula on the menu. 

2.      Calculation formula dialog box opens with already existing formula.

3.      Edit the formula, and then click Ok.

 

Customizing the formula

 

The User can type their own formula using different operators.

 

To create a custom formula

 

1.      In the Name box, type the name of the formula.

2.      In the Datatype box, select/type the required data type. 

3.      Drag and drop the first required column from Business Object area onto the Formula area in Calculation Formula dialog window.

4.      From the operators (highlighted as “1”), select the required operator.

5.      Drag and drop the second required column or type a number.

 

                     ?   Example: First Required Column: Sales

                                              Operator:  +

                                 Second Required Column: Shipping Cost

 

?  Example: Formula: ["SuperStore"."Sales”] + ["SuperStore"."Shipping Cost"]

 

            

 

6.      To validate the formula, click Evaluate.

7.      To save the changes, click Ok.
