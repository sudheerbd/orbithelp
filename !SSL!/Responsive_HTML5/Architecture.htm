Architecture

ORBIT (Operational Reporting Business Intelligence Tool) helps the users to view and analyze data, analysis and dashboard reports. 

 

 

 

 

                                                   

The above figure depicts the ORBIT Architecture. The data is sent from the database to Application Server and vice-versa. ORBIT application is deployed onto the Application Server. ORBIT can be deployed as a client-premise product or as cloud-based product.

 

ORBIT can be deployed onto the following application servers

  • Tomcat

  • Weblogic

  • IBM Web Sphere

  • JBOSS(Wildfly)

 

ORBIT is certified on the following databases:

 

  • Oracle Database 11g, 12c

  • Microsoft SQL Server 2008+

  • MySQL

  • IBM DB2

 

The above architecture encompasses the following scenarios:

Scenario 1: Combination of Application Server and Metadata Database  

Scenario 2 : Application Server with separate database

Scenario 3: Clustered Servers and separate database.

 

Clustered Servers are needed for 100+ concurrent users for distributing workload.
