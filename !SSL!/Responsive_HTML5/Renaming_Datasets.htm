Renaming Datasets

 

To rename a dataset

 

1.      In the Datasets area, a list of available datasets appears.

 

                                      

 

2.      To rename a dataset, click Rename.

 

                                                  

 

3.      Edit <Dataset name> dialog box appears.

 

       

                       

 

4.       In the Edit<Dataset name> dialog box, perform the following:

a.       In the Name box, type the new name for the dataset.

b.      In the Description box, type/edit the description for the dataset.

5.      To save the changes, click Save.

6.      To close the dialog box, click Cancel.
