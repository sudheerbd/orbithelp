Creating a Dashboard

To Create a Dashboard Report

1.      In Reports, click New Report, and then click Dashboard .

2.      Business Object area appears along with Run Results area.  

3.      Dashboard reports can be created in either of the following ways:

a.       Drag and drop the already existing reports from Reports area to Run Results area.

 

?   Examples: First Report: PO Analysis

                             Second Report: hierarchy Test                           

 

                                 

(OR)

 

b.      Select a business object and then drag and drop the required columns from Business Object area to Report Configurator area.

 

      Example: Required Columns: Y- Axis: Amount_Paid    

                                                   X-Axis: Invoice_Id          

                                      Color: Invoice_Date
