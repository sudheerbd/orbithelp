Creating a Report Token

To Create a Report Token

1. Open the required report.

  • Example: Required Report: SuperstoreToken

2. The report can be anyone of the following:

  • Data Report

  • Analysis

  • Dashboard

  • Publisher Report

  • Script Report

3. In Edit Mode, click Options , and then click Generate Link .

 

  

4. Generate Report Link dialog appears.

 

  

 

5. In the Generate Report Link dialog, select the required option from the following:

  • Private

  • Public

 

  •    Example: Required Option: Public

 

6. In the Expiry Date box, select/type the required expiry date of the report link.

 

  • Example: Expiry Date: 01/09/2018

 

7. In the Expiry Time box, select/type the required expiry time of the report link.

 

  •  Example: Expiry Time: 3:00 PM

 

  

 

8. To generate the report link, click Generate.

  • When the user clicks on Generate, a Public link is generated with the expiry date/time mentioned in the expiry date/time box. Once the Expiry time is complete, the report token in the public link gets expired.

  • When the expired token is run in the browser and then if the Report Token page is refreshed, the expired token gets deleted from the list of tokens under Report Tokens.

 

 

9. To close the dialog, click Close.

 

Viewing Report Token

 

To view the report token

 

1.  In the left-hand side menu, point to Monitoring, and then click Report Tokens

2. Report Tokens page opens with a list of report tokens.

 

 

 

3. The report token that appears in the report link appears as shown above.

4. My Tokens displays only the tokens that are created by the specific user.

5. All Tokens displays all the report tokens.

  • When the Token is in the active state, "Active" appears under the Status; when the token gets expired, "Expired" appears under the Status.  

 

Deleting a Token

 

To delete a Token

1. Select the required report token.

2. For the required token, a checkmark appears in the checkbox before the Report Name.  

3. To delete the token, click Delete.
