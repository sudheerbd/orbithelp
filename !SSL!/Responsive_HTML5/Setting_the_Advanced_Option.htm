Setting the Advanced Option

To set the Advanced Option

1.      On the Advanced tab (highlighted as “1”), under Months section (highlighted as “2”, select the month in which the request has to be run.

 

                                       Figure: Displaying Advanced Tab

 

2.      Under Dates area (highlighted as “3”), select the required dates of every month.

3.      Under Days area (highlighted as “4”), select the required days of every month.

4.      To submit the request, click Submit (highlighted as “5”).

  

      

                                                             Figure : Displaying Days

 

 

  • The user has to select either Simple tab or Advanced tab. The system takes the options that have been entered at the end.
