Changing Date Field Format in Report

To change Date field format in report

 

1.      On the left-hand side menu, point to Business Models, and then click Meta Objects. 

2.      Meta Objects page appears with a list of Meta Objects (highlighted as “1”) on the left-hand side. 

 

 

3.      To change the formatting of Date Field in the reports, perform the following:

a.       Under the Base Objects (highlighted as “2”), select the Date Field (highlighted as “3”).

b.      Name and Value columns (highlighted as “4”) for the Date Field appear.

 

 

c.       Under the Format tab, in the Format box, type the required format.

¥    In the Format tab, specify if the number has decimals.

 

  Example: Required Date Format: dd/MM/yyyy

 

d.      Under the Style tab, perform the following:

                                                                    i.            In the Alignment box, type/select the required option.

 

 Example: Required Option: left

                                                                

                                ii.       In the Column Width box, type/select the required value.

 

Example: Required Value: 130

 

                                                                iii.            In the Font Family box, type/select the required values by clicking .

                                iv.      A dialog window appears with multiple fields.

                                v.     Edit the required fields, and then click Ok.

 

 

 

      Example: Font Family: Arial

                    Font Size (px): 14

                    Font Style: Bold

                    Effects: Underline

                    Color: 638888

 

                                                                vi.            In the Hide box, type/select the required value.

 

?   Example: Hide: false

 

 

4.      To save the changes, click Save.
