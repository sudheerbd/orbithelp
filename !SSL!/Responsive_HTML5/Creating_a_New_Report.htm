Creating a New Report

To create a new report

1.      To create a new report, click New Report (highlighted as “1”).

2.      New Report provides the following options (highlighted as “2”):

o   Data Report

o   Analysis

o   Dashboards

o   Publisher Report

      3.  On the Reports section, perform the following to create various reports.

·         To create a data report, click  New Report, and then click Data Report.

·         To create an analysis report, click New Report, and then click  Analysis.

·         To create a dahsboard, click New Report, and then click Dashboard.

·         To create a publisher report, click  New Report, and then click Publisher Report.

 

Alternate Procedure

In the Reports section, in the main menu on the top, click the downward arrow  next to New (highlighted as “1”),  click the required report from the following, and then click Ok.

·         Data Report

·         Analysis

·         Dashboards

·         Publisher Report

·         Script Report
