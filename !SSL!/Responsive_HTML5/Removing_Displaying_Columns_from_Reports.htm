Removing/Displaying Columns from Reports

 

To remove/display the columns from a report

 

1.  In the Run Results area, move the cursor over the required column.

2. Downward arrow  appears beside the column name.

3. To remove the column from a report, click  beside the required name, point to Columns, and then remove the checkmark before the required value, if the value is checked.

  

      

                                                  

 

4.  To display the column in a report, click  beside the required name, point to Columns, and then select the checkbox before the required value in the list.
