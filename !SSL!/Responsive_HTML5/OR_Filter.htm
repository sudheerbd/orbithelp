OR Filter

The OR Filter is used when there are more than one condition and when at least one of the conditions must be true for the statement to be true. It is denoted by a pipe symbol “|”.

Consider the following example, where the OR Filter is used.

The search filter (|(givenName =Test)(sn=Kir*)) retrieves all the users whose given name starts with Test; also, it retrieves the users whose lastname starts with “Kir”.
