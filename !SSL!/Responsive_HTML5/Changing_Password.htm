Changing Password

 

To change the password

1.      On the top menu, click the image on the top right-hand side of the window.

2.      A window opens as shown below:

 

                                 

3.      To change the password, click Change Password.

4.      Change Password dialog box appears.

 

      

                           

5.      To change the password, perform the following:

o   In the Old Password box, type the existing/old password.

o   In the New Password box, type the new password.

o   In the Confirm New Password box, re-type the new password.

6.      To change password, click Submit (highlighted as “2”).

7.      To reset the fields, click Reset (highlighted as “3”).

8.      To close the dialog box, click Cancel (highlighted as “4”).
