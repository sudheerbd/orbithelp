Migrating Worksheets into ORBIT Analytics

Prior to migrating the worksheets into ORBIT Analytics application, the user must have exported the. eex file from discoverer desktop administrator. See exporting the eex file for more information.

 

To migrate the worksheets

 

¥Generate EEX file from discoverer desktop administrator. Once the EEX file is generated, you need to import the file into the monitoring section in ORBIT Analytics application.

     

1.      On the left-hand side, point to Monitoring, and then click Processes.

2.      Scheduled Processes page appears with a list of available processes.

3.      To import the eex file, click New Process on the right-hand side corner of the window, and then click Import Discoverer Object(s) (highlighted as “1”).

 

 

4.      Import Discoverer Object(s) dialog window appears.

 

    

    

 

5.      Under Parameters, click Upload (highlighted as “2”) next to the Upload File box.

6.      Open dialog box appears with required files.

7.      Select the required eex file, and then click Open.

 

8.      The selected eex file (highlighted as “3”) appears in the Upload File box.

9.      To submit the file, click Submit.

10.  Under Scheduled Processes, the newly submitted file appears.
