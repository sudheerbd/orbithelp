Editing Datasets

To edit datasets

1.      On the required dataset, click Edit (highlighted as “1”).

 

                                                     

 

2.      Domain Hierarchy (highlighted as “2”) appears for the selected dataset.

 

       

                    

3.      For the selected dataset, Physical Model, Logical Model and Business Object appear.

4.      To edit the existing dataset, make changes to the physical model, logical model, or business object.

5.      To save the changes, click Save.

 

¥    While editing the dataset, you can make any changes to Physical Model, Logical Model, and Business Object including importing objects and creating joins.
