Create an Analysis Report

To create an Analysis Report

 

In the Analysis section, multiple charts are available to display the data.

To create an analysis report

1.      In Reports, click New Report, and then click Analysis.

2.      Data Explorer appears along with Run Results area.

3.      In Data Explorer, click  next to Select Business Object.

4.      Search area appears with available business objects .

5.      Select the required business object, and then click OK.

(OR)

Double-click the required business object.

 

            ?   Example: Required Business Object: Sample Business Object

 

6.      In Business Object area, the selected business object or the dataset appears with the required columns.  

           7.      Drag and drop the required columns from Business Object area to different sections in report configurator area.

·         Report Configurator area is divided into various sections depending on the type of chart. In this scenario, Column Basic chart is used

 

8.      Under Values (Y-axis), drag and drop the column that needs to appear on the Y-axis, from Business Objects area.

 

                   ?   Example: Required Column for Y-axis: AMOUNT_PAID

 

9.      Under Category (X-axis), drag and drop the column that needs to appear on the X-axis, from the Business Objects area.

 

                   ?   Example: Required Columns for X-axis: INVOICE_ID                                                                                

 

10.  Under Color, drag and drop the fields from Business Objects area that need to appear in color.

?   Example: Required Value for Color: INVOICE_NUM

11.  The report is automatically generated in run results area as follows:

 

¥     In Query Settings, if Auto Refresh is not selected, then the report is not generated automatically. You need to click Apply beneath Color section in the Configurator tab or click Run above the run result area to run the report.

 

12.  To change the type of chart, click Change widget in the report configurator area next to Query Settings icon.

 

13.  Select the required type of chart from the various available options in the following ways:

1.      Select the required chart option, and then click Ok.

(OR)

2.      Double click the required chart option.

14.  Following are the various available chart options:

·         Table

·         Pivot/Matrix

·         Bar Basic

·         Stacked Bar

·         100% Stacked

·         Column Basic

·         Stacked Column

·         100% Stacked

·         Multi Axis

·         Negative Values

·         Line Basic

·         Spline

·         Step Line

·         Area

·         Area Spline

·         Combination

·         Pie Basic

·         Pie Donut

·         Scatter

·         Bubble

·         Radar

·         Gauge

·         Tree Map

·         Sun Burst

·         Heat Map

·         Map

·         Card

¥     Widget Settings  change for every selected chart options

 

15.  The report appears in the run result area.

16.  To save the report, click Save.
