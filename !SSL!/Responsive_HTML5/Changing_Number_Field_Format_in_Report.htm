Changing Number Field Format in Report

 

To change Number Field Format in Report

 

1.      On the left-hand side menu, point to Business Models, and then click Meta Objects.

2.  Meta Objects page appears with a list of Meta Objects (highlighted as “1”) on the left-hand side.

     

        

3.      To change the formatting of number fields in the reports, perform the following:

 

a.       Under the Base Objects (highlighted as “2”), select the Number Field.

b.      Name and Value fields (highlighted as “3”) for the Number Field appear.

 

c.       Under the Format tab, in the format box, type the required format.

                                                                                                               

d.      Under the Style tab, perform the following:

 

                                                                    i.            In the Alignment box, type/select the required option.

                                                                  ii.            In the Background Color box, select the required color.

                                                                iii.            In the Column Width box, type/select the required value.

                                                                iv.            In the Font Family box, type/select the required values by clicking .

                                 v.      A dialog window appears with multiple fields.

                                 vi.     Edit the required fields, and then click Ok.

 

                                          

 

                                                                  vii.            In the Hide box, type/select the required value.

 

4.      To save the changes, click Save.
