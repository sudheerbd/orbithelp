Importing Users from Authority

To Import Users from Authority

1.      On the required authority, click Import users from authority.

 

    •  Example: Required authority: Test Authority

 

      

                                          

2.      Import Users dialog box appears.

 

      

                                               

3.      In the textbox before the Search button (highlighted as “1”), type the user name or the search term, and then click Search.

4.      A list of available users appears under Available Users list.

5.      Drag the required user and drop it under Selected Users.

7.      To import the user, click Import.

 

       

                                        

8.      To close the dialog box, click Cancel.

9.      To sync the users and roles to the selected authority, click Sync users and roles .
