Editing a Report

 

To Edit a Report

1.      Under the Reports area, the required report appears.

2.      Right-click the required report, and then click Edit (highlighted as “1”).

 

        

             

3.      The required report opens in Edit Mode (highlighted as “2”).

 

      

                                      

4.      Edit/Update the report, and then click Save.

 

5. To edit the name of the report on the run results area, you can click it once and edit it with required name.

 

 

Browser Constraint: As opposed to other browsers, in Internet Explorer, to edit the name of the report on the run results area, you must double-click the name.; the name that appears unavailable initially, becomes available once it is double-clicked.
