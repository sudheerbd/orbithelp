Creating an Analysis Report

 

In the Analysis section, multiple charts are available to display the data.

To create an analysis report

1.      On Reports, click New Report, and then double-click Analysis (highlighted as "1") .

 

      (Or)

     On Reports, click New Report, click Analysis (highlighted as "1") , and then click Ok.

 

       

              

 

2.  Analysis report appears with the following two tabs on the left-hand side (highlighted as "1").

 

  • Data Explorer (highlighted as "2")  –  allows to select the required business object

  •  Widgets (highlighted as "3") –  displays the available reports.

3. Run results area appears on the right-hand side (highlighted as "4").

 

                           

 

3.     On the Data Explorer tab , click  next to Select Business Object (highlighted as”4”).

4.      Search area appears with available business objects (highlighted as “5”).

 

                                       

 

5.      Select the required business object, and then click OK.

(OR)

Double-click the required business object.

(OR)

To create a dataset, click Upload File to upload dataset file. 

6.      Under the Data Explorer, the selected business object or the dataset appears with the required columns.  

        

       

      

 

7.      Drag and drop the required columns from Data Explorer to different sections in report configurator area.

8.    Report Configurator area is divided into the following sections:

  •  Measures (Y-axis) (highlighted as “5”)

  •  Dimensions(X-axis) (highlighted as “6”)

  •  Color (highlighted as “7”)     

 

 

 

9.      Under Measures (Y-axis), drag and drop the column that needs to appear on the Y-axis, from Business Objects area.

           

                        Example: Required Column for Y-axis: SUM(Sales)

10.      Under Dimensions (X-axis), drag and drop the column thatneeds to appear on the X-axis, from the Business Objects area.

 

                          Example: Required Columns for X-axis: Region

                                                                                    State

11.      Under Color, drag and drop the fields from Business Objects area that need to appear in color.

  

                            Example: Required Value for Color: Product Category

 

  12.      The report is automatically generated in run results area as follows:

   
                                

 

9.      Under Values (Y-axis), drag and drop the column that needs to appear on the Y-axis, from Business Objects area.

10.  Under Category (X-axis), drag and drop the column thatneeds to appear on the X-axis, from the Business Objects area.

11.  Under Color, drag and drop the fields from Business Objects area that need to appear in color.

12.  The report is automatically generated in run results area as follows:

 

      

                                                                 

 

¥     In Query Settings, if Auto Refresh is not selected, then the report is not generated automatically. You need to click Apply beneath Color section in the Configurator tab or click Run above the run result area to run the report.

 

¥         Data in analysis reports does not come in the same order always. If the user needs the same order every time, then sorting can be enabled.

 

1.      To change the type of chart, click Change widget (highlighted as “9”) in the report configurator area next to Query Settings icon.

 

 

1.      Select the required type of chart from the various available options (highlighted as “10”) in the following ways:

a.       Select the required chart option, and then click Ok.

(OR)

b.      Double click the required chart option.

2.      Following are the various available chart options:

·         Table

·         Pivot/Matrix

·         Bar Basic

·         Stacked Bar

·         100% Stacked

·         Column Basic

·         Stacked Column

·         100% Stacked

·         Multi Axis

·         Negative Values

·         Line Basic

·         Spline

·         Step Line

·         Area

·         Area Spline

·         Combination

·         Pie Basic

·         Pie Donut

·         Scatter

·         Bubble

·         Radar Basic

·         Radar Filled

·         Sun Burst

·         Gauge

·         Tree Map

·         Map

·         Card

¥     Widget Settings change for every selected chart option

 

See Different Chart Outputs and their Settings

3.      The report appears in the run result area.

4.      To save the report, click Save.

5.      Save Report dialog box appears. 

 

 

6.      If the required report folder already exists, then perform the following:

·         In the Name box, type the report name.

·         In the Description box, type the description.

·         To save the report, click Save.

7.      If the required report folder does not exist, then create a new reports folder, and then save the report in it. See Creating a Reports Folder.
