Creating a New Group

 

To create a new group

  1. On the User Management menu, click Groups.

  2. Group List appears with a list of available groups.

  3. To create a new group, click New.

  4. New Group dialog box appears.

 

          

                                        

 

  1. To create a new group, perform the following:

路         In the Name box, type the name.

    • Example: Name : Test Group

The Name must be of at least 4 characters

路         In the Description box, type the description.

    • Example: Description: This is a Test Group

 

  • Name and Description are mandatory.

 

路         To save the changes, click Save.

路         To close the dialog box, click Cancel.
