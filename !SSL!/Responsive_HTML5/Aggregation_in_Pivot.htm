Aggregation in Pivot

To set the aggregate functions in Pivot

1.      From the All fields area, drag and drop the required fields to Values.

2.      Under Values area, click the required field, and then click the required aggregate function (highlighted as “1”).

 

  

                                                              

 

3.      The following aggregate functions are available under Values area.

·         Sum

·         Avg

·         Count

·         Max

·         Min

·         Group sum percentage

·         Group count percentage

·         StdDev

·         StdDevp

·         Var

·         Varp
