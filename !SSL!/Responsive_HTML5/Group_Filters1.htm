Group Filters

To create group filter

1. Open the analysis report, and place required columns as user/data filters.

2. Assign the values to the user/data filters.

 

To work with group filters/change the Logical conditions, perform the following:

3. On the menu above the run reports area, click  Options, and then click Group Filters.

4. Group Filters dialog appears.

5. In the Group Filters dialog, the specified parameters appear with the default logical operator assigned.

 

 

¥  AND operator is assigned, by default.

 

 

      

 

6. To change the logical operator, click the already existing logical operator as shown above.

 

   

                        

7. A menu appears with the list of available logical operators.

8. From the menu, click the required logical operator, and then click Ok.
