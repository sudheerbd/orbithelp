Properties

Data Reports, Analysis, and Dashboards have the same properties. Publisher Reports and Script Reports do not have the properties.

 

To set the Properties 

 

1.      Under Reports area, right-click the required report, and then click Edit/Run.

 

¥    Properties appear only in Edit Mode.

¥    The report can be either Data Report, Analysis or Dashboard.

2.      To set the properties, click Options menu, and then click  (highlighted as “1”) above the Run Results area.

 

 

 

3.      Properties dialog window appears.

4.      On the left-hand side, the selected report name (highlighted as “1”) and the columns (highlighted as “2”) under it appear.

 

 

5.      On the right-hand side, the properties of report name and the columns appear.  

6.      For the selected report (highlighted as “3”), the following properties (highlighted as “4”) appear:

a.       Under Name, type the name of the report. 

b.      Under Description, type the description of the report.  

c.       Under the Report Run Type, from the Run Type box, select any one option from the following:

                                                              i.      Schedule Only

                                                            ii.      Run & Schedule (BOTH)

Schedule Only: This option only schedules the report.

Run & Schedule: This option schedules and runs the report as well.

 

7.      For the required Measures (Sales), the following properties appear:

a.       Under Name, type the name of the attributes/dimensions/measures.

b.      Under Description, type the description of attributes/dimensions/measures.

c.       Under Grouping & Summary tab, perform the following:

                                                        i.            In Grand Total list, click one of the following options:

o   True

o   False

                                                      ii.            In Group Total list, click one of the following options:

o   True

o   False

                                                    iii.            In Summary list, click one of the following options:

o   Sum

o   Avg

o   Min

o   Max

d.      Under Format tab, perform the following:

                                                             i.      In the Format box, the format of the measure should be as follows:

                               

                                    Example: Required Format: ####,##,##0.00

 

        For example, if there is a number column in the report with three decimals ".123"; to specify it as "0.123", the number format has to be changed from "#,###,###.000" to "#,###,##0.000".

 

           This reflects in the report properties, and the format is applied to the values in the report.

 

            Initially, the format appears as "####,##,###.00", you must change the required format to ####,##,##0.00.

                

  •  To edit the format field, click in the format text box, and then click .

  •  Format dialog appears.

  •  In the Decimal Places box,  type/select the required number of decimal place.

  •  In the Currency Symbol box,type/select the required currency symbol.

  • To use the 1000 Separator, select the Use 1000 Separator  (,) check box.

  • To show the negative numbers in brackets, select the Show Negative Numbers in Brackets checkbox.

  •  The preview of the selected fields appears in the Preview box.

  • To save the changes, click Ok.

 

 

                              

 

At the report level, in the properties dialog, the "date/time" field can have the various formats which can be used using the following notation:

 

y : Year ( eg. 12); also use either yy or yyyy

M: Month (eg: Nov); also use either MM, MMM, or MMMM

d : Day in a Month ; can also use dd

h: hour of the day- 1 to 12 (AM/PM); can also use hh

H : Hour of the day - 0 to 23 ; can also use HH

m: minute in hour ( 0-59 ); can also use mm

s: seconds in a minute (0-59); can also use ss

SSS: Milliseconds in a second; 0-999

E: Day of the week; also use either EE, EEE, EEEE, or EEEEE

a: Uppercase Ante Meridiem (AM) /Post  Meridiem (PM)

Z: Difference to Greenwich time (GMT) in hours and minutes

 

The user can either type the above notations or simply select the required date/time options from the format box.

 

f.        Under Style tab, perform the following:

                                                              i.      In the Alignment box, type/select the required option from the following:

o   Left

o   Center

o   Right

                                                            ii.      In the BackGround Color box, type/select  the background color.

                                                          iii.      In the Cell Wrap box, select the required option from the following:

·         True

·         False

 

                                                          iv.      In the Column Width box, type/click /  to enter the column width.  

                                                            v.      In the Font Family box, type the font family.

(OR)

Select  to perform the following:

·         In the Font Family box, type/select the font family.

·         In the Font Size box, type/select the font size.

·         In the Font Style, select the required checkbox from the following:

o   Italic

o   Bold

·         Under Effects area, select Underline checkbox to underline.

·         To set the color for the underline, click the required color from the color palette. 

·         Under Preview area, the preview of the selected changes appears.

·         To save the changes, click Ok.

                                                          vi.      To hide the dimension/measure, click Hidden box and then select the required option from the following:

o   True

o   False 

g.      To save the changes, click OK.

h.      To close the dialog box, click Cancel.          

 

8.      For the required Dimensions (State), the following properties appear:

·         All the properties of Dimensions are similar to the properties of Measures; the only difference is that the Dimensions have a Grouping column under Grouping & Summary tab.

·         Under the Grouping & Summary tab, in the Grouping box, select the required option from the following:

§  True

§  False 

 

The only difference here is that the measures do not have the Grouping option under Grouping & Summary tab; while dimensions have grouping option under Grouping & Summary tab.
