Active Directory

Active Directory (AD) is a Windows OS directory service that facilitates working with interconnected, complex, and different network resources in a unified manner.

When the user enters only Base DN, the server starts its search from the root, and retrieves all the users. Under the Base DN, Common Name (CN) is created in Active Directory. Here, the common names are created as follows:

Once the structure is organized, the users/roles can be imported into Orbit.

In Orbit application, creating an AD authority in the Authorities section under User Management, enables authorization and authentication to import users and roles into Orbit from Active Directory.

Orbit enables authentication to import users from AD and authorization to import roles from authorities including Oracle EBS, PeopleSoft, Active Directory, and LDAP.
