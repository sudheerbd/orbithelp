Pie Basic

In this type of chart, the output is generated as Pie Basic.

 

Widget Settings

To set the widget settings

 

1.      Under the 3D Properties tab, perform the following:

a.       To set the thickness, slide the Thickness slider left or right.

b.      To set the distortion, slide the Distortion slider left or right.

c.       To set the bevel width, slide the Bevel Width slider left or right.

d.      To set the color spread, slide the Color Spread slider left or right.

e.        To set the opacity, slide the Opacity slider left or right.

2.      Under the General tab, perform the following:

a.       In the Title box, type the title for the report.

b.      In the Chart Type box, type the required chart type.

c.       In the Type box, type/select the required option from the following:

                                                  i.      Normal

                                                ii.      3D

d.      To set the donut size, slide the Donut slider left or right.

e.       In the Theme box, select/type the required option from the list.

3.      Under the Labels tab, perform the following:

a.       In the Display box, type/select the required option from the following:

                                            i.            Outside

                                          ii.            Inside

                                        iii.            None

4.      Under the Legends tab, perform the following:

a.       In the Hide box, type/select the required option from the following:

                                                  i.      True

                                                ii.      False

 

5.       The following types of charts have the same widget settings:

·         Pie Basic

·         Pie Donut

·         Radar Basic

·         Radar Filled

·         Sunburst

·         Gauge

·         Tree Map

·         Map

·         Card
