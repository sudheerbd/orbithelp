Viewing Recently Run Reports

Procedure 1

 

To view the recently run reports

1.       In the Home page, click plus icon, and then click Recent icon at the bottom-right corner of the page.

 

2.      The recently run reports appear under Recent section.

 

        

                               

3.      To run a report that appears under Recent area, click Run report.

 

         

                                           

4.      To refresh the recent reports, click Refresh (highlighted as “1”). 

 

 

Procedure 2

 

To view the recently run reports

1.       In the Reports module, click Recent tab that appears in the right-hand side section.

2.      The recently run reports appear under Recent tab.

 

 

                                     

3.      To run a report that appears under Recent tab, click Run report.

  

          

                                    

4.      To refresh the recent reports, click Refresh (highlighted as “1”).
