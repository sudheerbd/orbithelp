Creating a Complex Join

To create a Complex Join

 

1.      Select the source table.

¥  When the hand symbol appears on the selected table, a join can be created.

 

2.      Drag the arrow from the selected source table to destination table.

3.      Relationship (Source table =>Destination table) dialog box appears.

4.    To create a complex join, in the Relationship dialog box, select the Complex Join check box. 

5.    Complex Join box appears as shown below:

 

        

 

  6.    In the Complex Join box, perform the following:

· In the Complex Join box, click  to populate the complex join.

 

·  Expression Builder (Complex Join) dialog box appears.

 

 

 

    · In the Expression Builder, under the Category area, select the required option from the following:

o   Available Objects

o   Functions

 

      ? Example: Category: Available Objects

 

· Under the Sub-Category area, select the required table/view/synonyms.

 

Example: Sub Category: AP_INVOICES_V

                                   AP_INVOICE_PAYMENTS_V

                   

      

 

 

¥          Based on the selected sub-category, the columns appear on the right-hand side.

 

·         Double-click to insert the required columns.

 

Example: Required Column: INVOICE_ID

                           

· You can use the available Operators such as +, -,* ,/, !,=,>,<,(,),[,],,;,and %  to create an expression.  

  Based on the selection, the expression appears in the expression box.

 

 

·         To save the changes, click Ok.

 

 

·    The expression appears in the Complex Join box.

 

 

·         To save the join, click OK.

·         To close dialog box, click Cancel.
