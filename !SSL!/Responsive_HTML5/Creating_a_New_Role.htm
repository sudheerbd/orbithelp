Creating a New Role

  1. On the User Management menu, click Roles .

  2. A list of available roles appears.

 

          

                                                         

     3. To create a new role, click New.

     4. New Role dialog box appears.

            

         

                                       

  1. To create a new role, perform the following:

路         In the Name box, type the name.

    • Example: Name : Test Invoice Role

路         In the Description box, type the description.

    •  Example:Description: This is a sample description

路         To save the changes, click Save.

路         To close the dialog box, click Cancel.
