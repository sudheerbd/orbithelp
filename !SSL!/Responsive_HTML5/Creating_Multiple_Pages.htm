Creating Multiple Pages

 

To create multiple pages in a dashboard

1.      In the Run Results area of the dashboard, click  New (highlighted as “1”) beside first page tab at the bottom.

 

                                                    

 

2.        The new page appears with page number.

3.      To change the name of the page, double-click the page number label, and then type the new name for the page (highlighted as “2”).
