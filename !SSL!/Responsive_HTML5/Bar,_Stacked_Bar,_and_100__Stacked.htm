Widget Settings of Bar Charts

To set the widget settings

1.      On the report configurator, click Widget Settings.

2.      Under the Widget Settings the following tabs appear:

·         General

·         Category Axis

·         Value Axis

·         Data Labels

·         Legend

·         Styles

 
 General Tab

 

1. Under General tab, perform the following:

a)      In the Title box, type the title for the report type.

¥    Title of the chart gets displayed in the dashboards.

b)      In the 3D box, select/type the required option from the following:

                                i.            true

                              ii.            false

?   Example: 3D: True

 

c)      In the Theme box, select the required theme.

?   Example: Theme: Sky

 

d)                  In the Horizontal Grid box, select the required option.

?   Example: Horizontal Grid: False

 

e)      In the Vertical Grid box, select the required option.

?  Example: Vertical Grid: False

 

f)       In the Zoom box, select the required option from the following:

a.       True

b.      False

¥    If Zoom is set to True, the user can zoom into the report. To zoom into an area in the report, select the area and drag the mouse. To zoom out of the area, double click the area.

 

g)      In the Background box, pick the required color from the color palette.

?   Example: Background: d3c8c8

Value Axis

1.      Under the Value Axis, perform the following:

a.       In the Position box, type/select the required option from the following:

                                i.            Top

                              ii.            Bottom

¥    By default, the Value Axis appears in the bottom (highlighted as “1”)

 

b.      In the Title box, type the required title.

?   Example: Title Box: Y-Axis

 

c.       In the Display list, select the required option from the following:

                                i.            True

                              ii.            False

¥   When Display is set to true, Value Axis title is displayed.

d.   In the Y2-Axis box, select the required option from the following:

                                i.            True

                              ii.            False

¥    When Y2-Axis is set to True, second Y-axis is displayed (highlighted as “2”).

e.       In the Rotate box, type/select the required value.

?   Example: Rotate: 45

 

f.        In the Scale box, type/select the required option from the following:

                                i.            Thousands

                              ii.            Millions

                            iii.            Billions

                            iv.            Trillions

                              v.            None

?     Example: Scale: Millions

 

·         The report appears as follows when the above widget settings in the Value Axis tab are set:

 

Data Labels

1.      Under Data Labels tab, perform the following:

a.                                                                                                                   a.  In the Bar Labels box, type/select the required option from the following:

                                            i.         None

                                          ii.         Inside Start

                                        iii.       Inside End

                                        iv.         Outside 

?     Example: Bar Labels: Outside

¥    When Bar Labels is set to Outside, Bar labels appear outside the bar.

 

b.      In the Color box, pick the required color from the color palette.

?     Example: Color: d43535

¥     The bar label color changes to the above-mentioned color.

 

c.       In the Weight box, type/select the required option from the following:

                                                              i.      Normal

                                                            ii.      Bold

                                                          iii.      Bolder

                                                          iv.      Initial

                                                            v.      Inherit

                                                          vi.      Lighter

?     Example: Weight: Bolder

 

d.      In the Orientation box, type/select the required option from the following:

 

                                                        i.            Horizontal

                                                      ii.            Vertical

?     Example: Orientation: Vertical

 

¥  The bar labels are displayed vertically.

 

·          The report appears as follows when the above widget settings in the Data Labels tab are set:

 

Legend

1.      Under the Legend tab, perform the following:

a)      In the Hide tab, select the required option from the following:

a.      True

b.      False

?   Example: Hide: False

 

¥    When Hide is set to false, then the legend is displayed (highlighted as”1”).

¥    Legend is displayed only when two or more columns are present in the Values (Y-Axis) section.

 

b.      In the Position box, type/select the required option from the following:

                                            i.            Bottom

                                          ii.            Right

                                        iii.            Left

                                        iv.            Top

c. In the Title box, type the title for the legend.
