Generating Report Link

To generate report link

 

1. Under the Reports section, point to the required report, right-click, and then click Generate Link.

 

    Example: Required Report: Combination

 

2. Generate Link dialog box appears.

3. In the Generate Link dialog box, two report link options are available.

  • Private Link - this type of link requires login credentials to view the report.

  • Public Link - this type of link does not require any login credentials.

4. To generate private link, select the radio button before Private, and then click Generate.   

 

       

5. This generates a private link as shown above.

6. Copy and paste the generated link in the desired browser.

 

# Since, it is a private link, user needs to enter the login credentials.

 

7. To generate the public link, select the radio button before Public, and then click Generate.   

 

     

 

8. This generates a public link as shown above.

9. Copy and paste the generated link in the desired browser.

10. This generates the report without prompting for any login credentials.

 

# Since, it is a public link, user does not need any login credentials.
