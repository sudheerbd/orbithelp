Importing Discoverer Object(s)

Exercise: Importing Discoverer Object(s)

Using Import Discoverer Object(s) option, import schedule xml which loads the business areas into ORBIT Analytics application. Also, importing the eex files uploads the worksheet for a specific business area.

 

Importing Schedule XML

 

Schedule XML is an XML file generated using ORBIT Discoverer Migration. The contents of the file include the selected business areas in the discoverer migration. Once the schedule.xml file is generated using ORBIT discoverer migration, this file can be imported into the ORBIT Analytics.

To Import Schedule XML

 

1.      To upload schedule xml file, click New Process on the right-hand side corner of the Scheduled Processes page, and then click Import Discoverer Object(s).  

 

        

 

2.       Import Discoverer Object(s) dialog window appears.

 

     

 

.      3. Under Parameters, click Upload (highlighted as “2”) next to the Upload File box.

    4. Open dialog box appears with required files.

    5.      Select the required xml file, and then click Open.

   ?  Example: Required XML File: disco_job_ba (2).xml

 

    6.      In the Upload File box, the selected file path appears.

 

 

             7. To submit the file, click Submit.

       8. Under Scheduled Processes, the newly submitted file (highlighted as “3”) appears.

 

        

9.      On the left-hand side menu, point to  Business Models, and then click Domains.

10.      Domains page appears with a list of available domains.

11.      Under the Domains & Data Models, the newly created domain (highlighted as “4”) appears.

              ?    Example: Newly Created Domain: Cash Forecasting

 

 

Importing EEX Files

 

Importing EEX Files migrates the worksheets of selected business area into ORBIT Analytics.

 

To import EEX Files

 

Generate EEX file from discoverer desktop administrator. Once the EEX file is generated, you can import the file into the monitoring section of ORBIT Analytics application.

     

1. To import the eex file, click New Process on the right-hand side corner of the Scheduled Processes page, and then click Import Discoverer Object(s) (highlighted as “1”).

 

    

 

2. Import Discoverer Object(s) dialog window appears.

 

 

3 Under Parameters, click Upload (highlighted as “2”) next to the Upload File box.

4. Open dialog box appears with required files.

5. Select the required eex file, and then click Open.

 ?  Example: Required EEX file: CE_Cash_Forecasting.eex

 

 

6. The selected eex file (highlighted as “3”) appears in the Upload File box.

7. To submit the file, click Submit.

8. Under Scheduled Processes, the newly submitted file appears.
