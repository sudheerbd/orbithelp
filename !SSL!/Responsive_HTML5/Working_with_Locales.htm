Working with Locales

To work with locales

1.      Beside the Domain Hierarchy area, Semantic Layers area appears.

2.      In the Semantic Layers area, the domain menu appears.

3.      In the domain menu, click  Locales (highlighted as “1”).

 

 

                                            Figure: Displaying Locales

 

4.      Locale Configuration dialog box appears.

 

        

                                Figure: Locale Configuration Dialog Box

 

 

5.      To configure the locale settings, click  Search for items to add (highlighted as “1”) that appears on the upper right-hand corner of the Locale Configuration dialog box. 

6.      A list with various languages (highlighted as “3”) appears.

 

                                         Figure: List of Various Languages

 

7.      Select the checkbox before the required language.

8.      The selected language appears under Locale Configuration (as shown in the figure).

 

         

        

                                          Figure: Locale Configuration

 

9.      To save the changes, click Save.

10.  To close the dialog box, click Cancel.
