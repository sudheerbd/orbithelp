Adding a New Authority Type

1. To add a new authority type, click  Add Authority.

2. Authorization dialog appears.

3. In the Authority Type, select the required authority type.

¥     Here, the authority type does not need to be Active Directory essentially. The authority type in the Authorization section can vary from the authority type in the Authentication.

¥     Based on the selected Authority Type, the fields under it change.

4. To save the changes in the Authorization section, click Ok.

 5. To save the changes, click Save.
