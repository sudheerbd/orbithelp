Grid Filters

Grid Filters can be applied using specific conditions on every individual column of the report at the output level.

 

To apply grid filter on a report

 

1.      Under the Reports area, right-click the required report, and then click Open.

  

       

                 

 

2.      The selected report appears.

 

        

                                                      

 

3.      In the Run Results area above the report, click Hide/show grid filters (highlighted as “1”).  

 

                                     

 

4.      Grid filter (highlighted as “2”) appears above the selected report.

                                                       

 

5.      Apply the filter for the required column (highlighted as “3”).

6.      The report is filtered based on the condition specified in the grid filter (highlighted as “4”).

7.      The report appears as shown below:

                                            

 

¥    Grid filters are available in View Mode only.
