Exporting Report

To Export a Report

 

1. Login to the source instance, and then under Reports section, select the required report that must be imported, and then click Export.

 

 

¥    To import the reports, you must first import its corresponding domain, only then the reports will get exported from source instance and get imported into the target instance.

¥    Here, we have exported and imported “Test Domain -export-import” domain. Now, export the reports under this domain.

 

 

     

 

2. Export Report dialog appears.

 

   .

 

3. To export the report, click Submit. 

4. A success message appears once the report is exported, and a new process is created under Processes under Monitoring.

5. Select the required process (highlighted as “1”), and its corresponding output appears under Output(s) & Delivery.

 

.

                                                   

6. Select the required output.

7. The output gets downloaded as an xml file.

8. Place the xml file of the exported report in an appropriate location on the system.
