Properties

Operational Reports, Analysis, and Dashboards have the same properties

 

To set the Properties

1.      Under Reports area, right-click the required report, and then click Edit/Run.

2.      To set the properties, click Properties (highlighted as “1”) above the Run Result area.

 

                                            

3.      Properties dialog box appears.

4.      On the left hand side, the selected report name (highlighted as “1”) and the columns (highlighted as “2”) under it appear.

 

                                            Figure: Displaying Properties Dialog Box

 

5.      On the right-hand side, the properties of report name and the columns appear.  

6.      For the selected report (highlighted as “3”), the following properties (highlighted as “4”) appear:

a.       Under Name, type the name of the business object. 

b.      Under Description, type the description of the business object.

 

                      

 

7.      For the required Attributes/Dimensions/Measures (highlighted as “5”), the following properties (highlighted as “6”) appear:

 

                                                  

a.       Under Name, type the name of the attributes/dimensions/measures.

b.      Under Description, type the description of attributes/dimensions/measures.

c.       Under Grouping & Summary tab, perform the following:

                                                        i.            In Grand Total list, click one of the following options:

o   True

o   False

                                                      ii.            In Grouping list, click one of the following options:

o   True

o   False

                                                    iii.            In Grouping Summary list, click one of the following options:

o   True

o   False

d.      Under Format tab, perform the following:

                                                              i.      In the Format box, type the format of attribute/measure/column.

e.       Under Sort tab, perform the following:

                                                              i.      In the Sort Column Order box, type/click / to enter the order number.

                                                            ii.      In the Sort Type box, type/select the required option from the following:

o   ASC

o   DESC 

f.       Under Style tab, perform the following:

                                                              i.      In the Alignment box, type/select the required option from the following:

o   Left

o   Center

o   Right

                                                            ii.      In the BackGround Color box, type/select  to enter background color.

                                                          iii.      In the Column Width box, type/click / to enter the column width.  

                                                          iv.      In the Font Family box, type the font family.

(OR)

Select  to perform the following:

·         In the Font Family box, type/select the font family.

·         In the Font Size box, type/select the font size.

·         In the Font Style, select the required checkbox from the following:

o   Italic

o   Bold

·         Under Effects area, select Underline checkbox to underline.

·         To set the color for the underline, click the required color from the color palette.  

·         Under Preview area, the preview of the selected changes appears.

·         To save the changes, click Ok.

                                                            v.      To hide the dimension/attribute/measure, select the required option from the following:

o   True

o   False 

g.      To save the changes, click OK.

h.      To close the dialog box, click Cancel.
