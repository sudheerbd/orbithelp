Creating a Simple Join

To create a simple join

 

1.      Select the source table.

¥  When the hand symbol appears on the selected table, a join can be created.

 

2.      Drag the arrow from the selected source table to destination table.

3.      Relationship (Source table =>Destination table) dialog box appears.

 

                      

 

4.      To create a simple join, perform the following:

·         In the From Column combo box , select/type the required option.

·         In the To Column combo box, select/type the required option.  

·         In the Relation Type combo box, select/type the required option.

·         In the Join Type combo box, select/type the required option.

·          In the Join Order Key box, type the order key for the join

·         In the Description box, type the description for the join.

¥   The values in the From Object and To Object are populated by default.

 

·         To save the join, click OK.

·         To close dialog box, click Cancel.
