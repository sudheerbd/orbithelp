Multiple Drilldowns on Same Column

To work with multiple drilldowns on same column

 

1.      In the Drilldowns dialog box, on the Drilldown Column box, drag and drop the column from the report configurator.

2.      To work with multiple drilldowns on same column, drag and drop the second column on the same Drilldown Column box from the report configurator. 

     

       

                            

 

3.      To save the changes, click Ok.

4.      The report appears as shown below:

 

                                     

 

5.      To view multiple drilldown on the same column, click the drilldown column.

6.      A list of drilldown reports appears for the selected drilldown column as shown below:

                  

 

7.      Select the required drilldown report from the list.
