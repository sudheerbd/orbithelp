Uploading Schedule XML into ORBIT Analytics

To upload Schedule.xml file into ORBIT Analytics application

 

1.      Navigate to ORBIT URL.

2.      In the login page, type the user name and password, and then click Login.

3.      The home page of the ORBIT Analytics application appears.

4.      On the left-hand side menu, point to Monitoring, and then click Processes.

 

5.      Scheduled Processes page appears with a list of available processes. 

6.      To upload schedule xml file, click New Process on the right-hand side corner of the window, and then click Import Discoverer Object(s).  

 

 

7.     Import Discoverer Object(s) dialog window appears.

 

 

8.      Under Parameters, click Upload (highlighted as “2”) next to the Upload File box.

9.      Open dialog box appears with required files.

10.  Select the required xml file, and then click Open.

11.  In the Upload File box, the selected file path appears.

 

 

12.  To submit the file, click Submit.

13.  Under Scheduled Processes, the newly submitted file (highlighted as “3”) appears.

14.  On the left-hand side menu, point to Business Models, and then click Domains.

15.  Domains page appears with a list of available domains.

16.  Under the Domains & Data Models, the newly created domain (highlighted as “4”) appears.
