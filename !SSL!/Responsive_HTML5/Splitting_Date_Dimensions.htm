Splitting Date Dimensions

An available date dimension is split into various date dimensions, when the date dimension is consolidated and there is no single date dimension available to create a report. For example, if order date is the only available date dimension in the report and you need to create a report year and quarter-wise, then you can split the order date into multiple date dimensions, including calendar year, month, quarter, and so on.   

 

To split the date dimensions

1.      Under Data Explorer, right-click a date dimension, and then click Split Date Dimensions.

 

      

               

 

2.      Date Dimensions dialog box appears with the following options:

 

                                          

 

·         Calendar Year

·         Calendar Month

·         No of Days in Month

·         Week of the Month

·         Day Name

·         Leap Year

·         Month Name

·         Day of the Month

·         Day of the Week

·         Day Short Name

·         Calendar Quarter

·         Month Short Name

·         Calendar Week

·         Day of the Year

 

3.      Select the required date dimensions, and then click OK.

4.      To close the dialog box, click Cancel.

5.      The selected dimensions (highlighted as “2”) in Date Dimensions dialog box appear under the selected date dimension with <Date Dimension Name> Hierarchy.
