Getting Started with Excel Edge

To get started with Excel Edge

 

1.      Navigate to the ORBIT Analytics URL.

 

    2.      In the ORBIT Analytics Login Page, perform the following:

 

a.       In the Username box, type the required username.

 

b.      In the Password box, type the required password.

 

c.       To login to the ORBIT Analytics application, click Login.

 

d.      ORBIT Analytics Home page appears.

e.       On the top right-hand side of the Home page, click  Download excel edge (highlighted as “1”).
