Line, Spline, Step Line, and Area Charts

Under this section, the following charts appear with same widget settings:

1.      Line Basic

2.      Spline

3.      Step Line

4.      Area

5.      Area Spline

  •  Line Basic Chart has similar settings as of Column Basic Charts.

 

Widget Settings of Line Basic Chart

To set the Widget Settings

1.      In the report configurator area, click Widget Settings.

2.      The following tabs appear:

a.       General

b.      Category Axis

c.       Value Axis

d.      Data Labels

e.       Legend

f.        Styles 

 

General Tab

## To view the common widget settings, see General Tab  under Column Basic Charts.

 

Apart from the widget settings in the above-mentioned link, the following are the additional settings available under the General tab.

1.      In the Show Markers box, type/select the required option from the following:

·         True

·         False

?     Example: Show Markers: True

 

¥   When True is selected in Show Markers box, markers (highlighted as “1”) appear on the report.

 

      

 

¥    In Line, Spline and Step Line Charts, markers are visible on the chart. However, In Area and Area Spline, the markers are visible when the pointer hovers over the curve.

 

 

 

2.      In the Marker Type box, type/select the required option from the following.

a.      Cross

b.      Circle

c.       Arrow

?     Example: Marker Type: Cross

 

¥     Based on the selection in the Marker Type box, the marker types change.

Data Labels

To view the other common widget settings, see Data Labels tab under Column Basic Charts.

Apart from the widget settings in the above-mentioned link, the following are the additional settings available under the Data Labels tab.  

1.      Under the Data Labels tab, perform the following:

a.       In the Marker Labels box, type/select the required option from the following:

                                                           i.         None

                                                         ii.         Under

                                                       iii.         Over

                                                       iv.         Rotate

?     Example: Marker Labels: Under

 

¥    When None option is selected, no marker labels are displayed.

¥    When Under option is selected, marker labels are displayed below the marker.

¥    When Over option is selected, marker labels are displayed above the marker.

¥    When Rotate option is selected, the marker appears rotated.

b.      With the above widget settings, the report appears as shown below:

 

 

¥    In the above report, the marker label appears under the marker (highlighted as “2”)
