Combination Charts

The combination charts are combination of multiple Cartesian charts.

To set the Widget Settings

1.      In the report configurator area, click Widget Settings .

2.      Under Widget Settings, the following tabs appear:

a.       General

b.      Category Axis

c.       Value Axis

d.      Data Labels

e.       Legend

f.        Styles 

g.      Series

 

General

The widget settings under General tab are similar to that of Column Basic Charts. See General Tab of Column Basic Charts for more information.

 

Category Axis

The widget settings under Category Axis tab are similar to that of Column Basic Charts. See Category Axis tab of Column Basic Charts for more information.

 

Value Axis

The widget settings under Value Axis tab are similar to that of Column Basic Charts. See Value Axis tab of Column Basic Charts  for more information.

Series

1.      Under the Series tab, in the Widgets box, bar option is available by default.

2.      The first column under the Y-axis is displayed as bar (highlighted as “1”), by default. If there are more than one column under Y-axis, the columns starting from the second appear as line, if no option is entered in Widgets box.

3.      In the Widgets box, the values are entered as comma separated values.

6.      To apply the widget settings, click Apply.

!        In case you misspell the type of chart in Widgets box, then you will see a bar chart if the misspelt value is the first value. You will see a line chart, if you misspell the chart option from the second one. For Example, in the Widgets box, if you enter “Are” instead of “Area” as the first value (highlighted as “1”), then you will see a bar chart by default. If you enter “Are” instead of “Area” as the third value (highlighted as “2”), then you will see a line chart by default. This scenario can be seen in the figure below.
