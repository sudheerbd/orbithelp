Scheduling the Process

To schedule the process

 

1. In the ORBIT Excel Edge application, in the Submit/Schedule tab, click Expand panel (highlighted as "1").

 

        

   

 2. Submit/Schedule panel expands.

3. To schedule the process at a required interval, remove the checkmark from the   Run Immediately checkbox (highlighted as "2").

   

          

      

 4. The various fields to schedule the request appear under Submit/Schedule tab.

 

                

 

 

5. In the Start Date box, type/select the required start date of the request.

6. In the Start Time box, type/select the required start time of the request.

7. In the End Date box, type/select the required end time of the request.

8. In the End Time box, type/select the required end time of the request.

9. The user can schedule the request using the following two options:

·         Simple

·         Advanced

10. If the Simple tab is selected, perform the following:

   a. Under Recurrence, select the required option from the following:

        i. Never Repeat  

        ii. Repeat       

  

¥   The user can schedule the recurrence of the request using "Never Repeat" and "Repeat" options. "Repeat" option enables the user to schedule the request at a required interval. When the user selects Never Repeat, the request is never repeated.

 

 b. Under Apply the Interval tab, select the required option from the following:  

       i. From the completion of prior run

       ii.From the start of the prior run

c. To submit the request, clickSubmit.

          11. If the Advanced tab is selected, perform the following:

    a. Under the Months section, select the checkbox before the required month to select the required month.

    b. Under the Dates section, select the required dates.

    c. Under the Days section, select the required days.   

 

           

 

¥    To notify the user about the end of the process, select the Notify me checkbox.  

 

        d. To submit the request, click Submit.
