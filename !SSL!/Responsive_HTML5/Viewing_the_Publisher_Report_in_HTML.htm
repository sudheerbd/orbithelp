Viewing the Publisher Report in HTML

To View the Publisher Report in HTML

 

1.   Perform the process as in Applying Grouping to the Publisher Report.

 

¥     Prior to uploading the template in HTML format, you must have already created the template. See Creating a HTML Template for more information. 

 

2.   Upload the HTML template as in Applying Excel Template for Publisher Report .

3.   Save the report.

4.   To run the publisher report, perform the following:

a. On the report menu, click View Mode.

b. The View Mode of the publisher report appears.

c. In the Select template box, select the attached HTML template.

?    Example: Template: Customer Details_HTML

 

d. In the Select Output box, select the required outputs from the following options:

·         HTML

·         PDF

?    Example: Output: HTML

 

e. To run the report, click Run.
