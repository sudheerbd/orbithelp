Setting Output and Delivery Types

¥         The Output& Delivery tab can be seen under Schedule option in Reports

1.      On the Output & Delivery tab, on the Outputs and Deliveries section, click  Add output (highlighted as “1”) to set a new output type.

 

        

 

2. Templates section with its associated output types appear under Outputs and Deliveries.

3. The following are the available output types for a data report:

·         CSV

·         XSLX

·         XML

·         PDF

·         HTML

·         DOCX

·         Tabular

        For Analysis reports, the following output formats are available:

·         PDF

·         HTML

·         Image

      For Dashboards, the following output formats are available:

·         PDF

·         HTML

      For Publisher Reports, the following output formats are available:

·         HTML

·         PDF

·         Excel(XLS )

·         Excel(XLSX)

·         DOCX

·         DOC

     For Script Reports, the following output formats are available:

·         CSV

·         XSLX

·         XML

·         PDF

·         HTML

·         DOCX

·         Tabular
