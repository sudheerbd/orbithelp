To set the Widget Settings

1.      In the report configurator area, click Widget Settings .

2.       Under the Widget Settings tab, the following tabs appear:

a.       General

b.      Category Axis

c.       Value Axis

d.      Data Labels

e.       Legend

f.        Styles 

General

To view the other common widget settings, see Column Basic Charts under General tab.

Apart from the widget settings in the above-mentioned link, the following are the additional settings available under the General tab.  

1.      Under General tab, perform the following:

a.       In the Marker Type, type/select the required option from the following:

                                                                          i.      Cross

                                                                        ii.      Circle

                                                                      iii.      Arrow

?     Example: Marker Type: Circle

 

?   Marker Type is highlighted as “1” in the figure below.

 

¥    Columns must be present under Values (Y-axis), Values(X-axis), and Category sections.

 

 

The report appears as follows when the additional settings under General Tab are applied.

·         Here, Values in Y-axis has Sales column; Values in X-Axis has Profit Column.

·         Category has State column.

Category Axis

The widget settings under Category Axis tab are similar to that of Column Basic Charts. See Category Axis of Column Basic Charts tab for more information.

Value Axis

The widget settings under Value Axis tab are similar to that of Column Basic Charts. See Value Axis of Column Basic Charts tab for more information.

Data Labels

To view the other common widget settings, see Data Labels tab under Column Basic Charts.

 

Apart from the widget settings in the above-mentioned link, the following are the additional settings available under the Data Labels tab.  

1.      Under the Data Labels tab, perform the following:

a.       In the Marker Labels box, type/select the required option from the following:

                                                                          i.      None

                                                                        ii.      Under

                                                                      iii.      Over

                                                                      iv.      Rotate

?     Example: Marker Labels: Over

 

?   Marker Labels is highlighted as “1” in the figure below:

 

  

Legend

The widget settings under Legend tab are similar to that of Column Basic Charts. See Legend Tab of Column Basic Charts for more information.

 

Styles

The widget settings under Styles tab are similar to that of Column Basic Charts. See Styles Tab of Column Basic Charts for more information.
