Setting Outputs

 

To set the Outputs

1.      On the Output & Delivery tab, in the Outputs area, click  (highlighted as “1”) to set a new output type.

 

                                         Figure: Displaying New Output Type

 

2.      In the Name box, type the required name for the output.

3.      In the Template box, type/select the required option.

4.      In the Output Type box, type/select the required option.

 

      

                                                 Figure: Displaying Delete Option

 

 

5.      To delete an output option, click (highlighted as “2”).

 

¥    Once the output is set, adding delivery option (highlighted as “3”) is enabled.
