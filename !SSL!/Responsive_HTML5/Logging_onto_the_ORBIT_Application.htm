Logging onto ORBIT Application

 

To log on to the ORBIT Application

 

      1. Navigate to ORBIT URL.

      2. ORBIT Login page appears.

                Figure : Displaying Username and Password

 

     3.  In the Username box (highlighted as “1”), type the username.           

    • Example: Username: admin

       4.  In the Password box (highlighted as “2”), type the password.

    • Example: Password: admin

 

 

  •  Selecting Remember Password checkbox enables the system to remember the currently entered password. If you have forgotten password, click Forgot Password.

 

     5.  To log on to ORBIT application, click Login.

 

 

  • The Password changes to <••••••>

 

 

      6. Home page appears.
