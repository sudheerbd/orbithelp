Authentication

1. Under the Authentication section, in the Authority Type box, type/select the required authority type from the following:

 

·         Oracle ERP Security

·         LDAP

·         Active Directory

·         PeopleSoft

?  Example: Required Authority Type: Active Directory

¥     Based on the selected authority types, the fields in the Authentication section change.

2.  In the Server URL box, type the URL of the Server.

?     Example: Server URL: ldap://192.168.2.126:389

¥ Here, ldap is the protocol, "192.168.2.126" is the ip, and 389 is the default port number.

3. In the User DN box, type the Username.

?     Example: User DN: ORBIT8\Administrator

 4. In the Password box, type the password.

?   Example: Password: Welcome@123

 5. In the Base DN box, type the Base DN.  

¥     Base DN: This is a point from where the server starts its search.

?      Syntax: dc=orbit8, dc=com

¥     When the user enters only Base DN, the server starts its search from the root, and
