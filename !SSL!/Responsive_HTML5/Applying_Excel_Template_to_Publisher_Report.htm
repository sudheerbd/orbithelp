Applying Excel Template to Publisher Report

Prior to applying an Excel Template to a publisher report, you must have already created a template in Microsoft Excel.  See Creating an Excel Template for more information.

 

To Apply Excel Template to Publisher Report

 

1. On the extreme right-hand side, click Options, and then click Templates (highlighted as “1”).

 

 

 

2. Templates dialog appears.

3.  In the Templates dialog, perform the following:

4. On the upper right-hand side, click  Add Template.

5. The required fields appear.

6. In the Name box, type the name for the template.

?  Example: Name: Customer Details-Excel

 

7. In the Description box, type the description.

?   Example: Description: This is an Excel template for Customer Details.

 

8. In the Type box, “ORBIT” is selected by default.

 

9. To upload the required template, click Upload file that appears next to the Upload textbox.

10. Upload the required excel template, and then click Save.

    

    

 11. The uploaded template appears under Templates.

 

12. To make the template as a default template, select the template, and then click Default (highlighted as “2”).

 

13. A checkmark (highlighted as “3”) appears in the Default checkbox for the selected template.  

   

    

 

14. To close Templates, click Close.

 

15. Save the Report, and then click View Mode (highlighted as “4”).

 

 

 

16. View Mode appears.

 

17. In the View Mode, perform the following:

 

18. In the Select template box, select the required template.

 

?  Example: Select template: Customer Details-Excel

 

19. In the Select Output box, select the required output from the following:

 

·      PDF

·      Excel(xlsx)

 

?  Example: Select Output: Excel

 

20. To run the report, click Run.

 

¥   For Excel template, the outputs are generated in PDF and Excel.

 

21. The report appears in the excel sheet as follows:
