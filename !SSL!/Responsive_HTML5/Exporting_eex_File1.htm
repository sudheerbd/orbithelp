Exporting eex File

1. Launch Discoverer Administrator.

 2.Choose File --> Export to display the Export Wizard: Step 1 dialog window.

3. Select the Selected objects in the End User Layer option, and then click Next.

 

    

 

4. Export Wizard: Step 2 dialog window appears.

5. Use the “Which object do you want to export?” list to display the EUL object     

      types that you want to export to a file.

 

      @     Example: EUL Object: Workbooks 

 

 

 6. Move the object that needs to be exported from Available list to the Selected list by clicking , and then click Next.

 

¥       You can select more than one object at a time by holding down the <Ctrl> key and clicking another object.

 

 

   7. Export Wizard: Step 3 dialog window appears.

    8. Click Browse to display the Save As dialog window.

    

    

 

  9. Provide an appropriate location and name to save the. eex file.

  10. Select the Generate XML for exported workbooks check box to save the

     required workbooks in XML format.

¥       The export file (. eex file) is always in XML format.

 

 11. To create the export file, click Finish.
