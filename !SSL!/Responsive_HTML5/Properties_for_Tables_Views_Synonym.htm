Properties for Tables/Views/Synonym

 

1.      For the required table/view/column, perform the following:

a.       Under the Name field, type the name of the table/view/synonym.

b.      Under the Description field, type the description for the table/view/synonym.

c.        Under the Table tab, perform the following:

                                                                i.      In the Table Type box, type/select the required option.

Properties of Column under a Table/View/Synonym

1. Under the Name tab, perform the following:

 a.  In the Name box ,type or select the required name.

2. In the Description box, type/select the required description.

3.Under the Field tab, perform the following:

 a. In the Default Aggregation box, type/select the required aggregation from the following:

  • NONE

  • SUM

  • AVERAGE

  • COUNT

  • COUNT_DISTINCT

  • MINIMUM

  • MAXIMUM

b. In the Field Type box, type/select the required field type from the following:

  • OTHER

  • DIMENSION

  • FACT

  • KEY

  • ATTRIBUTE

c. In the Searchable box, type/select the required option from the following:

  • True

  • False

4. Under the Formula tab, perform the following:

a. In the Column Type box, type/select the required option from the following:

  •   Column

  •   Formula

b. In the Database Formula box, type/select the required option from the following:

  • True

  • False

c. In the Datatype box, type/select the required option from the following:

  • Unknown

  • String

  • Date

  • Datetime

  • Boolean

  • Integer

  • Decimal

d. In the Target Column/Expr box, click in the bow, and then click .

Selecting opens Expression Builder(Physical Column )dialog. From the dialog select the required expression or column.

5. Under the Key tab, perform the following:

a. In the Foreign Key box, type/select the required option from the following:

  • True

  • False

b. In the Primary Key box, type/select the required option from the following:

 

  • True

  • False

c. In the Unique Key, type/select the required option from the following:

  • True

  • False

6. Under the Format tab, in the Format box type/select the required option.

7. Under the Style tab, perform the following:

a. In the Alignment box, type/select the require option from the following:

  • Left

  • Right

  • Center

b. In the Cell Wrap box, type/select the required option from the following:

  • True

  • False

c. In the Column Width box, type/select the required value.

d. In the Font Family box, select the required option.

e. In the Hidden box, type/select the required option from the following:

  • True

  • False
