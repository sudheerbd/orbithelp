Pie Basic and Pie Donut

Pie Basic and Pie Donut have the same widget settings.

 

Widget Settings of Pie Basic

 

To set the widget settings

1.      On the report configurator, click Widget Settings.

2.      Under the Widget Settings, the following tabs appear:

a.       General

b.      Value Axis

c.       Data Labels

d.      Legend

e.       3D Styles

 

  General

1.      Under the General tab, perform the following:

a.       In the Title box, type the title for the report.

?   Example: Title: Pie Basic Chart

 

? This title is displayed in the dashboard.

b.      In the 3D box, type/select the required option from the following:

                                                  i.      True

                                                ii.      False

?   Example: 3D: True

c.       To set the donut size, slide the Donut slider left or right.

?   Example: Donut: 44

 

d.      In the Theme box, select/type the required option from the list.

?   Example: Theme: category1

 

?   The report appears as follows when the widget settings under General Tab are applied.

 

Value Axis

1.      Under the Value Axis tab, perform the following:

a.       In the Scale box, type/select the required option from the following:

                                                  i.      None

                                                ii.      Thousands

                                              iii.      Millions

                                              iv.      Billions

                                                v.      Trillions

?     Example: Scale: Millions

 

The report appears as follows when the above-mentioned setting is applied:

 

Data Labels

1.      Under the Data Labels tab, perform the following:

a.       In the Display box, type/select the required option from the following:

                                            i.            None

                                          ii.            Outside

                                        iii.            Inside

                                        iv.            Rotate

                                          v.            Horizontal

                                        vi.            Vertical

?     Example: Display: Outside

 

¥    The data labels are displayed outside the pie (highlighted as “1”).

 

b.       In the Wrap box, type/select the required option from the following:

                                                              i.      True

                                                            ii.      False

?     Example: Wrap: True

¥   When Wrap is set to True, then the percentage value is wrapped to the next line (highlighted as “2”).

¥   In this example, percentage represents the percentage of sales in a particular state.

c.       In the Donut Label box, select the required option from the following options:

                                                        i.            True

                                                      ii.            False

?     Example: Donut Label: True

 

¥    Donut Label appears in the middle of the pie (highlighted as “3”).

                        

 

Legend

1.      Under the Legend tab, perform the following:

a.    In the Hide box, type/select the required option from the following:

                                                     i.      True

                                                   ii.      False

¥    If Hide is set to true, then the legend will be hidden. To view the legend, set the option to False.

b.   In the Position box, type/select the required option from the following:

                                                  i.      Bottom

                                                ii.      Right

                                              iii.       Left

                                              iv.      Top

This defines the position of the legend in the chart.

 

3D Styles

1.      Under the 3D Properties tab, perform the following:

                             i.      To set the thickness, slide the Thickness slider left or right.

?     Example: Thickness: 63

 

                           ii.      To set the distortion, slide the Distortion slider left or right.

?     Example: Distortion: 65

 

                         iii.      To set the bevel width, slide the Bevel Width slider left or right.

?     Example: Bevel Width: 23

 

                         iv.      To set the color spread, slide the Color Spread slider left or right.

?     Example: Color Spread: 2

 

                           v.       To set the opacity, slide the Opacity slider left or right.

?     Example: Opacity: 0.7

 

?   To apply the widget settings, click Apply.

 

?   The report with settings under 3D Styles tab appears below.
