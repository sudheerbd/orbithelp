Setting Aggregate Functions on Reports

To set aggregate functions on a report

1.      In the Run Results area, different columns of the required report appear.

2.      To set aggregate functions in a report, click  next to the required column, point to Summary (highlighted as “1”), and then click the required aggregate function from the available aggregate functions (highlighted as “2”).   

 

                                             

 

 

¥    Types of aggregate functions depend on the data types of the column.

 

 

3.      The aggregate function is applied to the report as shown below:

 

 

                                

 

4.      Integers/Decimals have the following aggregate functions:

·         Sum

·         Average

·         Min

·         Max

5.      String /Date have the following aggregate function:

·         Count
