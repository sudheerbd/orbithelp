Running a Report in Excel Edge

To Run a Report in Excel Edge

 

1.  In ORBIT Excel Edge application, Reports section appears with the available report folders. 

 

¥  Reports available under the Reports section are the data, analysis, publisher, and script reports already present in the ORBIT Analytics application.

 

2.  Under the Reports section, select the required report (highlighted as "1”).

 

?     Example: Required Report: drilldown1         

 

     

 

3. To run the report, click Run (highlighted as "2").

 

4. If the report does not have any parameters, the following section appears indicating there are no available parameters for the selected report.

 

 

 

5. To run the report, click Run (highlighted as “3”).

6. The output of the report appears on the left-hand side of the excel edge application (highlighted as “4”).

 

 

 

 7. When a report with parameters is run, then Parameters - <report name> tab appears with already existing parameters as shown below:

 

 

8. In the parameters box, click .

9. A list of available parameter values appears.

10 For the required report, type the required parameter value, and then click Run.

?  Example: Required Report: drilldown2

                              

                                Required Parameter: Central

 

11. The output of the report appears in the excel worksheet on the left-hand side of ORBIT Excel Edge application.

 

  

 

  

¥    You can select different parameters and the excel sheet gets updated as per the selected parameters.
