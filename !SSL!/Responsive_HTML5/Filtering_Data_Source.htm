Filtering Data Source

To filter a data source

1.   Under Data Sources, click.

  1. Textboxes to filter a domain appear under Name, Description, and Created By.

          

          

  1. Under the required column, type the search term to filter.

    • Example: Required Column: Created By

    • Example: Search term: adm

  1. Based on the filter value (highlighted as "2"), the search results (highlighted as "3") appear as shown:

 

          

                                                            

  1. To refresh the list of data sources, click Refresh.
