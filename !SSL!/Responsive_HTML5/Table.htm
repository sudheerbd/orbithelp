Tabular Charts

In Tabular Charts, data is presented in rows and columns. The following type of charts come under tabular charts:

·         Table

Widget Settings of Tabular Charts

To set the widget settings

1.         On the report configurator, click Widget Settings.

2.         The following tabs appear:

                                            i.            General

                                          ii.            Paging

General

 

1.      Under General tab, perform the following:

a.       In the Title box, type the title.

?     Example: Title: Table Widget Settings

 

? This field is the title for the table widget.  This title is reflected when the report is viewed in the dashboard.

b.      In the Row Number box, select/type the required option from the following:

                                                     i.            True

                                                   ii.            False

?     Example: Row Number: True

 

¥ This field displays the row number on the left-hand side (highlighted as “1”).

c.       In the Column Lines box, select/type the required option from the following:

                                                     i.            True

                                                   ii.            False

?     Example: Column Lines: False

 

¥    When Column Lines option is set to False, the columns in the report disappear. Columns Lines appear only when the Column Lines option is set to True.

d.      In the Row Lines box, select/type the required option from the following:

                                                              i.      True

                                                            ii.      False

?     Example: Row Lines: True

 

¥    When Row Lines option is selected as False, the rows in the report disappear. Row Lines appear only when the Row Lines option is set to True.

 

e.       In the Row Banding box, select/type the required option from the following:

                                                              i.      True

                                                            ii.      False

?     Example: Row Banding: True

 

¥     Row banding is applying shading to every alternate row in a report.

 

f.        In the Locked Column box, select/type the number of columns to be locked.

?      Example: Locked Columns: 2

 

¥ This locks the first two columns in the report (highlighted as “2”)

 

              

 

 

In the above report, the columns are not visible, as the Column Lines option is set to False.  Every alternate row has a shading as the Row Banding is set to True.

 

Paging

1.      Under the Paging tab, perform the following:

a.       In the Paging box, type/select the required option from the following:

                                                              i.      True

                                                              i.      False

Example: Paging: True

 

¥   When Paging is set to true, this enables the paging option at the bottom, by default (highlighted as “1”).

b.      In the Position box, type/select the required option from the following:

                                                        i.            Bottom

                                                      ii.            Top

                                                    iii.            Right

                                                    iv.            Left

?     Example: Position: Top

 

¥    When the Position is set to Top, the paging option is displayed on the top.

¥    Likewise, the paging option can be displayed on the Left and Right as well.

¥    By default, the paging option is displayed in the Bottom.

 

c.       In the Size box, type/select the required option.

?     Example: Size:5

¥    The size option provides the number of records that must be displayed from the existing report. Here, the size is mentioned as 5, this displays 5 records from the current report (highlighted as “2”).

 

4.      To apply the widget settings, click Apply.
