Introduction

Script files are combined with the reporting services to create the script reports. Users can customize the script files, upload them, and create complex and interactive reports in the ORBIT Analytics application.

Supported Scripts

Currently, Script Reports in ORBIT Analytics supports Python 2, Python 3, Rstats, Perl 6, Ruby, Ada, Cobol, and Pascal.

Supported Outputs

Script Reports generated using the Python 2 and Python 3 currently support the following outputs:

·         PDF

·         XLS

·         XLSX

·         DOC

·         DOCX

·         PNG

·         HTML
