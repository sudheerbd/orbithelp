Search Filter

To apply a search filter on groups

1. Under Group Search Information section, perform the following:

      a. In the Search Base box, type the required search base.

?  Example: Search Base: ou=Roles                 

     b. In the Search Filter box, type the required filter condition.

?  Example: Search Filter: (&(objectClass=groupOfNames)(objectClass=userSecurityInformation))

The above condition retrieves the records where objectClass equals “groupOfNames” and where the objectClass equals “userSecurityInformation”.  

·         Import the required users once they are retrieved.

·         Verify the existence of users and their associated roles under Users, in the User Management section as shown below.

Result

The above group search has been explained using AND Filter. Similarly, OR Filter and Not Filter can be applied on the group search as in user search.
