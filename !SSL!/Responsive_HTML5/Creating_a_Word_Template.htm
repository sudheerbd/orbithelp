Creating a Word Template

To create a Word template, perform the following process:

 

1. Open the Word document.

2. Mention the header as “${Header.COL0}".

 

   

 

 

¥   Here, State is the filter.

 

3. As shown in the figure above, to retrieve the customer data, specify the band as “##band=Data” (highlighted as “1”).

4. Type the required column names.

5. To extract the column values under the required columns, enter the column numbers as ${COL0}, ${COL1}, and so on.

 

¥   To retrieve the data in multiple rows, it essential to create a table in Word template.

 

6. Save the word document with an appropriate name.
