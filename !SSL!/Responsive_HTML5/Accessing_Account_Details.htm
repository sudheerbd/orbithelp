Accessing Account Details

 

To access the account details

1.      On the top menu, click the image on the top right-hand side of the window.

2.      A window opens as shown below:

       

       

                        

 3. To access/update the account details, click My Account (highlighted as “1”).

 4. User<User Name> dialog box appears.

 

       

                         

5. To update the account details, perform the following:

 

o   In the First Name box, edit the required first name.

o   In the Last Name box, edit the required last name.

 

  •   You cannot edit the User Name and Email fields

 

         

 

             6. To save the changes, click Save (highlighted as “2”).

         7. To close the dialog box, click Cancel (highlighted as “3”).
