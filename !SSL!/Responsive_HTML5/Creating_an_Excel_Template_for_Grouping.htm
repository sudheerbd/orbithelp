Creating an Excel Template for Grouping

To Create an Excel Template for Grouping

 

1. Open the Excel Sheet. 

2. In the Excel sheet, enter the required columns (highlighted as “1”).

  

3. To extract the column values under the required columns, enter the column numbers as ${COL0}, ${COL1}, and so on (highlighted as “2”).

4. Enter the column name that must be used as the filter/parameter (highlighted as “3”).

5. Now select the first two rows and group them under one band as “Header” as shown in the figure below.
