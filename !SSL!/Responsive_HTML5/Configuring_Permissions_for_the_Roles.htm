Configuring Permissions for the Roles

To configure permissions for the roles

 

1.      On the User Management menu, click Roles.

2.      Role List appears with a list of available roles.

3.      For the required role, under Actions, click permissions.

4.      Permissions - <Role Name>  dialog box appears.

     

      

                    

5.      In the Permissions - <Role Name> dialog box, under the Menu Root folder (highlighted as "1"), various menu options (highlighted as "2") appear.

6.      For the required role, select the menu options from the available menu permissions.

7.      To save the changes, click OK.

8.      To close the dialog box, click Cancel.
