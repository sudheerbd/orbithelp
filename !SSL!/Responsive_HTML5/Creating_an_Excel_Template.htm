Creating an Excel Template

 

To Create an Excel Template

 

  1. Open the Excel Sheet.
  2. In the Excel sheet, enter the required columns (highlighted as “1”).

 

    

 

3. To extract the column values under the required columns, enter the column numbers as ${COL0}, ${COL1}, and so on (highlighted as “2”).

 

      

 The column values ${COL0}, ${COL1} and so on are the same as the columns under the Query section in publisher report.

 

For example, as shown in the figure below, ${COL0} under “Customer Name” in the excel sheet is same as the “COL0” under Query section which maps to “Customer Name” under Query Details in publisher report. 

 

 

Now, create different names (for example header, data etc.) that will be used as bands with the same name (as in excel sheet) in the ORBIT Publisher Report.
 
 
 Creating a Name
 
 

  To Create a Name

 

1. On the Formulas menu, in the Defined Names group, click Name Manager.

2. Name Manager dialog appears.

 

     

 

3. To create a new band, click New (highlighted as “1”).

4. New Name dialog appears.

5. In the New Name dialog, perform the following:

 

  • In the Name box, type the name of the band.

 

        Example: Name: Header

 

  • To enter the values in the Refers to box, click beside the Refers to.

 

           

 

  • New Name – Refers to dialog appears with the cursor inside the textbox.
  • Select the required cells from the excel sheet (highlighted as “2”).

 

          

 

  • In the New Name -Refers to dialog, the selected values appear in the textbox as shown below:

 

          

   

  • Click to navigate to the New Name. 

 

  • The selected cell values appear in the Refers to textbox (highlighted as “3”).

 

              

 

 

  • To save the changes, click OK.
  • The band name appears in the Name Manager dialog (highlighted as “4”).

 

        

 

  • To close the dialog, click Close.

 

  • Now create a new band “Data”, and place the column values ${COL0}, ${COL1} and so on in the Data band using the above process.
  • Save the Workbook with an appropriate name and close it.

 

  • Table and Pivot cannot be used for creating templates in Excel.
