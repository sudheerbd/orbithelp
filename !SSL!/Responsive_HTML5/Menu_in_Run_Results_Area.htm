Menu in Run Results Area

Field

Description

 Run

To run the report

 Save

To save the report.(This option is available in Edit Mode.)

 View Mode/  Edit Mode

In View Mode, you can only view the report, but you cannot edit/save the report.

In Edit Mode, you can edit the report. Edit Mode is available only to report owner. If the report is shared, then you only have View Mode.

Hide/Show User Filters

To hide/show User Filters
Hide/Show Grid Filters To hide/show Grid Filters. This option is available only in view mode.

Clear grid filters

To clear grid filters. This option is available only in view mode.  

 

 
Table Displays the report as a table.
Pivot Displays the report as a pivot.

Maximize/Minimize

To maximize/minimize the report window.

Options

Options provides the following options:

  • Properties (available in Edit Mode )

  • Save as New View (available in Edit/View Mode)

  • Save this View( available in Edit/View Mode)

  • Manage Views (available in  Edit/View Mode)

  • Auto Column Size (available in Edit/View Mode)

  • Export (available in Edit/View Mode)

  • Excel xlsx

  • Conditional Formats (available in Edit Mode)

  • Drilldowns( available in edit mode)

  • Group Filters( available in Edit Mode)

  • Generate Link (available in Edit/View Modes)

  • View SQL (available in Edit/View Modes)
