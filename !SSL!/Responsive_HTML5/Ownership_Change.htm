Change Ownership

To change the ownership

1.      In the Monitoring section, click New Process, and then click Change Ownership.

2.     Change Ownership dialog window appears.

3.      In the Parameters tab, perform the following:

·         In the From User box, type/select the required user.

 

? Example: From User: XXFORX

 

·         In the To User box, type/select the required user. 

 

? Example: To User: admin

 

4.      To submit the process, click Submit.

 # If the process must be run immediately, select the check box before Run Immediately. If the process must be run a later time, then remove the checkmark from the Run Immediately checkbox.  When the Run Immediately checkbox is deselected, the Schedule tab becomes available. Using the Schedule tab, schedule the process at a required time.
