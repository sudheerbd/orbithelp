Viewing Outputs

 

To view the output for a report

1.      Under Scheduled Processes, select the required report (highlighted as "1").

2.      For the required report, output is displayed under Output(s) & Delivery area (highlighted as "2").  

3.      Under Output(s) & Delivery area, the following columns appear:

  •          Output Type

  •           Phase

  •      Status

  •      Output

  •       Log

 

4. Under Process Details area (highlighted as "3"), the details of the process appears.

5. Under Parameters area (highlighted as "4"), the parameters/parameter values of the selected report appear.
