Menu Options in Reports

 

·         To create a new report, click New Report (highlighted as “1”).

·         New Report provides the following three options (highlighted as “2”):

o  Data Report

o   Analysis

o   Dashboards

o   Publisher Report

o   Script Report

 

 

             

·         To unpin the reports area, click Unpin this item (highlighted as “3”).

·         To search the reports, click Search (highlighted as “4”).

 

  •     A Search box appears.  

  •     In the Search box, type the name of the required report.   

  •    The required report appears below the Search box.

          

   

           

                    

·         To change the view of the structure under Reports area, click Change view  (highlighted as “5”)

·         Change view provides the following two options:

o   Folder view

o   View by widget

        

 

Folder View

 

             

 

View by widget

                

 

·         To collapse the complete folder structure, click Collapse all (highlighted as “1”).  

·         To expand the complete folder structure, click Expand all (highlighted as “2”).

·         To refresh the folder structure, click Refresh (highlighted as “3”).

 

         

To share the reports, click Share Reports.
