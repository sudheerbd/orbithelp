Renaming Labels

To rename labels

 

1.      Under Domain Hierarchy, Physical Models folder appears.  

2.     Under the folder, right-click the required folder, and then click  Rename Labels.

 

       

                        

 

3.      Rename Label Criteria-<Physical Model Name> dialog box appears.

 

                                

 

4.      A list of various label criteria (highlighted as “1”) appears.

5.      Select the required label criteria, and then click Apply.  

 

¥   If the desired criterion is not present in the available list, then you create a new criterion by clicking (highlighted as “2”).

 

To create new criterion

 

  1. In the Rename Label Criteria-<Physical Model Name> dialog box, click New Criterion.

 

                

                              

 

 

  1. New Criterion dialog box appears.

 

            

 

 

¥  In the New Criteria dialog box, you can set your own criterion on renaming the labels of the physical model.

 

 

  • Under New Criterion dialog box, perform the following:

 

·         In the Find box, type the terms in the label that need to be replaced.

                  Example: Find: Test

·         In Replace With box, type the term that needs to replace the existing term. 

                  Example: Replace With: Sample

·         Under Options area, select one or more from the following:

  • Case Sensitive
  • Whole Word
  • Regular Expression

               Example: Options: Whole Word

  • To save the changes, click OK.
  • To discard the changes, click Cancel.

  

            

 

          The new criterion (highlighted as “2”) appears under the list of existing criteria.

 

           

 

  • To apply the new criterion, click Apply.
