Import MetaObject Security

To Import MetaObject Security

Prior to importing metaobject security it is essential to import the domains and reports having the corresponding securities.

For more information refer to Importing Domain and Importing Report.

 

To Import MetaObject Security  

 

 1. Open the target instance, and then login with the required credentials.

   2. On the left-hand side menu, point to Monitoring, and then click Processes.

 3.  Scheduled Processes page appears with available processes.

       4.   On the right-hand side, click New Process , and then click Import MetaObject Security (highlighted as “1”).

 

            

 

 

      5. Import MetaObject Security dialog appears.

      6. In the Import MetaObject Security dialog, under the Parameters tab, click Upload beside the Upload Security text box.

      7. Upload the exported file.

      8. To merge with already existing securities,select the Merge Security checkbox.

c    9. To import the security details of metaobjects, click Submit.

     10. This generates a process under the Scheduled Processes in the Processes section.

     11. The imported metaobjects appear under the Business Object section in the Security menu.
