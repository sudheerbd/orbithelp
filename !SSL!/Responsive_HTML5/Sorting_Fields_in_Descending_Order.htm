Sorting Fields in Descending Order

To sort reports in descending order

1.      In the Run Results area, move the cursor over the required column.

2.      Downward arrow  appears beside the column name.

3.      To sort the required column, click  beside the required name.

4.      From the menu, click  Sort Descending (highlighted as “1”).  

 

         

                                              

5.      The values under the required column are sorted in descending order.
