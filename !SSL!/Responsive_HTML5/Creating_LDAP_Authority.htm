What is LDAP?  

 

To Create LDAP Authority

To create a new authority

1.      On the User Management menu, click Authorities.

 

              

                       

2.      Authorities page appears with a list of authorities (highlighted as “1”).  

   

       

 

3.     To create a new authority, click New (highlighted as "2") on the top right-hand side of the Authorities page.  

   

              

4.      New Authority dialog box appears.

     

 

5. Under New Authority, enter the following to create a new authority

  • In the Name box, type the name of the authority.              

  • In the Authority Key box, type the authority key;

  • To sync the users/settings on login, select Sync on Login checkbox.

  • In the Description box, type the description for the authority.

  • Under the Authentication section, in the Authority Type box, type/select the required authority type from the following:

    • Oracle ERP Security

    • LDAP

    • Active Directory

    • People Soft

      • Example: Required Authority Type:  LDAP

    •  Based on the selected authority types, the fields in the Authentication section change.

  • In the Server URL box, type the URL of the Server.

  • In the User DN box, type the User DN.

    •      In the Password box, type the password.

    •      In the Base DN box, type the Base DN.

  • On the Authorization section, click Add Authority.

Authorization dialog appears.  

         

             In the Authority Type list, select the required option from the following:

  • Oracle ERP Security

  • LDAP

  • Active Directory

  • People Soft

From the Data Source list, select the required data source or click Create new datasource to create a new data source.

To save the changes, click Ok.
