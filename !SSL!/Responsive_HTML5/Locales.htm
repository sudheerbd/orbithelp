Locales

Locales provides localization for the required features/options.

Localization enables a product to adapt the language of a specific region or a country.

 

To Work with Locales

1. On the left-hand side menu, click Settings.

2. In the Settings page, click the Locales tab.

3. On the Locales tab, select the required language (highlighted as “1”) from the Language list.

 

 

 

4. Under the Translated Labels section, the following fields appear:

·         Screen Name

·         Key

·         Label

·         Default Label (change the spelling in the screen ).

5. Under the Translated Labels section, the values(Screen Names/Key) of the  different screens appear.

6. Under the Label section, the user can change the name of the required object(s) in the desired language.

 

 

 

For example, as seen in the above example, if the user wants to rename “My Account” in Spainsh, then the user must type the relevant label (highlighted as “1”) under Label section.

 

7. To save the changes, click Save.

 

 

To Change the Language Settings

 

1. On the top right-hand side corner, click the user details section , and then click Settings.

2.Settings dialog box appears.

3.In Settings dialog , in the Language box, type/select the required language.

 

 

 

4. To save the changes, click Ok.

Once the settings have been changed in the Settings dialog, the user will be able to see the required language changes (highlighted as “1”).
