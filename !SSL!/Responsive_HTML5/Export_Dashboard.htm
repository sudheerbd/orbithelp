Exporting Output in Dashboard

To export output in dashboard

 

1.      On the top of the Run Results area, click  Options (highlighted as “1”), click Export (highlighted as “2”), and then click any one of the following options (highlighted as “3”):

a.       PDF

b.      HTML

 

Example: Required Output: PDF

 

 

 

2. A message " Process submitted successfully" appears.

 

3. Export is done as a process, and once the export is completed  a message "<report name><output type>  export is completed successfully. Please check in downloads" appears.

4. To view the exported report, click Downloads (highlighted as "4"), and click the downloaded output.

    (or)

     Navigate to Processes in the Monitoring ,under Scheduled Processes, select the required process; click the output that appears in the Output(s) & Delivery section.

 

5. The output appears in the required output format.
