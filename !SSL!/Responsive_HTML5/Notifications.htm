Notifications

To create notifications

·         On the Notifications tab, in the Notifications area, click Add to set new notification.

 

        

    

                                                      Figure: Displaying Notification Tab

 

·         In the Status list, click the required status option from the following:

a.      Success

b.      Failure

c.       Warning

·         In the Message box, type the required message.

·         In the Users box, type/select the required option

 

  •  Delete option becomes available only when a notification is created.

 

·         To delete the notification, click (highlighted as “2”).

 

                                           Figure: Displaying Delete Option

 

 

·         To submit the notification, click Submit (highlighted as “3”).

·         To notify about the end of the process, select the Notify me when this process ends checkbox (highlighted as “4”).

·         In the Notes box (highlighted as “5”), type the notes for the notification.

·         To submit the changes, click Submit.

 

  • The Notes entered in the Notes box, is sent as a notification when the process ends.
