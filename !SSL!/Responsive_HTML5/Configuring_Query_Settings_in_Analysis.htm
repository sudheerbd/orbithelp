Configuring Query Settings

To Configure Query Settings

1.      On the Query Settings tab, perform the following:

a.       To select distinct, select Select Distinct checkbox (highlighted as “1”).

b.      To set the row limit, select the Row Limit checkbox (highlighted as “2”), and then type/select the required number in the textbox beneath Row Limit checkbox.

c.       To set the query timeout session, select the Query Timeout checkbox (highlighted as “3”), and then type/select the required number in the textbox beneath Query Timeout checkbox.

d.      To auto-refresh the report, click Auto Refresh (highlighted as “4”).

 

        

            

2.      To save the changes, click Apply.
