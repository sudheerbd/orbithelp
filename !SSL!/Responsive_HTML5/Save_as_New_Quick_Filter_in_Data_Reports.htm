Save as New Quick Filter

To Save the New Quick Filter

 

1.      Drag and drop the required report onto the run results area.

?  Example: Required Report: Test Superstore

 

The report here has the parameter "Region" and the parameter values "East" and "Mountain"

2.      On the top right-hand corner of the run results area, click Options, and then click Save as New Quick Filter (highlighted as “1”).

  

        

 

 

3. Report Quick Filter dialog box appears.

 

 4. Under the Report Quick Filter, perform the following:

 

a. In the Name box, type the name of the quick filter.

 

?  Example: Quick Filter Name: Mountain and East Zones

 

b. To save the changes, click Save.

 

 

  •  The quick filter is saved as "Mountain and East Zones" and then the quick filter appears in the Select Quick Filter list (highlighted as "2") as shown below

           

                      

 

6. Select the required quick filter from the list, and then click Run.

7. The report appears with the selected quick filter as shown below:
