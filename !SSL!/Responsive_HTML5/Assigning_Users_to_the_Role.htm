Assigning Users to the Role

To assign Users to the Role

  1. Select the required role, and then click  next to Assigned Users.

  2. A list of users appears.

  3. Select the required user from the list.

    • Example: Required User: XXFORX

 

     

                                            

  1. Under Assigned Users, the selected user appears.

  2. To save the changes, click Save.
