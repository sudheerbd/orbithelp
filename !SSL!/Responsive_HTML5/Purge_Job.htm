Purging Processes

To purge a process

 

1. In the Monitoring section, click New Process, and then click Purge Processes.

2. Purge Processes dialog window appears.  

    

   

                                                  Figure: Purge Process Dialog Box

 

3. The Parameters tab appears by default.

4. Under the Parameters tab, the following options appear:

 

  • In the Till Date box, type the till date to delete the processes that are scheduled up to that date.

  • in the Users box, type the names of the users to delete the processes  that are scheduled by them.

  • In the Processes box, type/select the number of processes.

  • In the Week(s) box, type/select the number of weeks.

  • To report the statistics with the above-mentioned criteria (Till Date, Users, and Processes), select the Report Statistics (with above criteria) checkbox.

 

5. To purge the process, click Submit.
