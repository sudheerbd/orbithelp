Renaming Physical Model

 

To rename a physical model

 

1.     Under Domain Hierarchy, Physical Models folder appears.  

2.     Under the folder, right-click the required folder, and then click Rename.

 

    

                                  

 

 

3.     Edit -<Physical Model> dialog box appears.

 

    

                            

 

4.      In the Name box, edit the existing physical model name.

5.      In the Description box, edit the description.

6.      In the Data Source box, type/select the required data source.

(OR)

Create a new data source, by clicking Create new datasource.

7.      To save the changes, click Save.

8.      To close the dialog box, click Cancel.
