Displaying ORBIT Analytics Reports in Excel Edge

To display ORBIT Analytics Report in Excel

1.      In the ORBIT Excel EDGE application, perform the following:

a.       In the username box, type the required username.

b.      In the password box, type the required password.

c.       To logon to the application, click Login.

  

  

 

 

2.      In ORBIT Excel Edge, Reports section appears with the available report folders.

3.      Under the Reports section, select the required report.

 

4.      The parameters for the selected report appears as shown below:

 

5.      In the Parameters box, type the search term or click  to view the parameters, and then click Run.

                         State: Georgia
