Multiple Drilldowns in a Report

 

To perform multiple drilldowns

 

1.      In the Drilldowns dialog box, under the parameters of the first drilldown report, drag and drop the second report (highlighted as “1”).

 

 

                                         

2.      Drilldown Column and Navigate To appear for the second report, along with the available parameters of the report.  

3.      From the Report Configurator area, drag and drop the required column into Drilldown Column box.

4.      In the Navigate To box, type/select the required option.

5.      In the box under the Parameters (highlighted as “2”), type/select the required parameter value.

 

             

                                                

6.      To save the changes, click Ok.

7.      The drilldown report appears as shown below:

 

                  

Example:

 

·         To view the drilldown report State-wise, click Washington, under State Column.

·         The drilldown column name appears beside the report name (highlighted as “3”).

 

        

                                                  

 

 

·         To view the drilldown report Region-wise, click West, under Region Column.
