Renaming a Logical Model

To rename a logical model

1.      Under the Domain Hierarchy, Logical Models folder appears. 

2.      In the Logical Models folder, right-click the required logical model, and then click Rename

 

       

                                  

3.      Edit - <Logical Model Name> dialog box appears.

 

       

                                

4.      In Edit- <Logical Model Name> dialog box, perform the following:

a.       In the Name box, type a new name for the logical model.

b.      In the Description box, type the description.

c.       To save the changes, click Save.

d.      To cancel the changes, click Cancel.
