Exporting Domain

To export domain  

 

1. Login to the source instance, and then navigate to Domains & Data Models section.  

 

2. Under the Domains & Data Models select the required domain that must be exported, and then click Export.  

 

 

     Example: Required Domain: Test Domain -export-import

 

 

 

 

3. Export Domain dialog appears.  

 

    

 

  

4. To submit the request and export the domain, click Submit.  

5. A success message appears once the domain is exported, and a new process is created under Processes under Monitoring.  

6. Select the required process (highlighted as “2”), and its corresponding output appears under Output(s) & Delivery.  

 

  

  

 

7. Select the required output.

8. The output gets downloaded as an xml file.

9. Place the xml file of the exported domain in an appropriate location on the system.
