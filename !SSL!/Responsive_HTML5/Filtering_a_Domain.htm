Filtering a Domain

To filter a domain

1.      Under Domains & Data Models area, click .

2.      Textboxes to filter a domain (highlighted as “1”) appear under Name, Description, and Created By.

 

       

                                 

3.      Under the required column, type the search term to filter.

 

    •  Example: Required Column: Name

    •  Example: Search Term: AP

4.      Based on the filter value (highlighted as “2”), the search results appear as shown:

 

       

                                                  

5.      To refresh the domains, click Refresh.
