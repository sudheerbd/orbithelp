Configuring Query Settings

 

To Configure Query Settings

1.      In Report Configurator, click Query Settings (highlighted as “1”).

 

   

   

2.      To make the fields distinct, select the Select Distinct checkbox.

3.      To set the row limit, select the Row Limit checkbox.  

4.      Select/type the number of rows in the box beneath the Row Limit checkbox.

5.      To set the timeout session of the query, select Query Timeout.

6.      Select/type the number of seconds in the box beneath the Row Limit checkbox.

7.      To auto-refresh the report, click Auto Refresh.

8.      To save the changes, click Apply.
