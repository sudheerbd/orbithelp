Editing Roles

To edit Roles

1.      On the User Management menu, click Roles.

2.      Role List appears with a list of available roles.

3.      For the required role, under Actions, click  Edit (highlighted as "1") .

 

                                                            

 

  • For the seeded roles â€“Admin, Analyst, and User -- and the imported roles, the option to edit is unavailable.

 

 

 

4. Edit Role- < Role Name> dialog box appears.

 

      

                          

5. In the Edit Role dialog box, perform the following:

  • In the Name box, type/edit the role name.

  • Example: Name: Test Invoice Role

  • In the Description box, type/edit the role description.

    • Example: Description: This is a sample description.  

  • To save the changes, click Save.

  • To close the dialog box, click Cancel.
