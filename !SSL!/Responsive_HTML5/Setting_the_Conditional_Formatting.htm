Setting the Conditional Formatting

To set the conditional format

 

1.      For a selected report, on the reports menu, click Options, and then click Conditional Formats. 

    

                     ?     Example: Required Report: superstore sample

 

            

 

2.      Conditional Formats dialog box appears.

 

              

 

3. To add the required field to the Conditional Formats dialog, drag and drop the required field from the Columns section under Configurator.

4.      The selected field appears as shown below:

                 ?     Example: Selected Field: Sales

 

 

 

5.      To perform the conditional formatting for the selected field, perform the following:

a.       In the Operator list, click the required option.

 

?     Example: Operator: is greater than or equal to

 

b.      In the Value box, type the required option.

?     Example: Value: 10000

c.       In the Font Family box, type/select the required option.

?     Example: Font Family: Times New Roman

d.      In the Font Size box, type/select the required option.

?     Example: Font Size: 14

e.       In the Font Color box, type the required option in the box or select the required option from the color palette.

?     Example: Font Color: ea4747

f.        In the Background box, type the required option in the box or select the required option from the color palette.

?     Example: Background: eae5e5

g.      In the Alignment box, type/select the required option from the following:

a.       Left

b.      Center

c.       Right

 

?     Example: Alignment: Center

h.      Under the Style, click the required style from the following options:

a.        Bold

b.      Italic

c.       Underline

       Example: Style: Bold

                          Underline

i.        To save the changes, click Ok.

j.        To close the dialog box, click Cancel.

k.      The formatted field appears as shown below (highlighted as “2”):

 

 

Note:

  • When the report with conditional formatting is exported with excel, then font family,font color and alignment are not getting applied.
