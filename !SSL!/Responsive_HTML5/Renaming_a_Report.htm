Renaming a Report

 

To Rename a Report

1.      Under the Reports area, the required report appears.

2.      Right-click the required report, and then click Rename (highlighted as “1”).

 

          

              

3.      Rename <report name> dialog box appears.

a.       In the Name box, type the name.

b.      In the Description box, type the description.

 

                       

 

4.      To save the changes, click Save.

5.      To close the dialog box, click Cancel.

6.      The renamed report appears under Reports area.
