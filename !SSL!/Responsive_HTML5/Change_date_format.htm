Changing Date Format Globally

To change the date format globally

 

 1.      On the left-hand side menu, point to Monitoring, and then click Processes.

 2.     Scheduled Processes page appears with available processes.

       3.      On the right-hand side, click New Process , and then click Change date format (highlighted as “1”).

 

             

 

      4. Change date format dialog appears.  

 

         

 

       5. In the Change date format dialog, perform the following to change the date format globally:

 

    • In the Date Format box, type/select the required option.

 

                    

 

             Example: Date Format: yyyy-MM-dd

 

    • To update the date format of all the users, select All Users checkbox (highlighted as "2").

 

    • If All Users checkbox is unchecked, users with default date format (MM-dd-yyyy) will be updated.

 

  • The default date format (highlighted as "3") is displayed as shown below:

    

         

 

    •  When the date format is changed to the required format, it appears as shown below (highlighted as "4"):

 

 n               

 

             Here, the date format is changed to yyyy/mm/dd format.

 

 

      When All Users checkbox is selected,  the date format changes across all the users. You can login into your user, and verify the date format.

 

      When no date format is selected in the Settings section, then the global date format in the Processes section overrides the default date format in the Settings section.
