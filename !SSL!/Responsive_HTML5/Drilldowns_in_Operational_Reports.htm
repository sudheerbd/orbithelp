Drilldowns in Data Reports

To work with Drilldowns

 

1.      Create a Report on which the drilldown needs to be performed.

2.     Save the report by clicking Save.

3.     The report (highlighted as “1”) appears under Reports on the left-hand side.

 

           

 

4.     Open the newly created report. 

5.     On the top right-hand corner of the Run Results area, click Options, and then click Drilldowns (highlighted as "2").

 

                                  

 

6.      Drilldowns dialog box appears.

       

       

                                    

 

7.      Under Drilldowns dialog box, perform the following:

 

·         Drag and drop the required drilldown report that has the required parameters (highlighted as “3”) under the Drilldowns area.

 

                                               

 

·         Drilldown Column and Navigate To options appear.

 

                                               

 

·         Under Parameters area, the required parameter (highlighted as “4”) appears with a list of available values for the parameter.

·         From the required report, drag and drop the required columns from report configurator area of the required report to the Drilldown Column box (highlighted as “5”).

 

                                               

 

·         In the Navigate To box (highlighted as “6”), click the required option from the following:

a.      New Tab

b.      Window

 

·         From the Parameters Value box (highlighted as “7”), select the required option from the list.

·         To save the changes, click Ok.

8.      The drilldown report appears as selected in Navigate To box.

9.      The drilldown down column appears as shown below:

 

    

                                                         

 

10.  Select the drill down column to view the result.
