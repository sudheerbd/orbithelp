Applying Filters in Logical Model

Filters set here in the logical model are pre-defined conditions and the user must enter these conditions

 

Applying Filters in Logical Model

 

1. For the selected logical model, Columns and Filters appear in the Logical Model workspace.  

 

 

      

 

                                      

2. Drag and drop the tables/views/synonyms from the required physical model on the left-hand side.

 

 

     

 

3. Select the table/view for which the columns under it should be displayed.

4. The columns for the selected table/view appear under the Columns -<Synonym Name>.  

 

  • To view the columns under the selected table/view/synonym, you need to first save the logical model.

 

5. The columns appear as show below:

 

    

                                         

 

6. Drag and Drop the required column from Columns area to Filters area to set the filter conditions.

7. To view the filter conditions for the selected column, click for the selected column (highlighted as "3").

8. Filter-<Column Name> dialog box appears.

 

                                                                              

9. By default, Select a range option is selected.

 

  • If you want to change a condition, click Specify a condition.

 

10. To save the changes. click Ok.

11. To save the domain, click Save on the menu area.

        

      

                

  •  The report appears with the specified filter column when it is run. The user must enter the condition to run the report.
