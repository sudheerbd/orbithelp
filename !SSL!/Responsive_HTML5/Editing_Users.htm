Editing Users

To edit users

1.      On the User Management menu, click Users.

2.      The User List appears.

3.      For the required user, under Actions, click  Edit.

4.      Edit User <user name> dialog box appears.

 

      

                   

5.      Under Edit User <user name> dialog box, perform the following:

 

a.      In the First Name box, type the first name of the user.

b.       In the Last Name box, type the last name of the user.

c.      In the Email box, type the email of the user.

 

    •  You cannot edit the User Name.

 

6.      To save the changes, click Save.

7.      To close the dialog box, click Cancel.
