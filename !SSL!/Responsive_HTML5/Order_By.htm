Order By

To order by a column

1.      Under Conditions area, the Order By area appears.

2.      Drag and drop the required column from the Business Objects area to Order By area.

3.      Under the Order By area, perform the following:

a.       In the Aggregation box, type/select the required aggregate function from the following:

·         NONE

·         SUM

·         AVERAGE

·         COUNT

·         COUNT_DISTINCT

·         MINIMUM

·         MAXIMUM

b.      In the Column Name box, the column name appears automatically.

c.       In the Order box, type/select the required option from the following:

·         ASC

·         DESC
