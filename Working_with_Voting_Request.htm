Working with Voting Request

To work with Voting Request

1. In the chat window of the selected thread, click  in the textbox at the bottom of the window.

2. A menu appears with the following options:

a.    Snapshot

b.    Document

c.    Voting Request

3.    To create a voting request, click Voting Request.

                     

                   

 

4.    Voting Request dialog box appears.

5.    In the Voting Request dialog box, do the following:

a.    In Message box, type the voting message.

 

   

 

b.    Click the arrow next to the Voting Values box, and select the option from the list

      • Yes-No-Neither

      • Agree-Disagree-Maybe

Note: The selected voting configuration values appear beneath the Voting Values box.

     c.     To save the changes, click Save.

Note: A Voting request saved successfully notification appears briefly at the top of the ORBIT application window.

6.    The Voting Request appears as shown below:

 

          

                  

7.    To vote, select the required option, and then click Vote.

           

8.    The vote appears as follows:

 

          

 

9.    To Refresh the thread chat, click  Refresh.
