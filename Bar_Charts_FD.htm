Bar and Column Charts

 

The Cartesian charts utilize the coordinate system: X and Y axes in displaying the data. The following charts come under Cartesian charts.

  • Bar Basic, Stacked Bar, and 100% Stacked

  • Column Basic, Stacked Column, 100% Stacked, Multi Axis, and Negative Values charts

  • Line Basic, Spline, Step Line, Area, and Area Spline charts

  • Combination charts

In this topic, visualization options and visual settings are explained using Bar basic, Column basic, and Combination chart examples.

 

Visualization (Chart type) Options

A bar chart displays data using rectangular bars. One axis of a bar chart measures a value, while the other axis shows the specific categories being compared.

A bar chart usually consists of row, column, and data (value) fields.

1. Bar charts has the following visualization options

  • Values (Y-Axis)

  • Category (X-Axis)

  • Color

2. Drag and drop the required fields from the selected Business Object into the Values (Y-Axis), Category (X-Axis), and Color sections.

Note: Drag fields with numerical values into the Values section for several types of calculations (Sum, Average, Min, Max, etc.) However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the bar chart analysis report dynamically in the run results area.

 

Manage Colors

In the Color section, user can choose the color scheme of the chart and change color of individual series.

To change color scheme of the chart

1. Click the expander arrow next to Color in the Report Configurator area and then click Manage Colors.

2. Manage Colors dialog box appears.

3. Click on the Color Palette. Color Palette list appears.

4. Select desired color palette from the list.

5. To change default color of the Series

  1. Click the Color box next to the series name to open the Color Picker dialog box.

  2. Option 1: In the Color Picker, drag the vertical slider above H to choose a hue (color). Then click a spot in the large color box to choose the brightness and saturation of that color.

  3. Option 2: In the Color Picker, drag the vertical sliders above H (hue - color), S (Saturation), and V (Value - Brightness) to choose a color.

  4. Option 3: Enter exact color values into the HSV, RGB, or Hexadecimal color fields.

  5. To change opacity, drag the vertical slider above A to choose alpha (opacity).

  6. Click OK to close the Color. The color you chose appears in the color box.

6. Click Reset series colors to reset colors to default color of the selected color palette.

7. To reset colors to default color palette, click the expander arrow next to Color and then click Reset Colors.

  2. Axes: See Customize Axes Visual Settings for more details.
  3. Series: See Customize Series Visual Settings for more details.
  4. Legend: See Customize Legend Visual Settings for more details.
