Creating a New Data Source

To create a Data Source

 

  1. On the left-hand side, point to  Business Models, and then click Data Sources.

  2. Data Sources page appears with a list of available data sources.

 

         

                                                            

  1. To create a new data source, click New.

  1. A list of database management systems supported by ORBIT appears.

 

          

                          

  1. From the list, select the database you want to connect to and click Ok.

You can connect to any database from the list, for which you have connection details.

  1. New Data Source dialog box appears.

 

  1. To create a new data source, do the following:

    • In the Name box, type the data source name.

    • In the Description box, type the description for the data source.

    • In the Data Source Type, select the required data source type radio button from the following:

      • JDBC

      • JNDI

    • In the Select Database, enter or select the database from the list of available database management systems.

By default, name of the database selected in step 5 appears in the Select Database box.

    • In the Username box, type the user name.

    • In the Password box, type the password.

    • In the Host Name box, enter the host name of the computer where database is installed.

    • In the Port box, enter the port number.

    • In the Database Name box, enter the name of the database.

Based on type of the database selected, settings fields may differ under <Database> Settings.

Alternatively, select the Enter custom JDBC URL checkbox to enter the custom JDBC URL.

 

 

    • To test the data source connection, click Test Connection. If the test is successful, the word Connection success is displayed in the Status field.

If the test is unsuccessful, correct any configuration errors and retry the test.

    • To save the data source, click Save.

A Datasource saved successfully notification appears briefly at the top of the ORBIT application window.

    • To discard the changes, click Cancel.

Driver Class and Data Source Class are automatically populated based on the Select Database option.
