Pivot/Matrix Charts

Pivot tables are interactive tables that allow the user to group and summarize large amounts of data in a concise, tabular format for easier reporting and analysis.

Visualization (Chart type) Options

A pivot table usually consists of row, column, and data (value) fields.

1. Table chart has the following visualization options

  • Values

  • Rows

  • Columns

2. Drag and drop the required fields from the selected Business Object into the Values, Rows, and Columns sections.

Note: Drag fields with numerical values into the Values section for several types of calculations (Sum, Average, Min, Max, etc.)

However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the Pivot/matrix analysis report dynamically in the run results area.

4. Expand and Collapse buttons are displayed so that you can display or hide details in pivot  chart.

 

Visual Settings for Tabular Charts

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for pivot chart has the following tabs:

  1. General
  2. Sub Totals
  3. Grand Totals

General

General tab has the following visual settings:

A. Detail Data: Show/hide .

Default Value: False

In the Detail Data box, select True to show .

 

B. Layout: Changes pivot table layout.

Default Value: Outline

In the Layout box, select the required option from the following:

  • Outline

  • Compact

  • Tabular

  •  Tabular: Displays one column per field and provides space for field headers.

  2. Compact: Displays items from different row area fields in one column and uses indentation to distinguish between the items from different fields.

    Row labels take up less space in compact form, which leaves more room for numeric data. Compact form is saves space and makes the pivot table more readable.

  3. Tabular: Displays one column per field and provides space for field headers. When Layout value is set as Tabular, the report appears as follows:

 

C. Measure on Row: Show/hide measures column.

Default Value: False

In the Measure on Row box, select True to show the Measures column.

Note: Measures column appears in the report only when the Values section in the Report Configurator area has at least two fields.

Grand Totals

Grand Totals tab has the following visual settings:

A. Column Totals: Changes Column total position in the Pivot chart.

Default Value: Last

In the Column Total box, select the required option from the following:

  • Last: When Column Totals value is set as Last, the column grand totals appear at the end of the pivot chart.

  • First: When Column Totals value is set as First, the column grand totals appear at the start of the pivot chart.

  • None: When Column Totals value is set as None, the column grand totals do not appear in the pivot chart.

 

B. Row Totals: Changes Row total position in the Pivot chart.

Default Value: Last

In the Row Total box, select the required option from the following:

  • Last: When Row Totals value is set as Last, the row grand totals appear at the end of the pivot chart.

  • First: When Row Totals value is set as First, the row grand totals appear at the start of the pivot chart.

  • None: When Row Totals value is set as None, the row grand totals do not appear in the pivot chart.
