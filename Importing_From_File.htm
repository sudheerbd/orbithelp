Importing File

To import file

 

  1.  Under Domain Hierarchy, Physical Models folder appears.  

  2.  Under the folder, right-click the required folder, and then click Import from File.

 

       

                

  1. Create Object From File dialog box appears.

 

                               

  1. In the File Type box, type/select the required option.

  2. File Type field has the following options:

    • CSV or Other Delimited Text Format

    • Excel


File Type: CSV or Other Delimited Text Format
Here, CSV or Other Delimited Text Format option is selected.

 

  1. To select the required file, click Select File next to File box.

  2. Open dialog box appears,

  3. Select the required file, and then click Open.

  4. The selected file appears in the File box.

 

                                                              

  1. To proceed with uploading file, click Next.

  2. The columns in the CSV file appear with their associated data types.

 

    • Edit the data types as required.

    • Update the fields as required.

  1. To import the fields, click Import.
