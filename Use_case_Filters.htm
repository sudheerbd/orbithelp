Use case Filters

Please use below information for use-case validation.

Example group search filters:

Group name as group4 (&(objectCategory=Group)(objectClass=group)(cn=group4))

Group name starts with grou (&(objectCategory=Group)(objectClass=group)(cn=grou*))

Group name not as group4 (&(objectCategory=Group)(objectClass=group)(!(cn=group4)))

 

Note:

Like condition cannot be used for Group Filters as it has to be a particular case.
