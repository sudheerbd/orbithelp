Configuring the Settings

To configure the account settings

  1. On the top menu, click the icon or icon, on the top right-hand side of the window.

  2. A list of options appears as shown below:

  3. To configure the account settings, click Settings (highlighted as “1”).

  4. Settings dialog box appears.

  5. Under the Settings dialog box, perform the following:

    • In the Language box, type/ select the required language
    • In the Date Format box, type/select the required date format.
    • In the Default Page box, select the required page as the default page.

    The user can see the required page as the default page, once the page is refreshed or if the user logs out and logs in again.

  6. To save the changes, click Ok.

  7. To close the dialog box, click Cancel.
