Auto Refresh

To refresh the report automatically

 

  1. Open the required dashboard report in run results area.

Dashboard report Options available in both View and Edit Mode

  1. On the top right-hand corner of the run results area, click Options, and then click Options.

  2. Dashboard Options dialog appears.

 

  1. To auto refresh the report, select Auto Refresh check box.

  2. Once the auto refresh is selected, Duration(Seconds) field appears.

Duration field is taken in seconds.

 

  1. Enter the required time to refresh the report automatically.

  2. To save the changes, click Ok.

  3. To cancel the changes, click Cancel.
