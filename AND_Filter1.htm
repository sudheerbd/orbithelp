AND Filter

The AND Filter is used when there are more than one condition and where all the conditions in the series must be true. It is represented by an ampersand ‘&’.

Consider the following example, where the there is a need to retrieve all the users with different last names and whose first name is "User".

  1. Under the User Search Information section, in the Search Filter box, type the required search filter.

Example: Search Filter: (&(givenName=user) (sn=*))

  1. The above search filter retrieves the users with the First Name(givenName)— “user” and all the last names (*) such as User1, User2,and User3.
