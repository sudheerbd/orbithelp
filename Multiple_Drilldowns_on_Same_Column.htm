Multiple Drill-through on Same Column

To work with multiple Drill-through on same column

 

  1. In the Drill-through dialog box, on the Drill-through Column box, drag and drop the column from the report configurator.

  2. To work with multiple drill-through on same column, drag and drop the same column on the second Drill-through Column box from the report configurator. 

     

                            

  1. To save the changes, click Ok.

  1. The report appears as shown below:

 

                                     

  1. To view multiple drill-through on the same column, click the drill-through column.

  1. A list of drill-through reports appears for the selected drill-through column as shown below:

 

                  

  1. Select the required drill-through report from the list.
