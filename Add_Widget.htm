Adding widget to a dashboard report

 

In the dashboard report, on the top-menu, Add widget list appears with the following options:

 

  • Visualization

  •  Image

  • Styled Text

  • Audio

            

Note: Add Widget option will appear when Dashboard Report is in Edit mode.

 

 

Output of a Dashboard Report with combination of all the widgets

 

 

  • Visualization widget has been highlighted as "1"

  • Image widget has been highlighted as "2"

  • Styled Text widget has been highlighted as "3"

  • Audio widget has been highlighted as "4"
