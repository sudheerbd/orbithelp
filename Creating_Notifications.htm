Creating Notifications

To create notifications

 

  1. On the Notifications tab, in the Notifications area, click   Add (highlighted as "1") to set new notification. 

 

  1. In the Status list, click the required status option from the following:

a.    SUCCESS

b.    FAILURE                  

c.    WARNING

d.    SKIPPED

 

  1. In the Message box, type the required message.

  2. In the Users box, type/select the required option.

         

To delete notification

 

  1. To delete the notification, click (highlighted as “2”).

 

 

  1. To submit the notification, click Submit (highlighted as “3”).

  2. To notify about the end of the process, select the Notify me when this process ends checkbox (highlighted as “4”).

  3. In the Notes box (highlighted as “5”), type the notes for the notification.

Sample Notes

  1. To submit the changes, click Submit.

  2. The Notes entered in the Notes box, is sent as a notification when the process ends.

 

Also, the user is also notified via e-mail about the process completion.
