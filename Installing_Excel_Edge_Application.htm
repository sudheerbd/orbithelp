Installing XLEdge Application

To Install XLEdge Application

  1. Click the “OrbitXLEdge_<version>.msi” file.

  2. Installer dialog opens as shown below:

  3. Click Next and complete the installation process. Once the installation process is complete click Close.

  4. In case, any excel sheets are already open, close them, and re-open them to find the ORBIT Analytics tab (highlighted as “1”) in the Excel sheet.
