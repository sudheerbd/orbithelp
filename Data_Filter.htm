Local Filters

Local Filters are used to segregate huge data in the database while building reports. However, the local filters are unavailable to external users.

 

To apply a local filters on a report

 

  1. Drag and drop the required columns from the Select Business Object area into Local Filters (highlighted as “1”) section under Configurator tab (highlighted as “2”).

 

                        

  1. From the Business Object area, drag and drop the required column into Local Filters section.

 

 

  1. Select the required column from Local Filters section.

  2. To select the filter options, click on column name in Local Filter.

  3. A dialog appears as shown below:

 

  1. Here, the filter can be selected in any one of the following two processes:

 

Process 1

 

  • On the filter column dialog, click Specify a condition.

  • The following fields appear under "Show items when the value".

 

 

  • From the condition list, select the required condition.

  • In the textbox under the condition list, click Select value.

  • Choose Value dialog appears with the following options:

    • Static

    • Expression

    • Query

 

 

  • From the available options, select the required option to choose a value for the data filter.

  • Here, Static radio button has been selected.

  • For the Static option, in the Value box, type the required value for the user filter as shown below:

 

 

 

  • You can as well choose the value using Expression and Query options.

  • To save the changes, click Ok.

  • The value appears as shown below:

 

 

  • To save the changes, click Ok.

  • Run the report by clicking Run.

  • The report appears with the specified data filter.

 

 

Process 2

 

  • On the filter column dialog, On the filter column, click  Search for items to add.         

  • A list of filter values appears.

Example: Required Value: Central

 

 

  • Select the required value from the list.

  • The filter values appear under filter column dialog as shown above.

  • To run the report with data filter applied, click Run.

  • The report is generated in the Run Results area with filter applied on it. 

 

 

To clear applied value from filter section

 

  1. To clear the applied filter values, click next to Local Filters (highlighted as "3").

  2. Clear All Filter Selections option appears.

  3. This option clear the applied filter value as shown below:

 

 

To remove item from filter section

 

  1. To remove a filter column from Local Filters, click Remove this item next to the column name (highlighted as "4").
