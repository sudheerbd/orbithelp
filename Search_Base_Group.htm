Search Base Group

To retrieve the Users under a Group

  1. Under the Group Search Information section, in the Search Base box, type the starting point of the search.

E.g: Search Base: OU=orbit2

 

Prior to this fill all the required Attributes. See Attributes Active Directory

2. This retrieves all the Uers under the mentioned value and  in the User Name box, type the required Username, and then click Search.

3. The selected user appears under Available Users.

4. Select the required user and then click  . The selected user appears under Selected Users as shown below.
