Scatter, Bubble Charts

 

The Scatter and Bubble chart displays pairs of numerical data on the graph as a dot, with one variable on each axis. This type of chart can be used in to visually describe relationships between two numerical parameters or to represent distributions.

In this topic, visualization options and visual settings are explained using Scatter Charts examples. These visual settings are also same for Bubble charts.

Visualization (Chart type) Options

1. Scatter charts has the following visualization options

  • Y Axis

  • X Axis

  • Category (Points)

  • Color

2. Drag and drop the required fields from the selected Business Object into the X Axis, Y Axis, and Category (Points) sections.

Note: Drag fields with numerical values into the Values section for several types of calculations (Sum, Average, Min, Max, etc.) However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the scatter maps analysis report dynamically in the run results area.

4. Drag and drop field from the selected Business Object into the Color section.

 

Visual Settings for Scatter Charts

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for scatter chart has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Axes: See Customize Axes Visual Settings for more details.
  3. Series: See Customize Series Visual Settings for more details.
  4. Legend: See Customize Legend Visual Settings for more details.
  5. Tooltip: See Customize Tooltip Visual Settings for more details.

 

Threshold

Default Value: 1000

Enter a number in Threshold box, to limit maximum number of scatter points to show in the chat.
