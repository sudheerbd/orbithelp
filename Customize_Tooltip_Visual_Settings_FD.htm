Customize Tooltip Visual Settings

This topic explains different ways to customize the tool tips of your charts.

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

Note: You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Tooltip to open Tooltip visual settings.

Tooltip

Tooltip has following visual settings:

  1. Tooltip: Show/Hide tooltip when mouse hover over on a chart element.

  2. Shared between Series: Show combined tooltip of the selected series.

  3. Show Values: Show/Hide value of the slice in the tooltip.

  4. Show Percentage: Show/Hide percentage value of the slice in the tooltip.

  5. Background Color: Changes background color of the tooltip.

  6. Border Width: Changes border width of the tooltip.

  7. Border Corner Radius: Changes border corner radius of the tooltip.

  8. Border Color: Changes border color of the tooltip.

The visual settings available depend on the visualization type selected.

Tooltip

  1. Select Tooltip check box to show tooltip when you hover over a bar on the chart.

  2. Clear Tooltip check box to hide tooltip when you hover over a bar on the chart.

Shared between Series

Select Shared between Series check box to show combined tooltip of selected series.

Show Value

 

  1. Select Show Value check box to show value of the slice in the legend.

  2. Clear Show Value check box to hide value of the slice in the legend.

Show Percentage

  1. Select Show Percentage check box to show percentage value of the slice in the legend.

  2. Clear Show Percentage check box to hide percentage value of the slice in the legend.

Background Color

Default color: White (RGB 255, 255, 255).

Select Background Color box to change the color of tooltip background. See Change default color in Negative Color for detailed information.

Border Width

Default Value: 1 pixels

  1. Enter any numerical value in the Border width box, to change border width of the tooltip.

Border Corner Radius

Default Value: 5 pixels

  1. Enter any numerical value in the Border Corner Radius box, to change border corner radius of the tooltip.

Border Color

Default color: White (RGB 255, 255, 255).

Select Border Color box to change border the color of tooltip. See Change default color in Negative Color for detailed information.
