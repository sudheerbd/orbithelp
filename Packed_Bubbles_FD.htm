Packed Bubbles

 

Packed bubble charts display data in a cluster of circles. Dimensions define the individual bubbles, measures define the size, Color defines color of the individual circles.

Visualization (Chart type) Options

1. Packed Bubble charts has the following visualization options

  • Value (Size)

  • Category

  • Color

2. Drag and drop the required fields from the selected Business Object into the Value and Category sections.

Note: Drag fields with numerical values into the Values section for several types of calculations (Sum, Average, Min, Max, etc.)

However, using text values would also not be wrong; instead of Sum, it will give a Count.

3. Orbit application generates the packed bubbles analysis report dynamically in the run results area.

 

Visual Settings for Packed Bubble Charts

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for packed bubble chart has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Series: See Customize Slice Visual Settings for more details.
  3. Value: See Customize Series Visual Settings for more details.
  4. Legend: See Customize Legend Visual Settings for more details.
  5. Tooltip: See Customize Tooltip Visual Settings for more details.
