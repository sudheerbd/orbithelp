Customize Slice Visual Settings

This topic explains different ways to customize the Slices of your charts.

1. Navigate to the Reports page.

2. Open an existing analysis report in edit mode or create a new one.

 You can customize visual settings of reports for which you have edit permissions.

3. Click Visual Settings > Slice to open Slice visual settings.

 

Slice

Slice tab has the following visual settings:

  1. Color Palette: Change color palette of chart elements.

  2. Max slices before other: Maximum slices to show in pie chart before combining remaining categories into Other group.

  3. Prefix: Adds text before data labels.

  4. Suffix: Adds text after data labels.

  1. Hierarchy view (Click on labels to drilldown to next level): Enables drilldown hierarchy (Applicable for Tree Maps)

  2. Layout: Change layout of tree map. (Applicable for Tree Maps)

  3. Date Format: Customize the date format. (Applicable for Tree Maps)

The visual settings available depend on the visualization type selected.

Color Palette

Default Value: Default

Select desired color palette from the list.

 

 

Max slices before other:

Default Value: 10

  • Enter any numerical value in the Max slices before other box, to set as max number of slices to show in pie chart.

  • Tip: Click on Other to drilldown into hidden categories.

 

Prefix

Enter text in the Prefix box to append before the labels.

Suffix

Enter text in the Suffix box to append after the labels.

 

Hierarchy view

Default checkbox will be unchecked

  1. To enable the hierarchy select Hierarchy view (Click on labels to drilldown to next level) checkbox

 

Layout

Default Value: Squarified

Layout has following options:

  1. Squarified: Select Squarified to change tree maps visualization to nested squares.

  1. Slice and Dice: Select Slice and Dice to change tree maps visualization to nested slices.

  1. Stripes: Select Stripes to change tree maps visualization to nested vertical stripes.

  1. Strip: Select Strip to change tree maps visualization to nested rectangles.

 

Date Format
  1. Customize the date format. Select any one of the date format from the drop down list.
