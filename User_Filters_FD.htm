User Filters

Procedure 1

To apply User Filters on a report

1. To enable User Filters, click Hide/Show User Filters (Highlighted as "1").

2. User Filters area (Highlighted as "2") appears above the Run Results area.

3. From the available fields, drag and drop the required fields into the User Filters area.

4. User Filter columns (highlighted as “3”) appears in the User Filters area.

5. Click on a filter column to add user filter. A pop-up menu appears.

6. In the Search box, enter the search string and click icon or press Enter.

Note: Click icon or press Enter in the empty Search box, to show all the filter items for the selected column.

7. List of items appear below the Search box.

8. Double click or drag the items into the selected item(s) list and click Ok.

Hold Ctrl and click on required fields to make multiple selections.

Use Ctrl + A keyboard shortcut to select all items.

9. The selected items appears in the User Filter Column and the analysis report updates with the user filters.

10. To add advanced filters, click Filter (highlighted as “4”) on the required user filter column.

11. Filter - <Field Name> dialog box (highlighted as “5”) appears.

12. To apply filter on the selected report, perform the following:

  1. Under Filter – <Field Name> dialog box, in the Label box, type the filter label.

  2. To make the parameter required, select Required check box.

  3. In the Name box, type the name of the parameter.

  4. To apply the filter, select any one of the radio button from the following:

    • Select from list

    • Match a specific string

    • Specify a condition

  5. Based on the selected radio button, fields underneath change.

  6. If no condition is required, select No Condition.

  7. Select Skip validation while searching, to skip validation.

  8. Type the required fields, and then click Ok.

  9. To close the dialog box, click Cancel.

13. To remove user filter, click Remove on the required user filter column.

 

Procedure 2

To apply User Filters on a report

1. In the Run Result Area, right click on the desired chart element.

2. A pop-up menu appears with following options

  • Include

  • Exclude

Note: Hold Ctrl to select multiple chart elements.

3. Select any one the options to add the selected value to the user filter.

4. The selected value appears in the User Filter Column and the analysis report updates with the user filters.

 

Note:In the View Mode, Hide/Show User Filters option is available. This enables the end users to add user filters.
