Scheduler Operations

Working with scheduler operations

 

  1. On the left-hand side menu, point to   system, and then click Maintenance.

  2. Scheduler page appears with Operations tab.

  3. Under Operations tab a list of options appears:      

    • State

    • Pause/Resume All

 

  1. Under State section, user have the below option:

    • Start: This option allows user to start a scheduler.

      1. To start the scheduler, click Stand By

      2. A message "Are you sure you want to Start Scheduler?" appears.

      3. To standby the scheduler, click Yes.

    • Stand By: This option allows user to standby a scheduler.

      1. To change the scheduler to standby, click Stand By

      2. A message "Are you sure you want to Standby Scheduler?" appears.

      3. To standby the scheduler, click Yes.

    • Shutdown: This option allows user to stop a scheduler.

      1. To shutdown the scheduler, click Shutdown

      2. A message "Are you sure you want to Shutdown Scheduler? The Scheduler cannot be restarted after shutdown has been called. " appears.

      3. To shutdown the scheduler, click Yes.

To restart scheduler, the server needs to restart.

  1. Under Pause All/Resume All, user have the below option:

    • Pause All: This option allow user to pause all running scheduler.

      1. To pause the scheduler, click Pause ALL.

      2. A message "Are you sure you want to paused Scheduler?" appears.

      3. To pause scheduler, click Yes.

    • Resume All:  This option allow user to resume all the paused scheduler.

      1. To resume the paused scheduler, click Resume ALL.

      2. A message "Are you sure you want to resume Scheduler?" appears.

      3. To resume the scheduler, click Yes.

  2. To refresh the page, click Refresh.

  3. Under Scheduler Metadata, scheduler name and descriptions displays.

Under description field, scheduler metadata displays the current state of the scheduler
