Template Based

To select a default template

 

1.  Open the required data report.

2.  On the top menu, click Options, and then click Templates. 

3. Templates dialog appears.

4.  To generate a default template automatically, click Generate Default Template.   

    

 

5. To edit the generated default template, click Edit under Actions section.

6. To create a new template, click New Template.

7. Templates box appears.

8. In the Templates, upload the desired type of file by clicking Upload file.

 

  

 

9. Based on the uploaded file, the output types differ.

10. If the selected file is a word file, the available output types are:

  • Docx

  • PDF

  • HTML

11.If the selected file is an excel file, the available output types are:

  • XLSX

  • PDF

12. If the selected file is a HTML file, the available output types are

  • HTML

  • PDF

13. To save the template, click Save. 

14. The newly created template appears under Templates.

 

 

The above process remains the same for Analysis report; the output type is based on the uploaded template.

Viewing the Default Output

 

1. On the top menu, click Options, point to Export, and then click With Template.

2. Export with Template dialog appears.

3. In the Select Template box, select the required template.

4. In the Select Output box, the default output appears.

 

  

 

5. Here, Excel(Xlsx) has been selected as the default output for the selected template.

6. To export the template in the required output, click Export.

 

Default Template in Output & Delivery

To view the default template in output & delivery

 

1. Right-click the required report, and then click Schedule.

2. Schedule- Report dialog appears.

3. On the Output & Delivery tab, in the Outputs and Deliveries section, click Add Output.

4. Under Outputs and Deliveries section, Template, Output Type and Delivery Plan appear.

5. From the Template section, select the required template


The default output type for the selected template appears.
Here, Excel(Xlsx) is the default output.

6. To submit the process, click Submit.
