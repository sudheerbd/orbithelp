Heat Map

 

Heat map displays different values of data using variation in color.

Visualization (Chart type) Options

1. Heat Maps has the following visualization options

  • Values

  • X Axis

  • Y Axis

2. Drag and drop the required fields from the selected Business Object into the Values, X Axis, and Y Axis sections.

3. Orbit application generates the packed bubbles analysis report dynamically in the run results area.

 

Visual Settings for Heat Maps

To set visual settings

1. On the report edit page, click Visual Settings next to the Data tab.

2. Visual Settings for Heat Maps has the following tabs:

  1. Chart: See Customize Chart Visual Settings for more details.
  2. Axes: See Customize Axes Visual Settings for more details.
  3. Series: See Customize Series Visual Settings for more details.
  4. Legend: See Customize Legend Visual Settings for more details.
  5. Tooltip: See Customize Tooltip Visual Settings for more details.

 

This section explains unique visual settings for Heat Maps

Series

Series tab has the following unique visual settings for Heat Maps:

  1. Axis Color: Change axis color of heat map.
  2. Grid Lines Color: Change grid line color of heat map.

For remaining Series visual settings see Customize Series Visual Settings for more details.

Axis Color

Select Axis Color box to change color of chart axis. See Change default color in Negative Color for detailed information.

Grid Lines Color

Select Grid Lines Color box to change color of grid lines. See Change default color in Negative Color for detailed information.
