Resetting Password

To reset password

1.      On the left-hand side menu, point to User Management, and then click Users.

2.      The User List appears.

 

       

                                                                       

3.      For the required user, click icon (highlighted as “1”) under Actions, and then click Reset Password (highlighted as "2").  

 

4.      Reset Password <user name> dialog box appears. 

 

         

5.      Under the Reset Password <user name> dialog box, in the Password box, type the new password or click Auto Generate.

6.      To view the password, click Show Password.   

7.      To save the changes, click Save.

8.      To close the dialog box, click Cancel.
