Themes

To change the report theme

 

  1. Open the required dashboard report in run results area.

Dashboard report Options available in both View and Edit Mode

  1. On the top right-hand corner of the run results area, click Options, and then click Options.

  2. Dashboard Options dialog appears.

 

  1. To change the report Themes, select the required theme from the dropdown list (highlighted as "1").

 

Here, user selected Themes as Night Sky.

  1. To save the changes, click  Ok.

  2. To cancel the changes, click Cancel.

  3. The selected theme applies and the report appears as shown below:

 

 

Base theme with custom theme settings

 

  1. To enable custom theme setting, click Custom (highlighted as "2").

 

 

  1. Once custom theme is selected, Theme Settings button enables as shown below:

 

  1.  To select the required theme settings, click Theme Settings  (highlighted as "3").

  2. Custom Theme Settings dialog appears.

  1. By default Base theme appears with Dashboard section.

  1. User can also edit the base themes which are already existed

    • Example, if a user want to change the Night Sky theme.

  2. Select Night Sky theme from Base theme dropdown list.

  3. A confirmation message "Modified settings will be lost. Do you want to continue?" dialog box appears

  1. To modify the default settings of night sky, click Yes.

  2. Once the night sky theme is selected the custom theme settings appears with night sky options

  3. To edit night sky theme, click Portlet tab and apply the required changes.

  1. To apply the changes, click Ok.

  2. To revert the changes back/cancel the changes, click Cancel.

  3. The changes apply are highlighted as shown below:

Dashboard

To set dashboard custom theme

  1. To set dashboard custom theme, select Dashboard tab from Custom Theme Setting dialog.

    • In the Background Color box, select the required option from the color palette.

    • In the Space Between widgets box, type/click  to enter the column width.

    • Space Between widgets allows to give space between the visuals.

    • To view the applied changes, click OK.

    • To cancel the changes, click Cancel.

Portlet

To set widget portlet custom theme

  1. To set visual portlet theme, select portlet tab from Custom Theme Setting dialog.

    • To select the visualization background color, click Toolbar Background Color box and then select the required color from the color palette.

This will be observed when we refresh the report

    • To select the visualization title background color, click Title Background Color box and then select the required color from the color palette.

    • To select the action icon color of a visualization, click Action Icons Color box and then select the required color from the color palette.  
    • To select the action icon hover color, click Action Icons Hover Color box and then select the required color from the color palette.  

    • To select the Tile Align, select the required alignment next to tile align..

    • To select the Boarder Radius of an visualization, type/click  to enter the widget boarder radius.

    • To select the title font, click Title Styles dropdown list and then select the required font.

      1. Roboto

      2. Arial

      3. Arial-Black

      4. Times New Roman

      5. Tahoma

      6. Verdana

      7. Calibri

      8. Georg

    • To select the font size px, type/click  to enter title font size.

    • To select the font style, click the required style buttons

      1. Bold

      2. Italic

      3. Underline

    • To select the title color, click the color box and then select the required color from the color palette.

    • Under Border section, perform the following:

      1. To apply the Width of the visualization boarder, type/click  to enter visualization boarder size.

      2. To select the boarder color, click Color box and then select the required color from the color palette.

      3. To apply the visualization boarder style, select Style dropdown list and then select the required style.

      2. To select the boarder color, click Color box and then select the required color from the color palette.

      3. To apply the visualization boarder style, select Style dropdown list and then select the required style.

Context Menu

To set context menu custom theme

  1. To set visualization context menu theme, select Context Menu tab from Custom Theme Setting dialog.

    • To select the menu background color, click Background Color box and then select the required color from the color palette.

    • To select the text color, click Text Color box and then select the required color from the color palette.

    • To select the mouse hover color of a context menu, click Hover Color box and then select the required color from the color palette.  
    • To select the mouse hover text color of a context menu, click Hover Text Color box and then select the required color from the color palette.  

    • To select the menu separator color, click Separator Color and then select the required color from the color palette.  

    • Under Border section, perform the following:

      1. To apply the Width of the menu boarder, type/click  to enter menu boarder size.

      2. To select the boarder color, click Color box and then select the required color from the color palette.

      3. To apply the menu boarder style, select Style dropdown list and then select the required style.

        • none

        • hidden

        • dotted

        • dashed

        • solid

        • double

        • groove

        • ridge

        • inset

        • outset

 

    • To view the applied changes, click OK.

    • To cancel the changes, click Cancel.

To view the context menu, click Options on the title visualization. (OR) Right click on the visualization.

    • To select the buttons boarder color, click Border Color box and then select the required color from the color palette.

    • To select the button text color, click Text Color box and then select the required color from the color palette.  
    • To apply the Border Width of the buttons, type/click  to enter menu boarder size.

 

 

    • To view the applied changes, click OK.

      2. Roboto

      3. Arial

      5. Times New Roman

      6. Tahoma

      7. Verdana

      8. Calibri

      9. Georg

By default, the Main text font set to Helvetica Neue
