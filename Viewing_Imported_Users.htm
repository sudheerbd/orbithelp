Viewing Imported Users

To view imported users

 

  1. On the left-hand side menu, point to  User Management, and then click  Users.

  2. User list appears.

  3. Under the User List, the newly imported user (highlighted as “1”) appears.

Newly Imported User: XXFORX

  1. To view the assigned roles of the specified user, select the required user.

  2. The assigned roles of the selected user (highlighted as “2”) appear under Assigned Roles.

 

 

  1. To add more roles to the selected user, perform the following:

    • On the Assigned Roles area, click Search for items to add .

    • A list of available roles appears.

    • Select the checkbox before the required role, and then click Save.
