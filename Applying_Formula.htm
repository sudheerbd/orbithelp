Calculation Column

 

To Work with Calculation Column

 

  1. In the Reports section, select the required folder to view the reports under it.

  2. Right-click the required report, and then click Open.

      (or)                  

     For the required report, click icon, and then click Open.          

  1. The required report opens in the Run Results area.

  2. To view the column area, click Edit Mode.

  3. The report appears with the Business Object and report column areas.

 

    

 

  1. In the report section, Columns area (highlighted as “1”) and My Calculations (highlighted as “2”).

  2. To create a new calculation column,click next to My Calculations and then click New Calculation.

  3. Calculation Formula dialog box appears.

   

  1. In the Name box, type the name of the formula.

  2. In the Datatype list, click the required data type from the following:

    • Number

    • Text

    • Date

    • Integer

Required Data type: Number

  1. In the Formula area (highlighted as “3”), enter the required formula.  

  2. To enter a formula in the calculation area, perform the following:

    1. In the Function Types section on the right-hand side of the Calculation Formula dialog window, click the required function.

Required Function Types: Aggregation Functions

    1. The functions based on the function types appears in the Functions section below the Function Types section.

 

    

    1. In the textbox under Functions, type the search term for the required function.

    1. The relevant functions matching the search term appear (as shown below) under Functions.

 

    

    1. In the Functions section, double-click the required function.

Aggregation Function: SUM

    1. In the Formula area, the selected function appears.

    2. In the formula, drag and drop the column name or type the column name.

Formula: SUM(["SuperStore"."Sales"]) + SUM(["SuperStore"."Profit"])

    2. To save the changes to the formula, click Ok.

  1. The created calculation column appears under My Calculations.

The functions that are supposed to be used in “WHERE” clause of a query cannot be used in the Calculation Formula.     

 

Editing the Formula

 

 To edit the formula of the newly created column, perform the column:

  1. To edit the formula, double-click on the column in My calculation section.

  2. Calculation Formula dialog box opens with already existing formula.

  3. Edit the formula, and then click Ok.

 

 

Customizing the formula

 

The User can type their own formula using different operators.

 

To create a custom formula

  1. In the Name box, type the name of the formula.

  2. In the Datatype box, select/type the required data type.

  3. Drag and drop the first required column from Business Object area onto the Formula area in Calculation Formula dialog window.

  4. From the operators (highlighted as “1”), select the required operator.

  5. Drag and drop the second required column or type a number.


First Required Column: Sales
Operator:  +
Second Required Column: Shipping Cost


Formula: ["SuperStore"."Sales”] + ["SuperStore"."Shipping Cost"]

 

            

  1. To validate the formula, click Evaluate Expression.

  2. To save the changes, click Ok.
