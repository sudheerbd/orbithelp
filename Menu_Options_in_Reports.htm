Menu Options in Reports

1.      In the Reports section, the following menu options appear:

 

  • To refresh the folders, click Refresh folders (highlighted as "1").

  • To switch to the grid view, click Grid view (highlighted as "2").

  • To switch to the list view, click List view (this appears in the place of Grid view).  

  • To view the details of a report, click   View details (highlighted as "3").

  • To view the different options, click Options (highlighted as "4").

 

 

2. Under Options, the following options appear. 

  • To share the reports, click Share (highlighted as "5").

  • To organize the reports, Organize (highlighted as "6").

  • To get the information about the reports that are scheduled, click Schedules (highlighted as "7").

  • To view the different options of a folder, click icon (highlighted as "8")  for the required folder.

  • Under icon, the following options appear:

 

 

    • New (highlighted as "9") - This option displays option to create the following new reports:

      • New Data Report

      • New Analysis

      • New Dashboard

      • New Publisher Report

      • New Script Report

      • New Finance Report

      • New Custom Report

 

    • New Folder (highlighted as "10") : This option is to create a new folder.  

    • Rename (highlighted as "11") : This option is to rename a folder.  

    • Share (highlighted as "12"): This option is to share a folder.

    • Export (highlighted as "13") : This option is to export a folder .

    • Delete (highlighted as "14"): This option is to delete a folder.  

 

 

Report Options

 

To view the report options

 

1. Double-click the required folder.

2. The reports under the selected folder appear.

3. For a required report, click icon to view different options of a report.

 

 

    • Open (highlighted as "1"): This option is to open a report

    • Schedule (highlighted as "2") : This option is to schedule a report.

    • Add to favorites (highlighted as "3") : This option is to add the required reports to Favorites.

    • Edit (highlighted as "4"): This option is to edit a report.

    • Rename (highlighted as "5"): This option is rename a report.

    • Copy (highlighted as "6") : This option is to copy a report.

    • Delete (highlighted as "7"): This option is to delete a report.

    • Share (highlighted as "8"): This option is to share a report.

    • Export (highlighted as "9") : This option is to export a report.

    • Generate Link (highlighted as "10") : This option is to generate public and private report links.
