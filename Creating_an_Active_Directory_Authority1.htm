Creating an Active Directory Authority

To Create a new Active Directory Authority

  1. On the left-hand side menu, point to  User Management, and then click  Authorities.

  2. Authorities page appears with a list of authorities.

  3. To create a new authority, click New (highlighted as "1") on the top right-hand side of the Authorities page.

  4. New Authority dialog box appears.

  5. In the New Authority dialog, enter the following to create a new authority.

    • In the Name box, type the name of the authority.

    • In the Authority Key box, type the authority key.

    • To sync the users/settings on login, select Sync on Login checkbox.

    • In the Description box, type the description for the authority.

  6. Under the Authentication section, in the Authority Type box, type/select the required authority type from the following:

    • Oracle ERP Security

    • LDAP

    • Active Directory

    • PeopleSoft

      Example: Required Authority Type: Active Directory.
