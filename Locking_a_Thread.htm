Locking a Thread

To lock a thread

 

1. In the chat window of the selected thread, click  Thread Chat Settings Menu, and then click Lock Thread.

2.Change Thread Status dialog box appears.

3. To lock the thread, click Yes.

Note: A Collaboration thread status change request processed notification appears briefly at the top of the ORBIT application window.

4. To cancel the lock thread request, click No.

5. When the thread is locked, color of thread status indicator changes to grey.

 

6. Also, in Threads section Thread Status appears as “CLOSE”.

 

 

 

Note: Once the thread is locked, users cannot perform any actions on it. However, the user can view the history of the thread.
