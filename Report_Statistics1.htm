Report Statistics

In this module, you will learn about the Report statistics feature of the application.

 

Report Statistics contain the following fields:

 

S.no

Fields

Description

1.       

Report Name

This field displays the report name.

2.       

Process Id

This field displays the process id 

3.

Run Date

 This field displays the Run Date of the report

      4.

Execution Time(ms)

 This field displays the execution time of the report in milli seconds

      5.

Rows Received

This field displays the number of rows retrieved.

6.       

 Error Message

This field displays any error message.

7.       

Output Type

 This field displays the output type.

8.

Component

 This field displays the type of component.

 

 

1. The reports that run in the reports section appear under the report statistics.

 

2. My Reports option shows the reports only under the current user (highlighted as “1”).  

 

3. All Reports option shows all the reports under all users (highlighted as “2”).

 

4. To refresh the reports, click Refresh.
