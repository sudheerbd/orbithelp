Meta Objects

Meta-Objects are objects that allow you to apply conditional formats to the columns in the reports. Under the Meta-Objects section, you can find the Base object. Base object has the following fields:

  • Base

    • Boolean

    • Number Field

      • Id Field

    • String

    • Date Field

      • Time Field
