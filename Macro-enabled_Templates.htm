Macro-enabled Templates

The .xlsm (macro-enabled) template here is designed in a such a way that when it is opened it generates a message written using Macros (Microsoft VBA) as shown below:

The above template is designed in such as way that it resembles the excel template for a publisher report where the data for the columns is denoted as ${COL0}, ${COL1}, and so on.  

Its associated names are entered in the Name Manager under Formula tab.

 The figure below illustrates the rows and its associated names in the Name Manager. The mapping between the rows and its associated names is shown below:

  

 

Here, TableHeader is mapped to the first row (highlighted as “1”) and ReportData is mapped to the second row (highlighted as “2”).
