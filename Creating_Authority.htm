Creating Authority

To create a new authority

 

  1. On the left-hand side menu, point to  User Management, and then click  Authorities.

  2. Authorities page appears with a list of authorities (highlighted as “1”).

 

  1. To create a new authority, click New (highlighted as "2") on the top right-hand side of the Authorities page.

  2. New Authority dialog box appears.

 

  1. Under New Authority, enter the following to create a new authority

    • In the Name box, type the name of the authority.              

    • In the Authority Key box, type the authority key; this would be used in profile options that would be created in EBS. This is a required field.

    • To sync the users/settings on login, select Sync on Login checkbox.

    • In the Description box, type the description for the authority.

    • Under the Authentication section, in the Authority Type box, type/select the required authority type from the following:

      • Oracle ERP Security

      • LDAP

      • Active Directory

      • People Soft

      • Based on the selected authority types, the fields in the Authentication section change.

    • In the Data Source box, type/select the required data source or click Create new datasource to create a new data source.

    • If the authority type and the data source of the new authority is same as the authentication, then select Authorization same as Authentication checkbox.

    • If the authority type and data source of the new authority is not same as the authentication, then deselect the Same as Authentication checkbox.
