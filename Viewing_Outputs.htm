Viewing Outputs

To view the output for a report

  1. Under Processes tab, select the required report (highlighted as "1").

  2. For the required report, output is displayed under Output(s) & Delivery area (highlighted as "2").

  3. Under Output(s) & Delivery area, the following columns appear:

    • Output Type

    • Phase

    • Status

    • Output

    • Log

  4. Under Parameters area (highlighted as "3"), the parameters/parameter values of the selected report appear.


    i. The output for a specific report appears in this section, only when the output is set in Output & Delivery tab in Schedule dialog box.
    ii. To view the schedule option, right-click the required report, and then click Schedule.
    iii. To set the Outputs, see Setting Outputs.

  5. Under Schedule tab (highlighted as "4"), the details of the process appears if the processes is scheduled for multiple times.
